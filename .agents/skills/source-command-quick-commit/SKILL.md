# source-command-quick-commit

Use this skill when the user asks to run the migrated source command `quick-commit`.

## Command Template

# /quick-commit - 빠른 커밋 (v6)

---

## 0단계: 메시지 확인

커밋 메시지가 인자로 제공되었는지 확인한다.
메시지가 없으면 에러를 출력하고 종료한다.

```
사용법: /quick-commit "커밋 메시지"
```

---

## 1단계: 변경사항 확인

```bash
git status --short
git diff --stat
```

변경사항이 없으면 에러를 출력하고 종료한다.

변경 규모 체크:
- 변경 파일 3개 초과 또는 20줄 초과 시 경고:
  ```
  변경이 큽니다. 전체 워크플로우 권장:
  /handoff-verify → /commit-push-pr

  그래도 빠른 커밋하려면 "계속"이라고 입력하세요.
  ```

---

## 2단계: 빠른 검증

`git diff`로 변경 내용을 확인한다:
- 명백한 오류만 체크
- 문제 있으면 안내 후 중단

---

## 3단계: 스테이징 확인

커밋에 포함할 파일 목록을 명시적으로 보여준다:

```bash
git status --short
```

사용자에게 스테이징할 파일을 확인받는다:

```
위 파일들을 커밋에 포함할까요?
확인 후 "스테이징"이라고 입력하세요. 제외할 파일이 있으면 파일 경로를 알려주세요.
```

확인 없이 `git add -A`를 실행하지 않는다.
승인된 파일만 경로를 명시해서 스테이징한다:

```bash
git add path/to/file1 path/to/file2
```

스테이징 후 포함 파일을 다시 확인한다:

```bash
git diff --cached --stat
git diff --cached --name-only
```

---

## 4단계: 빌드 검증

커밋 전 반드시 프로덕션 빌드를 실행한다:

```bash
npm run build
```

`npm run build`가 exit 0이 아니면 커밋하지 않고 중단한다.
빌드 실패 시 실패 로그를 요약하고, 스테이징 상태를 유지한 채 사용자에게 수정이 필요하다고 안내한다.

---

## 5단계: 커밋 실행

```bash
git commit -m "{메시지}"
```

자동으로 `git push`를 실행하지 않는다.
push는 커밋 이후 사용자와 별도로 결정한다.

---

## 6단계: 출력

### 성공 시

```
════════════════════════════════════════════════════════════════
  Quick Commit v6
════════════════════════════════════════════════════════════════

  커밋: {short_hash}
  메시지: {메시지}
  변경: +{N} -{N} ({N} files)

  빌드: npm run build 통과
  다음 단계: 필요 시 별도로 push 여부 결정

════════════════════════════════════════════════════════════════
```

### 메시지 누락 시

```
════════════════════════════════════════════════════════════════
  Quick Commit v6
════════════════════════════════════════════════════════════════

  커밋 메시지가 필요합니다.

  사용법: /quick-commit "fix: 오타 수정"

════════════════════════════════════════════════════════════════
```

```
════════════════════════════════════════════════════════════════
  Quick Commit v6
════════════════════════════════════════════════════════════════

  커밋할 변경사항이 없습니다.
  git status: clean

════════════════════════════════════════════════════════════════
```
