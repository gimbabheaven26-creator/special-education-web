## 핵심개념

**표준화 검사(standardized test)**: 검사의 실시 절차, 채점 방법, 결과 해석 기준이 표준화되어 있는 검사이다. 대표적인 규준 집단(norm group)과 비교하여 개인의 수행 수준을 파악할 수 있으며, **규준참조검사(norm-referenced test)**의 대표적인 형태이다.

**지능검사(intelligence test)**: 개인의 인지적 능력을 측정하는 표준화 검사이다. 특수교육대상자 선정·배치 과정에서 중요한 역할을 하며, 전체 지능지수(FSIQ)뿐 아니라 인지적 강점과 약점을 파악하는 데 활용된다.

**적응행동검사(adaptive behavior assessment)**: 개인이 일상생활에서 기대되는 사회적·실용적 기능을 얼마나 잘 수행하는지를 측정하는 검사이다. **AAIDD(2021)**의 지적장애 정의에서 지능과 함께 적응행동이 핵심 진단 기준이 된다.

**신뢰도(reliability)**: 검사가 측정하고자 하는 것을 **일관성 있게** 측정하는 정도이다. 동일한 대상에게 반복 측정했을 때 유사한 결과를 산출하는 정도를 의미한다.

**타당도(validity)**: 검사가 측정하고자 하는 것을 **정확하게** 측정하는 정도이다. 검사 결과가 그 검사의 목적에 부합하는지를 나타낸다.

## 주요이론

### 1. K-WISC-V (한국판 웩슬러 아동 지능검사 5판)

**Wechsler**가 개발한 아동용 개인 지능검사의 한국 표준화판이다. 만 6세~16세 11개월 아동을 대상으로 하며, 특수교육대상자 진단·평가에서 가장 널리 사용되는 지능검사이다.

#### K-WISC-V의 5가지 지표(Index)

| 지표 | 영문 | 측정 내용 | 주요 소검사 |
|------|------|----------|-----------|
| **언어이해(VCI)** | Verbal Comprehension | 언어적 개념 형성, 추론, 표현 능력 | 공통성, 어휘, 이해 |
| **시공간(VSI)** | Visual Spatial | 시각적 자극의 분석·통합, 공간 관계 파악 | 토막짜기, 퍼즐 |
| **유동추론(FRI)** | Fluid Reasoning | 새로운 상황에서의 추론, 규칙 발견 능력 | 행렬추리, 무게비교 |
| **작업기억(WMI)** | Working Memory | 정보를 일시적으로 저장·조작하는 능력 | 숫자, 그림기억 |
| **처리속도(PSI)** | Processing Speed | 단순한 시각적 정보를 빠르고 정확하게 처리하는 능력 | 기호쓰기, 동형찾기 |

#### K-WISC-V 결과 해석

- **전체 지능지수(FSIQ)**: 평균 100, 표준편차 15
- 지적장애 진단 기준: FSIQ 약 70 이하 (평균에서 2 표준편차 이하)
- **지표 간 차이 분석**: 인지적 강점과 약점 파악에 활용. 예를 들어, 언어이해(VCI)가 낮고 시공간(VSI)이 높은 학생은 언어적 지원이 필요할 수 있다.

### 2. K-ABC-II (한국판 카우프만 아동용 지능검사 2판)

**Kaufman & Kaufman**이 개발한 아동용 개인 지능검사의 한국 표준화판이다. 만 2세 6개월~12세 5개월을 대상으로 한다.

#### K-ABC-II의 이론적 기반

K-ABC-II는 **Luria의 신경심리학 이론**과 **CHC(Cattell-Horn-Carroll) 이론** 두 가지 모델에 기반하여 해석할 수 있다.

| Luria 모델 척도 | 측정 내용 |
|----------------|----------|
| **순차처리** | 자극을 순서대로 처리하는 능력 (청각적·시간적 순서) |
| **동시처리** | 여러 자극을 동시에 통합·처리하는 능력 (시각적·공간적 통합) |
| **계획** | 문제 해결 과정에서의 계획·조직·실행 능력 |
| **학습** | 새로운 정보를 학습하고 저장·인출하는 능력 |

#### K-WISC-V와 K-ABC-II 비교

| 비교 기준 | K-WISC-V | K-ABC-II |
|----------|----------|---------|
| 개발자 | Wechsler | Kaufman & Kaufman |
| 대상 연령 | 6세~16세 11개월 | 2세 6개월~12세 5개월 |
| 이론적 기반 | CHC 이론 | Luria 모델 / CHC 이론 이중 해석 |
| 인지와 학업 분리 | 인지 능력 중심 | 인지처리와 습득도(학업) 분리 |
| 언어 의존도 | 상대적으로 높음 | 상대적으로 낮음 (비언어 척도 포함) |
| 특장점 | 가장 널리 사용, 풍부한 연구 자료 | 문화적 편향 최소화, 저연령 아동 평가 가능 |

### 3. 적응행동검사

#### (1) ABAS-3 (적응행동평가체계 3판)

**Harrison & Oakland**이 개발한 적응행동검사로, **AAIDD(2021)**의 적응행동 3영역 모델에 기반한다.

| 영역 | 하위 기술 |
|------|----------|
| **개념적 적응행동** | 언어(수용·표현), 읽기·쓰기, 수 개념, 자기지시 |
| **사회적 적응행동** | 대인관계, 사회적 책임, 자존감, 규칙 준수, 속임 회피 |
| **실제적 적응행동** | 일상생활활동, 건강·안전, 직업 기술, 이동, 일과/일정 관리 |

#### (2) K-Vineland-II (한국판 바인랜드 적응행동검사 2판)

**Sparrow, Cicchetti & Balla**가 개발한 적응행동검사의 한국 표준화판이다. 출생~90세까지 전 연령대를 대상으로 하며, 반구조화된 면담 형식으로 실시한다.

| 영역 | 하위 영역 |
|------|----------|
| **의사소통** | 수용, 표현, 읽기·쓰기 |
| **일상생활기술** | 개인, 가정, 지역사회 |
| **사회성** | 대인관계, 놀이·여가, 대처기술 |
| **운동기술** | 대근육, 소근육 |
| **부적응행동** (선택) | 내재화, 외현화, 기타 문제행동 |

### 4. 학력검사: BASA (기초학습기능 수행평가체제)

**김동일** 교수가 개발한 한국형 학력검사 및 교육과정중심측정 도구이다.

| 검사 영역 | 측정 내용 |
|----------|----------|
| **BASA: 읽기** | 1분간 정확하게 읽은 음절 수 (읽기 유창성) |
| **BASA: 쓰기** | 3분간 정확하게 쓴 음절 수 또는 완성한 문장 수 |
| **BASA: 수학** | 2분간 정확하게 풀은 수학 연산 문제 수 |
| **BASA: 초기 문해** | 음운인식, 글자 지식, 단어 읽기 등 초기 읽기 기술 |

BASA는 표준화 검사이면서 동시에 **교육과정중심측정(CBM)** 도구로도 활용된다. 규준 자료와 비교하여 학업 성취 수준을 판별하고, 반복 측정을 통해 학습 진전도를 모니터링할 수 있다.

### 5. 신뢰도와 타당도

#### 신뢰도의 종류

| 신뢰도 유형 | 정의 | 확인 방법 |
|------------|------|----------|
| **검사-재검사 신뢰도** | 동일 검사를 일정 시간 간격으로 두 번 실시한 결과의 일관성 | 두 검사 점수 간 상관계수 |
| **동형검사 신뢰도** | 동등한 두 검사 간 결과의 일관성 | 두 동형 검사 점수 간 상관계수 |
| **반분 신뢰도** | 하나의 검사를 두 부분으로 나누었을 때 결과의 일관성 | 두 반분 점수 간 상관계수 |
| **내적 일관성 신뢰도** | 검사 문항들이 동일한 구인을 일관성 있게 측정하는 정도 | Cronbach alpha 계수 |
| **채점자 간 신뢰도** | 서로 다른 채점자가 동일 답안을 채점한 결과의 일관성 | 채점자 간 일치도 |

#### 타당도의 종류

| 타당도 유형 | 정의 | 확인 방법 |
|------------|------|----------|
| **내용 타당도** | 검사 문항이 측정하고자 하는 내용 영역을 대표하는 정도 | 전문가 판단 |
| **준거 타당도** | 검사 결과가 외부 준거와 관련되는 정도 | 준거와의 상관계수 |
| - 공인 타당도 | 검사와 준거를 동시에 측정했을 때의 관련성 | 동시 실시 후 상관 분석 |
| - 예언 타당도 | 검사 결과가 미래의 수행을 예측하는 정도 | 시차를 두고 상관 분석 |
| **구인 타당도** | 검사가 이론적 구인(construct)을 정확하게 측정하는 정도 | 요인분석, 수렴·변별 타당도 |

## 출제경향분석

### 기입형 출제 패턴

- "K-WISC-V의 5개 지표 중 새로운 상황에서의 추론과 규칙 발견 능력을 측정하는 지표는 ( )이다." - **유동추론(FRI)**
- "AAIDD(2021)의 적응행동 3영역은 개념적, ( ), ( ) 적응행동이다." - **사회적**, **실제적**
- "검사가 측정하고자 하는 것을 일관성 있게 측정하는 정도를 ( )이라 하고, 정확하게 측정하는 정도를 ( )이라 한다." - **신뢰도**, **타당도**
- "K-ABC-II에서 자극을 순서대로 처리하는 능력을 측정하는 척도는 ( )이다." - **순차처리**

### 서술형 출제 패턴

**[패턴 1] 지능검사 결과 해석 및 교육적 시사점**
- K-WISC-V 검사 결과(5개 지표 점수)가 제시되고, 인지적 강점·약점을 분석하여 교육적 지원 방안을 서술하는 문제가 출제된다.
- 답안 포인트: 지표별 의미, 지표 간 차이 분석, 강점 기반 교육 전략

**[패턴 2] 적응행동검사의 필요성과 영역 서술**
- 지적장애 진단 과정에서 적응행동검사가 필요한 이유와 3영역의 내용을 서술하는 문제가 출제된다.
- 답안 포인트: AAIDD 정의(지능+적응행동), 3영역 명칭과 각 영역의 하위 기술

**[패턴 3] 신뢰도·타당도 종류 구분**
- 검사 상황이 제시되고, 해당 상황에서 확인하고자 하는 신뢰도 또는 타당도의 종류를 판별하는 문제가 출제된다.

### 최근 출제 경향

- K-WISC-V의 **5가지 지표 명칭과 측정 내용**이 기입형에서 빈출된다.
- 지적장애 진단에서 **지능검사와 적응행동검사의 관계**를 묻는 문제가 증가하고 있다.
- **신뢰도와 타당도의 개념 구분** 및 하위 유형 분류가 꾸준히 출제된다.
- BASA를 활용한 **학업 성취 수준 판별**과 CBM 연계 문제가 출제 빈도가 높아지고 있다.

## 자가점검

- [ ] K-WISC-V의 5가지 지표(VCI, VSI, FRI, WMI, PSI)의 명칭과 측정 내용을 설명할 수 있다.
- [ ] K-ABC-II의 Luria 모델 척도(순차처리, 동시처리, 계획, 학습)를 설명할 수 있다.
- [ ] K-WISC-V와 K-ABC-II의 차이점을 이론적 기반과 대상 연령 측면에서 비교할 수 있다.
- [ ] AAIDD의 적응행동 3영역(개념적, 사회적, 실제적)과 각 영역의 하위 기술을 설명할 수 있다.
- [ ] 신뢰도의 5가지 유형과 각각의 확인 방법을 구분할 수 있다.
- [ ] 타당도의 3가지 유형(내용, 준거, 구인)을 구분하고, 준거 타당도의 하위 유형을 설명할 수 있다.
- [ ] BASA의 측정 영역과 활용 방법을 설명할 수 있다.
- [ ] 지적장애 진단에서 지능검사와 적응행동검사가 모두 필요한 이유를 서술할 수 있다.
