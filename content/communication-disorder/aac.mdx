## 핵심개념

**보완대체의사소통(AAC, Augmentative and Alternative Communication)**: 구어(말)를 통한 의사소통이 불가능하거나 제한적인 사람이 자신의 의사를 표현하고 타인과 소통하기 위해 사용하는 **구어 이외의 모든 의사소통 방법**을 총칭한다. **보완(augmentative)**은 기존의 구어 능력을 보충하는 것이며, **대체(alternative)**는 구어를 완전히 대신하는 것을 말한다.

**도구적 AAC(aided AAC)**: 의사소통을 위해 **외부 도구나 장치**를 사용하는 방식이다. 의사소통판, 음성출력 의사소통기기(VOCA), 태블릿 기반 AAC 앱 등이 해당된다.

**비도구적 AAC(unaided AAC)**: 외부 도구 없이 **자신의 신체**만을 사용하여 의사소통하는 방식이다. 수어(수화), 제스처, 몸짓, 표정, 발성 등이 해당된다.

**상징체계(symbol system)**: AAC에서 의미를 전달하기 위해 사용하는 시각적·청각적·촉각적 표상 체계를 말한다. 실물, 사진, 그림상징(PCS), 블리스상징(Bliss), 문자 등 다양한 유형이 있다.

> ⚠️ **용어 검증 필요**: AAC 관련 용어(aided/unaided 등)의 한국어 번역은 학자에 따라 '도구적/비도구적' 또는 '보조적/비보조적'으로 상이할 수 있음

## 주요이론

### 1. AAC의 4구성요소

AAC 체계는 **상징(symbols)**, **도구(aids)**, **기법(techniques)**, **전략(strategies)**의 네 가지 구성요소로 이루어진다.

| 구성요소 | 정의 | 예시 |
|---------|------|------|
| **상징(symbols)** | 의미를 표상하는 시각적·청각적·촉각적 방법 | 실물, 사진, PCS, Bliss, 문자, 수어 |
| **도구(aids)** | 상징을 전달하기 위해 사용하는 물리적 장치나 도구 | 의사소통판, VOCA, 태블릿 앱, 눈응시판 |
| **기법(techniques)** | 상징을 선택하고 전달하는 방법 | 직접선택, 스캐닝(간접선택), 부호화(encoding) |
| **전략(strategies)** | 의사소통 능력을 향상시키기 위한 운영 방법 | 대화 상대자 훈련, 의사소통 기회 확대, 환경 배열 |

> **KICE 기출 연계 (2025 전공A 문제2)**: "밑줄 친 ㉠은 어떤 측면에서 보조공학 기기를 구분하고 있는지 [A]를 참고하고, 밑줄 친 ㉡은 보완 대체 의사소통(AAC)의 구성 요소 중 무엇에 해당하는지 쓰시오." — 도구의 연속적 구성(로우테크~하이테크)과 AAC 4구성요소를 구분하는 문항이 출제되었다.

### 2. 참여모델(Participation Model)

**참여모델(Beukelman & Mirenda)**은 AAC 사용자의 의사소통 참여를 방해하는 장벽을 체계적으로 분석하고 중재 계획을 수립하기 위한 틀이다. 장벽은 크게 **기회장벽**과 **접근장벽**으로 나뉜다.

#### 기회장벽(Opportunity Barriers) — 5가지

기회장벽은 AAC 사용자의 능력이 아닌 **외부 환경과 타인의 태도**에 의해 의사소통 기회가 제한되는 경우를 말한다.

| 기회장벽 유형 | 내용 |
|-----------|------|
| **정책 장벽(policy)** | AAC 사용을 제한하는 제도·규정·법령. 예: 학교에서 AAC 기기 사용을 허용하지 않는 교칙 |
| **실제 장벽(practice)** | 현장에서 AAC를 실제로 적용하지 않는 관행. 예: 교사가 AAC 사용 시간을 별도로 할애하지 않음 |
| **지식 장벽(knowledge)** | AAC에 대한 정보와 이해 부족. 예: 부모가 AAC를 사용하면 구어 발달이 더 지연된다는 잘못된 믿음 |
| **기술 장벽(skill)** | AAC를 효과적으로 사용하도록 지도하는 기술 부족. 예: 교사가 AAC 기기 조작법을 모르는 경우 |
| **태도 장벽(attitude)** | AAC 사용에 대한 부정적 인식이나 편견. 예: "AAC를 사용하면 학생이 아예 말을 할 수 없게 된다"는 신념 |

> **KICE 기출 연계 (2026 전공B 문제4)**: "밑줄 친 ㉣은 AAC 참여 모델의 기회 장벽 중 무엇에 해당하는지 그 유형을 쓸 것" — 학생 K의 어머니가 "AAC를 사용하면 학생 K가 아예 말을 할 수 없게 된다"는 **신념(태도 장벽)** 또는 "반 친구들이 학생 K가 AAC를 사용해도 제대로 대화하기 어렵다고 생각해서 대화를 시도하지 않는" **태도 장벽** 유형을 판별하는 문항이 출제되었다.

#### 접근장벽(Access Barriers)

접근장벽은 AAC 사용자의 **능력·기능적 제한**으로 인해 AAC 체계를 효과적으로 사용하기 어려운 경우를 말한다.

| 접근장벽 유형 | 내용 |
|-----------|------|
| **운동 능력 제한** | 상지 기능, 미세 운동 능력의 제한으로 직접선택이 어려운 경우 |
| **인지·언어 능력 제한** | 상징을 이해하거나 문장을 구성하는 인지적·언어적 능력의 제한 |
| **감각 능력 제한** | 시각·청각 등 감각 기능의 제한으로 상징 인식이 어려운 경우 |
| **문해 능력 제한** | 글자 기반 AAC를 사용하기 위한 읽기·쓰기 능력의 부족 |

### 3. 상징체계의 유형과 도상성

AAC에서 사용하는 상징은 **추상성 수준**에 따라 다양한 유형으로 분류되며, 상징이 지시 대상과 얼마나 닮았는지를 나타내는 **도상성(iconicity)**이 중요한 선정 기준이 된다.

#### 상징체계 유형 (구체→추상 순서)

| 상징 유형 | 특성 | 예 |
|---------|------|-----|
| **실물(real object)** | 실제 사물 또는 그 일부를 상징으로 사용 | 우유팩으로 '우유' 표현 |
| **축소 모형(miniature object)** | 실물의 축소판 | 장난감 자동차로 '자동차' 표현 |
| **사진(photograph)** | 실제 대상의 사진 | 컵 사진으로 '마시다' 표현 |
| **그림상징(PCS)** | Picture Communication Symbols. 단순화된 선화(line drawing) | PCS '먹다' 그림 |
| **블리스상징(Bliss)** | Charles Bliss가 개발한 의미 기반 상징 체계. 기본 요소의 조합으로 의미 표현 | 기본 형태 조합으로 복잡한 개념 표현 |
| **문자(text/orthography)** | 글자, 낱말, 문장 | 한글, 영어 알파벳 |

#### 도상성(Iconicity) 수준

**도상성**은 상징이 그것이 나타내는 지시 대상과 시각적으로 얼마나 유사한지를 나타내는 정도이다.

| 도상성 수준 | 정의 | 예 |
|---------|------|-----|
| **투명(transparent)** | 상징과 지시 대상의 관계가 **누구나** 쉽게 알 수 있을 정도로 명확 | 컵 사진 → '마시다' |
| **반투명(translucent)** | 상징과 지시 대상의 관계가 **설명을 들으면** 이해할 수 있는 수준 | 물방울 그림 → '마시다' |
| **불투명(opaque)** | 상징과 지시 대상의 관계가 **자의적**이어서 설명 없이는 알 수 없음 | 블리스 상징 '행복' |

> **KICE 출제 핵심**: 기출에서 특정 상징을 제시하고 도상성 수준(투명/반투명/불투명)을 판별하는 문항이 출제된다. 판별 기준은 "상징만 보고 의미를 추측할 수 있는가"(투명), "설명을 들은 후 관련성을 이해할 수 있는가"(반투명), "설명 없이는 전혀 추측이 불가능한가"(불투명)이다.

> **KICE 기출 연계 (2026 전공B 문제4)**: "학생 K가 '손답'과 같은 손짓기호를 사용할 때는 **도상성이 높은 상징**을 우선적으로 활용한다" — AAC 상징 선정 시 도상성 수준을 고려해야 한다는 원칙이 출제되었다.

### 4. 선택기법(Selection Techniques)

AAC 사용자가 상징을 선택하는 방법은 크게 **직접선택**과 **간접선택(스캐닝)**으로 나뉜다.

| 선택기법 | 정의 | 적합한 사용자 |
|--------|------|-----------|
| **직접선택(direct selection)** | 사용자가 원하는 상징을 **직접 가리키거나 터치**하여 선택 | 상지 운동 능력이 충분한 사용자 |
| **스캐닝(scanning)** | 상징이 순차적으로 제시(강조)되면 원하는 상징이 나타났을 때 **스위치를 눌러** 선택 | 운동 능력이 매우 제한된 사용자 |

#### 스캐닝의 유형

| 유형 | 방법 |
|------|------|
| **자동 스캐닝(automatic)** | 기기가 자동으로 상징을 순차 강조하며, 사용자는 원하는 상징에서 스위치를 누름 |
| **단계 스캐닝(step)** | 사용자가 스위치를 누를 때마다 다음 상징으로 이동하고, 원하는 상징에서 다른 스위치를 눌러 선택 |
| **행-열 스캐닝(row-column)** | 먼저 행(줄) 단위로 스캔 → 원하는 행에서 선택 → 해당 행 내 열(칸) 단위로 스캔 → 선택. 선택 시간 단축 |
| **그룹-항목 스캐닝** | 상징을 그룹으로 묶어 스캔한 후, 선택된 그룹 내에서 개별 항목을 스캔 |

### 5. HAAT 모형(Human Activity Assistive Technology Model)

**HAAT 모형(Cook & Hussey)**은 보조공학 적용을 위한 체계적 틀로, AAC를 포함한 보조공학 전반에 적용된다. 네 가지 구성요소의 상호작용을 강조한다.

| 구성요소 | 내용 |
|---------|------|
| **인간(Human)** | AAC 사용자의 신체적·인지적·감각적·정서적 능력과 요구 |
| **활동(Activity)** | 사용자가 수행하고자 하는 의사소통 과제(자기 표현, 요구하기, 대화 등) |
| **보조공학(Assistive Technology)** | AAC 기기 및 관련 기술 (하드웨어, 소프트웨어, 상징 체계) |
| **맥락(Context)** | 사용 환경 — 물리적(교실, 가정), 사회적(대화 상대자), 문화적, 제도적 맥락 |

HAAT 모형은 보조공학 기기를 선정할 때 **기기의 성능**만이 아니라 **사용자의 능력, 활동 목표, 사용 맥락**을 종합적으로 고려해야 함을 강조한다.

### 6. AAC 중재 시 고려사항

| 고려사항 | 내용 |
|---------|------|
| **의사소통 상대자 훈련** | AAC 사용자의 대화 상대자(교사, 부모, 또래)에게 AAC 기기를 함께 사용하고 반응하는 방법을 교육 |
| **어휘 선정** | 사용자의 연령, 생활환경, 관심사를 고려하여 **핵심 어휘(core vocabulary)**와 **주변 어휘(fringe vocabulary)**를 균형 있게 선정 |
| **상징 선정 시 도상성 고려** | 인지 수준이 낮은 사용자는 도상성이 높은(투명한) 상징부터 시작. 문해 능력이 있으면 문자 상징 활용 |
| **사용 빈도 고려** | 음성출력 의사소통기기의 상징 선정 시 학생의 **사용 빈도**를 고려 (2026 전공B 문제4 참조) |
| **기기 배치** | 사용자의 운동 능력, 시각 능력을 고려한 기기 위치와 크기 조정 |

> **KICE 기출 연계 (2026 전공B 문제4)**: "학생 K가 사용하는 ㉤ 음성출력 의사소통 기기의 상징을 선정할 때는 학생의 **사용 빈도**를 고려한다"와 "㉥ 학생 K가 '손답'과 같은 손짓기호를 사용할 때는 **도상성이 높은 상징**을 우선적으로 활용한다"는 AAC 중재의 핵심 원칙으로 출제되었다.

## 기출연계

### 기입형 출제 패턴

- "구어를 통한 의사소통이 제한적인 사람이 사용하는 구어 이외의 모든 의사소통 방법을 ( )(이)라 한다." → **보완대체의사소통(AAC)**
- "AAC의 4구성요소는 상징, ( ), 기법, ( )이다." → **도구**, **전략**
- "참여모델에서 외부 환경과 타인의 태도에 의해 의사소통 기회가 제한되는 장벽을 ( ) 장벽이라 하고, 구체적으로 ( ), 실제, 지식, 기술, 태도의 5가지로 나뉜다." → **기회**, **정책**
- "상징이 지시 대상과 시각적으로 유사한 정도를 ( )(이)라 하며, 누구나 쉽게 알 수 있는 수준을 ( ), 설명 후 이해할 수 있는 수준을 ( )이라 한다." → **도상성(iconicity)**, **투명(transparent)**, **반투명(translucent)**
- "외부 도구나 장치를 사용하는 AAC를 ( ) AAC, 자신의 신체만을 사용하는 AAC를 ( ) AAC라 한다." → **도구적(aided)**, **비도구적(unaided)**

### 서술형 출제 패턴

**[패턴 1] 기회장벽 유형 판별 (2026 전공B 문제4 유형)**
- 사례에서 나타난 AAC 참여 제한 요인이 기회장벽 5가지 중 어디에 해당하는지 판별하고 근거 서술
- 답안 포인트: ① 기회장벽 5가지(정책/실제/지식/기술/태도)의 정의 제시 ② 사례와 정의의 대응 ③ 해결 방안 제안

**[패턴 2] AAC 구성요소 구분 및 적용 (2025 전공A 문제2 유형)**
- 특정 AAC 활용 장면에서 4구성요소(상징/도구/기법/전략) 각각에 해당하는 내용을 구분하여 서술
- 답안 포인트: ① 상징의 유형 명시 ② 도구의 로우테크/하이테크 구분 ③ 선택기법 명시 ④ 전략의 구체적 내용

**[패턴 3] 도상성 수준 판별 및 상징 선정 근거 서술**
- 제시된 상징의 도상성 수준을 판별하고, 대상 학생에게 적합한 상징을 선정하는 근거를 서술
- 답안 포인트: ① 투명/반투명/불투명의 정의와 판별 기준 ② 학생의 인지 수준 고려 ③ 도상성이 높은 상징 우선 사용 원칙

### 최근 출제 경향

- **참여모델의 기회장벽 5가지**(정책/실제/지식/기술/태도)를 사례에 적용하여 판별하는 문항이 매년 출제
- AAC 4구성요소를 **구분하여 기입**하는 문항 꾸준히 출제
