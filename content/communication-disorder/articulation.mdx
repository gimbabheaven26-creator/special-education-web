## 핵심개념

**조음(articulation)**: 폐에서 나온 공기가 성대를 지나 구강·비강을 통해 나오면서 혀, 입술, 치아, 경구개, 연구개 등의 **조음기관**에 의해 특정 말소리로 산출되는 과정을 말한다. 조음은 말소리의 **운동적·물리적 산출** 측면에 초점을 둔다.

**음운(phonology)**: 언어 체계 내에서 말소리가 어떤 규칙과 패턴에 따라 조직되고 사용되는지를 다루는 **언어학적 체계**이다. 음운론은 개별 소리의 물리적 산출보다는 말소리의 **체계적 패턴과 규칙**에 관심을 둔다.

**말소리장애(speech sound disorder)**: 조음장애와 음운장애를 포괄하는 상위 개념으로, 말소리의 산출이나 음운 체계의 사용에 어려움을 보이는 경우를 총칭한다. [「장애인 등에 대한 특수교육법」](/subjects/laws/special-education-act) 제15조에서는 의사소통장애의 하위 유형으로 '조음' 영역의 제한을 명시하고 있다.

> ⚠️ **용어 검증 필요**: 말소리장애(SSD)와 조음음운장애의 용어 사용은 학자·기관에 따라 차이가 있으므로 최신 교재 확인 필요

## 주요이론

### 1. 조음의 기초: 조음위치와 조음방법

말소리는 **조음위치(place of articulation)**와 **조음방법(manner of articulation)**의 조합으로 분류된다. 한국어 자음 체계를 이해하는 데 핵심이 된다.

#### 조음위치

| 조음위치 | 조음기관 | 한국어 자음 예 |
|--------|--------|-----------|
| **양순음(bilabial)** | 두 입술 | ㅂ, ㅃ, ㅍ, ㅁ |
| **치경음(alveolar)** | 혀끝과 윗잇몸(치경) | ㄷ, ㄸ, ㅌ, ㄴ, ㄹ, ㅅ, ㅆ |
| **경구개음(palatal)** | 혀의 앞부분과 경구개(센입천장) | ㅈ, ㅉ, ㅊ |
| **연구개음(velar)** | 혀의 뒷부분과 연구개(여린입천장) | ㄱ, ㄲ, ㅋ, ㅇ(종성) |
| **성문음(glottal)** | 성문(성대 사이) | ㅎ |

#### 조음방법

| 조음방법 | 특성 | 한국어 자음 예 |
|--------|------|-----------|
| **파열음(stop/plosive)** | 구강 내 공기를 완전히 막았다가 터뜨림 | ㄱ, ㄲ, ㅋ, ㄷ, ㄸ, ㅌ, ㅂ, ㅃ, ㅍ |
| **마찰음(fricative)** | 조음기관 사이를 좁혀 공기가 마찰하며 나옴 | ㅅ, ㅆ, ㅎ |
| **파찰음(affricate)** | 파열 후 마찰이 이어지는 복합 방식 | ㅈ, ㅉ, ㅊ |
| **비음(nasal)** | 연구개를 내려 공기가 비강으로 나옴 | ㅁ, ㄴ, ㅇ(종성) |
| **유음(liquid)** | 혀끝이 치경에 가볍게 접촉하거나 측면으로 공기 흐름 | ㄹ |

> ⚠️ **용어 검증 필요**: 한국어 자음 분류 체계는 국어학과 언어병리학 간 세부 차이가 있을 수 있으므로 교재 확인 필요

### 2. 조음음운장애의 유형

조음음운장애는 **원인**에 따라 크게 기능적 조음장애와 기질적 조음장애로 나뉘며, **오류 유형**에 따라 대치·생략·왜곡·첨가로 분류된다.

#### 원인에 따른 분류

| 유형 | 특성 | 예 |
|------|------|-----|
| **기능적 조음장애(functional)** | 말소리를 산출하는 조음 기관에 구조적·신경학적 이상이 **없음**에도 불구하고 조음 오류를 보임 | 또래에 비해 특정 음소의 습득이 지연된 경우 |
| **기질적 조음장애(organic)** | 구개파열, 설소대 단축, [청각장애](/subjects/hearing-impairment/audiology), [뇌성마비](/subjects/physical-disability/cerebral-palsy) 등 **기관의 구조적 또는 신경학적 이상**으로 인한 조음 오류 | 구개파열로 인한 비누출(nasal emission) |

#### 조음 오류 유형

| 오류 유형 | 정의 | 예시 |
|---------|------|------|
| **대치(substitution)** | 목표 음소를 **다른 음소**로 바꿔 산출 | /사과/ → [타과] (ㅅ→ㅌ) |
| **생략(omission)** | 목표 음소를 **빠뜨림** | /사과/ → [아과] (ㅅ 생략) |
| **왜곡(distortion)** | 목표 음소와 유사하나 **정확하지 않은 소리**로 산출 (해당 언어의 음소 목록에 없는 소리) | /ㅅ/를 혀가 치간으로 나오며 산출 |
| **첨가(addition)** | 목표 단어에 **불필요한 음소**를 추가 | /사과/ → [스사과] |

> **KICE 출제 핵심**: **왜곡(distortion)**은 대치·생략과 달리 목표 음소와 비슷하지만 **해당 언어의 정상적인 음소 범주에 속하지 않는 소리**로 산출되는 경우이다. 기출에서 왜곡과 대치를 구분하는 문제가 빈출된다. 또한 **기능적 조음장애**는 "조음 기관에 이상이 없기 때문에(ㅇ), 조음·음운 장애를 보인다"는 표현이 핵심이다 (2025 전공B 문제5 참조).

### 3. 조음음운 평가: U-TAP 검사

**U-TAP(우리말 조음음운 평가, Urimal Test of Articulation and Phonology)**은 한국어 말소리장애 아동의 조음음운 능력을 평가하는 대표적 표준화 검사 도구이다.

#### U-TAP의 구성

| 구분 | 내용 |
|------|------|
| **단어 수준 검사** | 그림카드를 보고 목표 단어를 자발적으로 산출하게 하여 각 음소의 조음 정확성을 평가 |
| **문장 수준 검사** | 연결 발화(문장) 속에서의 조음 능력을 평가. 단어 수준보다 자연스러운 발화 맥락을 반영 |
| **자음정확도(PCC)** | Percentage of Consonants Correct. 전체 자음 중 정확하게 산출된 자음의 비율(%) |

#### 자음정확도(PCC) 산출 공식

```
PCC(%) = (정조음한 자음 수 / 전체 목표 자음 수) × 100
```

#### PCC에 따른 말 명료도 분류 (Shriberg & Kwiatkowski, 1982)

| PCC 범위 | 분류 |
|---------|------|
| 85~100% | 경도(mild) |
| 65~85% | 경도-중도(mild-moderate) |
| 50~65% | 중도-심도(moderate-severe) |
| 50% 미만 | 심도(severe) |

### 4. 변별자질 분석

**변별자질(distinctive features)**은 음소를 구별하는 데 사용되는 이진적(+/−) 자질로, 음운장애 아동의 오류 패턴을 체계적으로 분석하는 데 활용된다.

| 변별자질 | 설명 | 예 |
|--------|------|-----|
| **[+/−유성(voiced)]** | 성대 진동의 유무 | [+유성]: ㅁ, ㄴ / [−유성]: ㅍ, ㅌ |
| **[+/−비음(nasal)]** | 비강 공명의 유무 | [+비음]: ㅁ, ㄴ, ㅇ / [−비음]: ㅂ, ㄷ, ㄱ |
| **[+/−설정(coronal)]** | 혀끝/혀날 사용 여부 | [+설정]: ㄷ, ㄴ, ㅅ / [−설정]: ㅂ, ㄱ |
| **[+/−지속(continuant)]** | 공기 흐름의 지속 여부 | [+지속]: ㅅ, ㅎ / [−지속]: ㄱ, ㄷ, ㅂ |
| **[+/−전방(anterior)]** | 치경 이전에서 산출 여부 | [+전방]: ㅂ, ㄷ, ㄴ / [−전방]: ㄱ, ㅈ |

변별자질 분석을 통해 아동이 **어떤 자질 차원에서 체계적으로 오류를 보이는지** 파악할 수 있으며, 이는 **최소대립쌍(minimal pair) 치료**의 목표 선정에 직접 활용된다.

### 5. 음운변동 분석

**음운변동(phonological processes)**은 아동이 성인의 목표 말소리를 단순화하여 산출하는 체계적 패턴이다. 정상 발달 과정에서도 나타나지만, 일정 연령을 넘어서도 지속되면 음운장애로 진단된다.

#### 주요 음운변동 유형

| 음운변동 | 설명 | 예시 |
|--------|------|------|
| **경음화** | 평음을 경음으로 대치 | /바나나/ → [빠나나] |
| **파열음화(stopping)** | 마찰음·파찰음을 파열음으로 대치 | /사탕/ → [타탕] (ㅅ→ㄷ) |
| **탈비음화** | 비음을 파열음으로 대치 | /나비/ → [다비] (ㄴ→ㄷ) |
| **전설음화(fronting)** | 연구개음을 치경음 등 앞쪽 위치의 소리로 대치 | /가방/ → [다방] (ㄱ→ㄷ) |
| **후설음화(backing)** | 치경음을 연구개음 등 뒤쪽 위치의 소리로 대치 | /다리/ → [가리] (ㄷ→ㄱ) |
| **탈기식음화** | 격음(기식음)을 평음으로 대치 | /코/ → [고] (ㅋ→ㄱ) |
| **종성생략(final consonant deletion)** | 받침 자음을 생략 | /밥/ → [바] |
| **어중자음생략(medial consonant deletion)** | 단어 중간의 자음을 생략 | /안경/ → [아경] |
| **유음의 단순화** | /ㄹ/을 다른 소리로 대치하거나 생략 | /라면/ → [나면] (ㄹ→ㄴ) |

> **KICE 기출 연계 (2025 전공B 문제5)**: "학생 A는 조음 기관에 이상이 없기 때문에, 조음·음운을 보인다고 할 수 있으며, 학생 A의 말소리의 조합을 규정하는 규칙을 다루는 (음운론)에 대한 지식이 필요하다"는 형태의 문항이 출제되었다. 음운변동 분석 시 경음화 규칙, 탈비음화, 구개음화 등의 구체적 음운변동 명칭을 기입하는 문항이 빈출된다.

### 6. 음운치료 접근법

| 접근법 | 핵심 원리 | 적용 |
|------|---------|------|
| **최소대립쌍(minimal pairs)** | 하나의 변별자질만 다른 두 단어 쌍을 대비시켜 의미 차이를 인식하게 함 | 예: '달'과 '갈' (ㄷ↔ㄱ, [+전방]↔[−전방]) |
| **최대대립쌍(maximal opposition)** | 여러 변별자질이 다른 두 음소 쌍을 대비 | 예: 'ㅁ'과 'ㅅ' (비음↔마찰음, 양순↔치경) |
| **빈자리쌍(empty set)** | 아동이 산출하지 못하는 두 음소를 대비시켜 자질 습득을 유도 | 두 목표 음소 모두 아동의 음소목록에 없는 경우 |
| **음소인식(phonological awareness)** | 음소를 의식적으로 분리·합성·조작하는 상위 언어 능력을 훈련. [교육과정 중심 측정(CBM)](/subjects/assessment/cbm)에서도 읽기 유창성 평가의 기초가 된다 | 음절 분리, 첫소리 인식, 음소 합성 등 |
| **핵심어휘(core vocabulary)** | 비일관적 음운장애에 적합. 아동의 핵심 어휘를 선정하여 일관된 산출을 목표로 함 | 매 회기 동일한 단어 세트 반복 연습 |
| **의사소통 중심 접근(naturalistic)** | 의사소통 자체를 중시하고, 자연스러운 환경에서 말소리를 가르침. 의사소통에 기초한 반응과 강화를 강조 | 2025 전공B 문제5 [B] 참조 |

> **KICE 기출 연계 (2026 전공B 문제4)**: "학생 H를 조음중재할 때, 전통적인 조음중재 방법보다는 **변별자질 접근법**이나 **음운변동 접근법**을 활용하여 조음 개선 효과가 유사한 음이나 음운으로 전이되도록 함"이 출제되었다. 변별자질 접근법과 음운변동 접근법의 공통 장점(일반화/전이 효과)을 서술하는 문항이 핵심이다.

## 키워드정리

| 용어 | 설명 | 출제빈도 |
|------|------|---------|
| 조음(articulation) | 조음기관을 사용하여 말소리를 물리적으로 산출하는 과정 | ★★★ |
| 음운(phonology) | 말소리가 언어 체계 내에서 조직되고 사용되는 규칙과 패턴 | ★★★ |
| 말소리장애(SSD) | 조음장애와 음운장애를 포괄하는 상위 개념 | ★★★ |
| 기능적 조음장애 | 조음 기관에 구조적 이상 없이 조음 오류를 보이는 경우 | ★★★ |
| 기질적 조음장애 | 구개파열·설소대 단축 등 구조적/신경학적 원인에 의한 조음 오류 | ★★★ |
| 왜곡(distortion) | 목표 음소와 유사하지만 해당 언어의 음소 목록에 없는 소리로 산출 | ★★★ |
| 대치(substitution) | 목표 음소를 다른 음소로 바꿔 산출 | ★★★ |
| 생략(omission) | 목표 음소를 빠뜨리고 산출 | ★★★ |
| U-TAP | 우리말 조음음운 평가: 단어/문장 수준의 조음 검사 | ★★★ |
| 자음정확도(PCC) | 전체 목표 자음 중 정확하게 산출된 자음의 비율 | ★★★ |
| 변별자질 | 음소를 구별하는 이진적(+/−) 자질 (유성, 비음, 지속 등) | ★★★ |
| 음운변동 | 목표 말소리를 체계적으로 단순화하는 패턴 | ★★★ |
| 파열음화(stopping) | 마찰음·파찰음을 파열음으로 대치하는 음운변동 | ★★★ |
| 전설음화(fronting) | 연구개음을 치경음 등 앞쪽 소리로 대치하는 음운변동 | ★★ |
| 경음화 | 평음을 경음으로 대치하는 음운변동 | ★★ |
| 최소대립쌍(minimal pairs) | 하나의 변별자질만 다른 두 단어 쌍을 이용한 치료법 | ★★★ |
| 음소인식(phonological awareness) | 음소를 의식적으로 인식·분리·조작하는 상위 언어 능력 | ★★ |

## 기출연계

### 기입형 출제 패턴

- "학생 A는 조음 기관에 이상이 없으므로 ( ) 조음장애라 할 수 있다." → **기능적**
- "목표 음소와 유사하지만 해당 언어의 정상적 음소에 속하지 않는 소리로 산출하는 오류 유형은 ( )이다." → **왜곡(distortion)**
- "마찰음이나 파찰음을 파열음으로 대치하는 음운변동을 ( )(이)라 한다." → **파열음화(stopping)**
- "전체 목표 자음 수 대비 정조음한 자음 수의 백분율을 ( )(이)라 한다." → **자음정확도(PCC)**
- "하나의 변별자질만 다른 두 단어 쌍을 대비시켜 의미 차이를 인식하게 하는 치료 접근법은 ( )이다." → **최소대립쌍(minimal pairs)**

### 서술형 출제 패턴

**[패턴 1] 조음 오류 유형 분석 (2025 전공B 문제5 유형)**
- 학생의 말소리 산출 자료를 제시하고 나타난 조음 오류의 유형(대치/생략/왜곡)과 음운변동의 명칭을 묻는 문항
- 답안 포인트: ① 각 오류의 유형을 정확히 판별 ② 해당 음운변동의 명칭 기입 ③ 기능적 vs 기질적 원인 구별

**[패턴 2] 변별자질 접근법 vs 전통적 조음치료 (2026 전공B 문제4 유형)**
- 학생의 조음 오류 패턴을 분석하고 변별자질 접근법 또는 음운변동 접근법의 적합성과 장점(일반화/전이)을 서술
- 답안 포인트: ① 전통적 조음치료는 개별 음소 단위의 반복 연습 ② 변별자질/음운변동 접근은 체계적 패턴을 목표로 하여 전이 효과가 큼 ③ 최소대립쌍 활용법 설명

**[패턴 3] 의사소통 중심 조음치료 접근 (2025 전공B 문제5 [B] 유형)**
- 의사소통 자체를 중시하는 자연주의적 접근에서의 조음 지도 방법을 서술
- 답안 포인트: ① 의사소통에 기초한 반응과 강화 강조 ② 자연스러운 발화 맥락에서 목표 음소의 모델링 제공 ③ 확대(expansion) 기법 활용

### 최근 출제 경향

- **기능적 조음장애**의 개념과 음운변동 분석은 매년 출제되는 핵심 영역
- **변별자질 접근법**의 원리와 장점(일반화 효과)을 묻는 서술형 문항 증가
- U-TAP 검사 결과(PCC)를 해석하고 중재 방향을 제안하는 **통합형 문항** 출제
- 조음위치·조음방법 분류와 음운변동 명칭을 **기입형**으로 묻는 문항 꾸준히 출제

- [ ] 대치, 생략, 왜곡, 첨가의 차이를 예시와 함께 설명할 수 있다.
- [ ] 기능적 조음장애와 기질적 조음장애의 차이를 구분하여 설명할 수 있다.
- [ ] U-TAP 검사의 단어 수준/문장 수준 검사 차이를 설명할 수 있다.
- [ ] 자음정확도(PCC) 산출 공식을 제시하고 결과를 해석할 수 있다.
- [ ] 주요 변별자질(유성, 비음, 설정, 지속, 전방)을 설명하고 음소를 분류할 수 있다.
- [ ] 파열음화, 전설음화, 탈비음화, 경음화 등 주요 음운변동의 정의와 예시를 들 수 있다.
- [ ] 최소대립쌍 치료의 원리와 목표 선정 방법을 설명할 수 있다.
- [ ] 음소인식 훈련의 활동 유형(음절 분리, 첫소리 인식, 음소 합성 등)을 나열할 수 있다.
