## 핵심개념

**언어중재(language intervention)**: 언어 발달에 어려움을 보이는 아동의 수용·표현 언어 능력을 향상시키기 위해 체계적으로 계획된 교수 활동을 말한다. 언어중재는 **구조화된 접근(clinician-directed)**부터 **자연주의적 접근(child-directed/naturalistic)**까지 연속선상에 위치한다.

**자발화분석(spontaneous language sample analysis)**: 아동이 자연스러운 상황에서 자발적으로 산출한 발화를 수집하여 언어 능력을 분석하는 **비표준화 평가** 방법이다. [표준화 검사](/subjects/assessment/standardized-tests)와 함께 사용하면 더욱 종합적인 평가가 가능하다. 표준화 검사로 파악하기 어려운 실제 의사소통 능력을 평가할 수 있다.

**환경중심 언어중재(milieu teaching)**: 아동의 자연스러운 일상 환경과 관심사를 활용하여 의사소통 기회를 만들고, 그 맥락 속에서 언어를 가르치는 자연주의적 중재 접근이다. **강화된 환경중심 언어중재(EMT, Enhanced Milieu Teaching)**는 이를 더욱 체계화한 모형이다.

> ⚠️ **용어 검증 필요**: '환경중심 언어중재'의 영문 표기(milieu teaching)와 하위 기법 분류는 학자에 따라 다소 차이가 있을 수 있음

## 주요이론

### 1. 강화된 환경중심 언어중재(EMT) — 4가지 기법

**EMT(Enhanced Milieu Teaching)**는 아동의 자연스러운 환경에서 의사소통 기회를 의도적으로 구성하고, 아동의 관심과 주도(initiative)를 따르면서 언어를 가르치는 체계적 접근이다. EMT는 다음의 **4가지 핵심 교수 기법**으로 구성된다.

| 기법 | 절차 | 예시 |
|------|------|------|
| **시범(modeling)** | 아동이 관심을 보이는 사물이나 활동에 대해 교사가 **목표 언어 형태를 직접 보여줌** | 아동이 공을 보면 교사가 "공이야. 빨간 공." 이라고 말함 |
| **요구-모델(mand-model)** | ① 아동이 관심을 보이면 교사가 **언어적 요구(mand)**를 함 → ② 아동이 반응하지 못하면 **모델**을 제공 → ③ 아동이 모방하면 **강화** | ① "뭐라고 해야 하지?" → ② (무반응 시) "공 주세요." → ③ 아동이 "공 주세요" 하면 공을 줌 |
| **시간지연(time delay)** | 아동이 원하는 것이 있을 때 교사가 **기대하는 눈빛으로 기다림**(시간을 지연시킴)으로써 아동의 자발적 의사소통을 유도 | 아동이 좋아하는 장난감을 눈앞에 놓고 기대하는 표정으로 약 5~10초 기다림 |
| **우발교수(incidental teaching)** | 아동이 **자발적으로 요구**를 시작했을 때, 좀 더 정교하거나 확장된 언어 형태를 사용하도록 촉진 | 아동이 "주스" 하면 교사가 "주스 주세요, 라고 해 보자." |

#### EMT의 물리적 환경 조절 전략

EMT에서는 아동의 의사소통 기회를 극대화하기 위해 **물리적 환경을 의도적으로 조절**한다.

| 환경 조절 전략 | 내용 |
|-----------|------|
| **보이지만 닿지 않는 곳에 배치** | 아동이 원하는 물건을 눈에 보이지만 손이 닿지 않는 곳에 두어 요청을 유도 |
| **불충분한 자료 제공** | 활동에 필요한 자료를 일부만 제공하여 추가 요청을 유도 |
| **선택 기회 제공** | 두 가지 이상의 선택지를 제시하여 아동이 선호하는 것을 언어로 표현하도록 유도 |
| **예상치 못한 상황 만들기** | 익숙한 활동에서 예상과 다른 상황을 만들어 아동의 반응을 유도 |
| **아동의 관심 따르기** | 아동이 주도하는 활동과 관심사를 따르며 그 맥락에서 언어 모델 제공 |

> **KICE 기출 연계 (2025 전공A 문제8)**: "강화된 환경 중심 언어 중재(Enhanced Milieu Teaching: EMT)에 근거하여 [A]에서 교사가 사용한 교수 기법의 명칭을 쓰고, 밑줄 친 ㉠에서 제시하고 있는 물리적 환경 조절 전략 중 1가지를 쓸 것." — 문항에서 교사가 학생이 좋아하는 물건을 눈에 보이는 곳에 두고 "뭐라고 해야 하지?"라고 묻는 장면이 제시되었고, 이는 **요구-모델(mand-model)** 기법에 해당한다. 환경 조절 전략으로는 "보이지만 닿지 않는 곳에 배치" 등이 정답이다.

### 2. 자발화분석 지표

자발화 표본을 수집하여 분석할 때 사용하는 핵심 지표는 다음과 같다. 각 지표는 언어의 **서로 다른 영역**을 반영한다.

| 지표 | 정의 | 반영하는 언어 영역 | 산출 공식 |
|------|------|-------------|---------|
| **MLU (Mean Length of Utterance)** | 평균발화길이. 발화의 평균적인 길이를 측정 | **구문론(syntax)** — 문장의 복잡성과 길이를 반영 | 전체 형태소(또는 낱말) 수 / 전체 발화 수 |
| **TTR (Type-Token Ratio)** | 유형-토큰 비율. 전체 낱말 중 서로 다른 낱말의 비율 | **의미론(semantics)** — 어휘의 다양성을 반영 | 다른 낱말 수(type) / 전체 낱말 수(token) |
| **NDW (Number of Different Words)** | 다른 낱말 수. 발화에서 사용된 서로 다른 낱말의 총 수 | **의미론(semantics)** — 어휘의 다양성(절대량)을 반영 | 발화에서 사용된 서로 다른 낱말의 수 |

#### MLU-w(낱말 단위 평균발화길이) 계산 예시

한국어에서는 형태소보다 **낱말 단위**로 MLU를 산출하는 경우가 많으며, 이를 **MLU-w (Mean Length of Utterance in words)**라 한다.

**발화 예시:**

| 발화 번호 | 발화 내용 | 낱말 수 |
|---------|---------|---------|
| 1 | "나도" | 1 |
| 2 | "우유 줘" | 2 |
| 3 | "우유 좋아" | 2 |

```
MLU-w = 전체 낱말 수 / 전체 발화 수
       = (1 + 2 + 2) / 3
       = 5 / 3
       = 1.67
```

> **KICE 기출 연계 (2026 전공A 문제6)**: 자발화 분석 문항에서 교사와 교육실습생의 대화를 통해 자발화 표본 수집 방법(자연스러운 장소, 대화 주제, 발화 단위 구분 기준)과 자발화 분석을 통해 파악할 수 있는 언어학의 하위 영역(예: **평균발화길이(MLU)** → 구문론, **어휘다양도(TTR/NDW)** → 의미론)을 묻는 문항이 출제되었다.

#### 자발화 표본 수집 시 유의사항

| 항목 | 내용 |
|------|------|
| **수집 장소** | 학생이 주로 생활하는 **자연스럽고 익숙한 장소** 2곳 이상에서 수집 |
| **대화 주제** | 학생이 **주도하도록** 하는 것이 좋음 |
| **표본 크기** | 최소 50~100개 발화 수집이 권장됨 |
| **발화 단위 구분** | 분석자가 **일관된 발화 구분 기준**을 적용해야 함 (억양, 쉼, 문법적 완성도 등) |

### 3. 언어촉진 기법

언어중재에서 사용하는 대표적 **간접 언어촉진 기법**은 다음과 같다.

| 기법 | 정의 | 예시 |
|------|------|------|
| **혼잣말(self-talk)** | 교사가 **자신의 행동이나 경험**을 언어로 묘사 | (교사가 블록을 쌓으며) "선생님이 블록을 높이 쌓고 있어." |
| **평행적 발화(parallel talk)** | 교사가 **아동의 행동이나 경험**을 언어로 묘사 | (아동이 그림을 그리고 있을 때) "00이가 빨간색으로 그림을 그리고 있구나." |
| **확대(expansion)** | 아동의 발화를 **문법적으로 완성된 형태**로 다시 말해 줌. 의미는 바꾸지 않고 문법적 요소를 추가 | 아동: "엄마 밥" → 교사: "엄마가 밥을 먹어." |
| **확장(extension/expatiation)** | 아동의 발화에 **새로운 의미 정보**를 추가하여 다시 말해 줌 | 아동: "강아지 자" → 교사: "응, 강아지가 소파에서 자고 있네." |

> **확대(expansion)와 확장(extension)의 구분**: **확대**는 아동의 발화를 문법적으로 완성하는 것(구문 수정)이고, **확장**은 아동의 발화에 새로운 의미 정보를 더하는 것(의미 확장)이다. KICE에서 두 개념을 혼동하지 않도록 정확히 구분하는 것이 핵심이다.

> **KICE 기출 연계 (2025 전공B 문제5)**: 의사소통장애 학생 A에 대해 통합학급 교사와 특수 교사의 대화에서, 특수 교사가 "의사소통 자체를 중시하고, 자연스러운 환경에서 말소리를 가르치는 의사소통중심법을 사용해 보세요. 의사소통에 기초한 반응과 강화를 강조해요. 예를 들어, 학생이 오류를 보이는 /ㅅ/의 발음을 '손 소독제 주세요.'라고 정확하게 소리 내었다면, 선생님이 (ㅇ)와/과 같이 반응하시는 겁니다." — **확대(expansion)** 기법의 교사 강화 반응 예를 묻는 문항이 출제되었다.

### 4. 유창성장애(말더듬, stuttering)

**유창성장애**는 말의 흐름(유창성)이 비정상적으로 방해되는 의사소통장애이다. [장애 영역별 이해](/subjects/introduction/disability-types)에서 의사소통장애의 정의와 유형을 다루고 있다. 가장 대표적인 유형이 **말더듬(stuttering)**이다.

#### 말더듬의 핵심행동과 부수행동

| 구분 | 정의 | 하위 유형 |
|------|------|---------|
| **핵심행동(core behaviors)** | 말더듬의 **직접적·필수적** 특성으로, 말의 흐름이 방해되는 행동 | **반복(repetition)**: 음절/낱말/구 반복 ("나-나-나는") |
| | | **연장(prolongation)**: 소리를 비정상적으로 길게 늘임 ("ㅅㅅㅅ사과") |
| | | **막힘/차단(block)**: 말이 나오지 않고 멈추는 현상 (공기가 막힘) |
| **부수행동(secondary/associated behaviors)** | 핵심행동에 수반되어 나타나는 **부차적** 행동. 말더듬을 회피하거나 빠져나오려는 시도에서 학습됨 | **탈출행동(escape)**: 막힘에서 벗어나려는 행동 (눈 깜빡임, 고개 끄덕임, 발 구르기 등) |
| | | **회피행동(avoidance)**: 말더듬이 예상되는 상황이나 단어를 피하는 행동 (대체어 사용, 말하기 상황 회피) |

#### 말더듬 중재 접근

| 접근 | 대상 | 핵심 내용 |
|------|------|---------|
| **유창성 완성법(fluency shaping)** | 모든 연령 | 이완된 발성, 느린 말 속도, 부드러운 시작 등 **유창한 말 산출 기술**을 직접 가르침 |
| **말더듬 수정법(stuttering modification)** | 주로 학령기 이상 | 말더듬의 순간을 인식하고 **더 쉬운 형태의 말더듬**으로 수정하는 기술 습득. 4단계: 확인(identification) → 둔감화(desensitization) → 수정(modification) → 안정화(stabilization) |
| **간접치료(indirect therapy)** | 주로 학령전기 | 아동에게 직접 유창성 기술을 가르치기보다 **환경을 조절**(말 속도 늦추기, 발화 차례 기다리기, 압력 줄이기)하여 유창성을 촉진 |
| **리드콤 프로그램(Lidcombe Program)** | 학령전기 | 부모가 가정에서 아동의 **유창한 발화에 긍정적 피드백**, 비유창한 발화에 교정적 피드백을 제공하는 행동주의적 접근 |

### 5. 언어중재의 연속체

언어중재는 **교사 주도(clinician-directed)**에서 **아동 주도(child-directed)**까지 연속선상에 위치한다.

```
교사 주도 ←──────────────────────────────→ 아동 주도
(구조화됨)                                     (자연주의적)

직접 교수법    시범-모방    요구-모델    시간지연    우발교수    환경 배열
(drill)       (model)    (mand)     (delay)   (incidental) (milieu)
```

| 접근 | 특성 | 장점 | 단점 |
|------|------|------|------|
| **교사 주도(구조화)** | 교사가 목표, 자극, 반응, 강화를 통제 | 체계적·효율적, 집중적 연습 가능 | 일반화 어려움, 동기 저하 가능 |
| **아동 주도(자연주의)** | 아동의 관심과 주도를 따르며 자연스러운 맥락에서 중재. [긍정적 행동지원(PBS)](/subjects/behavior-support/pbs)의 선행사건 중재와 유사한 원리 적용 | 일반화 촉진, 동기 유발, 의미 있는 맥락 | 목표 행동 연습 기회가 적을 수 있음 |
| **하이브리드(EMT 등)** | 자연주의적 맥락 + 체계적 교수 기법 결합 | 일반화와 체계성 모두 확보 | 교사의 숙련된 기술 필요 |

| EMT(강화된 환경중심 언어중재) | 환경중심 언어중재를 체계화한 모형. 4가지 교수 기법 포함 | ★★★ |
| 시범(modeling) | 교사가 목표 언어 형태를 직접 보여주는 기법 | ★★★ |
| 요구-모델(mand-model) | 언어적 요구 → 무반응 시 모델 제공 → 모방 시 강화 | ★★★ |
| 시간지연(time delay) | 기대하는 표정으로 기다려 아동의 자발적 표현 유도 | ★★★ |
| 우발교수(incidental teaching) | 아동의 자발적 요구를 확장·정교화하는 기법 | ★★★ |
| MLU(평균발화길이) | 발화의 평균 길이. 구문론적 발달 지표 | ★★★ |
| MLU-w | 낱말 단위 평균발화길이 = 전체 낱말 수 / 전체 발화 수 | ★★★ |
| TTR(유형-토큰비율) | 다른 낱말 수(type) / 전체 낱말 수(token). 어휘 다양성(의미론) 지표 | ★★★ |
| NDW(다른 낱말 수) | 발화에서 사용된 서로 다른 낱말의 수. 어휘 다양성(의미론) 지표 | ★★ |
| 혼잣말(self-talk) | 교사가 자신의 행동을 언어로 묘사하는 기법 | ★★ |
| 평행적 발화(parallel talk) | 교사가 아동의 행동을 언어로 묘사하는 기법 | ★★ |
| 확대(expansion) | 아동 발화를 문법적으로 완성하여 다시 말해 줌 (구문 수정) | ★★★ |
| 확장(extension) | 아동 발화에 새로운 의미 정보를 추가하여 다시 말해 줌 (의미 확장) | ★★★ |
| 핵심행동(core behaviors) | 말더듬의 직접적 특성: 반복, 연장, 막힘 | ★★★ |
| 부수행동(secondary behaviors) | 말더듬에 수반되는 부차적 행동: 탈출행동, 회피행동 | ★★★ |
| 반복(repetition) | 음절/낱말/구를 반복하는 핵심행동 | ★★ |
| 연장(prolongation) | 소리를 비정상적으로 길게 늘이는 핵심행동 | ★★ |
| 막힘(block) | 말이 나오지 않고 멈추는 핵심행동 | ★★ |
| 유창성 완성법 | 유창한 말 산출 기술을 직접 가르치는 중재 접근 | ★★ |
| 말더듬 수정법 | 말더듬을 인식하고 더 쉬운 형태로 수정하는 기술 습득 | ★★ |

## 기출연계

### 기입형 출제 패턴

- "아동이 관심을 보일 때 교사가 '뭐라고 해야 하지?'라고 언어적 요구를 하고, 반응이 없으면 모델을 제공하는 EMT 기법은 ( )이다." → **요구-모델(mand-model)**
- "전체 낱말 수를 전체 발화 수로 나눈 값으로, 구문론적 발달 수준을 반영하는 지표는 ( )이다." → **평균발화길이(MLU)** 또는 **MLU-w**
- "아동의 발화를 문법적으로 완성하여 다시 말해 주는 기법은 ( )이고, 새로운 의미 정보를 추가하여 다시 말해 주는 기법은 ( )이다." → **확대(expansion)**, **확장(extension)**
- "말더듬의 직접적·필수적 특성인 반복, 연장, 막힘을 총칭하여 ( )이라 하고, 말더듬을 회피하거나 빠져나오려는 부차적 행동을 ( )이라 한다." → **핵심행동**, **부수행동**
- "자발화 표본에서 사용된 서로 다른 낱말의 수를 전체 낱말 수로 나눈 값을 ( )(이)라 하며, 이는 어휘의 ( )을/를 반영한다." → **유형-토큰비율(TTR)**, **다양성**

### 서술형 출제 패턴

**[패턴 1] EMT 교수 기법 판별 및 환경 조절 전략 서술 (2025 전공A 문제8 유형)**
- 교사와 학생의 대화 장면을 제시하고 사용된 EMT 기법의 명칭과 물리적 환경 조절 전략을 묻는 문항
- 답안 포인트: ① 4가지 기법(시범/요구-모델/시간지연/우발교수) 중 해당 기법 명칭 ② 환경 조절 전략의 구체적 내용 서술 ③ 각 기법의 절차(요구→모델→강화) 설명

**[패턴 2] 자발화분석 지표 산출 및 해석 (2026 전공A 문제6 유형)**
- 아동의 발화 기록을 제시하고 MLU-w, TTR 등을 산출하거나, 자발화 분석으로 파악할 수 있는 언어 영역을 묻는 문항
- 답안 포인트: ① MLU-w 산출 공식과 계산 과정 ② MLU → 구문론, TTR/NDW → 의미론 대응 ③ 자발화 표본 수집 시 유의사항(자연스러운 환경, 일관된 발화 구분 기준)

**[패턴 3] 확대(expansion)와 확장(extension) 구분 적용**
- 교사의 언어촉진 발화를 제시하고 확대인지 확장인지 판별하는 문항
- 답안 포인트: ① 확대 = 문법적 완성(의미 변화 없음), 확장 = 새로운 의미 정보 추가 ② 구체적 사례 적용

### 최근 출제 경향

- **EMT 4가지 기법** 중 특히 **요구-모델(mand-model)**의 절차를 묻는 기입형·서술형 문항이 핵심
- **자발화분석 지표(MLU, TTR, NDW)**의 정의와 반영 언어 영역(구문론/의미론)을 묻는 문항 꾸준히 출제
- **확대(expansion)와 확장(extension)**의 구분은 매년 출제되는 단골 주제
- **유창성장애**의 핵심행동 3가지(반복/연장/막힘)와 부수행동(탈출/회피)을 구분하는 문항 출제
- EMT의 **물리적 환경 조절 전략**(보이지만 닿지 않는 곳 배치, 불충분한 자료 제공 등)을 서술형으로 묻는 문항 증가
- 의사소통중심법(2025 전공B 문제5)에서 **확대(expansion)** 기법의 교사 반응 예를 묻는 문항 출제
