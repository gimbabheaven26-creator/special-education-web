## 핵심개념

**보청기(hearing aid)**: 잔존 청력이 있는 청각장애인의 청취 능력을 향상시키기 위해 소리를 **증폭(amplification)**하여 전달하는 전자 장치이다. 보청기는 소리를 **증폭**하는 기기로, 음향 에너지를 더 큰 음향 에너지로 변환하여 전달한다.

**인공와우(cochlear implant)**: 고도 이상의 감각신경성 난청으로 보청기의 효과가 제한적인 경우, 와우(달팽이관) 내에 전극을 삽입하여 **소리를 전기 신호로 변환**한 후 청신경을 직접 자극하는 장치이다. 보청기가 소리를 증폭하는 것과 달리, 인공와우는 소리를 **전기 자극으로 변환**하는 기기이다.

**증폭(amplification)**: 입력된 소리의 강도를 높여 출력하는 것을 말한다. 보청기의 핵심 기능이며, 증폭량(이득, gain)은 **출력 음압 - 입력 음압**으로 계산된다.

> **용어 검증 필요**: 보청기와 인공와우의 정의는 「장애인 보조기기 활용촉진법」 및 최신 청각학 교재와 대조할 것

## 주요이론

### 1. 보청기의 구성요소

보청기는 기본적으로 **마이크로폰(microphone)**, **증폭기(amplifier)**, **수화기(receiver)**의 세 가지 핵심 구성요소로 이루어져 있다.

| 구성요소 | 기능 | 변환 과정 |
|---------|------|---------|
| **마이크로폰(microphone)** | 외부의 음향 에너지를 **전기 에너지**로 변환하는 **입력 변환기(input transducer)** | 소리(음향) → 전기 신호 |
| **증폭기(amplifier)** | 마이크로폰에서 전달된 전기 신호를 **증폭**하는 장치. 디지털 보청기에서는 신호처리(noise reduction, feedback cancellation 등)도 수행 | 전기 신호 → 증폭된 전기 신호 |
| **수화기(receiver)** | 증폭된 전기 에너지를 다시 **음향 에너지**로 변환하는 **출력 변환기(output transducer)** | 전기 신호 → 소리(음향) |

보청기 작동 과정 요약:
```
외부 소리 → [마이크로폰] 음향→전기 → [증폭기] 전기 신호 증폭 → [수화기] 전기→음향 → 귀
```

여기서 **변환기(transducer)**란 에너지의 형태를 바꾸는 장치를 의미하며, 마이크로폰은 입력 변환기, 수화기는 출력 변환기에 해당한다.

### 2. 보청기의 유형

보청기는 착용 위치와 크기에 따라 다양한 유형으로 분류된다.

| 유형 | 명칭 | 착용 위치 | 특징 |
|------|------|---------|------|
| **BTE** | 귀걸이형(Behind-The-Ear) | 귓바퀴 뒤에 본체, 이어몰드를 통해 소리 전달 | 가장 넓은 출력 범위. 모든 난청 정도에 적용 가능. 소아에게 가장 많이 사용. 조작과 관리가 용이 |
| **ITE** | 귓속형(In-The-Ear) | 이개(귓바퀴) 전체에 삽입 | 중등도~고도 난청에 적용. BTE보다 크기가 작아 미관상 유리 |
| **ITC** | 외이도형(In-The-Canal) | 외이도 입구에 삽입 | 경도~중고도 난청에 적용. ITE보다 작아 눈에 덜 띔 |
| **CIC** | 고막형(Completely-In-the-Canal) | 외이도 깊숙이 삽입, 외부에서 거의 보이지 않음 | 경도~중등도 난청에 적용. 가장 작은 크기. 배터리 크기 작아 수명 짧음. 피드백 적음 |
| **RIC** | 수화기외이도형(Receiver-In-Canal) | 본체는 귀 뒤, 수화기만 외이도에 삽입 | BTE와 유사하나 수화기가 외이도에 위치하여 더 자연스러운 음질 제공 |

> **용어 검증 필요**: 최신 보청기 유형(RIC 등)과 디지털 보청기의 신호처리 기능은 보청기 전문 교재 확인 필요

### 3. 인공와우의 구조

인공와우는 **체외부(external part)**와 **체내부(internal part)**로 나뉜다.

| 구분 | 구성요소 | 기능 |
|------|---------|------|
| **체외부** | **마이크로폰** | 외부의 소리를 수집하여 전기 신호로 변환 |
| | **어음처리기(speech processor)** | 마이크에서 수집된 전기 신호를 분석·부호화(coding)하여 전기 자극 신호로 변환 |
| | **전송코일(transmitter coil)** | 어음처리기에서 변환된 전기 자극 신호를 **피부를 통해(경피적으로)** 체내부 수신기에 전달 (전자기 유도 방식) |
| **체내부** | **수신기(receiver/stimulator)** | 전송코일에서 받은 신호를 수신하고 전극에 전달 |
| | **전극(electrode array)** | 와우(달팽이관) 내에 삽입되어 **청신경을 직접 전기 자극** |

인공와우 작동 과정 요약:
```
외부 소리 → [마이크] → [어음처리기] 신호 분석·변환 → [전송코일] 경피 전달 → [수신기] → [전극] 청신경 자극 → 대뇌
```

**보청기와 인공와우의 핵심 차이**:

| 비교 항목 | 보청기 | 인공와우 |
|---------|-------|---------|
| 원리 | 소리를 **증폭**(음향→음향) | 소리를 **전기 자극으로 변환**(음향→전기) |
| 대상 | 경도~고도 난청(잔존 청력 활용) | 보청기로 효과가 제한적인 고도 이상 감각신경성 난청 |
| 구조 | 체외부만 존재 | 체외부 + 체내부(수술 필요) |
| 변환기 | 음향 신호를 음향 신호로 출력 (수화기 = 출력 변환기) | 음향 신호를 전기 신호로 출력 (전극 = 출력 변환기) |

2026학년도 전공A 9번 문항에서 보청기와 인공와우의 기본 구조와 기능 차이가 출제되었다. 보청기와 인공와우 모두 **음향 신호를 전기 신호로 바꿔 주는** 입력 변환기(마이크로폰)를 가지고 있으나, **출력 과정**에서 보청기는 소리를 증폭하여 전달하고 인공와우는 소리를 전기 자극으로 변환하여 전달한다는 점이 핵심 차이이다.

> **용어 검증 필요**: 인공와우의 수술 적합 기준(나이, 청력 기준 등)과 재활 과정은 최신 임상 가이드라인 확인 필요

### 4. FM시스템과 보조청취장치

**FM시스템(Frequency Modulation System)**은 교실 등 소음이 있는 환경에서 화자(교사)의 말소리를 FM 전파로 직접 청각장애 학생에게 전달하는 보조청취장치이다.

| 구성 | 기능 |
|------|------|
| **송신기(transmitter) + 마이크** | 화자(교사)가 착용. 말소리를 FM 전파로 송신 |
| **수신기(receiver)** | 학생이 보청기나 인공와우에 연결하여 착용. FM 전파를 수신하여 음향 신호로 변환 |

FM시스템의 핵심 효과는 **신호대잡음비(SNR: Signal-to-Noise Ratio)**를 개선하는 것이다.

---

### 5. 신호대잡음비(SNR: Signal-to-Noise Ratio)

**SNR**은 목표 신호(말소리)의 크기와 배경 소음의 크기 차이를 dB로 나타낸 것이다.

```
SNR(dB) = 신호 수준(dB) - 소음 수준(dB)
```

| SNR 값 | 의미 |
|--------|------|
| **양수(+)** | 말소리가 소음보다 큼 → 듣기 유리 |
| **0** | 말소리와 소음이 같은 크기 |
| **음수(-)** | 소음이 말소리보다 큼 → 듣기 매우 어려움 |

- 정상 청력인은 SNR이 0 dB 이상이면 대화 이해에 큰 어려움이 없음
- 청각장애 학생은 SNR이 **+15~+20 dB 이상** 되어야 효과적인 청취가 가능
- FM시스템은 교사의 말소리를 직접 전달하여 SNR을 **+15~+20 dB** 이상으로 개선

교실 환경에서 SNR을 개선하는 방법:
1. FM시스템, 사운드필드 시스템 등 보조청취장치 활용
2. 교실 내 **소음 줄이기**: 카펫, 커튼, 천장 흡음재 설치
3. 교실 내 **반향(reverberation) 줄이기**: 흡음 자재 설치
4. 교사가 학생에게 가까이 다가가서 말하기

> **용어 검증 필요**: SNR 개선 수치와 FM시스템의 효과 범위는 청각학 교재 확인 필요

### 6. 양이효과(Binaural Effect)

**양이효과**란 **양쪽 귀로 동시에 소리를 들을 때** 한쪽 귀로만 들을 때보다 얻는 이점을 말한다.

| 양이효과 유형 | 내용 |
|-----------|------|
| **양이합산(binaural summation)** | 양쪽 귀로 들으면 한쪽 귀로 들을 때보다 약 3~6 dB 더 크게 인식됨 |
| **양이오징(binaural squelch)** | 양쪽 귀의 정보를 종합하여 소음 속에서 말소리를 더 잘 분리·인식할 수 있음 |
| **두영효과(head shadow effect)** | 머리가 소음을 차단하여 소음 반대편 귀에서 더 나은 SNR을 얻을 수 있음 (엄밀히 양이효과라기보다 양이 착용의 이점) |
| **소리 방향 변별(localization)** | 양쪽 귀에 도달하는 소리의 시간차·강도차를 이용하여 소리의 방향을 판별할 수 있음 |

2025학년도 전공B 11번 문항에서 학생 A가 양쪽 귀에 보청기를 착용하는 이유로 **양이 효과**가 제시되었다. 한쪽 귀에만 보청기를 착용하면 소리의 방향을 판별하기 어렵고, 소음 속에서의 말소리 이해력이 떨어지므로 가능한 양쪽에 착용하는 것이 권장된다.

- 2026학년도 전공A 9번: 보청기와 인공와우의 **구조·기능 비교**, 인공와우 변환기(transducer), **지문자** 활동
- 2025학년도 전공B 11번: 청각장애 학생의 보청기 착용과 **양이효과**, 교실 내 반향 문제, **수어 문법**
