## 핵심개념

**청력(hearing)**: 소리를 감지하고 인식하는 능력을 말한다. 청력은 소리의 **강도(intensity)**와 **주파수(frequency)**에 의해 결정되며, 청각기관(외이·중이·내이·청신경·청각중추)을 통해 소리가 전달·처리되는 과정에서 어느 부위에 손상이 있느냐에 따라 청력 손실의 유형과 정도가 달라진다.

**데시벨(dB, decibel)**: 소리의 **강도(크기)**를 나타내는 단위이다. 청력검사에서는 **dB HL(Hearing Level)**을 사용하며, 이는 정상 청력을 기준(0 dB HL)으로 얼마나 큰 소리가 있어야 들을 수 있는지를 나타낸다. 숫자가 클수록 청력 손실이 심한 것을 의미한다.

**주파수(Hz, Hertz)**: 소리의 **높낮이(pitch)**를 나타내는 단위이다. 1초에 진동하는 횟수를 의미하며, 주파수가 높을수록 고음, 낮을수록 저음이다. 인간의 가청 주파수 범위는 약 **20~20,000 Hz**이며, 일상 대화에서 주로 사용되는 주파수 범위는 **250~4,000 Hz**이다.

**청력역치(hearing threshold)**: 특정 주파수의 소리를 **겨우 들을 수 있는 최소한의 강도(dB)**를 말한다. 청력역치가 높을수록 더 큰 소리가 있어야 들을 수 있으므로 청력 손실이 큰 것이다.

**청력도(audiogram)**: 각 주파수별 청력역치를 그래프에 표시한 것으로, 가로축에 주파수(Hz), 세로축에 청력역치(dB HL)를 나타낸다. 청력도를 통해 난청의 유형, 정도, 형태를 파악할 수 있다.

> **용어 검증 필요**: dB HL, dB SPL, dB SL 등 데시벨 단위의 구분은 최신 청각학 교재를 참고하여 확인할 것

## 주요이론

### 1. 소리의 전달 경로

소리가 귀에 도달하여 뇌에서 인식되기까지의 경로는 크게 **기도(air conduction)**와 **골도(bone conduction)** 두 가지가 있다.

| 전달 경로 | 과정 | 검사 의의 |
|---------|------|---------|
| **기도(air conduction)** | 외이(이개·외이도) → 중이(고막·이소골) → 내이(와우) → 청신경 → 대뇌 청각중추 | 소리의 전체 전달 경로를 평가. 외이·중이·내이 중 어디에 문제가 있어도 기도 청력역치가 상승함 |
| **골도(bone conduction)** | 두개골 진동 → 내이(와우) → 청신경 → 대뇌 청각중추 | 외이·중이를 거치지 않고 내이 기능을 직접 평가. 감각신경 기능만을 반영함 |

기도는 외이부터 뇌까지 전달되는 소리의 경로이고, 골도는 두개골을 진동시켜 (외이와 중이를 건너뛰고) 내이부터 소리가 전달되는 경로이다. 따라서 **기도검사는 전체 청각 경로**를, **골도검사는 내이 이후의 감각신경 기능**만을 평가한다.

> **용어 검증 필요**: 소리 전달 경로의 세부 해부학적 구조(이소골 3개: 추골·침골·등골 등)는 청각학 교재 확인 필요

### 2. 청력검사의 유형

#### 순음청력검사(PTA: Pure Tone Audiometry)

순음청력검사는 **오디오미터(audiometer)**를 사용하여 각 주파수별로 순음(pure tone)을 들려주고 피검자가 반응하는 최소 강도(청력역치)를 측정하는 검사이다.

| 검사 유형 | 측정 방법 | 평가 내용 |
|---------|---------|---------|
| **기도검사(air conduction test)** | 헤드폰 또는 삽입형 이어폰을 통해 순음 제시 | 외이~대뇌까지 전체 청각 경로의 기능 평가 |
| **골도검사(bone conduction test)** | 골진동자(bone vibrator)를 유양돌기에 부착하여 순음 제시 | 중이를 건너뛰고 내이 이후의 감각신경 기능 평가 |

- 검사 주파수: 보통 **250, 500, 1000, 2000, 4000, 8000 Hz**에서 측정
- 검사 강도: 0~110 dB HL 범위

#### 어음청력검사(SRT/WRS)

| 검사 | 내용 |
|------|------|
| **어음청취역치검사(SRT: Speech Reception Threshold)** | 이음절어(spondee words)를 50% 정확하게 따라 말할 수 있는 최소 강도를 측정. 순음 청력역치(PTA)와 대략 일치해야 함 |
| **어음명료도검사(WRS: Word Recognition Score)** | 쾌적 청취 수준에서 단음절어를 제시하고 정확하게 인식하는 비율(%)을 측정. **어음명료도(어음변별력)**가 낮으면 보청기 착용 후에도 말소리 이해에 한계가 있을 수 있음 |

> **용어 검증 필요**: 어음청력검사의 세부 절차와 SRT-PTA 일치 기준은 최신 청각학 교재 확인 필요

### 3. 청력도(Audiogram) 해석

#### 청력도 기호

청력도에서는 좌·우 귀 및 기도·골도 검사를 구분하기 위해 국제 표준 기호를 사용한다.

| 기호 | 의미 | 색상 |
|------|------|------|
| **O** | 우측 귀 기도(right air conduction) | 빨간색 |
| **X** | 좌측 귀 기도(left air conduction) | 파란색 |
| **&lt;** | 우측 귀 골도(right bone conduction) | 빨간색 |
| **&gt;** | 좌측 귀 골도(left bone conduction) | 파란색 |
| **[** | 우측 귀 골도 차폐(right bone masked) | 빨간색 |
| **]** | 좌측 귀 골도 차폐(left bone masked) | 파란색 |
| **△** | 우측 귀 기도 차폐(right air masked) | 빨간색 |
| **□** | 좌측 귀 기도 차폐(left air masked) | 파란색 |

- 가로축: 주파수(Hz) — 왼쪽이 저주파(125 Hz), 오른쪽이 고주파(8000 Hz)
- 세로축: 청력역치(dB HL) — 위쪽이 0 dB(정상), 아래로 갈수록 청력 손실이 큼

#### 기도-골도차(Air-Bone Gap, ABG)

**기도-골도차(ABG)**란 동일 주파수에서 **기도 청력역치와 골도 청력역치의 차이**를 말한다. 이 차이를 통해 난청의 유형을 감별할 수 있다.

- ABG가 **10 dB 이하**: 기도와 골도 역치가 거의 같으므로 **외이·중이에는 문제가 없음**
- ABG가 **15 dB 이상**: 기도 역치가 골도 역치보다 높으므로 **외이·중이에 전음 문제가 있음**

> **용어 검증 필요**: ABG의 임상적 기준(10 dB vs 15 dB)은 청각학 교재마다 차이가 있을 수 있으므로 확인 필요

### 4. 난청의 유형

청력 손실은 손상 부위에 따라 세 가지 유형으로 분류된다.

| 유형 | 손상 부위 | 기도 청력역치 | 골도 청력역치 | ABG | 특징 |
|------|---------|-----------|-----------|------|------|
| **전음성 난청(conductive hearing loss)** | 외이 또는 중이 | 상승 | 정상 | 큼 (15 dB 이상) | 소리가 내이까지 전달되는 과정에 문제. 보청기로 증폭하면 효과적. 의학적 치료 가능한 경우 많음 |
| **감각신경성 난청(sensorineural hearing loss)** | 내이(와우) 또는 청신경 | 상승 | 상승 | 없음 또는 매우 작음 | 감각 과정에 문제. 기도와 골도 역치가 함께 상승. 보청기 효과가 제한적일 수 있음 |
| **혼합성 난청(mixed hearing loss)** | 외이/중이 + 내이/청신경 | 상승 | 상승 (기도보다는 낮음) | 있음 | 전음성 + 감각신경성 난청이 동시에 존재. 기도와 골도 역치 모두 상승하되, 기도 역치가 골도 역치보다 높음 |

2025학년도 KICE 기출(전공A, 4번)에서 학생 A(감각신경성 청각장애)와 학생 B(중추청각처리장애)의 청력도를 비교하는 문항이 출제되었다. **전음성 청력손실**의 경우 기도 청력역치는 높게 나오지만 골도 청력역치는 정상이며, **감각신경성 청력손실**의 경우 기도와 골도 청력역치가 모두 같은 수준으로 높게 나온다.

#### 중추청각처리장애(CAPD: Central Auditory Processing Disorder)

중추청각처리장애는 외이부터 청신경까지를 포함한 **청각 기관에는 문제가 없고**, **중추청각신경계에 문제**가 있는 경우이다. 기도와 골도 청력역치가 모두 정상 수준이지만, 소리가 어느 방향에서 나는지 구분하기 어렵거나 소리의 리듬이나 높낮이 인식에 어려움이 있을 수 있다.

### 5. 평균청력 산출: 6분법

청각장애의 정도를 판별하기 위해 **순음청력역치의 평균**을 산출하며, 한국에서는 일반적으로 **6분법**을 사용한다.

**6분법 공식**:

```
평균청력(dB) = (a + 2b + 2c + d) / 6
```

- a = 500 Hz의 청력역치
- b = 1,000 Hz의 청력역치
- c = 2,000 Hz의 청력역치
- d = 4,000 Hz의 청력역치

6분법은 일상 대화에서 가장 중요한 **1,000 Hz와 2,000 Hz에 2배의 가중치**를 두어 산출한다.

| 방법 | 공식 | 비고 |
|------|------|------|
| **3분법** | (a + b + c) / 3 | 500, 1000, 2000 Hz |
| **4분법** | (a + 2b + c) / 4 | 1000 Hz에 2배 가중 |
| **6분법** | (a + 2b + 2c + d) / 6 | 1000, 2000 Hz에 각 2배 가중 |

> **용어 검증 필요**: 장애인복지법 시행규칙의 청각장애 등급 판정 기준(6분법 적용)은 최신 법령을 확인할 것

### 6. 청각장애의 분류 (청력 손실 정도)

순음청력역치 평균(PTA)에 따른 청각장애 분류는 다음과 같다.

| 분류 | 청력 손실 정도(dB HL) | 듣기 특성 |
|------|-------------------|---------|
| **정상(normal)** | 0~15 | 대부분의 소리를 어려움 없이 들음 |
| **경도 난청(mild)** | 16~40 | 작은 소리, 먼 거리의 말소리를 듣기 어려움. 1:1 대화는 가능 |
| **중등도 난청(moderate)** | 41~55 | 보통 대화 음량에서 듣기 어려움. 보청기 필요 |
| **중고도 난청(moderately severe)** | 56~70 | 큰 소리만 들을 수 있음. 보청기 효과 있음 |
| **고도 난청(severe)** | 71~90 | 귀 가까이에서 큰 소리만 겨우 들림. 보청기 효과 제한적 |
| **농(profound)** | 91 이상 | 보청기로도 말소리 청취가 매우 어려움. 인공와우 고려 대상 |

2025학년도 전공B 11번 문항에서 학생 A의 평균순음청력역치가 좌측 85 dB HL, 우측 80 dB HL로 제시되어 **고도 난청** 수준에 해당하는 사례가 출제되었다.

## 기출연계

### 기입형 출제 패턴

- "기도는 외부의 소리가 외이부터 뇌까지 전달되는 소리 전달 경로이고, 골도는 두개골을 진동시켜 ( )부터 소리가 전달되는 경로이다." → **내이(와우)**
- "소리 전달 경로 중 외이와 중이는 ( ) 기능을 담당하고, ( )은/는 감음 기능을 담당한다." → **전음**, **내이(감각신경)**
- "기도 청력역치와 골도 청력역치의 차이를 ( )(이)라 하며, 이를 통해 난청의 유형을 감별할 수 있다." → **기도-골도차(ABG, air-bone gap)**
- "전음성 난청의 경우 기도 청력역치는 ( ), 골도 청력역치는 ( ) 수준이다." → **상승(높게 나옴)**, **정상**
- "6분법에 의한 평균청력 산출 공식은 (a + 2b + 2c + d) / ( )이다." → **6**

### 서술형 출제 패턴

**[패턴 1] 난청 유형 비교 서술** (2025 전공A 4번 유형)
- 전음성·감각신경성·혼합성 난청의 기도·골도 청력역치 특성과 ABG 유무를 비교 서술
- 답안 포인트: ① 각 난청 유형의 손상 부위 명시 ② 기도·골도 역치 관계 ③ ABG 유무 및 크기 ④ 임상적 특성(치료 가능성, 보청기 효과 등)

**[패턴 2] 청력도 해석** (2025 전공A 4번 유형)
- 제시된 청력도를 보고 난청의 유형과 정도를 판별하는 문항
- 답안 포인트: ① 기도·골도 기호 구분(O/X, &lt;/&gt;) ② 주파수별 역치 읽기 ③ ABG 확인 ④ 난청 유형 판별 ⑤ 6분법 평균청력 산출

**[패턴 3] 중추청각처리장애와 감각신경성 난청의 비교**
- 두 장애의 청력검사 결과 차이와 듣기 특성의 차이를 서술
- 답안 포인트: ① CAPD는 기도·골도 역치 정상, 감각신경성은 모두 상승 ② CAPD는 소리 방향 변별, 소음 속 말소리 변별 등에 어려움

### 최근 출제 경향

- 2025학년도 전공A 4번: 감각신경성 청각장애와 중추청각처리장애 학생의 **청력도 비교**, 기도·골도 전달 경로 빈칸 완성
- 전음성/감각신경성 난청의 **기도-골도차(ABG)** 해석 문제가 반복 출제
- 청력도 기호(O, X, &lt;, &gt;)의 의미를 묻는 기입형 출제 빈도 높음
