## 핵심개념

**수어(sign language)**: 손, 팔, 얼굴 표정, 몸의 움직임 등 **시각적 수단**을 사용하여 의미를 전달하는 독립적인 언어 체계이다. 한국수어(Korean Sign Language, KSL)는 2016년 「한국수어언어법」([관련 법률](/subjects/laws/anti-discrimination-act) 참조)에 의해 한국어와 동등한 자격을 가진 **공용어**로 인정되었다. 수어는 음성언어와 독립된 고유한 **문법 구조**를 가지고 있다.

**지문자(fingerspelling)**: 한글의 자모(자음·모음)를 **손 모양**으로 하나씩 표현하여 글자를 전달하는 수단이다. 수어 어휘에 없는 고유명사, 전문용어 등을 표현할 때 주로 사용하며, 한국 지문자 체계는 **자음 14자 + 모음 10자**로 구성된다.

**독화(speechreading/lipreading)**: 화자의 **입술 모양, 혀의 움직임, 얼굴 표정, 몸짓** 등 시각적 단서를 통해 말의 내용을 파악하는 것이다. 입술 읽기(lipreading)보다 넓은 개념으로, 입술뿐 아니라 전체 맥락(표정, 제스처, 상황)을 종합적으로 활용한다.

**청능훈련(auditory training)**: 보청기나 인공와우를 착용한 청각장애인이 **잔존 청력 또는 새롭게 얻은 청각 정보를 최대한 활용**할 수 있도록 체계적으로 듣기 능력을 훈련하는 것이다. 소리의 감지에서 이해까지 단계적으로 진행한다.

> **용어 검증 필요**: 한국수어언어법(2016) 제정 연도와 수어의 법적 지위 관련 최신 법령 확인 필요

## 주요이론

### 1. 의사소통 접근법

청각장애 교육에서의 의사소통 접근법은 크게 **구화법**, **수화법**, **통합적 의사소통법(TC)**으로 나뉜다.

| 접근법 | 주요 수단 | 핵심 원리 | 장점 | 제한점 |
|--------|---------|---------|------|--------|
| **구화법(oral method)** | 잔존 청력 활용, 독화, 발화 | 음성언어를 통한 의사소통 능력 개발에 집중. 보청기/인공와우 착용 전제 | 건청인과의 직접 의사소통 가능. 사회 통합에 유리 | 고도·농 학생에게 적용 어려움. 독화의 한계(동구어 등) |
| **수화법(manual method)** | 수어, 지문자 | 시각적 의사소통 수단을 주요 언어로 사용. 농문화(Deaf culture) 존중 | 시각적 정보 전달로 의미 파악이 직관적. 농사회 내 원활한 의사소통 | 건청인과의 소통 장벽. 수어를 모르는 사람과 의사소통 어려움 |
| **통합적 의사소통법(TC: Total Communication)** | 구어 + 수어 + 지문자 + 독화 + 필기 등 모든 수단 | 학생의 필요와 상황에 따라 가능한 **모든 의사소통 수단**을 사용 | 다양한 상황에 유연하게 대처 가능. 학생 개별 필요에 맞는 접근 | 특정 의사소통 수단의 깊이 있는 발달이 어려울 수 있음 |

> **용어 검증 필요**: 이중언어-이중문화 접근법(Bi-Bi approach)의 개념과 TC와의 차이점은 최신 교재 확인 필요

### 2. 수어의 문법 구조

수어는 음성언어와 독립된 **고유한 문법 체계**를 가진 자연언어이다. 수어의 주요 문법적 특성은 다음과 같다.

#### 수어의 구성 특성

| 문법 요소 | 설명 | 예시 |
|---------|------|------|
| **NMS(Non-Manual Signals, 비수지 신호)** | 손이 아닌 **얼굴 표정, 눈썹, 고개 움직임, 입 모양, 몸의 자세** 등으로 문법적 기능을 수행 | 의문문: 눈썹을 올림, 부정문: 고개를 좌우로 흔듦, 감정 표현: 얼굴 표정 활용 |
| **CL(Classifier, 분류사)** | 사물의 형태, 크기, 위치, 움직임을 나타내기 위해 특정 **손 모양**을 분류사로 사용하는 문법 체계 | 차량 이동: 손가락을 세워 움직임, 사람 이동: 검지를 세워 걸어가는 동작 |
| **공간활용(spatial grammar)** | 수어 공간에서 **지시점(referent locus)**을 설정하고, 동사의 방향·위치를 통해 주어·목적어 관계를 표현 | "나→너에게 주다"와 "너→나에게 주다"는 동사 방향으로 구분 |

2025학년도 전공B 11번 문항에서 수어 문장의 **목적어를 나타내는 데 사용된 수어의 구성 특성**을 묻는 문제가 출제되었다.

#### 한국수어의 어순

한국수어는 한국어(SOV)와 다른 독자적 어순을 가진다. 기본 어순은 **주제(Topic)-설명(Comment)** 구조를 많이 사용하며, 문장 유형(평서문·의문문·명령문)에 따라 NMS가 달라진다.

### 3. 지문자 체계

한국 지문자는 한글의 자모를 손 모양으로 표현하는 체계로, **자음 14자**와 **모음 10자**로 구성된다.

| 구분 | 개수 | 글자 |
|------|------|------|
| **자음** | 14자 | ㄱ, ㄴ, ㄷ, ㄹ, ㅁ, ㅂ, ㅅ, ㅇ, ㅈ, ㅊ, ㅋ, ㅌ, ㅍ, ㅎ |
| **모음** | 10자 | ㅏ, ㅑ, ㅓ, ㅕ, ㅗ, ㅛ, ㅜ, ㅠ, ㅡ, ㅣ |

- 지문자는 수어 대화 중 **고유명사**(사람 이름, 지명), **전문용어**, **수어 어휘에 없는 단어** 등을 표현할 때 보조적으로 사용됨
- 일부 지문자는 수형이 같은 모음이 있음 (예: 'ㄷ'와 'ㅏ'의 수형이 유사)

2026학년도 전공A 9번 문항에서 지문자 활동이 제시되었으며, 특정 지문자의 한글 표기와 **같은 수형을 가진 모음 지문자**를 묻는 문제가 출제되었다.

> **용어 검증 필요**: 지문자 수형의 유사성(같은 수형의 자음-모음 쌍)은 한국농아인협회 지문자 표준 자료 확인 필요

### 4. 독화(Speechreading)의 한계와 보완

#### 독화의 한계

| 한계 요인 | 내용 |
|---------|------|
| **동구어(homophene)** | 입술 모양이 같거나 유사하여 시각적으로 구별이 어려운 말소리가 존재. 예: /ㅂ/, /ㅃ/, /ㅍ/은 모두 양순음으로 입술 모양이 거의 같음 |
| **시각적 정보의 제한** | 전체 말소리의 약 **30~40%**만이 입술에서 시각적으로 관찰 가능. 나머지는 구강 내에서 조음되어 보이지 않음 |
| **환경 요인** | 화자와의 거리, 조명, 화자의 입 가림(마스크 등), 여러 사람이 동시에 말하는 상황 등에서 독화 어려움 증가 |
| **화자 요인** | 화자의 발음 습관, 말 속도, 수염·입 가림 등에 따라 독화 정확도가 달라짐 |

#### 독화의 보완 전략

| 보완 전략 | 내용 |
|---------|------|
| **큐드스피치(Cued Speech)** | 독화만으로 구별이 어려운 말소리를 **손 모양(handshape)**과 **손 위치(placement)**로 보완. 독화의 동구어 문제를 해결하기 위해 개발됨. [AAC](/subjects/communication-disorder/aac)와 함께 보완적 의사소통 체계의 하나로 이해할 수 있다 |
| **맥락 활용** | 대화 주제, 상황, 화자의 표정·제스처 등 맥락 단서를 종합적으로 활용 |
| **보청기/인공와우 병용** | 청각적 정보와 시각적 정보를 동시에 활용하여 이해도 향상 |
| **필기·문자 지원** | 핵심 내용을 글로 제시하여 시각적 보완 |

### 5. 큐드스피치(Cued Speech)

큐드스피치는 **Cornett(1967)**이 개발한 시각적 의사소통 보조 체계로, 독화에 **손 단서(cue)**를 추가하여 동구어 문제를 해결한다.

| 구성 요소 | 기능 | 체계 |
|---------|------|------|
| **손 모양(handshape)** | 자음을 구별하기 위한 단서 | 8가지 손 모양으로 자음 그룹 표시 |
| **손 위치(placement)** | 모음을 구별하기 위한 단서 | 4가지 위치(턱 옆, 목, 턱 아래, 입 옆)로 모음 그룹 표시 |

- 큐드스피치는 **언어 자체가 아니라** 구어를 시각적으로 보완하는 **보조 체계**임
- 독화와 함께 사용하여 말소리 지각의 정확도를 높임
- 구화법 교육에서 독화의 한계를 보완하는 수단으로 활용

> **용어 검증 필요**: 큐드스피치의 손 모양·위치 체계 세부 내용과 한국어 적용 방식은 전문 교재 확인 필요

### 6. 청능훈련(Auditory Training) 4단계

청능훈련은 [보청기나 인공와우](/subjects/hearing-impairment/amplification)를 통해 얻은 청각 정보를 최대한 활용할 수 있도록 **단계적으로 듣기 능력을 발달**시키는 체계적 훈련이다.

| 단계 | 명칭 | 내용 | 예시 |
|------|------|------|------|
| **1단계** | **감지(detection)** | 소리의 **유무**를 알아차리는 단계. 소리가 있는지 없는지를 구분 | "소리가 들리면 손을 들어 보세요" |
| **2단계** | **변별(discrimination)** | 두 개 이상의 소리가 **같은지 다른지**를 구분하는 단계 | "이 두 소리가 같나요, 다른가요?" (북소리 vs 피아노 소리) |
| **3단계** | **확인(identification)** | 여러 소리 중 특정 소리가 **무엇인지** 알아내는 단계. 소리를 듣고 대응하는 사물이나 단어를 지적 | "지금 들은 동물 소리는 어떤 동물인가요?" (선택지 제공) |
| **4단계** | **이해(comprehension)** | 들은 말의 **의미를 파악**하고 적절히 반응하는 단계. 실제 대화 상황에서 소리 정보를 활용하여 의사소통 | "오늘 날씨가 어떤가요?"에 적절히 대답하기 |

- 청능훈련은 반드시 **감지 → 변별 → 확인 → 이해** 순서로 단계적 진행
- 각 단계가 충분히 습득된 후 다음 단계로 이행
- 인공와우 이식 후 재활(매핑 후 훈련)에서 특히 중요

### 7. 어음명료도와 반향

#### 어음명료도(Speech Intelligibility)

**어음명료도**란 화자가 산출한 말소리를 청자가 **정확하게 알아듣는 정도**를 백분율(%)로 나타낸 것이다.

- 어음명료도가 낮을수록 화자의 말을 이해하기 어려움
- 청각장애인의 **말 산출(발화)** 측면에서의 명료도: 청각장애인이 말한 내용을 건청인이 얼마나 알아듣는지
- **말 수용(청취)** 측면에서의 명료도: 건청인의 말을 청각장애인이 얼마나 알아듣는지 (= 어음변별력/WRS)

어음명료도에 영향을 미치는 요인:
1. 청력 손실의 정도와 유형
2. 보청기/인공와우의 적합 여부
3. 환경 소음 수준(SNR)
4. 반향(reverberation)
5. 화자와의 거리

#### 반향(Reverberation)

**반향**이란 소리가 교실의 벽, 바닥, 천장 등 **단단한 표면에 반사**되어 울림이 지속되는 현상이다. 반향은 **반향시간(RT: Reverberation Time)**으로 측정하며, 소리가 60 dB 감쇠되는 데 걸리는 시간을 의미한다.

| 항목 | 내용 |
|------|------|
| 반향의 영향 | 원래 소리와 반사된 소리가 겹쳐서 말소리의 명료도가 떨어짐. 특히 자음의 변별이 어려워짐 |
| 권장 반향시간 | 교실에서 청각장애 학생을 위해 **0.4~0.6초 이하** 권장 |
| 반향 감소 방법 | 카펫, 커튼, 천장 흡음재, 벽면 흡음판 설치, 교실 문·창문 닫기 |

2025학년도 전공B 11번 문항에서 학생 A가 보청기 착용 후에도 **교실 안의 반향**이 심하면 듣는 데 어려움이 있다는 내용이 출제되었으며, 반향을 줄이기 위한 **소음 측면의 방안**을 서술하도록 하였다.

> **용어 검증 필요**: 교실 반향시간 권장 기준(ANSI/ASA S12.60)은 최신 기준 확인 필요

| NMS(비수지 신호) | 얼굴 표정·고개 움직임 등으로 문법적 기능 수행 | ★★★ |
| CL(분류사) | 사물의 형태·크기·움직임을 나타내는 손 모양 체계 | ★★ |
| 공간활용 | 수어 공간에서 지시점·동사 방향으로 문법 표현 | ★★ |
| 큐드스피치(Cued Speech) | 손 모양(자음)·손 위치(모음)로 독화 보완하는 체계 | ★★ |
| 동구어(homophene) | 입술 모양이 같아 독화로 구별 불가능한 말소리 | ★★ |
| 어음명료도 | 말소리를 정확하게 알아듣는 정도(%) | ★★★ |
| 반향(reverberation) | 소리가 반사되어 울림이 지속되는 현상 | ★★★ |
| 반향시간(RT) | 소리가 60 dB 감쇠되는 데 걸리는 시간 | ★★ |
| 감지(detection) | 청능훈련 1단계. 소리의 유무를 알아차리는 것 | ★★★ |
| 변별(discrimination) | 청능훈련 2단계. 소리의 같고 다름을 구분하는 것 | ★★★ |
| 확인(identification) | 청능훈련 3단계. 소리가 무엇인지 알아내는 것 | ★★★ |
| 이해(comprehension) | 청능훈련 4단계. 말의 의미를 파악하고 반응하는 것 | ★★★ |

## 기출연계

### 기입형 출제 패턴

- "청능훈련의 4단계를 순서대로 나열하면 ( ) → ( ) → ( ) → ( )이다." → **감지, 변별, 확인, 이해**
- "독화만으로 구별이 어려운 말소리를 보완하기 위해 손 모양과 손 위치를 추가하는 시각적 체계를 ( )(이)라 한다." → **큐드스피치(Cued Speech)**
- "입술 모양이 같거나 유사하여 독화로 구별이 어려운 말소리 쌍을 ( )(이)라 한다." → **동구어(homophene)**
- "한국 지문자 체계는 자음 ( )자와 모음 ( )자로 구성된다." → **14**, **10**
- "소리가 교실의 벽·바닥·천장에 반사되어 울림이 지속되는 현상을 ( )(이)라 한다." → **반향(reverberation)**
- "수어에서 얼굴 표정, 고개 움직임 등 손이 아닌 신체 부위로 문법적 기능을 수행하는 요소를 ( )(이)라 한다." → **비수지 신호(NMS)**

### 서술형 출제 패턴

**[패턴 1] 의사소통 접근법 비교 서술**
- 구화법·수화법·TC의 특성을 비교하고, 특정 학생에게 적합한 접근법을 근거와 함께 서술
- 답안 포인트: ① 각 접근법의 주요 수단 ② 장점과 제한점 ③ 학생의 청력·언어 수준에 따른 적합성 근거

**[패턴 2] 수어 문법 구조 서술** (2025 전공B 11번 유형)
- 수어 문장에서 NMS, CL, 공간활용의 역할을 구체적 예시와 함께 서술
- 답안 포인트: ① NMS의 문법적 기능(의문문·부정문 표지) ② CL의 형태소적 기능 ③ 공간활용의 통사적 기능

**[패턴 3] 교실 환경에서의 의사소통 지원** (2025 전공B 11번, 2026 전공A 9번 유형)
- 교실에서 청각장애 학생의 말소리 이해를 높이기 위한 환경 개선 방안과 의사소통 전략을 서술
- 답안 포인트: ① SNR 개선(FM시스템 등) ② 반향 감소(흡음 자재) ③ 독화 지원(시각적 정보 제공, 자리 배치) ④ 보조공학기기 활용. [협력교수(co-teaching)](/subjects/inclusive-education/co-teaching) 환경에서도 적용 가능

**[패턴 4] 지문자와 수어의 관계 서술** (2026 전공A 9번 유형)
- 지문자의 기능과 수어 대화에서의 역할을 서술
- 답안 포인트: ① 지문자 체계(자음14+모음10) ② 사용 상황(고유명사, 전문용어) ③ 지문자와 수어의 관계(보조적 수단)

### 최근 출제 경향

- 2026학년도 전공A 9번: **보청기·인공와우 구조**, **지문자** 활동(한글 표기, 같은 수형 모음 찾기)
- 2025학년도 전공B 11번: 청각장애 학생의 보청기 착용, **양이효과**, **교실 반향** 문제, **수어 문법**(목적어 표현에 사용된 구성 특성)
- **반향(reverberation)**과 **SNR** 관련 환경 요인이 서술형 문항에서 반복 출제
- 수어 문법(NMS, CL, 공간활용) 관련 문항이 증가 추세
- 지문자와 큐드스피치 등 **시각적 보조 수단**에 대한 출제도 등장
