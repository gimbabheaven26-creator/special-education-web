## 핵심개념

**보조공학(assistive technology, AT)**: 장애인의 기능적 능력을 유지·향상시키기 위해 사용되는 모든 기기(device), 장비, 소프트웨어 또는 서비스를 말한다. 보조공학은 로우테크(low-tech)부터 하이테크(high-tech)까지 연속적인 스펙트럼으로 이해해야 하며, **하이테크 기기가 로우테크 기기보다 항상 좋은 것은 아니다**.

**접근성(accessibility)**: 장애의 유무와 관계없이 모든 사람이 시설, 서비스, 정보, 기기 등에 동등하게 접근하고 이용할 수 있는 정도를 의미한다. 물리적 접근성(건물, 시설)과 정보 접근성(웹, 소프트웨어)을 포함한다.

**보편적 설계(universal design, UD)**: Ronald Mace가 제창한 개념으로, 장애 유무·연령·성별에 관계없이 **모든 사람이 이용할 수 있도록** 제품, 환경, 서비스를 설계하는 원리이다. 사후 수정(adaptation)이 아닌 처음부터 모든 사용자를 고려한 설계를 지향한다.

> ⚠️ **용어 검증 필요**: 보편적 설계의 7원칙(공평한 사용, 유연한 사용, 간단한 사용 등)에 대한 정확한 내용 확인 필요

## 주요이론

### 1. HAAT 모형(Human Activity Assistive Technology Model)

**HAAT 모형**은 Cook과 Hussey(1995, 2002)가 제안한 보조공학 적용의 이론적 틀로, 보조공학 선정과 적용 시 고려해야 하는 네 가지 요소를 체계적으로 제시한다.

| 구성 요소 | 설명 |
|---------|------|
| **사람(Human)** | 사용자의 신체적·인지적·감각적·정서적 능력과 요구. 장애 특성, 동기, 선호도 등 |
| **활동(Activity)** | 사용자가 수행하고자 하는 과제나 활동. 자기관리, 학습, 의사소통, 이동, 놀이 등 |
| **보조공학(Assistive Technology)** | 활동 수행을 지원하는 기기와 서비스. 인간-기술 인터페이스(human-technology interface), 처리기(processor), 활동 출력(activity output), 환경 인터페이스(environmental interface) |
| **맥락(Context)** | 보조공학이 사용되는 물리적·사회적·문화적·제도적 환경. 학교, 가정, 직장, 지역사회 등 |

HAAT 모형의 핵심은 보조공학 선정이 **기기 자체**가 아니라 **사람-활동-맥락의 상호작용** 속에서 이루어져야 한다는 것이다.

```
맥락(Context)
┌──────────────────────────┐
│   사람(Human)             │
│     ↕                    │
│   활동(Activity)          │
│     ↕                    │
│   보조공학(AT)            │
└──────────────────────────┘
```

> ⚠️ **용어 검증 필요**: HAAT 모형의 보조공학 하위 구성요소(인간-기술 인터페이스, 처리기, 활동 출력, 환경 인터페이스)의 정확한 명칭과 내용 확인 필요

### 2. 키보드 접근성(Keyboard Accessibility)

지체장애 학생이 컴퓨터 키보드를 사용할 때 겪는 어려움을 해결하기 위한 접근성 기능으로, 대부분의 운영체제(Windows, macOS)에 기본 내장되어 있다.

| 기능 | 영문명 | 설명 | 대상 학생 |
|------|-------|------|---------|
| **고정키(Sticky Keys)** | Sticky Keys | **Ctrl, Alt, Shift 등의 수정키(modifier key)를 한 번에 하나씩 입력**할 수 있게 함. 두 키를 동시에 누르지 않아도 됨 (예: Ctrl+C를 순서대로 누름) | 한 손만 사용 가능하거나, 두 키를 동시에 누르기 어려운 학생 |
| **반복키(Repeat Keys) / 느린키(Slow Keys)** | Filter Keys | **키를 누르고 있을 때 반복 입력되는 것을 방지**하거나, 일정 시간 이상 누르고 있어야 입력이 인정됨 | 불수의운동이 있어 키를 실수로 반복 입력하거나, 키를 정확하게 누르기 어려운 학생 |
| **토글키(Toggle Keys)** | Toggle Keys | **Caps Lock, Num Lock, Scroll Lock** 등의 상태 변경 시 소리로 알려줌 | 시각적으로 표시등을 확인하기 어려운 학생 |
| **마우스키(Mouse Keys)** | Mouse Keys | **키보드의 숫자 키패드로 마우스 포인터를 조작**할 수 있게 함 | 마우스 조작이 어렵지만 키보드 사용은 가능한 학생 |

### 3. 스캐닝(Scanning) 기법

스캐닝은 **직접 선택(direct selection)**이 어려운 중증 지체장애 학생을 위한 **간접 선택(indirect selection)** 방법이다.

#### (1) 선택 방식의 분류

| 방식 | 설명 | 특성 |
|------|------|------|
| **직접 선택(direct selection)** | 원하는 항목을 **직접** 가리키거나 터치하여 선택. 손가락, 시선, 머리 포인터 등 사용 | 빠르고 직관적. 일정 수준의 운동 조절 능력 필요 |
| **간접 선택(indirect selection)** | **스캐닝** 등을 통해 항목이 순서대로 강조될 때 원하는 항목에서 **스위치를 눌러** 선택 | 최소한의 운동 능력으로 선택 가능. 직접 선택보다 느림 |

#### (2) 스캐닝 유형

| 유형 | 방법 | 특성 |
|------|------|------|
| **원형 스캐닝(circular scanning)** | 항목들이 원형으로 배열되어 순서대로 강조됨 | 항목 수가 적을 때 적합. 단순함 |
| **선형 스캐닝(linear scanning)** | 항목들이 일렬로 배열되어 처음부터 끝까지 순서대로 강조됨 | 항목 수가 적을 때 효율적. 항목 수가 많으면 시간이 오래 걸림 |
| **행-열 스캐닝(row-column scanning)** | **1단계**: 행(row)이 순서대로 강조됨 → 원하는 행에서 스위치 선택. **2단계**: 선택된 행 내에서 열(column)이 순서대로 강조됨 → 원하는 열에서 스위치 선택 | 항목 수가 많을 때 효율적. **2번의 스위치 입력**으로 선택. 가장 널리 사용됨 |
| **그룹-항목 스캐닝(group-item scanning)** | 항목들을 그룹으로 나누어 먼저 그룹을 선택한 후, 그룹 내 개별 항목을 선택 | 행-열 스캐닝의 변형. 항목 수가 매우 많을 때 유용 |

#### (3) 스위치(Switch) 유형

스위치는 스캐닝이나 기기 조작 시 사용자의 운동 능력에 맞게 선택하는 입력 장치이다.

| 스위치 유형 | 작동 방식 | 사용 부위 예시 |
|---------|---------|------------|
| **압력 스위치(pressure switch)** | 누르면 작동 | 손, 발, 머리, 턱 등 |
| **근접 스위치(proximity switch)** | 가까이 가면 작동(접촉 불필요) | 움직임이 매우 제한적인 학생 |
| **호흡 스위치(sip-and-puff switch)** | 빨기(sip)와 불기(puff)로 작동. 효과 방향으로 미는 정도에 따라 조절 가능 | 사지 사용이 어려운 학생. 호흡 조절 가능한 학생 |
| **근전도 스위치(EMG switch)** | 근육의 미세한 수축을 감지하여 작동 | 수의적 근육 수축이 최소한 가능한 학생 |

> ⚠️ **용어 검증 필요**: 스위치의 분류와 명칭은 교재에 따라 다소 차이가 있으므로 확인 필요

### 4. 전동 휠체어 조작

전동 휠체어는 GMFCS Level IV~V에 해당하는 중증 지체장애 학생의 자기 이동성(self-mobility)을 확보하는 데 핵심적인 보조공학 기기이다.

#### 조이스틱(Joystick) 유형

| 유형 | 작동 방식 | 특성 |
|------|---------|------|
| **비례적 조이스틱(proportional joystick)** | 기울이는 **방향**과 **정도**에 따라 휠체어의 방향과 속도가 비례적으로 조절됨. 가고 싶은 방향으로 비례적 조이스틱을 움직이면 그 방향으로 휠체어가 움직임 | 섬세한 속도·방향 조절 가능. 일정 수준의 손 조절 능력 필요 |
| **디지털(비비례적) 조이스틱** | 정해진 방향(전후좌우)으로만 작동. 속도 단계가 정해져 있음 | 미세 조절이 어려운 학생에게 적합 |

비례적 조이스틱으로 **속도를 조절**할 수도 있다. 원하는 방향으로 미는 정도에 따라 속도가 조절된다.

#### 대안적 조작 방법

손으로 조이스틱 조작이 어려운 학생을 위한 대안적 방법:

- **턱 조이스틱(chin joystick)**: 턱으로 조이스틱 조작
- **머리 배열(head array)**: 머리의 움직임으로 방향 조절
- **호흡 조작(sip-and-puff)**: 빨기와 불기의 강도로 방향·속도 조절
- **스위치 조작**: 여러 개의 스위치를 배치하여 각 방향을 제어

### 5. 근이영양증(Muscular Dystrophy)

근이영양증은 근육이 점진적으로 약화되고 퇴행하는 **진행성(progressive)** 유전 질환이다. 뇌성마비(비진행성)와 대비되는 중요한 질환이다.

#### 뒤시엔느형 근이영양증(Duchenne Muscular Dystrophy, DMD)

| 항목 | 내용 |
|------|------|
| **유전 양식** | **X-연관 열성(X-linked recessive)**. 거의 남아에게만 발생 |
| **원인** | 디스트로핀(dystrophin) 단백질의 결핍 |
| **발현 시기** | 2~5세경 근력 약화 시작. 보통 3세 이전에 초기 증상 |
| **초기 증상** | 잘 넘어짐, 계단 오르기 어려움, 달리기 어려움, **가워스 징후(Gowers' sign)**: 바닥에서 일어날 때 양손으로 무릎·허벅지를 짚고 몸을 일으킴 |
| **진행 과정** | 하지 → 상지 순서로 근력 약화 진행. 보통 10~12세경 보행 불능 → 휠체어 사용. 20대 초반 호흡 부전 위험 |
| **특징적 소견** | 장딴지(비복근) 비대(가성비대: pseudohypertrophy), 척추(척추)만곡이 나타남, 점측보행을 하며 균형 감각이 불안하여 자주 넘어짐 |

**교육적 고려사항**:

- **진행성** 질환이므로 현재 기능 수준뿐 아니라 **향후 기능 저하를 예측**한 보조공학 지원 계획이 필요
- 근력 약화가 진행되면 수동 휠체어 조작도 어려워지므로, **착석 시스템 적용 전동 휠체어**를 적기에 제공
- 호흡 기능 관리, 심장 기능 모니터링 등 **건강 관리**가 매우 중요
- 인지 기능은 대체로 보존되므로, 학업 참여를 위한 보조공학 적극 활용

> ⚠️ **용어 검증 필요**: 뒤시엔느형 외 다른 유형(베커형, 안면견갑상완형 등)의 특성 확인 필요

### 6. 보치아(Boccia)

**보치아(boccia)**는 뇌성마비를 포함한 중증 지체장애인을 위한 대표적인 장애인 스포츠 종목이다.

| 항목 | 내용 |
|------|------|
| **정의** | 주어진 시간 내에 **6개의 빨간색 공과 6개의 파란색 공**을 던져 **흰색 표적구(잭볼, jack ball)**에 가장 가까이 던진 공에 1점을 부가하고, 이 점수를 합산하여 한 팀이 승리하는 경기 |
| **참가 대상** | 중증 지체장애인(주로 뇌성마비). 공을 던지기 어려운 선수는 **발로 차서 보내거나**, **마우스 스틱**과 **흠통(ramp/chute)**을 사용 |
| **올림픽 채택** | **1984년 하계 장애인 올림픽** 대회의 정식 종목으로 채택 |
| **분류** | 생활환경형 스포츠 종목에 해당 |
| **경기 방식** | 개인전, 페어전(2인), 팀전(3인). 경기장 크기 12.5m x 6m |

**교육적 활용**:

- 중증 지체장애 학생도 참여할 수 있는 대표적인 스포츠 활동
- 다양한 보조 기구(흠통, 마우스 스틱 등)를 활용하여 참여 보장
- 전략적 사고력, 집중력, 사회적 상호작용 기술 발달에 기여

### 7. 보완대체 의사소통(AAC)에서의 보조공학

지체장애 학생이 구어 의사소통에 어려움이 있을 때 **보완대체 의사소통(AAC: Augmentative and Alternative Communication)**을 활용한다. AAC에서 보조공학의 역할은 의사소통 기기의 **선택과 접근** 방법을 결정하는 것이다.

| 구성 요소 | 설명 |
|---------|------|
| **상징(symbols)** | 의미를 나타내는 그림, 사진, 글자, 음성 등 |
| **보조도구(aids)** | 의사소통판, 음성출력 의사소통기기(VOCA/SGD) 등 |
| **기법(techniques)** | 직접 선택, 스캐닝 등 상징에 접근하는 방법 |
| **전략(strategies)** | 의사소통 효율을 높이는 방법(예측, 약어 등) |

보조공학의 연속적 스펙트럼에 따른 AAC 도구:

- **로우테크**: 의사소통판, 그림 카드, 눈 응시판 → 간단한 구성으로 된 의사소통판을 사용하는 것이 더 효율적인 학생도 있음
- **하이테크**: 음성출력(text-to-speech) 기반 의사소통 기기, 태블릿 기반 AAC 앱

## 기출연계

### 기입형 출제 패턴

- "두 키를 동시에 누르기 어려운 학생이 Ctrl+C를 순서대로 누를 수 있게 해주는 키보드 접근성 기능을 ( )(이)라 한다." → **고정키(Sticky Keys)**
- "HAAT 모형의 4가지 구성 요소는 ( ), ( ), ( ), ( )이다." → **사람(Human)**, **활동(Activity)**, **보조공학(AT)**, **맥락(Context)**
- "뒤시엔느형 근이영양증 아동이 바닥에서 일어날 때 양손으로 무릎과 허벅지를 짚고 몸을 일으키는 것을 ( ) 징후라 한다." → **가워스(Gowers')**
- "보치아에서 공을 던지기 어려운 선수가 공을 굴려 보내기 위해 사용하는 기구를 ( )(이)라 한다." → **흠통(ramp/chute)**
- "전동 휠체어에서 가고 싶은 방향으로 미는 정도에 따라 속도가 조절되는 조이스틱을 ( ) 조이스틱이라 한다." → **비례적(proportional)**

### 서술형 출제 패턴

**[패턴 1] HAAT 모형에 기반한 보조공학 선정 서술**
- 특정 학생 사례를 주고 HAAT 모형의 4요소에 따라 보조공학을 선정하는 과정 서술
- 답안 포인트: ① 사람(학생의 장애 특성, 능력) ② 활동(수행하려는 과제) ③ 보조공학(적절한 기기 선택) ④ 맥락(사용 환경)

**[패턴 2] 키보드 접근성 기능과 적용 대상 서술** *(2025 전공A 2번 참고)*
- 보조공학의 측면에서 키보드 접근성 기능의 종류를 쓰고, 특정 학생에게 적합한 기능 서술
- 답안 포인트: ① 고정키/반복키/토글키/마우스키의 기능 ② 학생의 장애 특성과 매칭 ③ 하이테크 vs 로우테크의 연속성 고려

**[패턴 3] 전동 휠체어 조작 방식 서술** *(2024 전공B 4번 참고)*
- 학생 A와 B의 장애 특성에 따라 전동 휠체어의 조이스틱 유형과 대안적 조작 방식 서술
- 답안 포인트: ① 비례적 조이스틱의 작동 원리 ② 스위치 유형(호흡 방향으로 미는 정도로 작동 등) ③ 학생의 잔존 능력에 맞는 선택

**[패턴 4] 뒤시엔느형 근이영양증 학생의 지원** *(2024 전공B 4번 참고)*
- 뒤시엔느형 근이영양증의 진행성 특성을 고려한 교육적 지원 방안 서술
- 답안 포인트: ① 진행성 질환의 특성 ② 착석 시스템 적용 전동 휠체어 ③ 향후 기능 저하를 예측한 지원 계획

### 최근 출제 경향

- 2025학년도 전공A 2번: 보조공학 기기를 구분하는 **측면**(하이테크 vs 로우테크의 연속성), **AAC의 구성 요소** 중 무엇에 해당하는지 출제
- 2025학년도 전공A 12번: **보치아** 종목의 설명, **외전기**(가위자세 교정), STNR 잔존 학생의 **자료 제시 위치** 출제
- 2024학년도 전공B 4번: **뒤시엔느형 근이영양증** 학생(척추만곡, 점측보행, 착석 시스템 적용 전동 휠체어)과 **경직형 뇌성마비** 학생의 전동 휠체어 조작 비교. **비례적 조이스틱**, **스위치 유형** 출제
- 2024학년도 전공B 11번: 뇌성마비 학생의 **식사 보조**(실리콘 소재 숟가락, 순가락 홀더), **모로반사** 잔존 시 안정된 환경 제공
- 2026학년도 전공B 6번: **프론스탠더와 수파인스탠더** 선택 기준, **웨지** 사용, 서기 자세보조기기의 **정서적·예방적 효과** 출제
- 2026학년도 전공B 11번: 뇌병변장애 학생의 보조공학 지원 계획 - **TTS(text-to-speech)** 기반 음성출력 보조기기, **토글키** 설정, **화면확대 프로그램**, **플랫베드 확대경** 등 출제
