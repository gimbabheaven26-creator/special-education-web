## 핵심개념

**지역사회 중심 교수(CBI, Community-Based Instruction)**: 장애 학생에게 필요한 기능적 기술을 **실제 지역사회 환경**에서 직접 교수하는 접근이다. 교실 내 모의 환경이 아닌 실제 상황에서 교수함으로써 기술의 일반화와 유지를 촉진한다.

**지역사회참조교육과정(community-referenced curriculum)**: 학생이 현재와 미래에 기능해야 할 지역사회 환경을 분석하여, 그 환경에서 요구되는 기술을 교육과정의 내용으로 선정하는 접근이다. 생태학적 목록(ecological inventory)을 통해 교수 내용을 결정한다.

**기능적 교육과정(functional curriculum)**: 학생의 현재 및 미래 생활에 실제적으로 필요한 기술을 중심으로 구성된 교육과정이다. 학업적 기술보다는 독립적인 일상생활, 직업, 지역사회 참여에 필요한 실용적 기술을 강조한다.

**지원고용(supported employment)**: 경쟁 고용이 어려운 중도 장애인이 **통합된 환경**에서 급여를 받으며 일할 수 있도록 지속적인 지원을 제공하는 고용 형태이다.

## 주요이론

### 1. CBI의 정의와 원리

CBI는 Brown 등(1983)의 연구에 기반하며, 중도 장애 학생 교육에서 특히 중요하게 다루어진다.

#### CBI의 핵심 원리

| 원리 | 설명 |
|------|------|
| **연령 적합성(age-appropriateness)** | 학생의 생활연령에 적합한 활동과 환경을 선택한다 |
| **기능성(functionality)** | 학생의 실제 생활에 즉시 필요하고 유용한 기술을 교수한다 |
| **자연적 환경(natural environment)** | 실제 사용될 환경에서 직접 교수한다 |
| **부분 참여(partial participation)** | 완전한 독립 수행이 어려워도 활동의 일부에 참여할 수 있도록 한다 |
| **자연적 비율(natural proportion)** | 지역사회 내에서 장애인과 비장애인의 자연스러운 비율을 유지한다 |
| **영차이 접근(zero-reject)** | 장애의 정도와 관계없이 모든 학생에게 CBI 기회를 제공한다 |

#### CBI 실시의 근거

- 교실에서 학습한 기술이 실제 환경으로 **자동 일반화되지 않는다**는 연구 결과
- 특히 중도 장애 학생은 일반화에 어려움이 크므로, 처음부터 실제 환경에서 교수하는 것이 효과적
- 자연적 단서(natural cue)와 자연적 결과(natural consequence)를 활용할 수 있다

### 2. CBI의 4가지 교수 영역

Brown 등(1979)은 CBI의 교수 내용을 4가지 주요 영역으로 체계화하였다.

| 영역 | 하위 기술 | 교수 장소 예시 |
|------|-----------|---------------|
| **가정생활(domestic)** | 요리, 청소, 세탁, 식사 준비, 개인위생 | 학교 실습실, 가정, 기숙사 |
| **지역사회(community)** | 대중교통 이용, 식당 이용, 은행 업무, 쇼핑 | 마트, 은행, 식당, 우체국 |
| **여가(recreation/leisure)** | 운동, 취미활동, 문화시설 이용, 사회적 활동 | 체육관, 도서관, 공원, 영화관 |
| **직업(vocational)** | 직무 기술, 작업 행동, 직장 사회성, 출퇴근 | 사업체, 작업장, 사무실 |

#### 생태학적 목록(Ecological Inventory) 과정

교수 내용 선정을 위한 생태학적 목록은 다음 절차로 진행된다:

1. **영역(domain) 선정**: 가정생활, 지역사회, 여가, 직업 중 우선 영역 결정
2. **환경(environment) 파악**: 학생이 현재/미래에 기능해야 할 구체적 환경 조사
3. **하위환경(subenvironment) 분석**: 환경 내 세부 공간 분석
4. **활동(activity) 목록**: 하위환경에서 요구되는 활동 나열
5. **기술(skill) 분석**: 각 활동에 필요한 구체적 기술 파악
6. **불일치 분석(discrepancy analysis)**: 또래 수행과 학생 수행 간 차이 분석

### 3. 현장학습 vs CBI 차이

현장학습(field trip)과 CBI는 모두 교실 밖에서 이루어지지만, 목적과 방법에서 근본적인 차이가 있다.

| 구분 | 현장학습(Field Trip) | CBI |
|------|---------------------|-----|
| **목적** | 학습 내용의 보충, 경험 확대 | 기능적 기술의 직접 교수 |
| **빈도** | 연 1~2회 (일회성 행사) | 주 1~3회 (정기적·반복적) |
| **교수 활동** | 관찰, 체험 중심 | 체계적 교수·연습·피드백 |
| **평가** | 감상문, 소감 발표 | 과제분석 기반 수행 평가 |
| **참여 형태** | 학급 전체 단체 활동 | 소그룹(2~4명) 또는 개별 |
| **일반화** | 일반화 계획 없음 | 일반화를 주요 목표로 설정 |
| **교육과정 연계** | 교과 내용 보충 | 기능적 교육과정의 핵심 요소 |

### 4. 일반화 촉진 전략

CBI의 궁극적 목표는 학습한 기술이 다양한 환경, 사람, 상황에서 **일반화**되고 **유지**되는 것이다.

#### Stokes와 Baer(1977)의 일반화 촉진 전략

| 전략 | 설명 | 적용 예시 |
|------|------|-----------|
| **충분한 예시 활용(train sufficient exemplars)** | 다양한 자극과 상황에서 교수 | 여러 마트에서 쇼핑 기술 교수 |
| **느슨한 교수(train loosely)** | 교수 조건을 다양화 | 다른 시간대, 다른 교사와 연습 |
| **공통 자극 프로그래밍(program common stimuli)** | 교수 환경에 자연적 환경의 자극 포함 | 실제 화폐, 실제 메뉴판 사용 |
| **자연적 유지 결과 활용(use natural maintaining contingencies)** | 자연적 강화를 활용 | 쇼핑 후 실제 물건 획득 |
| **자기관리 훈련(train to generalize)** | 학생 스스로 일반화 점검 | 자기점검표 활용 |
| **매개 일반화(mediate generalization)** | 일반화를 촉진하는 매개 전략 교수 | 그림 일정표, 자기교수 전략 |

### 5. 과제분석과 교수절차

#### 과제분석(Task Analysis)

복합적인 기술을 학습 가능한 작은 단계(steps)로 세분화하는 절차이다. CBI에서 핵심적인 교수 도구이다.

**과제분석 작성 방법:**

1. **목표 기술을 직접 수행**하면서 모든 단계를 기록한다
2. **숙련된 수행자를 관찰**하여 단계를 확인한다
3. **전문가 자문**을 통해 단계의 적절성을 검증한다
4. 학생의 능력 수준에 따라 **단계의 크기**를 조정한다

**과제분석 예시 - 편의점에서 음료 구매하기:**

| 단계 | 행동 |
|------|------|
| 1 | 편의점에 들어간다 |
| 2 | 음료 진열대를 찾아간다 |
| 3 | 원하는 음료를 선택한다 |
| 4 | 음료를 들고 계산대로 간다 |
| 5 | 음료를 계산대에 올려놓는다 |
| 6 | 점원의 금액 안내를 듣는다 |
| 7 | 지갑에서 돈을 꺼내 지불한다 |
| 8 | 거스름돈과 영수증을 받는다 |
| 9 | 음료를 가지고 편의점을 나온다 |

#### 교수절차의 적용

과제분석을 기반으로 **행동연쇄(chaining)**, **전체과제제시법(total task presentation)**, **촉구(prompt)** 등의 교수전략을 적용한다. 각 단계에서 학생의 수행을 기록하여 진전도를 모니터링한다.

### 6. 직업교육과 지원고용 모델

#### 직업교육의 단계

| 단계 | 내용 | 시기 |
|------|------|------|
| **직업인식(career awareness)** | 다양한 직업의 존재와 역할 인식 | 초등학교 |
| **직업탐색(career exploration)** | 자신의 흥미와 적성에 맞는 직업 탐색 | 중학교 |
| **직업준비(career preparation)** | 구체적 직업 기술 훈련 | 고등학교 |
| **직업배치(career placement)** | 실제 직업 환경에 배치 | 고등학교~전환기 |

#### 지원고용(Supported Employment) 모델

| 모델 | 설명 | 특징 |
|------|------|------|
| **개별배치모델(individual placement)** | 직무지도원이 1:1로 현장 지원 | 가장 통합적, 가장 널리 사용 |
| **소집단모델(enclave)** | 3~8명이 사업체 내에서 집단 근무 | 지속적 감독 제공, 사업체 내 통합 |
| **이동작업대모델(mobile work crew)** | 소집단이 이동하며 계약 작업 수행 | 조경, 청소 등 이동성 직무 |
| **소기업모델(entrepreneurial model)** | 장애인을 위한 소규모 사업체 운영 | 비장애인과 함께 고용 |

#### 직무지도원(job coach)의 역할

- **직무분석**: 직무 환경과 과제 분석
- **직무 교수**: 체계적인 교수 절차를 활용한 직업 기술 훈련
- **자연적 지원(natural supports) 구축**: 동료 근로자의 지원 체계 형성
- **지원 용암(fading)**: 직무지도원의 지원을 점진적으로 줄이고, 자연적 지원으로 전환
- **사후 관리(follow-along)**: 배치 후 지속적인 모니터링과 필요 시 재교수

## 출제경향분석

CBI와 지역사회 중심 교수는 특수교육학 임용시험에서 **중도 장애 교육과정과 연계하여 빈출**되는 영역이다.

1. **개념 정의형**: CBI의 정의, 원리(연령 적합성, 부분 참여 등), 현장학습과의 차이를 묻는 기본 문항이 꾸준히 출제된다.

2. **과제분석 적용형**: 특정 기술(예: 대중교통 이용, 식당에서 식사하기)에 대한 과제분석을 작성하게 하는 문항이 출제된다.

3. **일반화 전략형**: Stokes와 Baer의 일반화 촉진 전략을 설명하거나 사례에 적용하는 문항이 빈출된다.

4. **직업교육 통합형**: 직업교육의 단계, 지원고용 모델(개별배치, 소집단, 이동작업대)의 비교 문항이 출제된다.

5. **생태학적 목록형**: 생태학적 목록의 절차(영역→환경→하위환경→활동→기술→불일치분석)를 순서대로 서술하게 하는 문항이 출제된다.

6. **최근 동향**: 통합교육 맥락에서 CBI를 실시할 때의 유의사항, 일반교육과정 접근과 기능적 교육과정의 균형 등을 묻는 문항이 증가하고 있다.
