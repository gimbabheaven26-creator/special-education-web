## 핵심개념

**점자(braille)**: 시각장애인이 촉각을 통해 읽고 쓸 수 있도록 고안된 문자 체계이다. 1829년 프랑스의 루이 브라유(Louis Braille)가 고안하였으며, 6개의 돌출된 점(2열 3행)의 조합으로 문자를 표현한다.

**한글점자(훈맹정음)**: 1926년 박두성 선생이 창안한 한글 점자 체계이다. 한글의 자모 구조(초성+중성+종성)에 맞추어 설계되었으며, 6점(⠿) 체계를 기반으로 한다. 2020년 개정된 「한국 점자 규정」(문화체육관광부 고시)이 현행 표준이다.

**6점 체계**: 점자의 기본 단위인 점자 칸(cell)은 세로 3행, 가로 2열의 6개 점으로 구성된다. 각 점에는 왼쪽 위부터 아래로 1·2·3, 오른쪽 위부터 아래로 4·5·6의 번호가 부여된다.

```
① ④
② ⑤
③ ⑥
```

6개의 점을 조합하면 이론적으로 2^6 = 64가지(빈 칸 포함) 표현이 가능하다.

**초성·중성·종성**: 한글점자는 묵자(일반 문자)의 한글 자모 구조를 반영한다. 초성 자음, 중성 모음, 종성 자음을 각각 별도의 점자 칸으로 표기한다. 예를 들어, '한'은 초성 'ㅎ' + 중성 'ㅏ' + 종성 'ㄴ'으로 3칸이 필요하다.

**약자(abbreviated braille)**: 자주 사용되는 글자를 한 칸으로 줄여 표기하는 방법이다. 예: '가' → ⠫(1칸), '그' → ⠈(1칸). 약자를 사용하면 점자 분량을 줄이고 읽기 속도를 높일 수 있다.

**약어(contracted word)**: 자주 사용되는 단어 전체를 축약하여 표기하는 방법이다. 예: '그래서' → ⠈⠎, '그러나' → ⠈⠉.

> ⚠️ **용어 검증 필요**: 약자·약어의 구체적 점형은 2020년 개정 「한국 점자 규정」과 대조 필요

## 주요이론

### 1. 한글점자의 구성 원리

한글점자는 자음과 모음을 별도의 점자 칸으로 표기하며, 초성-중성-종성 순서로 배열한다.

#### 초성 자음

| 자음 | 점번호 | 자음 | 점번호 |
|------|--------|------|--------|
| ㄱ | 4 | ㄴ | 1-4 |
| ㄷ | 2-4 | ㄹ | 5 |
| ㅁ | 1-5 | ㅂ | 4-5 |
| ㅅ | 8(특수) | ㅇ | 1-2-4-5 |
| ㅈ | 4-6 | ㅊ | 5-6 |
| ㅋ | 1-2-4 | ㅌ | 1-2-5 |
| ㅍ | 1-4-5 | ㅎ | 2-4-5 |

> ⚠️ **용어 검증 필요**: 초성 자음의 점번호는 「한국 점자 규정」(2020) 기준. 교재와 반드시 대조할 것

#### 중성 모음

| 모음 | 점번호 | 모음 | 점번호 |
|------|--------|------|--------|
| ㅏ | 1-2-6 | ㅑ | 3-4-5 |
| ㅓ | 2-3-4 | ㅕ | 1-5-6 |
| ㅗ | 1-3-6 | ㅛ | 3-4-6 |
| ㅜ | 1-3-4 | ㅠ | 1-4-6 |
| ㅡ | 2-4-6 | ㅣ | 1-3-5 |

#### 종성 자음

종성은 초성과 같은 점형을 사용하되, 위치(중성 뒤)로 구별한다. 받침이 없는 글자 뒤에 모음으로 시작하는 글자가 오면 연음(이어 읽기)이 적용되므로 주의해야 한다.

---

### 2. 수표(숫자 표기)

한글점자에서 숫자를 표기할 때는 **수표(⠼, 3-4-5-6점)**를 먼저 적고, 그 뒤에 숫자에 해당하는 점자를 적는다. 숫자는 초성 자음 'ㄱ'부터 'ㅈ'까지의 점형을 재활용한다.

| 숫자 | 대응 자음 | 점번호 |
|------|----------|--------|
| 1 | ㄱ | 4 |
| 2 | ㄴ | 1-4 |
| 3 | ㄷ | 2-4 |
| 4 | ㄹ | 5 |
| 5 | ㅁ | 1-5 |
| 6 | ㅂ | 4-5 |
| 7 | ㅅ | - |
| 8 | ㅇ | 1-2-4-5 |
| 9 | ㅈ | 4-6 |
| 0 | (특수) | 2-4-5 |

수표의 효력은 빈 칸이나 다른 표지(문장부호, 글자표 등)가 나올 때까지 유지된다.

2026학년도 KICE 전공B 11번에서 시각장애 학생이 **숫자 점자를 활용**하면서, 수표 뒤에 오는 점형이 **영문자가 아니라 숫자**임을 인식하는 규칙이 출제되었다. 특히 'IDEA'와 같은 영어 단어를 점자로 표기할 때, 대문자 표기와 숫자 표기의 차이를 구분하는 것이 핵심이었다.

> ⚠️ **용어 검증 필요**: 숫자 7과 0의 정확한 점번호는 「한국 점자 규정」 확인 필요

---

### 3. 영어점자와 대문자 표기

영어점자에서 **대문자**를 표기할 때는 해당 글자 앞에 **대문자표(⠠, 6점)**를 붙인다.

- 한 글자만 대문자: 대문자표 1개 + 글자 (예: ⠠⠊ = I)
- 단어 전체가 대문자: 대문자표 2개 + 단어 (예: ⠠⠠⠊⠙⠑⠁ = IDEA)

영어점자는 기본적으로 알파벳 a~j가 1~0의 숫자와 동일한 점형을 사용하므로, 수표와 글자표의 구분이 매우 중요하다.

### 4. 점자 읽기·쓰기 지도법

#### 점자 읽기 지도

| 단계 | 내용 |
|------|------|
| **점자 전 단계(pre-braille)** | 촉각 변별력 훈련, 점자 칸 구조 이해, 줄 따라가기 등 기초 촉각 기술 습득 |
| **초기 읽기 단계** | 점자 문자 식별, 단어 수준의 읽기, 양손 읽기 기법 도입 |
| **유창성 단계** | 읽기 속도 향상, 양손 분리 읽기(한 손은 다음 줄 탐색), 약자·약어 활용 |

**양손 읽기 기법**: 효율적인 점자 읽기를 위해 양손의 검지를 모두 사용한다. 왼손은 줄의 시작과 다음 줄의 위치를 탐색하고, 오른손은 줄의 끝부분까지 읽는다. 숙련되면 두 손이 역할을 분담하여 읽기 속도가 크게 향상된다.

#### 점자 쓰기 지도

| 도구 | 특징 |
|------|------|
| **점자판(slate)과 점필(stylus)** | 가장 전통적인 쓰기 도구. 점자판에 종이를 끼우고 점필로 점을 찍는다. **오른쪽에서 왼쪽으로 거꾸로** 써야 뒤집었을 때 왼쪽에서 오른쪽으로 읽힌다 |
| **점자 타자기(braille writer)** | 퍼킨스 브레일러(Perkins Brailler)가 대표적. 6개 키를 동시에 눌러 한 칸을 완성한다. **왼쪽에서 오른쪽으로** 정방향 쓰기가 가능 |
| **점자정보단말기** | 전자 점자 기기로, 점자 입력과 출력이 모두 가능. 컴퓨터·스마트폰과 연결하여 활용 |

---

### 5. 점자정보단말기

**점자정보단말기(braille display, 점자 디스플레이)**: 컴퓨터나 스마트폰의 화면 정보를 점자로 변환하여 촉각적으로 출력하는 보조공학 기기이다.

주요 기능:
- 점자 입출력(읽기·쓰기)
- 문서 작성 및 편집
- 인터넷 검색, 이메일
- 스마트폰 연동(블루투스)
- 음성출력 병행 가능

2026학년도 KICE 전공B 11번에서는 시각장애 학생의 교과 활동별 보조공학 지원 계획이 출제되었다. 사회 교과에서 **점자정보단말기를 활용하여 생성형 AI로 제작한 사막 사진의 장면 해설을 듣는** 활동이 제시되었으며, 체육 교과에서는 **화면확대프로그램**과 **플랫베드 확대경**이 언급되었다.

> ⚠️ **용어 검증 필요**: 점자정보단말기의 구체적 기종(한소네 등)과 기능은 최신 제품 정보 확인 필요

---

### 6. 묵자-점자 전환 규칙

묵자(일반 인쇄 문자)를 점자로 전환할 때 적용되는 주요 규칙:

| 규칙 | 설명 | 예시 |
|------|------|------|
| **풀어쓰기** | 한글을 초성·중성·종성으로 분리하여 각각 점자 칸으로 표기 | '한' → ㅎ+ㅏ+ㄴ (3칸) |
| **약자 적용** | 자주 사용되는 글자는 약자로 축약 | '가' → 1칸 약자 |
| **띄어쓰기 준수** | 묵자의 띄어쓰기를 그대로 적용 | - |
| **줄바꿈** | 점자는 단어 중간에서 줄바꿈하지 않음을 원칙으로 함 | - |
| **수표·글자표 삽입** | 숫자 앞에 수표, 영문 앞에 글자표를 삽입 | '3월' → 수표+3+월 |
| **문장부호 변환** | 묵자의 문장부호를 점자 문장부호로 변환 | 마침표, 쉼표 등 각각의 점형 존재 |

## 기출연계

### 기입형 출제 패턴

- "한글점자는 세로 ( )행, 가로 ( )열의 ( )개 점으로 한 칸을 구성한다." → **3**, **2**, **6**
- "점자에서 숫자를 표기하기 위해 숫자 앞에 붙이는 표지를 ( )(이)라 한다." → **수표**
- "영어점자에서 대문자를 표기할 때 해당 글자 앞에 붙이는 ( )점 표지를 대문자표라 한다." → **6**
- "점자판과 점필로 점자를 쓸 때 ( )쪽에서 ( )쪽으로 써야 뒤집었을 때 정상적으로 읽힌다." → **오른**, **왼**
- "한글점자를 창안한 사람은 ( )이며, 이 점자의 이름은 ( )이다." → **박두성**, **훈맹정음**

### 서술형 출제 패턴

**[패턴 1] 점자 읽기·쓰기 지도 단계 서술**
- 점자 전 단계부터 유창성 단계까지의 지도 내용을 체계적으로 서술
- 답안 포인트: ① 점자 전 단계 — 촉각 변별력 훈련 ② 초기 읽기 — 문자 식별, 양손 읽기 도입 ③ 유창성 — 속도 향상, 양손 분리 읽기

**[패턴 2] 점자 쓰기 도구의 비교** (2026 KICE 전공B 11번 연계)
- 점자판·점필, 점자 타자기, 점자정보단말기의 특징을 비교
- 답안 포인트: ① 쓰기 방향(역방향 vs 정방향) ② 입력 방식 ③ 활용 범위

**[패턴 3] 묵자-점자 전환 규칙 적용**
- 제시된 묵자 문장을 점자로 전환할 때 적용해야 할 규칙을 서술
- 답안 포인트: ① 풀어쓰기 원칙 ② 약자 적용 여부 ③ 수표·글자표 삽입 위치

### 최근 출제 경향

- 2026학년도 KICE 전공B 11번에서 시각장애 학생의 **점자 표기**(숫자점자, 영어점자 대문자표)와 **점자정보단말기 활용**이 직접 출제
- 2025학년도 KICE 전공A 7번에서 시각장애 학생의 **확대 독서기**, **시간표 촉각 인식**이 출제
- 점자 자체의 점형을 묻기보다는, **점자 관련 보조공학 기기**와 **점자 지도법**을 상황에 적용하는 문항이 증가 추세
