## 핵심개념

**시력(visual acuity)**: 두 점 또는 두 선을 분리하여 볼 수 있는 눈의 능력을 말하며, 일반적으로 중심시력(central visual acuity)을 의미한다. 시력은 교정(안경, 콘택트렌즈) 후의 최대 교정시력으로 측정하며, 분수시력(예: 20/200)이나 소수시력(예: 0.1)으로 표기한다.

**시야(visual field)**: 눈을 고정한 상태에서 볼 수 있는 범위를 말한다. 정상 시야는 상방 약 60도, 하방 약 75도, 비측(코쪽) 약 60도, 이측(귀쪽) 약 100도이다. 시야 결손은 중심시야 결손(중심암점)과 주변시야 결손(터널시야)으로 구분된다.

**시기능(visual function)**: 시력과 시야를 포함하여, 시각 정보를 수용하고 처리하는 총체적인 능력을 말한다. 시기능에는 시력, 시야, 색각, 명암 순응, 대비감도, 안구운동 능력 등이 포함된다.

> ⚠️ **용어 검증 필요**: '시기능'과 '기능적 시력(functional vision)'의 차이에 대해 교재마다 정의가 다를 수 있음

## 주요이론

### 1. 시력측정 방법

시력측정은 시각장애의 정도를 판별하고 교육적 지원을 계획하는 기초가 된다.

#### 시력표를 이용한 측정

| 시력표 | 특징 |
|--------|------|
| **스넬렌차트(Snellen chart)** | 가장 널리 사용되는 시력표. 알파벳(E, C 등)을 사용. 검사 거리 20피트(6m)를 기준으로 시력을 측정. 20/20이 정상시력 |
| **란돌트 C차트(Landolt C chart)** | C자 모양(란돌트 고리)의 열린 방향을 식별하는 검사. 문맹이나 아동에게 적합. 국제 표준 시력표 |
| **스넬렌 E차트** | E자의 방향(상하좌우)을 식별. 알파벳을 모르는 아동에게 적합 |
| **LEA 기호 차트** | 원, 사각형, 집, 사과 등 단순한 기호 사용. 유아 및 지적장애 동반 시각장애 아동에게 적합 |

#### 스넬렌 분수시력의 이해

스넬렌 시력은 **검사 거리 / 정상인이 읽을 수 있는 거리**로 표현된다.

- **20/20**: 20피트 거리에서 정상인이 20피트에서 읽는 글자를 읽음 → 정상시력
- **20/200**: 20피트 거리에서 정상인이 200피트에서 읽는 글자를 읽음 → 법적 맹 기준
- **20/70**: 20피트 거리에서 정상인이 70피트에서 읽는 글자를 읽음 → 저시력 범위

소수시력으로 환산하면: 20/200 = 0.1, 20/70 ≈ 0.3, 20/20 = 1.0

#### 시력표 측정이 불가능한 경우

시력표로 측정할 수 없을 정도로 시력이 낮은 경우, 다음 단계로 측정한다.

| 단계 | 방법 | 시력 표기 |
|------|------|----------|
| **지수(finger counting, FC)** | 검사자의 손가락 수를 셀 수 있는 최대 거리를 측정 | FC 50cm (50cm에서 손가락 수를 셈) |
| **안전수동(hand movement, HM)** | 눈앞에서 손을 흔들 때 움직임을 감지할 수 있는지 확인 | HM (+) 또는 HM (-) |
| **광각(light perception, LP)** | 빛의 유무를 감지할 수 있는지 확인 | LP (+) 또는 LP (-) |
| **무광각(no light perception, NLP)** | 빛을 전혀 감지하지 못하는 상태 | NLP (전맹) |

2025학년도 KICE 전공A 7번에서 시각장애 학생의 시력이 "우안 시력은 50cm 앞에서 손가락 개수를 셀 수 있는 정도"라고 제시되었다. 이는 **지수(FC) 50cm**에 해당한다.

### 2. 법적 시각장애 기준

한국의 시각장애 기준은 **[「장애인복지법」](/subjects/laws/welfare-act) 시행령**에 규정되어 있다.

| 구분 | 기준 |
|------|------|
| **법적 맹(legal blindness)** | 좋은 눈의 최대 교정시력이 **0.04 이하**이거나, 시야가 주시점에서 **10도 이하**인 경우 |
| **저시력(low vision)** | 좋은 눈의 최대 교정시력이 **0.04 초과 ~ 0.3 이하**인 경우 |

**특수교육법상 시각장애**: [「장애인 등에 대한 특수교육법」](/subjects/laws/special-education-act) 제15조에서는 시각장애를 "시각계의 손상이 심하여 시각기능을 전혀 이용하지 못하거나 보조공학기기의 지원을 받아야 시각적 과제를 수행할 수 있는 사람으로서 시각에 의한 학습이 곤란하여 특정 광학기구·학습매체 등을 통하여 학습하거나 촉각 또는 청각을 학습의 주요 수단으로 사용하는 사람"으로 규정한다.

**교육적 분류**:
- **맹(blind)**: [점자](/subjects/visual-impairment/braille)를 주된 학습 매체로 사용하는 학생
- **저시력(low vision)**: 확대 자료, 광학기기 등을 활용하여 묵자(인쇄 문자)로 학습이 가능한 학생

> ⚠️ **용어 검증 필요**: 장애인복지법 시행령의 시각장애 등급 기준이 2019년 장애등급제 폐지 이후 변경되었으므로 최신 기준 확인 필요

### 3. 시기능훈련(visual function training)

저시력 학생이 잔존시력을 최대한 활용할 수 있도록 체계적으로 훈련하는 것이다. 시기능 훈련 목표는 [IEP(개별화교육계획)](/subjects/curriculum/iep)에 포함되어 체계적으로 지도해야 한다. 시기능은 훈련을 통해 향상될 수 있다.

#### 안구운동 훈련

| 훈련 유형 | 설명 |
|----------|------|
| **추적(tracking)** | 움직이는 대상을 눈으로 따라가는 능력. 수평·수직·원형 방향으로 천천히 움직이는 물체를 눈으로 추적하는 훈련 |
| **추종(pursuit)** | 부드럽게 움직이는 물체를 연속적으로 따라보는 능력. 추적과 유사하나, 보다 부드러운 안구운동을 강조 |
| **주시이동(saccade)** | 한 지점에서 다른 지점으로 시선을 빠르게 이동시키는 능력. 읽기 활동에 필수적인 안구운동 |
| **고정주시(fixation)** | 특정 대상을 일정 시간 동안 안정적으로 바라보는 능력 |

2025학년도 KICE 전공A 7번에서는 시각장애 학생별 교육적 고려 사항으로 **"효율적인 잔존 시각 활용을 위한 시기능 훈련하기"**가 제시되었으며, 학생C(망막색소변성)의 시기능 훈련으로 **움직이는 공 주고받기**와 **움직이는 단어 카드 읽기** 활동이 구체적으로 출제되었다.

#### 시기능 활용 훈련

| 훈련 | 내용 |
|------|------|
| **시각적 주의(visual awareness)** | 빛, 색, 움직임에 대한 인식 훈련. CVI 학생에게 특히 중요 |
| **시각적 변별** | 형태, 크기, 색상의 차이를 구별하는 훈련 |
| **시각적 기억** | 본 것을 기억하고 재인하는 훈련 |
| **시각-운동 협응** | 눈과 손의 협응 활동(쓰기, 그리기, 물건 잡기 등) |

### 4. 편심주시(eccentric viewing)

**편심주시**: 중심시야에 손상(중심암점)이 있는 경우, 손상되지 않은 망막 주변부를 활용하여 물체를 보는 기법이다. 예를 들어, 황반변성으로 중심시력이 손상된 학생은 보고자 하는 대상의 약간 옆이나 위/아래를 바라봄으로써 주변 망막으로 대상을 인식할 수 있다.

편심주시 훈련의 핵심:
- **선호망막부위(PRL, Preferred Retinal Locus)** 찾기: 가장 잘 보이는 망막 위치를 파악
- 해당 위치를 활용한 **읽기, 얼굴 인식, 물체 확인** 연습
- 시선을 의도적으로 편심 위치에 고정하는 습관화 훈련

2025학년도 KICE 전공A 7번에서 학생B(황반변성) 학생에 대해 **"중심부 시야가 손상이 되면 암점의 위치와 크기를 확인하여 ( ) 방법 지도하기"**라는 출제가 있었으며, 정답은 **편심주시(eccentric viewing)**였다.

> ⚠️ **용어 검증 필요**: '편심주시'와 '편심 보기' 용어가 혼용됨. 출제 시 사용된 용어 확인 필요

### 5. 확대경과 광학 보조기구

저시력 학생의 시기능을 보완하기 위한 다양한 광학 보조기구가 있다.

| 보조기구 | 특징 | 적합한 상황 |
|----------|------|----------|
| **손잡이형 확대경(hand-held magnifier)** | 손으로 잡고 사용. 배율과 초점 거리를 자유롭게 조절 가능. 간헐적 읽기에 적합 | 짧은 글 읽기, 가격표 확인, 약 설명서 등 |
| **스탠드형 확대경(stand magnifier)** | 받침대가 있어 초점 거리가 고정됨. 안정적인 사용 가능 | 장시간 읽기, 손 떨림이 있는 학생 |
| **안경형 확대경(spectacle magnifier)** | 안경에 렌즈를 장착. 양손이 자유로움 | 양손이 필요한 작업(쓰기, 공작 등) |
| **망원경(telescope)** | 원거리 사물을 확대하여 관찰. 단안/양안, 고정식/휴대식 구분 | 칠판 보기, 야외 활동, 원거리 표지판 |
| **확대 독서기(CCTV)** | 카메라로 촬영한 영상을 모니터에 확대 표시. 배율·대비·색상 조절 가능 | 장시간 학습, 높은 배율 필요 시 |
| **화면확대프로그램(screen magnifier)** | 컴퓨터 화면을 소프트웨어적으로 확대 | 컴퓨터 활용 학습 |

2025학년도 KICE 전공A 7번에서 학생B(황반변성)에 대해 **"상대적 크기 확대법을 적용하여 확대 독서기로 학습자료 접근성 높이기"**가 출제되었다. 또한 2026학년도 KICE 전공B 11번에서는 저시력 학생(시력 0.1)의 체육 교과 활동을 위해 **화면확대프로그램**으로 체육 전자 교과서를 읽게 하고, **플랫베드 확대경**을 사용하여 원거리 점수판을 확인하는 방안이 제시되었다.

### 6. CVI(피질시각장애, Cortical Visual Impairment)

**CVI(피질시각장애)**: 안구 자체에는 이상이 없으나, 대뇌의 시각 피질(후두엽) 또는 시각 전달 경로의 손상으로 인해 시각 정보를 처리하지 못하는 상태를 말한다. 선진국에서 소아 시각장애의 가장 흔한 원인이다.

#### CVI의 주요 특징

| 특징 | 설명 |
|------|------|
| **색상 선호** | 특정 색상(주로 빨간색, 노란색 등 밝은 색)에 대한 선호가 뚜렷함 |
| **움직임 선호** | 정지된 물체보다 움직이는 물체에 더 잘 반응 |
| **시각적 복잡성 회피** | 시각적으로 복잡한 환경에서 시기능이 저하됨 |
| **시야 선호** | 특정 시야(주로 하방 또는 측방)에서 더 잘 봄 |
| **빛 응시(light gazing)** | 밝은 빛을 응시하는 경향 |
| **시각적 잠복기(visual latency)** | 시각 자극에 대한 반응이 지연됨 |
| **시기능 변동** | 피로, 환경, 컨디션에 따라 시기능이 크게 변동 |
| **원거리 시각 제한** | 가까이 있는 물체는 보이나 먼 거리의 물체는 잘 보지 못함 |

2025학년도 KICE 전공A 7번에서 학생A의 원인이 **'대뇌피질시각장애'**로 제시되었으며, 교육적 고려 사항으로 **"단순한 수업 환경 제공하기"**, **"대상을 제시한 후 시각적으로 바로 반응하게 지도하기"**, **"교육 자료를 주기적으로 움직여 학생의 시각적 주의력 높이기"**가 출제되었다.

#### CVI 학생을 위한 교육적 지원

- 시각적으로 **단순한 환경** 제공 (배경 복잡성 최소화)
- 학생이 선호하는 **색상과 움직임** 활용
- **충분한 반응 시간**(시각적 잠복기) 허용
- 자료를 **주기적으로 움직여** 시각적 주의 유도
- **친숙한 물체**로 시작하여 점차 새로운 자극 도입

> ⚠️ **용어 검증 필요**: CVI의 특징 목록과 교육적 지원 방안은 Roman-Lantzy(2018) 등 최신 연구와 대조 필요

| 광각(LP) | 빛의 유무 감지 여부로 시력 표기 | ★★ |
| 법적 맹 | 최대 교정시력 0.04 이하 또는 시야 10도 이하 | ★★★ |
| 저시력 | 최대 교정시력 0.04 초과 ~ 0.3 이하 | ★★★ |
| 추적(tracking) | 움직이는 대상을 눈으로 따라가는 안구운동 훈련 | ★★ |
| 주시이동(saccade) | 한 지점에서 다른 지점으로 시선을 빠르게 이동 | ★★ |
| 편심주시(eccentric viewing) | 중심암점이 있을 때 주변 망막을 활용하여 보는 기법 | ★★★ |
| 선호망막부위(PRL) | 편심주시 시 가장 잘 보이는 망막 위치 | ★★ |
| 손잡이형 확대경 | 손으로 잡고 사용. 초점 거리 자유 조절 | ★★ |
| 스탠드형 확대경 | 받침대 고정. 안정적 사용. 손 떨림 학생에게 적합 | ★★ |
| 확대 독서기(CCTV) | 카메라+모니터. 높은 배율·대비 조절 가능 | ★★★ |
| 화면확대프로그램 | 컴퓨터 화면을 소프트웨어적으로 확대 | ★★ |
| CVI(피질시각장애) | 시각 피질 손상으로 인한 시각장애. 안구 자체는 정상 | ★★★ |

## 기출연계

### 기입형 출제 패턴

- "좋은 눈의 최대 교정시력이 ( ) 이하이거나, 시야가 주시점에서 ( )도 이하인 경우를 법적 맹이라 한다." → **0.04**, **10**
- "시력표로 측정이 불가능하여 검사자의 손가락 수를 셀 수 있는 거리로 시력을 표기하는 방법을 ( )(이)라 한다." → **지수(FC, finger counting)**
- "중심시야에 암점이 있을 때 손상되지 않은 망막 주변부를 활용하여 물체를 보는 기법을 ( )(이)라 한다." → **편심주시(eccentric viewing)**
- "안구 자체에는 이상이 없으나 대뇌 시각 피질의 손상으로 시각 정보를 처리하지 못하는 상태를 ( )(이)라 한다." → **CVI(피질시각장애)**

### 서술형 출제 패턴

**[패턴 1] 시각장애 학생별 교육적 지원 서술** (2025 KICE 전공A 7번 직접 출제)
- 대뇌피질시각장애, 황반변성, 망막색소변성 등 원인별 특성과 교육적 고려 사항을 서술
- 답안 포인트: ① 각 원인의 시각적 특성 ② 편심주시, 확대 독서기 등 보조 방법 ③ 시기능 훈련 구체적 활동

**[패턴 2] 보조공학 기기의 선정과 활용 서술** (2026 KICE 전공B 11번 연계)
- 저시력 학생의 교과 활동에 적합한 보조공학 기기를 선정하고 활용 방안을 서술
- 답안 포인트: ① 학생의 시기능 수준에 따른 기기 선택 ② 확대경 유형별 특징 비교 ③ 점자정보단말기·화면확대프로그램 병용

**[패턴 3] CVI 학생의 특성과 교육적 지원 서술**
- CVI의 주요 특징(색상 선호, 움직임 선호, 시각적 복잡성 회피 등)과 교육적 지원 방안을 서술
- 답안 포인트: ① CVI의 정의(안구 정상, 대뇌 손상) ② 주요 특징 3가지 이상 ③ 각 특징에 대응하는 교육적 지원

### 최근 출제 경향

- 2025학년도 KICE 전공A 7번에서 **CVI(대뇌피질시각장애)**, **황반변성**, **망막색소변성** 학생별 교육적 고려 사항이 직접 출제
- 2026학년도 KICE 전공B 11번에서 저시력 학생(시력 0.1)의 교과별 **보조공학 지원**(화면확대프로그램, 플랫베드 확대경, 점자정보단말기)이 출제
- 단순한 시력 수치 암기보다는 **원인별 특성에 맞는 교육적 지원**을 구체적으로 서술하는 문항이 주류
- **편심주시**, **시기능 훈련**, **확대 독서기** 등 실제 지도 상황에 적용하는 서술형 문항 증가

## 학습체크

- [ ] 시력, 시야, 시기능의 개념을 구분하여 설명할 수 있다.
- [ ] 스넬렌차트의 분수시력(20/200 등)의 의미를 설명할 수 있다.
- [ ] 시력표 측정이 불가능할 때의 측정 단계(지수→안전수동→광각→무광각)를 순서대로 설명할 수 있다.
- [ ] 법적 맹(0.04 이하 또는 시야 10도 이하)과 저시력(0.3 이하)의 기준을 정확히 서술할 수 있다.
- [ ] 추적, 추종, 주시이동, 고정주시의 차이를 설명할 수 있다.
- [ ] 편심주시의 개념과 훈련 방법을 설명할 수 있다.
- [ ] 확대경의 종류(손잡이형, 스탠드형, 안경형)와 각각의 적합한 상황을 구분할 수 있다.
- [ ] CVI(피질시각장애)의 주요 특징 5가지 이상을 나열하고 교육적 지원 방안을 서술할 수 있다.
- [ ] 시각장애 원인(대뇌피질시각장애, 황반변성, 망막색소변성)별 특성과 교육적 고려 사항을 비교할 수 있다.
