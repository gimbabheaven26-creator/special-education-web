## 1. 2015 → 2022 핵심 용어 변화

### 공통 변천사 (제7차~2022)

| 개정 시기 | 국가 기준 | 개별 성취기준 달성 정도 | 단원/영역별 달성 정도 |
|---|---|---|---|
| 제7차 | 내용 | 성취기준 | — |
| 2007 개정 | 중영역 | 성취기준 | 평가기준 |
| 2009 개정 | 교과과정 내용 | 성취기준 | 성취수준 |
| **2015 개정** | **교육과정 성취기준** | **평가준거 성취기준** | **평가기준 / 단원·영역별 성취수준** |
| **2022 개정** | **성취기준** | **삭제** | **성취수준 (성취기준별 + 영역별)** |

### 핵심 2개 변화 (시험 출제 포인트)

| 2015 개정 | 2022 개정 |
|---|---|
| 교육과정 성취기준 | **성취기준** |
| 평가준거 성취기준 | **삭제** (별도 개발 안 함) |
| 평가기준 | **성취수준** (성취기준별 + 영역별로 통합) |
| 단원/영역별 성취수준 | **성취수준** (동일 개념으로 통합) |

> **요약**: "평가준거 성취기준"이라는 중간 층위가 사라지고, "성취기준 → 성취수준"의 2단계로 간소화됨.

### 2022 기본 교육과정 전용 용어

| 용어 | 정의 |
|---|---|
| **교육과정 성취수준** | 성취기준의 특성에 따라 교육 내용 및 범위 설정을 위한 단계·위계. 지식·이해, 과정·기능, 가치·태도 3차원으로 진술. **풀(pool)** 형태로 제공 |
| **성취기준별 성취수준** | 성취기준 단위로 개별 학생이 실제 도달·수행한 정도를 기술. 개별 맞춤 설정 가능 |
| **세분화된 성취기준별 성취수준** | 3~5수준(A~E)으로 부족할 때 더 세분화하여 기술 (통합교육 환경 전용) |

## 2. 성취수준 풀(pool) 구조와 활용법

### 풀(pool)이란

성취기준을 **3개 부분으로 분해**하고, 각 부분에 대해 다수준 행동 기술어를 **풀(pool) 형태**로 제공하는 것.

- 단일 문장 획일화 → 다양한 특수교육 학생 실정 반영 불가
- 풀 형태 → 학생/학급/학교 상황에 따라 **다채롭게 조합** 가능

### 풀 표 구조 (예시)

**[9국어03-02] "기록의 필요성을 알고 일상생활에 필요한 내용과 정보를 메모한다."**

| 기록의 필요성을 알고 | 일상생활에 필요한 내용과 정보를 | 메모한다 |
|---|---|---|
| 필요한 정보를 기억하기 위해 | 수업 시간에 배운 내용을 | 글로 메모하고 활용하는 습관을 형성 |
| 필요한 정보를 전달하기 위해 | 일기를 쓸 때 있었던 일을 | 음성으로 메모하는 습관 형성 |
| 필요한 정보를 수집하기 위해 | 여행할 때 경험한 내용을 | 사진으로 메모하는 활동에 참여 |
| — | 약속이나 초대할 때 날짜·요일·시간·장소를 | 그림으로 메모하는 활동에 참여 |
| — | 책을 읽고 난 후 느낀 점을 | 영상으로 메모하는 활동에 참여 |

### 활용 절차 (6단계)

```
1단계. 성취기준 분석
   └ 교육과정 내용 체계 및 성취기준 분석
   └ 성취기준별 성취수준 분석
   └ 영역별 성취수준 분석

2단계. 성취기준별 교육과정 성취수준 설정
   └ 풀(pool)을 활용하여 지식·이해, 과정·기능, 가치·태도별 선택
   └ 개별 학생 A, B, C 각각에 맞게 다르게 조합 가능

3단계. 교수학습·평가에서의 준거 설정
   └ 학기 단위 성취수준 진술
   └ 평가 계획 수립 (요소 선정, 계획서 작성)

4단계. 평가도구 개발 (수행 중심 평가)
   └ 수행평가 과제 및 채점 기준 개발
   └ 관찰 활용 시 관찰 시점·기준 개발

5단계. 평가 척도에서 준거 설정
   └ 도달/미도달(pass/fail) 기준
   └ 3점 또는 5점 척도 기준

6단계. 성취평가 결과 산출 및 보고
   └ 피드백 제공 → 교수·학습 개선에 활용
```

## 3. 교육과정 성취수준 3축 구조

모든 성취기준에 대해 3가지 차원으로 교육과정 성취수준이 진술됨:

| 차원 | 설명 | 예시 (국어 읽기) |
|---|---|---|
| **지식·이해** | 학생이 알아야 하는 지식, 개념 | 단어의 의미 관계(유의어, 반의어, 상하위어) |
| **과정·기능** | 학생이 수행할 수 있는 과정과 기능 | 어절 단위로 띄어 읽기, 문장 의미 파악하기 |
| **가치·태도** | 학생이 가져야 할 가치관과 태도 | 자발적인 읽기 태도, 읽기 경험 공유 |

## 4. IEP 연계 방안

### 핵심 논리

> "IEP를 수립하는 것과 교육과정-수업-평가 일체화를 실현해 가는 것을 별도의 것으로 여기지 않고 서로 유기적으로 연계" (가이드북 p.3)

### IEP 작성 시 활용

1. **장·단기 목표 설정**: 풀(pool)에서 학생 수준에 맞는 행동 기술어를 선택하여 IEP 목표 문장 구성
2. **교육과정 재구성**: 교육과정 성취수준을 기반으로 개별 학생에 맞는 내용 범위 설정
3. **과정 중심 평가**: 성취기준별 성취수준을 평가 준거로 직접 활용
4. **수준별 교육**: target_level(기초/보통/우수)에 풀의 다수준 기술어를 매핑

## 5. 성취기준별 데이터 구조 (PDF 내부)

각 성취기준에 대해 PDF는 다음 4개 섹션을 제공:

```
┌─────────────────────────────────────┐
│ 1. 내용요소 / 성취기준 / 해설       │
│    + 적용 시 고려사항               │
├─────────────────────────────────────┤
│ 2. 교육과정 성취수준                │
│    (3열: 지식·이해 / 과정·기능 /    │
│     가치·태도, 다수준 행)           │
├─────────────────────────────────────┤
│ 3. 성취수준 풀(pool)               │
│    (3열: 성취기준 문장 3분할,       │
│     각 열 3~7행 행동 기술어)        │
├─────────────────────────────────────┤
│ 4. 교수·학습 참고자료               │
│    (교수 전략, 활동 예시, 유의점)    │
└─────────────────────────────────────┘
```

## 6. 프로젝트 활용 맥락 (nadaun IEP 도구)

| 용도 | 활용 데이터 | 위치 |
|---|---|---|
| AI 주간계획 생성 프롬프트 | 성취수준 풀(pool) → 수준별 활동 제안 | `achievement_pool` 필드 |
| target_level 매핑 | 기초/보통/우수 → 풀 행동 기술어 연결 | `curriculum_levels` 필드 |
| IEP 목표 문장 자동 생성 | 풀 열 조합 → 자연어 목표 문장 | `achievement_pool.columns` |
| 교수학습 전략 제안 | 교수·학습 참고자료 | `teaching_references` 필드 |
| 교육과정 정합성 검증 | 성취기준 코드 + 내용 | `code`, `content` 필드 |
