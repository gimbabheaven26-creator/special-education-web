## 0. 강선생 분석에서 빠진 것 (PDF 직접 분석 결과)

강선생의 kice-analysis.md는 영역별 비중·키워드 빈도 등 매크로 분석은 잘 되어있으나,
**실제 문항을 읽어보면 드러나는 구조적 특성**이 빠져 있다.

### 0.1 복합 영역 문항이 대부분

거의 모든 4점 서술형이 **2~3개 영역을 횡단**. 강선생의 "주된 영역 1개 분류"는 한계가 있다.

| 2026 문항 | 강선생 분류 | 실제 포함 영역 |
|---------|---------|-----------|
| 전공A-5 | curriculum | curriculum + inclusive(TAI) + assessment(평가조정) |
| 전공A-7 | assessment | assessment(RTI) + inclusive(학습장애 진단모델) |
| 전공A-10 | curriculum | curriculum(2022개정) + transition(일반화) + inclusive(학교스포츠클럽) |
| 전공B-8 | transition | transition(직업교육) + curriculum(2022개정총론) + behavior(자기교수법) |
| 전공B-10 | curriculum | introduction(자폐인지특성) + curriculum(교수적수정) |

→ **퀴즈도 단일 개념이 아닌 복합 시나리오 기반으로 만들어야 함**

### 0.2 <작성 방법> 하위 질문 구조

4점 서술형은 반드시 **3~4개 하위 질문**이 있음. 예시 (2026 전공A-8):
1. 핸드트레일링+상부보호법 함께 사용 이유 서술
2. 랜드마크로 부적합한 것 1개 찾아 기호+이유
3. 시력 용어(지수) 쓰기 + 촉각적 시간표 예시 1가지

→ **1문항 = 3~4개 독립 개념**. 퀴즈 1개당 1개념이면 실전과 괴리.

## 1. 핵심 발견: 출제 비중 vs 퀴즈 비중 불일치

| 영역 | KICE 출제 비중 | 현재 퀴즈 수 | 퀴즈 비중 | **갭** |
|------|:------------:|:----------:|:-------:|:-----:|
| introduction (개론) | **~24%** | 5문제 | 7.7% | **⚠️ -16.3%** |
| laws (법령) | ~20% | 10문제 | 15.4% | -4.6% |
| curriculum (교육과정) | ~13% | 10문제 | 15.4% | +2.4% |
| assessment (진단/평가) | ~12% | 10문제 | 15.4% | +3.4% |
| transition (전환교육) | ~12% | 10문제 | 15.4% | +3.4% |
| behavior-support (행동지원) | ~11% | 10문제 | 15.4% | +4.4% |
| inclusive-education (통합교육) | ~8% | 10문제 | 15.4% | +7.4% |

### 결론: **특수교육학 개론이 심각하게 부족**

KICE에서 가장 많이 나오는 영역(24%)인데 퀴즈는 5문제뿐(7.7%). 최소 **15~20문제 추가** 필요.

## 4. 콘텐츠 우선순위 제안 (강선생에게)

### 🔴 1순위: 즉시 필요 (KICE 고빈출 + 현재 0문제)

1. **장애유형별 특성 퀴즈팩** (introduction 영역 확장)
   - 시각장애: 점자 규정, 보행 훈련, 점자정보단말기 → 최소 5문제
   - 청각장애: 청력도 해석, 수어, 인공와우 → 최소 5문제
   - 지체장애: 뇌성마비 유형, GMFCS, 원시반사 → 최소 5문제
   - 자폐성장애: 반향어, 공동관심, TEACCH vs PRT → 최소 3문제 (일부 있음)
   - 의사소통장애: AAC, 보완대체 상징체계 → 최소 3문제

2. **법령 심화** (laws 영역 5문제 추가)
   - 2022 개정 특수교육법 조항
   - 건강장애 원격수업/병원학교 법적 근거
   - 장애인권리협약 관련

### 🟡 2순위: 다음 스프린트

3. **보조공학/디지털** — 점자정보단말기, 스위치, 화상키보드, KWCAG
4. **증후군 퀴즈** — 다운, 프래더-윌리, 엔젤만, 윌리엄스, 클라인펠터
5. **2022 개정 교육과정** — 총론 변화, 기본교육과정 개정 포인트

### 🟢 3순위: Phase 2

6. **IRIS 스타일 모듈형 학습** — Challenge → 학습 → 평가 사이클
7. **사례 기반 서술형** — KICE 4점 서술형 패턴 연습
8. **통합 문항** — 2~3개 영역 횡단 (KICE 최근 트렌드)

## 6. 문제 유형 제안: KICE 실전형 퀴즈 포맷

현재 퀴즈는 객관식/OX/단답/서술 4종인데, KICE 실전에 맞추려면:

### 새로 추가해야 할 퀴즈 유형

| 유형 | KICE 대응 | 예시 |
|-----|---------|-----|
| **시나리오 기반 복합** | 4점 서술형 | 교사 대화문 읽고 ㉠용어 + ㉡이유 + ㉢예시 3개 답하기 |
| **오류 찾기** | "틀린 것 N가지 찾아 바르게 고쳐 쓰시오" | ㉠~㉤ 중 틀린 2가지 찾기 |
| **도표/그래프 해석** | RTI 그래프, 청력도, 교육과정 편성표 | 그래프 보고 판별 근거 제시 |
| **법령 적용** | "~법에 근거하여" | 사례에 법 조항 적용 |
| **비교 서술** | "차이점을 ~측면에서" | 두 개념 비교 |

### 2026 전공A 문항별 실제 구조 (참고용)

| # | 배점 | 유형 | 핵심 주제 |
|---|:---:|------|---------|
| 1 | 2점 | 용어 2개 | 특성화 특수학교 + 통합학급 (법령) |
| 2 | 2점 | DSM-5 명칭 + 기법 명칭 | 사회불안장애 + 실제상황 둔감법 위계 |
| 3 | 2점 | 용어 2개 | 부적강화/정적벌 + 기능(도피) + 대체행동(기능적 의사소통) |
| 4 | 2점 | 영역 1개 + 용어 | 할펀 전환모델(주거) + 평생학습 |
| 5 | 4점 | 시나리오+3하위 | 키워드전략 + TAI 장점 + 평가조정 유형 구분 |
| 6 | 4점 | 시나리오+3하위 | 자발화 표본수집 이유 + 발화구분기준 + 어휘다양도→의미론 + 언어기능(요구) |
| 7 | 4점 | 그래프+3하위 | RTI 3단계(집중교수) + 학습장애 판별근거 + 강점-약점 진단모델 |
| 8 | 4점 | 도면+3하위 | 핸드트레일링+상부보호법 이유 + 랜드마크 + 시력(지수) + 촉각적 시간표 |
| 9 | 4점 | 시나리오+3하위 | 인공와우 변환기 + 전극 기능 + 지문자 + 모음 지문자 |
| 10 | 4점 | 시나리오+3하위 | 2022개정(학교스포츠클럽 오류찾기) + 기능적 기술 + 자극일반화/반응일반화 |
| 11 | 4점 | 평가지+3하위 | 자폐 DSM-5(동일성 고집) + 자리올림 오류 + 모두세기→이어세기 지도 |
| 12 | 4점 | 시나리오+3하위 | 보치아 + STNR 자료 위치 + 몸통 전방지지 + 가위자세 하지지지(외전기) |
