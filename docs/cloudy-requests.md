# 클루디 → 강선생 요청 목록

> 클루디가 강선생에게 필요한 UI/API 작업을 기록
> 강선생은 완료 시 상태를 업데이트하고 changelog.md에 기록

## 작성 규칙

```
### REQ-{번호}: 요청 제목
- **요청일**: YYYY-MM-DD
- **우선순위**: P0(즉시) / P1(이번 주) / P2(다음 Phase)
- **상태**: 대기 / 진행중 / 완료 / 보류
- **상세**: 구체적으로 필요한 것
- **완료 기준**: 어떻게 되면 완료인지
```
