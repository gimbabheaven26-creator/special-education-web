## 3. FSRS 연계 제안

### 3.1 현재 → FSRS 매핑

```
현재 난이도    FSRS difficulty (0~10)    반복 간격 특성
─────────────────────────────────────────────────
1 (쉬움)     → 1~2                     즉시 기억, 긴 간격
2 (보통-쉬움) → 3~4                     1~2회 반복 후 기억
3 (보통)     → 5~6                     기본 간격으로 반복
4 (어려움)   → 7~8                     짧은 간격 집중 반복
5 (매우 어려움) → 9~10                    최단 간격, 빈번 반복
```

### 3.2 문제: 현재 매핑하면?

FSRS difficulty 1~6 범위에만 문항이 몰림 → 간격반복의 효과가 극대화되지 않음.
실제 수험생은 KICE 4점 서술형에서 가장 고전하는데, 우리 퀴즈에 이 수준의 문제가 없음.

### 3.3 난이도 재설계 제안

| 조정 대상 | 현재 | 목표 | 방법 |
|-----------|------|------|------|
| 확인형 (fill_in, ox) | 1~2 | 1~3 | 함정/혼동 선지 추가, 법조문 세부 조항 출제 |
| 확인형 (multiple) | 1~3 | 1~4 | 복합 보기 ("모두 고른 것은?"), 소거법 필요 문제 |
| 사고형 (descriptive) | 2~3 | 3~5 | KICE 4점 구조 각색, 복합 영역 시나리오 |

### 3.4 난이도 4~5 기준 (신규)

**난이도 4 — 어려움**:
- 2개 이상 개념을 결합해야 풀 수 있는 문제
- 기출 2~3점 문항 수준
- 예: "RTI 2단계에서 사용하는 CBM의 실시 절차와 결과 해석 방법을 서술하시오"

**난이도 5 — 매우 어려움**:
- KICE 4점 서술형 동형
- 3개 이상 과목을 횡단하는 복합 시나리오
- (가)/(나) 자료 병치 + 3~4개 하위질문
- 예: 기출 뷰어의 동형문제를 퀴즈 형태로 압축한 것

## 4. 액션 아이템

1. **즉시**: 기존 descriptive 127건 중 20~30건을 난이도 4로 상향 (복합 시나리오 추가 시)
2. **단기**: 난이도 4 확인형 10~15건 신규 제작 (함정 선지/복합 보기)
3. **중기**: 난이도 5 사고형 5~10건 신규 제작 (KICE 4점 동형 압축)
4. **FSRS 연동**: 안선생에게 난이도→FSRS difficulty 매핑 테이블 전달
