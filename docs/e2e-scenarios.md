# E2E Test Scenarios

> 전체 사용자 여정 검증. Playwright 기반.
> 최종 수정: 2026-03-11

## 핵심 사용자 여정 (Critical Path)

### Journey 1: 학습 흐름
```
홈(/) → 과목 목록(/subjects) → 과목 상세(/subjects/[slug])
→ 챕터 학습(/subjects/[slug]/[chapter])
```

**검증 항목:**
- [ ] 홈에서 과목 목록 진입 가능
- [ ] 11개 과목이 모두 표시됨 (DB에서 로드)
- [ ] 과목 클릭 → 해당 과목의 챕터 목록 표시
- [ ] 챕터 클릭 → MDX 콘텐츠 렌더링 (빈 페이지 아님)
- [ ] 뒤로가기 정상 작동

### Journey 2: 퀴즈 흐름
```
퀴즈 목록(/quiz) → 과목 선택 → 퀴즈 풀기(/quiz/[subject])
→ 답 선택 → 정답/오답 피드백 → 결과 화면
```

**검증 항목:**
- [ ] 퀴즈 목록에서 과목별 문제 수 표시
- [ ] 퀴즈 시작 → 첫 문제 표시
- [ ] 객관식: 4개 선택지 렌더링, 클릭 시 선택 상태
- [ ] OX: O/X 버튼 렌더링
- [ ] 정답 선택 → 초록색 피드백 + 해설
- [ ] 오답 선택 → 빨간색 피드백 + 해설 + 오답 설명
- [ ] 마지막 문제 후 결과 화면 (정답률)

### Journey 3: 문제지 흐름
```
문제지 목록(/worksheets) → 과목/토픽 선택
→ 문제지 풀기(/worksheets/[id]) → 답안 확인(/worksheets/[id]/answers)
```

**검증 항목:**
- [ ] 과목별 토픽 목록 표시
- [ ] 토픽 선택 → 문제 목록 표시
- [ ] 문제에 답 입력 가능
- [ ] 답안 확인 페이지에서 정답/해설 표시

### Journey 4: 플래시카드 흐름
```
플래시카드 목록(/flashcards) → 카드 복습(/flashcards/review)
→ 카드 추가(/flashcards/add)
```

**검증 항목:**
- [ ] 플래시카드 목록 표시
- [ ] 카드 플립 동작 (앞면 → 뒷면)
- [ ] 복습 모드 진입/진행
- [ ] 새 카드 추가 폼

### Journey 5: 검색 & 북마크
```
검색(/search) → 키워드 입력 → 결과 클릭 → 해당 페이지 이동
북마크(/bookmarks) → 저장된 항목 표시 → 클릭 시 이동
```

**검증 항목:**
- [ ] 검색어 입력 → 퀴즈/챕터 결과 표시
- [ ] 결과 클릭 → 해당 페이지로 이동
- [ ] 북마크 추가/제거 동작
- [ ] 북마크 페이지에서 목록 표시

## 모바일 UI 검증

### Mobile Navigation
```
모든 페이지에서 하단 탭바 표시 → 탭 전환 → 정상 이동
```

**검증 항목:**
- [ ] 하단 탭바: 홈/학습/퀴즈/카드 아이콘 표시
- [ ] 더보기 탭: 문제지/북마크/검색 표시
- [ ] 각 탭 클릭 → 해당 페이지 이동
- [ ] 현재 페이지에 해당하는 탭 활성화 표시
- [ ] viewport 375px에서 레이아웃 깨지지 않음

### Header
- [ ] 로고 뱃지 표시
- [ ] 검색 아이콘 클릭 → /search 이동
