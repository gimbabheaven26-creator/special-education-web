### 1.2 전공A vs 전공B 차이점

| 구분 | 전공A (2교시) | 전공B (3교시) |
|------|--------------|--------------|
| **문항 유형** | 기입형(2점) + 서술형(4점) | 서술형(4점) + 논술형(10점, 초기) |
| **배점 구조 (초기: 2016~2019)** | 기입형 2점 x 8 + 서술형 4점 x 6 = 40점 | 서술형 4~5점 x 6~7 + 논술형 10점 x 1 = 40점 |

| **특징** | 용어, 명칭, 법령 등 정확한 지식 요구 | 종합적 사고력, 적용 능력, 논술 능력 평가 |
| **교시** | 2교시 | 3교시 |

### 1.3 구조 변화 추이

- **2016~2019**: 전공A 14문항(기입형 2점 x 8 + 서술형 4점 x 6), 전공B 8문항(서술형 + 논술형 10점 1문항)
- **2020~현재**: 전공A 12문항, 전공B 11문항으로 통일. 논술형 10점 문항 폐지. 전공B도 기입형 2점 문항 포함.
- 총점(40+40=80점)과 시험시간(각 90분)은 전 연도 동일.

## 5. 문제 유형 패턴

### 5.1 기입형 (2점) 자주 사용되는 질문 패턴

| 패턴 | 예시 | 빈도 |
|------|------|:----:|
| "~의 명칭을 쓰시오" | "㉠에 해당하는 중재 방법의 명칭을 쓰시오" | 매우 높음 |
| "~에 해당하는 용어를 쓰시오" | "괄호 안의 ㉠에 해당하는 용어를 쓰시오" | 매우 높음 |
| "~에 해당하는 내용을 순서대로 쓰시오" | "㉠, ㉡에 해당하는 내용을 순서대로 쓰시오" | 높음 |
| "~에 근거하여 쓰시오" | "특수교육법에 근거하여 ㉠에 들어갈 내용을 쓰시오" | 높음 |
| "틀린 것 N가지를 찾아 바르게 고쳐 쓰시오" | "㉠~㉤ 중 틀린 것 2가지를 찾아 바르게 고쳐 쓰시오" | 중간 |

### 5.2 서술형 (4점) 자주 사용되는 질문 패턴

| 패턴 | 예시 | 빈도 |
|------|------|:----:|
| "<작성 방법>에 따라 서술하시오" | 가장 보편적인 형식. 3~4개 하위 요구사항 제시 | 매우 높음 |
| "~의 이유를 1가지 서술할 것" | "밑줄 친 ㉠의 이유를 1가지 서술할 것" | 매우 높음 |
| "~에 해당하는 것을 쓰고 ~을 서술할 것" | "명칭을 쓰고, 장점을 1가지 서술할 것" | 높음 |
| "~중 적절하지 않은 것 N가지를 찾아 기호를 쓰고 이유를 서술할 것" | 오류 판별 + 근거 제시 유형 | 높음 |
| "~의 차이점을 ~측면에서 서술할 것" | "㉠과 ㉡의 차이점을 교수 집단 구성 측면에서 서술할 것" | 중간 |
| "~에 근거하여 ~의 예를 1가지 서술할 것" | 이론 적용 + 구체적 사례 제시 유형 | 중간 |

### 5.3 논술형 (10점, 2016~2019 전공B에만 존재) 패턴

| 패턴 | 특징 |
|------|------|
| "서론, 본론, 결론의 형식으로 작성할 것" | 필수 형식 조건 |
| 이론 비교 + 적용 | 예: 의료적 모델 vs 사회적 모델 비교 후 ICF 설명 |
| 대화 맥락 활용 | 시나리오 대화를 인용하여 논의 |
| 장단점 + 시사점 | 교수법의 장점, 학습자 입장 장점, 한계 및 시사점 |

## 6. 난이도 추세

### 6.1 연도별 난이도 변화 관찰

| 시기 | 난이도 경향 | 주요 특징 |
|------|-----------|----------|
| **2016~2017** | 중간 | 용어 암기 중심, 단답형 비중 높음. 장애 유형 기본 지식 중심 |
| **2018~2019** | 중간~높음 | 복합 문항 증가. 하나의 시나리오에서 다수 영역 통합 출제. 교육과정 개정(2015 개정) 관련 문항 급증 |
| **2020~2021** | 높음 | 문항 구조 변경(전공A 14->12문항, 전공B 8->11문항). 논술형 폐지. 서술형 문항의 하위 요구사항 증가(3~4개). COVID-19 관련 맥락 반영 |
| **2022~2023** | 높음 | 통합적 사고력 요구 증가. 법령 + 교육과정 + 교수법 연계 문항 증가. 점자 규정 개정(2020) 반영 |
| **2024~2026** | 매우 높음 | 2022 개정 교육과정 반영 시작. 디지털 리터러시, 보편적 학습설계 등 최신 개념 출제. 실제 교육 현장 시나리오 복잡도 증가 |

### 6.2 난이도 변화 요인

1. **통합형 문항 증가**: 단일 개념이 아닌 2~3개 영역을 횡단하는 문항이 증가
2. **법령 개정 반영**: 특수교육법 개정 주기에 맞춰 새로운 조항 출제
3. **교육과정 개정 반영**: 2015 개정(2018~), 2022 개정(2024~) 시점에 난이도 상승
4. **실제 적용 능력 강조**: 단순 암기에서 교육 현장 시나리오 기반 적용 능력으로 전환
5. **하위 질문 수 증가**: 서술형 1문항 내에서 3~4개의 세부 요구사항을 포함

### 7.3 핵심 변화 요약

1. **전환교육 비중 확대**: 자기결정, 지원고용, 직업평가, 개인중심계획(PCP) 등 전환교육 관련 문항이 2021년 이후 꾸준히 증가
2. **진단/평가 영역 심화**: 단순 검사도구 명칭이 아닌 결과 해석, 관찰기록법 적용, 일치도 계산 등 실질적 평가 역량 요구
3. **장애 유형의 다양화**: 초기에는 지적장애, 청각장애, 시각장애 중심이었으나, 건강장애, 중도중복장애, 감각통합 문제 등 다양한 장애 영역으로 확대
4. **법령 문항의 실무화**: 단순 조문 암기에서 실제 사례에 법령 적용하는 형태로 전환
5. **보조공학/AAC 비중 증가**: 점자정보단말기, 스위치, 스캐닝 기법, 화상키보드 등 보조공학 실무 관련 출제 확대
6. **교육과정 편성 운영 실무**: 자유학기제, 학교스포츠클럽, 집중이수제, 복식학급 운영 등 실제 학교 운영 맥락 문항 증가

## 학습 전략 제안

위 분석 결과를 바탕으로 다음과 같은 학습 전략을 제안한다.

### 필수 학습 영역 (배점 비중 순)

1. **특수교육학 개론 (~24%)**: 각 장애 유형별 특성(정의, 원인, 특징), 증후군(다운, 프래더-윌리, 엔젤만, 클라인펠터 등), 뇌성마비 유형, 의사소통 특성
2. **관련 법령 (~20%)**: 특수교육법 주요 조항(차별금지, 배치, IEP, 전환교육, 건강장애 지원), 시행령/시행규칙, 최신 개정 내용
3. **교육과정 (~13%)**: 현행 교육과정 총론(편성/운영 기준), 기본 교육과정 각론, 자유학기제, 교수법(직접교수, DTT, PRT, EMT 등)
4. **진단 및 평가 (~12%)**: 표준화 검사(K-WISC, SIS, CBM 등), 관찰기록법, 일치도 계산, 타당도/신뢰도
5. **전환교육 (~12%)**: 지원고용 유형, 개인중심계획, 직업평가, 자기결정, 지역사회중심교수
6. **행동지원 (~11%)**: 차별강화(DRL, DRO, DRI), 기능적 행동평가, 긍정적 행동지원, 강화계획, 행동연쇄법
7. **통합교육 (~8%)**: 협력교수 유형, 교수적 수정, UDL 원리, 또래지원

### 고효율 학습 포인트

- **점자 규정**: 거의 매년 출제. 한글 점자 규정, 약자, 수표, 점자정보단말기 활용
- **보행 훈련**: 시각장애 보행 기술(지팡이 보행, 안내법, 방향정위) 매년 출제
- **청력 검사**: 청력도 해석, 6분법/4분법, 어음청취역치, 어음명료도 매년 출제
- **뇌성마비 유형**: 경직형/무정위운동형/실조형, GMFCS, 원시반사 매년 출제
- **자폐성장애**: 반향어, 공동관심, 사회상황이야기, TEACCH, DTT vs PRT 빈출
