## 2. 지문 시작 패턴

| 패턴 | 빈도 | 설명 |
|------|------|------|
| `(가)는 ...이고, (나)는 ...이다` | **178건 (54%)** | 두 자료를 병치하여 비교/적용 요구 |
| `다음은 ...이다` | **145건 (44%)** | 단일 자료 제시 후 분석 요구 |
| 기타 | 4건 | - |

**핵심**: KICE 문항의 절반 이상이 **(가)/(나) 병치 구조**. 우리 사고형도 2개 자료 병치를 고려할 것.

## 3. 대화 구조

- 대화 문항: 46건 (14.1%)
- **기입형에 편중** (34.4%) — 서술형은 6.4%만 대화 포함
- 대화 턴: 중앙값 5턴, 최대 10턴
- 대화 텍스트: 평균 89자/턴, 중앙값 79자/턴
- 주요 화자: 특수 교사(58), 특수 교사 A/B(31), 신규 교사(12), 보호자(8)

**우리 서비스 적용**:
- 확인형(fill_in)에 짧은 대화 2~3턴 활용 → 효과적
- 사고형(descriptive)에는 대화보다 시나리오 서술이 KICE 원본에 가까움
- 대화 사용 시 1턴 평균 **80자 이내**로 유지

## 4. sub_items 구조

- 274건 (83.8%)이 sub_items 보유
- 레이블: (가) 174, (나) 174, (다) 32 — 대부분 **(가)+(나) 2개 세트**
- content 유형: 문자열 145, 배열 95, 객체 102 (다양한 형태)

**핵심**: 거의 모든 KICE 문항이 (가)+(나) 구조를 가짐. 우리 사고형도 최소 1개의 시나리오 자료를 case_context로 제공해야 자연스러움.

## 5. 복합 영역 문항 (47.4%)

상위 조합:

| 조합 | 빈도 | 의미 |
|------|------|------|
| behavior-support + introduction | 17 | 장애 특성 → 행동 중재 |
| curriculum + inclusive-education | 16 | 교육과정 수정 → 통합교육 |
| assessment + curriculum | 14 | 진단평가 → 교육과정 적용 |
| curriculum + transition | 14 | 교육과정 → 전환교육 연계 |
| curriculum + introduction | 8 | 장애 특성 → 교수적 수정 |

**핵심**: KICE의 절반이 2개 이상 과목을 횡단. 우리 사고형도 **단일 과목 문제에서 벗어나 복합 시나리오**를 만들어야 기출에 가까워짐.

## 7. 사고형 문제 제작 가이드라인 (분석 기반)

### 구조 템플릿

```
[case_context: 80~120자]
  ↓
[(가)/(나) 자료 제시 — 선택사항]
  ↓
[질문 + 1~2개 하위 과제]
```

### 체크리스트

- [ ] context 80~120자 (KICE 평균 105자 기준)
- [ ] 대화 사용 시 1턴 80자 이내, 최대 3턴
- [ ] 2개 이상 과목 횡단 (KICE 47.4% 비율 반영)
- [ ] 과제 1~2개 (KICE 평균 3개를 모바일용으로 압축)
- [ ] 최근 3년 기출 키워드 우선 반영
