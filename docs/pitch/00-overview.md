## 서비스가 해결하는 문제

특수교육 임용고시는 11개 세부 과목을 동시에 준비해야 하는 극고난도 시험이다.
평균 합격 소요 기간 3년, 경쟁률 최대 23:1 (서울 유치원특수, 2025).

| 수험생 페인포인트 | 우리 솔루션 |
|----------------|------------|
| 방대한 암기량 (11과목) | Leitner SRS — 시스템이 복습 스케줄을 자동 생성 |
| 기출 패턴 분석 툴 없음 | KICE 출제 경향 히트맵 + 2027 예측 문항 |
| 사례 기반 학습 불가 | BDS 시뮬레이터 (3종 시나리오, 분기형) |
| 공부 지속 동기 부족 | XP·스트릭·마스터리 트리 게임화 |
| 서술형 채점 기준 불명확 | 키워드 매칭 + 법령 인용 자동 채점 가이드 |
| 혼자 공부하는 고독감 | 커뮤니티 + 사용자 출제 시스템 |
