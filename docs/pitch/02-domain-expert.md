# 특수교육 전문가(교사·교수) 협력 제안서

> 특수교육학 전공 교수, 현직 특수교사, 특수교육 연구자를 위한 문서.
> **목적**: 콘텐츠 검수 협력, 공동 연구, 자문 제안.

---

## 서비스 철학

이 플랫폼은 "시험 문제를 많이 푸는 것"이 아니라
**"학습 과학 기반으로 특수교육학을 체화하는 것"** 을 목표로 합니다.

- Leitner 간격 반복 (Spaced Repetition System): 망각 곡선에 맞춰 복습 스케줄 자동화
- 정교화 질문 (Elaboration Interrogation): 정답 후 "왜?"를 묻는 메타인지 강화
- 행동 의사결정 시뮬레이터: 사례 기반 학습 (Case-Based Learning)
- 마스터리 학습 (Mastery Learning): 과목별 숙련도 추적, 합격 시뮬레이션

- FBA 기반 자해 행동 사례
- 통합교육 환경 문제행동 사례
- PBS 기반 공격 행동 중재 사례
- 각 시나리오: 20+ 분기 노드, 법령 근거 제시

|------|-----------|

## 협력이 필요한 영역

### 1. 콘텐츠 검수 (가장 시급)

현재 AI가 생성한 2,061개 문항의 정확성 검증이 필요합니다.

**검수 방법**:
- 과목별 10~20% 샘플링 검토
- 오류 발견 시 수정 요청 → 에이전트가 즉시 반영
- 검수 완료 문항에 "expert-reviewed" 배지 표시 예정

**보상**: 공동 개발자 크레딧, 서비스 라이프타임 프리미엄 이용

### 2. 시나리오 콘텐츠 기여

BDS 시나리오는 현재 3개 (확장 목표 20개+).
새 시나리오 제안 또는 기존 시나리오 의학적·교육학적 검증.

**형식**:
```json
{
  "id": "scenario_04",
  "title": "중증 지적장애 학생 자해 행동 감소",
  "context": "...",
  "nodes": [
    { "id": "n1", "text": "...", "choices": [...] }
  ]
}
```

### 3. 학습 효과 공동 연구

- 연구 설계: 플랫폼 사용자 vs 비사용자 임용 성적 비교
- 데이터 제공: 익명화된 학습 패턴 데이터 (동의 기반)
- 논문 공동 저자 가능

### 4. 교원 연수 콘텐츠 확장 (장기)

현직 특수교사를 위한 연수 콘텐츠 개발.
경기교원연수원, 티처빌 협력 예정.

## 제안하는 협력 형태

**A안 — 자문 계약**: 월 2~4시간 자문, 콘텐츠 검수 + 방향 조언
**B안 — 공동 연구**: NISE 또는 대학 연구비 활용, 논문 공동 게재
**C안 — 커뮤니티 기여**: 자유로운 오류 제보 (이슈 시스템)
