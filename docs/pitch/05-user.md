# 수험생 가이드 — 특수교육 임용 합격을 위한 학습법

> 임용고시를 준비하는 수험생을 위한 서비스 소개 + 최대 활용 전략.

---

## 한 줄 소개

**특수교육 임용 전용** 무료 학습 앱.
11개 과목을 스마트하게, 오래, 지치지 않게.

서비스: https://special-education-web.vercel.app

---

## 기존 공부법과 뭐가 다른가

### 기존 학원·인강의 한계

| 문제 | 현실 |
|------|------|
| "들었는데 기억이 안 남" | 수동 청취 → 장기 기억 미전환 |
| "11과목을 어떻게 다 외워" | 암기 우선순위 없이 무작정 반복 |
| "기출 보면 유형은 알겠는데 응용이 안됨" | 패턴 분석 없이 단순 암기 |
| "혼자 공부하면 지침" | 동기 유지 시스템 없음 |

### 우리 앱이 다른 이유

**"시스템이 복습을 대신한다"**

- 오늘 틀린 문제는 내일 다시 나옴
- 2주 전에 맞혔던 문제는 한 달 후에 다시 나옴
- 네가 개입하지 않아도 복습 스케줄이 자동으로 만들어짐

이것이 말해보카가 영어 단어 앱 MAU 2위가 된 원리. 우리는 이걸 특수교육 11과목에 적용했다.

## 핵심 기능 5가지

### 1. 과목별 퀴즈 (2,061문항)
- OX형 / 객관식 / 단답형 / 서술형 4가지 유형
- 11과목 39챕터 전체 커버
- 정답 후 상세 해설 + 관련 법령 근거

**활용법**: 챕터 하나 끝낼 때마다 해당 챕터 퀴즈 10분씩.

### 2. Leitner 플래시카드 (스페이스드 리피티션)
- 틀린 카드: 내일 복습 (박스 1)
- 맞힌 카드: 2일 후, 4일 후, 8일 후, 16일 후
- 오답 → 1클릭으로 Leitner 카드 저장

**활용법**: 매일 아침 15분. "오늘의 복습" 카드만 풀면 됨.

### 3. KICE 기출 모의고사
- 2020~2026 전공A+B 기출 520문제
- 실전 타이머 (100분) + 플래그 마크
- 출제 경향 히트맵: "올해 뭐가 나올까?"

**활용법**: 주 1회 모의고사 → 취약 챕터 확인 → 해당 챕터 퀴즈 집중.

### 4. 행동 의사결정 시뮬레이터 (BDS)
- 자해 행동 FBA, 통합교육 문제행동, PBS 공격행동 3가지
- 분기형 선택 → "이 상황에서 어떻게 해?" → 즉시 피드백
- 2차 시험 (수업 실연) 준비에 직접 도움

**활용법**: 1주일에 1~2시나리오. 선택 이유를 말로 설명해보기.

### 5. 마스터리 트리 + 합격 시뮬레이션
- 과목별 숙련도 (미시작/학습중/연습중/숙련/마스터)
- 현재 공부량 기준 합격 확률 계산
- "지금 어느 과목이 발목을 잡고 있나" 한 눈에 확인

**활용법**: 주 1회 마스터리 트리 확인 → 빨간 과목 우선 집중.

## 추천 학습 플로우

### 매일 루틴 (30~60분)

```
아침 15분: Leitner 복습 카드 (시스템이 골라준 것만)
낮 20분:   새 챕터 퀴즈 + 틀린 것 Leitner 저장
저녁 10분: 오늘의 문제 (/daily) — 오늘 배운 것 확인
```

### 주간 루틴

```
월·수·금: 3개 과목씩 돌아가며 챕터 퀴즈
화·목:    오답노트 복습 + Leitner 집중
토:       KICE 모의고사 1회 (100분)
일:       마스터리 트리 확인 + 다음 주 약점 과목 결정
```

## 자주 묻는 질문

**Q. 로그인 안 해도 쓸 수 있나요?**
A. 네, 모든 기능을 로그인 없이 사용할 수 있습니다. 학습 데이터는 브라우저에 저장됩니다. 로그인하면 다른 기기에서도 이어서 사용할 수 있습니다.

**Q. 유료인가요?**
A. 현재 완전 무료입니다. 나중에 AI 맞춤 플랜 등 프리미엄 기능이 추가되더라도 기본 기능(퀴즈, SRS, 오답노트)은 영구 무료 유지 예정입니다.

**Q. 기출 문제는 어디서 가져왔나요?**
A. KICE(한국교육과정평가원) 공식 배포 기출 문제를 디지털화했습니다. 2020~2026년 전공A+B 전체 포함.

**Q. 문항에 오류가 있는 것 같은데요?**
A. 리뷰 패널에서 오류 신고 가능합니다. 확인 후 수정합니다. (AI 생성 문항 포함 — 전문가 검수 진행 중)

**Q. 모바일에서도 되나요?**
A. 네, 모바일 최적화 완료. 하단 탭 바로 주요 기능 접근 가능합니다. (오프라인 모드는 추후 지원 예정)

**Q. 2차 시험(수업 실연)도 도움이 되나요?**
A. BDS 시뮬레이터와 서술형 채점 가이드가 2차 준비에 도움이 됩니다. 완벽한 2차 대비 도구는 아니지만, 사례 기반 사고력을 기르는 데 유용합니다.
