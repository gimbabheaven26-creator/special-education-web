# 수험생 일기: "특수교육 공부방" 써본 솔직 후기

**쓴 사람**: 25살, 졸업 1년차, 임용 D-242, 독서실 10시간러
**날짜**: 2026년 3월 24일 월요일, 밤 11시
**상태**: 모의고사 채점 끝나고 자괴감 오는 중

---

## 처음 들어갔을 때 (3초)

스터디 언니가 카톡으로 링크 보내줘서 폰으로 열었어. 3초 안에 느낀 거:

**"아, 이건 나를 위한 사이트다."**

"특수교사 임용시험까지 D-242"가 딱 뜨는데, 솔직히 심장이 철렁했어. 매일 머릿속으로만 세던 숫자를 이렇게 화면에서 마주치니까. 근데 그 아래에 "오늘 학습 시작하기" 파란 버튼이 있고, "OX 10, 단답 5, 서술 3문제"라고 써있으니까 — "아 일단 이거 누르면 되는구나" 하는 방향은 잡혔어.

근데 문제는, **처음 온 나한테 바로 D-day 숫자를 들이대는 건 좀 무서워.** 나 아직 이 사이트가 뭔지도 모르는데 "시간 없어"부터 말하는 느낌? 카페 문 열고 들어갔더니 메뉴판 대신 "폐점까지 242일 남았습니다" 써있는 거랑 비슷해.

"시험일은 언제야?" "지금 몇 회독이야?" "어떤 과목이 약해?" 이런 걸 먼저 물어봐주면 좋겠는데. 코드를 보니까 온보딩 페이지가 따로 있더라고, 5단계짜리 꽤 잘 만든 것도. 근데 처음 들어왔을 때 그게 자동으로 안 뜨더라. 이건 아깝다 진짜.

---

## "와, 이건 좋다!" 감탄한 순간들

### 1. 오늘 학습의 STEP 1-2-3 구조

"오늘 학습 시작하기" 누르면 나오는 `/daily` 페이지. OX 10문제 -> 단답 5문제 -> 서술 3문제, 쉬운 것에서 어려운 것으로 자연스럽게 올라가는 구조. 이거 진짜 좋아. 왜냐면 독서실에서 아침에 "오늘 뭐부터 하지..."로 30분 날리는 나한테, 이 사이트는 "이거 해, 순서대로" 딱 정해주니까. 아침에 뇌 안 돌아갈 때 OX부터 가볍게 풀고, 점점 어려운 거 풀면서 워밍업하는 느낌이 너무 자연스러워.

그리고 OX 채점 후에 "틀린 영역 OX 다시 풀기" vs "단답형으로 넘어가기" 선택지를 주는 것. 이건 진짜 세심해. 내가 다 맞았으면 바로 다음 단계로 넘어가고, 많이 틀렸으면 한 번 더 다지고 가는 거. 강사 수업에서도 이런 분기를 못 해주는데 사이트가 해주네.

### 2. 틀린 챕터 우선 출제

코드를 보니까 문제가 그냥 랜덤이 아니라, 내가 전에 틀린 챕터에서 먼저 뽑더라. 이건 화면에 안 써있어서 모르고 지나갈 뻔했는데, 알고 나니까 "오 이 사이트가 내 약점을 기억하고 있구나" 하고 신뢰가 생겨. 근데 이걸 왜 숨겨놔? "너의 취약 챕터에서 우선 출제됩니다" 한 줄만 써줘도 완전 다른 느낌인데.

### 3. 오답노트 + AI 약점 분석

오답노트 페이지가 생각보다 잘 돼있어. 그냥 틀린 문제 리스트가 아니라, 챕터별 오답 분포를 바 차트로 보여주고, AI가 "왜 틀리는지" 패턴 분석도 해줘. 임용 학원에서 1:1 상담받으면 강사님이 해주는 게 이런 건데, 그게 공짜로 돼있어. "오답 재시험"으로 틀린 것만 골라서 다시 풀 수 있는 것도 핵심. 기출문제집은 내가 직접 표시해놓고 찾아야 하는데, 여기는 자동이야.

### 4. 기출문제 분석의 깊이

기출 분석 페이지가 연도별, 영역별, 출제경향, 문제검색 네 탭으로 나뉘어 있어. 출제경향 분석이 있다는 것 자체가 대단해. 보통 이런 건 강사들이 유료 특강에서 하는 건데. "모의고사"로 실제 시험처럼 풀어볼 수도 있고.

### 5. 스트릭과 XP

"3일 연속 학습 중!" 이런 거 나오면 솔직히 은근 뿌듯해. 듀오링고에서 느꼈던 "스트릭 안 끊으려고 일단 한 판"하는 그 심리. 임용 공부는 매일 해야 하는 건데, 이런 작은 동기부여가 독서실에서 폰 잠깐 켜서 문제 5개라도 푸는 습관 만들어줄 수 있을 것 같아.

## "이게 뭐지?" 혼란스러웠던 순간들

### 1. "진단평가" 탭

하단 네비게이션에 "진단평가"라고 써있는데, 순간 "이게 학생 대상 진단평가를 공부하는 곳인가?" 했어. 우리가 배우는 진단평가랑 내가 문제를 푸는 건 완전 다른 건데. 누르고 보니까 OX 퀴즈, 단답형, 용어학습이야. 그냥 "문제풀기"라고 하면 바로 이해할 수 있는 걸 왜 진단평가라고 한 건지 모르겠어. 특수교육 전공자인 나도 헷갈리는데 비전공자가 보면 더 헷갈릴 거야.

### 2. 기출문제가 어디 있는지 모르겠어

기출문제를 찾으려고 탭을 하나하나 눌러봤어. "진단평가"에도 없고, "실력쌓기"에도 바로 안 보이고... "실력쌓기" -> "문제풀기" -> "모의고사"를 눌러야 기출로 갈 수 있더라. 3번이나 눌러야 하는 건 너무 깊어. 기출문제는 수험생한테 밥이야. 밥을 찬장 맨 윗칸에 넣어두면 안 돼.

### 3. "SRS 복습"이 뭔데

오답노트 안에 "전체 오답" 탭이랑 "SRS 복습" 탭이 있는데, SRS가 뭔지 아는 수험생이 몇 명이나 될까. Spaced Repetition System이라는 건 알겠는데, 그냥 "자동 복습 스케줄"이나 "간격 반복"이라고 써주면 안 돼? 플래시카드에서도 "박스 1 -> 박스 5"라는 말이 나오는데, 차라리 "잘 모름 -> 알겠어 -> 확실히 안다" 이렇게 한국어로 써주면 바로 이해할 텐데.

### 4. 서술형 채점이 애매해

서술형 문제에서 내가 답을 쓰고 "정답 키워드 확인"을 누르면, 내 답이랑 모범답안이 나란히 나와. 근데... 내가 맞은 건지 틀린 건지 판단을 내가 해야 해. 셀프 채점이라는 건 알겠는데, "이건 셀프 채점입니다. 키워드를 비교해보세요"라는 안내가 작은 글씨로 있을 뿐이야. 임용 서술형이 원래 셀프 채점이 어려운 건데, 최소한 "핵심 키워드 3개 중 몇 개 포함?" 같은 체크리스트라도 있으면 좋겠어.

### 5. BDS 시나리오?

"함께하기" 탭 안에 "BDS 시나리오"가 있는데, 누르면 "행동 의사결정 시뮬레이터"라고 나와. BDS가 Behavioral Decision Simulator인 것 같은데, 약어만 써놓으면 "뭔가 어려운 거구나" 하고 안 눌러. 근데 막상 들어가보면 교실 상황 시나리오로 의사결정 연습하는 거라 꽤 괜찮은 기능이야. 이름 때문에 손해보는 케이스.

## "이거 없으면 안 쓸 것 같아" — 핵심 기능

1. **오늘 학습 (Daily)**: 매일 아침 "이거 풀어" 해주는 것. 이게 이 사이트의 심장이야.
2. **오답노트 + 오답 재시험**: 틀린 거 모아서 다시 풀게 해주는 것. 기출문제집에 형광펜 치는 것보다 100배 편해.
3. **기출 분석 + 모의고사**: 기출을 연도별/영역별로 분석해주고, 실전처럼 풀어볼 수 있는 것.
4. **스트릭/연속 학습**: 매일 안 쓰면 스트릭 끊기니까 "일단 켜서 5문제라도" 하게 만드는 것.

이 4개가 없으면 그냥 강사 PDF 모아둔 사이트랑 다를 게 없어.

---

## "이건 지금 내게 필요 없어" — 솔직하게

- **커뮤니티 (수험생 제출 문제)**: 시간이 없어. 다른 사람 문제까지 풀 여유가 없어. 기출이랑 기본 문제도 안 끝났는데.
- **인터랙티브 학습 (매칭, 빈칸)**: 재밌긴 한데, 3개밖에 없어. 콘텐츠가 더 쌓이면 쓸 것 같은데 지금은 아니야.
- **배지 시스템**: 솔직히 임용 준비하면서 배지 모으는 건... 내 상황에서는 사치야. 합격이 최고의 배지지.
- **리뷰 페이지**: 관리자 전용이라 나는 볼 수도 없고, 있는지도 몰랐어.

오해는 하지 마. 이 기능들이 나쁜 게 아니야. 지금 D-242인 내 상황에서 우선순위가 아닐 뿐이야.

## "이것만 있으면 매일 올 텐데" — 바라는 것들

### 1. 기출 키워드가 문제랑 연결됐으면

기출 분석에서 "2026년 전공A 3번 문제의 키워드: 긍정적 행동지원, 기능적 행동평가"라고 알려주는데, 그 키워드가 퀴즈에서도 태그로 보이고, 내가 그 키워드 관련 문제를 얼마나 맞혔는지 추적해주면 진짜 쓸 것 같아. "이 키워드는 최근 5년간 매년 출제, 네 정답률 40%" 이런 거.

### 2. 합격 시뮬레이션에 "다음 행동" 알려줬으면

"합격 가능성 12%"가 빨간색으로 크게 뜨면... 솔직히 앱 닫고 싶어. 근데 "아직 학습 안 한 과목이 7개라서 낮은 거예요. 이번 주 행동지원 챕터 3~5만 끝내면 22%까지 올라가요"라고 써있으면? 그러면 나 이번 주에 행동지원부터 하지. 숫자만 보여주면 절망, 숫자 + 대안이면 동기부여야.

### 3. 타이머/뽀모도로

독서실에서 이 사이트로 공부할 때, "25분 집중 + 5분 휴식" 타이머가 내장돼 있으면 폰을 다른 앱으로 안 넘길 것 같아. 지금은 문제 풀다가 카톡 알림 오면 바로 나가거든. 집중 모드 같은 게 있으면 좋겠어.

### 4. 주간 리포트

일요일 밤에 "이번 주 성과: 문제 127개 풀었고, 정답률 5% 올랐고, 다음 주는 시각장애 집중 추천"이 푸시로 오면 다음 주도 열심히 하고 싶어질 텐데. 통계 페이지가 있긴 한데 내가 찾아 들어가야 하고, 데이터가 너무 많아서 뭘 봐야 할지 모르겠거든.

### 5. 온보딩이 자동으로 떴으면

처음 온 사람한테 "시험 언제야?", "지금 몇 회독이야?", "약한 과목 뭐야?" 물어보고 맞춤 계획 세워주는 거. 이미 만들어져 있더라, 5단계짜리로. 근데 내가 직접 /onboarding을 찾아가야 돼. 이게 첫 화면에서 자동으로 뜨면 "이 사이트 나를 위한 거구나" 느낌이 확 달라질 텐데.

## 친구한테 추천할까?

**추천한다. 근데 조건부로.**

스터디 단톡방에 링크 던지면서 이렇게 말할 것 같아:

> "야 이 사이트 써봐. 매일 문제 알아서 뽑아주고, 틀린 거 자동으로 모아서 다시 풀게 해줘. 기출 분석도 있어. 근데 처음에 좀 헤맬 수 있는데, /onboarding 먼저 가서 설정하고, 기출은 실력쌓기 > 문제풀기 > 모의고사에 있어. 그리고 합격 시뮬레이션에서 숫자 낮게 나와도 겁먹지 마, 아직 문제를 많이 안 풀어서 그런 거야."

이 설명이 필요하다는 게 문제이긴 해. 좋은 앱은 설명서가 필요 없는 법이니까.

근데 핵심은, **이 사이트가 제공하는 "매일 약점 기반 문제 출제 + 오답 자동 관리 + 기출 분석"은 현재 시중에 무료로 이 정도 퀄리티인 곳이 없어.** 유료 인강 사이트들도 이 정도 적응형 출제를 안 해. 강사 모의고사 1회에 3만원인데, 여기서 매일 공짜로 맞춤형 문제를 뽑아주면 안 쓸 이유가 없지.

## 마지막으로

개발자님, 이 사이트 만든 사람이 특수교육을 공부해본 적이 있는지 모르겠지만, 기능 하나하나에서 "수험생이 뭘 필요로 하는지" 고민한 흔적이 보여. OX -> 단답 -> 서술 순서, 틀린 챕터 우선 출제, 오답 자동 수집, 간격 반복. 이건 공부를 실제로 해본 사람이 아니면 설계 못 해.

근데 **동선이 좀 불친절해.** 좋은 식당인데 간판이 안 보이고, 메뉴판이 영어로 돼있고, 맛있는 메뉴가 뒤쪽 페이지에 숨어있는 느낌? 음식 자체는 맛있어. 그냥 앞에 "오늘의 추천 메뉴"를 크게 써놓고, 좌석 안내를 해주면 돼.

8개월 남았고, 이 사이트 매일 쓸 생각이야. 오답노트 자동 수집이 진짜 편해서, 이것만으로도 기출문제집에 형광펜 치던 시간을 아낄 수 있거든. 부탁이 있다면:

1. 온보딩 좀 자동으로 띄워줘
2. 기출문제 좀 찾기 쉽게 해줘
3. 합격 시뮬레이션 숫자가 낮을 때 "괜찮아, 이렇게 하면 돼"라고 말해줘
4. "진단평가"를 "문제풀기"로 바꿔줘

이 네 개만 바뀌어도 스터디 전원한테 강추할 수 있어.

고맙다. 화이팅.
