# 3주 스프린트 계획: 기술 구조 완성

> 기간: 2026-03-25 (화) ~ 2026-04-14 (화)
> 목표: 5월 전 모든 기술적 구조 완성, 이후 데이터 검증에만 집중
> 일일 작업량: 카이란 3~4시간 + 에이전트 자동 작업
> 작성: X

---

## 원칙

1. 카이란이 할 일과 에이전트가 할 일을 명확히 분리한다. 카이란은 의사결정과 도메인 검증만. 코드는 에이전트가 쓴다.
2. 매일 저녁 작업을 시키고 자면, 아침에 결과물이 준비되어 있다. 에이전트 야간 자동화를 최대 활용한다.
3. 하루 3~4시간 중 1시간은 검토, 2~3시간은 의사결정과 데이터 작업.
4. 병렬 가능한 작업은 반드시 병렬로 돌린다.
