## 1. AAC의 개념

**보완대체 의사소통(AAC)**: 말하기·쓰기 장애로 원활한 의사소통이 어려운 사람을 일시적 또는 영구적으로 돕는 방법

| 용어 | 정의 |
|------|------|
| **보완(Augmentative)** | 기존 의사소통 방법을 증가시키거나 첨가 |
| **대체(Alternative)** | 기존 의사소통 방법을 다른 방법으로 대신 |

**AAC 체계 4가지 필수 구성 요소**: 상징 / 도구 / 기법 / 전략

---

## 2. AAC 지도 원칙 (6가지)

> ⚠️ **기출 포인트**: 6가지 원칙 명칭과 내용 구별.

| 원칙 | 핵심 내용 |
|------|----------|
| **최대화** | 의사소통 빈도·양을 최대한 증가 |
| **기능화** | 화용론 중심; 사회적 결과에 초점 |
| **개별화** | 세밀한 평가 → 개별 아동·환경에 맞는 중재 결정 |
| **상호관계성** | 사회적 맥락 안에서 지도; 표현+반응 모두 필요 |
| **정상화** | 일반 아동과 유사한 방법 선택; 학생 선호도 고려 |
| **다중 의사소통 양식** | 한 가지 방법 고집 금지; 보조 도구+비도구 방법 병행 |

---

## 3. AAC 사용자의 4가지 의사소통 능력

| 능력 | 정의 |
|------|------|
| **언어 능력** | AAC 상징 및 대화 상대 언어에 대한 지식 |
| **조작 능력** | AAC 체계를 정확·효율적으로 조작하는 기계적 기술 |
| **사회적 능력** | 상호작용 시작·유지·진전·종료하는 사회적 기술 |
| **전략적 능력** | 제약 상황에서 유연하게 대처하는 보완 전략 |

## 4. 상징 체계

### 4-1. 도상성 순서

> ⚠️ **기출 포인트**: 도상성 높음→낮음 순서 암기.

**높음 → 낮음**: 실제 사물 → 컬러 사진 → 흑백 사진 → 리버스 상징 → 다이나심 → 픽토그램 상징 → 블리스 상징 → 단어

**음성 산출 도구 지도 순서**: 실제 사물 → 실물 축소 모형 → 컬러 사진 → 흑백 사진 → 선화 상징

### 4-2. 비도구적 상징

**정의**: 어떤 외부 기기도 사용하지 않는 상징 (제스처, 눈 응시, 수화, 지화 등)

| 장점 | 단점 |
|------|------|
| 도구 불필요; 전달 속도 빠름; 눈 접촉 기회 | 그 체계를 아는 사람과만 소통 가능; 신체 능력·인지 능력 요구 |

### 4-3. 도구적 상징 비교

> ⚠️ **기출 포인트**: PCS가 가장 널리 사용; 블리스가 도상성 가장 낮음.

| 상징 | 특징 | 도상성 |
|------|------|--------|
| **PCS** | 7,000개 이상 흑백/컬러 선화; 가장 널리 사용; 블리스보다 도상성 높음 | 높음 |
| **리버스** | 낱말/문장 선화; 음성 언어 통사 규칙 따름 → 복잡한 문장 표현 가능 | 중상 |
| **다이나심** | 5,000개 이상; 개념 조합 가능 (예: 침대+생각=꿈) | 중 |
| **픽토그램** | picture+telegram 합성어; 교육 없이도 직감적 이해 가능 | 중하 |
| **블리스** | 완전한 언어 체계; 시제·복수 표현 가능; **도상성 가장 낮음** | 낮음 |

**블리스 상징 장점**: 의미 기반 상징 조합 가능; 감정·전치사·접속사 표현 가능; 시제·복수 가능

**블리스 상징 단점**: 가장 도상성 낮음(자의성 높음); 배우기·기억하기 어려움; 높은 인지 능력 요구

### 4-4. 민스피크(Minspeak)

- **개념**: 아이콘 조합 코딩 시스템 (예: '사과' + '활동하는 사람' = '먹다')
- 단일 단어도 표현 가능; 아이콘 조합으로 구·문장 구성

| 장점 | 단점 |
|------|------|
| 상징 아래 단어 표기 → 익숙하지 않은 사람도 이해 | 의사 전달 속도 느림 |
| 이동 없이 판에서 지적으로 선택 가능 | 상대방과의 눈 접촉 기회 적음 |
| 배열 위치 기억 후 재생 불필요 | 휴대 불편 |
