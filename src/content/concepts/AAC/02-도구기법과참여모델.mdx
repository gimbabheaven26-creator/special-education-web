## 1. 도구(Aids)

### 1-1. 비도구형 vs 도구형

| 유형 | 방법 | 장점 | 단점 |
|------|------|------|------|
| **비도구형** | 손짓·몸짓·눈 응시 | 휴대 부담 없음 | 의사소통 어려움 |
| **도구형** | 저급·고급 기기 사용 | 상호 이해 가능성 높음 | 휴대 번거로움 |

### 1-2. 저급 테크놀로지

**종류**: 의사소통판 (단면·다면·의사소통 책), 그림판, 글자판, 블리스 상징판

| 장점 | 단점 |
|------|------|
| 저렴; 제작·수정 수월 | 어휘 저장 제한적 |
| 다른 도구와 병행 사용 쉬움 | 현장성 떨어짐; 의존도 높아짐 |

**의사소통판 제작 시 고려 사항**: 상징 형태/크기/구성 (왼쪽→오른쪽 배열), 사용자 능력·요구

### 1-3. 고급 테크놀로지 — 음성 출력 의사소통 기기

| 유형 | 특징 | 대상 |
|------|------|------|
| **원버튼식** | 버튼당 1메시지; 녹음·재생; '전체 메시지' 형식 | 인지·언어 장애, 실어증 환자 |
| **다버튼식** | 앨범식 다중 음성 카드; '폐쇄적' 기기 | — |
| **음성합성식** | 사용자가 직접 내용 구성; '개방적' 기기 | 스스로 내용 구성 가능한 인지·언어 능력자 |

## 2. 기법(Techniques)

### 2-1. 직접 선택

**정의**: AAC 이용자가 원하는 항목을 직접 가리키는 것

| 장점 | 단점 |
|------|------|
| 표현력 향상; 속도 빠름; 직관적 | 피로감 큼; 실패 시 스트레스·자신감 상실 |

**고려 요소**: 항목 수, 항목 크기, 항목 간격, 의사소통판 배치

### 2-2. 간접 선택 — 훑기(스캐닝)

**정의**: 손으로 직접 선택 불가 시 스위치로 선택하는 간접 방법

| 장점 | 단점 |
|------|------|
| 미세한 근육 활동만으로 조작 | 정보 입력 제한; 많은 시간 소요 |

#### 훑기 방법 3종

> ⚠️ **기출 포인트**: 3가지 훑기 방법 특징과 대상 구별.

| 방법 | 특징 | 적합 대상 |
|------|------|----------|
| **자동 훑기** | 포인터 자동 이동; 스위치 활성화로 멈춤 | 스위치 정확 활성화 가능하나 유지·멈춤 어려운 사람 |
| **수동(단계적) 훑기** | 스위치 활성화마다 한 항목씩 이동; 잦은 스위치 조작 | 운동조절·인지 제한 심한 사람; 초보자 |
| **반전 훑기** | 스위치 활성화하면 이동; 비활성화하면 선택 | 스위치 활성화 어렵지만 유지·떼기는 가능한 사람 |

#### 훑기 형태 3종

| 형태 | 특징 |
|------|------|
| **원형 훑기** | 가장 단순; 원형 배열; 아동·초보자; 시각 부담 있음 |
| **선형 훑기** | 줄 단위 이동; 간단하고 배우기 쉬움; 항목 많으면 비효율 |
| **집단-항목 훑기** | 집단→항목 순차 선택; 선형 훑기 피로감 보완 |

## 3. 전략 — 상징 활성화 3종

> ⚠️ **기출 포인트**: 3가지 활성화 전략 차이 구별.

| 전략 | 원리 | 적합 대상 |
|------|------|----------|
| **시간 활성화** | 일정 시간 접촉 유지 시 활성화 (예: 3초 이상) | 불수의 운동·의도치 않은 움직임 있는 사람 |
| **해제 활성화** | 접촉 중에는 선택 안 됨; 접촉 떼면 선택 | 느리거나 비효율적 움직임으로 시간 활성화 불가능한 사람 |
| **평균(여과) 활성화** | 가장 오래 머문 항목 선택 | 특정 항목에 안정적 접촉 유지 어려운 최중도 장애인 |

## 4. 참여 모델 (뷰켈만과 미렌다)

**정의**: 일반 또래의 기능적 참여 욕구를 기반으로 최적합한 AAC 중재 절차를 수립하는 평가 모델

- 생활 연령 동일 일반 학생 패턴을 근거로 AAC 평가 수행
- 참여를 방해하는 두 가지 장벽: **기회 장벽** + **접근 장벽**

### 4-1. 기회 장벽 (외부에 의해 강제되는 장벽)

> ⚠️ **기출 포인트**: 기회 장벽 5가지 명칭과 예시 정확히.

| 장벽 | 정의 | 예시 | 중재 방향 |
|------|------|------|----------|
| **정책 장벽** | 법률·규정으로 인한 제한 | 비행기 이륙 중 전자기기 금지 | 법률·규범 변경 |
| **실제 장벽** | 가정·학교·직장의 관습 | 교육청 기금 도구를 학교에서만 사용 제한 | 옹호 운동 |
| **지식 장벽** | 관련자의 정보 부족 | 교사의 AAC 사용법 부족 | 직원 교육·워크숍 |
| **기술 장벽** | 지식은 있으나 실행 어려움 | 새 상징 추가·분류 실습 어려움 | 실습·훈련 |
| **태도 장벽** | 타인의 시선·사고·태도 | AAC 사용자 주변 사람들의 부정적 태도 | 시간이 지나면서 개선 |

### 4-2. 접근 장벽 (AAC 대상자 능력/체계 제한)

| 요인 | 내용 |
|------|------|
| **대상자 내부 요인** | 사용자의 능력·태도·자원 제한 (예: 터치스크린 정확도, 자세 보조 기기 필요) |
| **AAC 체계 요인** | 도구 고장, 어휘 저장 용량 부족 |
| **환경 요인** | 관찰·인터뷰·직접 유도로 의사소통 능력 평가 |

### 4-3. 사용자 평가 영역

| 영역 | 내용 |
|------|------|
| **언어 능력** | 상징 체계 선택; 어휘 이해 정도; 비공식 언어 관찰 |
| **운동 능력** | 자세·이동 능력; 직접 선택 가능 신체 부위 확인 (손/팔→머리/목→발/다리) |
| **감각 능력** | 시력·청력; 상징 크기·배치 결정을 위한 진단 |
| **인지 능력** | 사물 영속성; 부분-전체 이해; 범주화; 사물-상징 대응 |
