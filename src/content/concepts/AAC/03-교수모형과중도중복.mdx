## 1. AAC 교수 모형 — 한경근 5단계

> ⚠️ **기출 포인트**: 단계 순서와 지도 효과 4개 지표 명칭.

| 단계 | 내용 |
|------|------|
| **1단계: 평가 및 고려사항** | 전문가 팀(언어치료사·심리학자·PT·OT·재활의학·특수교사) 협력; 다분야 평가 |
| **2단계: 어휘 선정** | 동일 연령대 고빈도 어휘 + 일상생활 분석; 기능적 어휘 우선 |
| **3단계: 훈련** | 의례적·익숙한 일상생활 활동에서 자연스러운 의사소통 기회 제공 |
| **4단계: 지도 효과 평가** | 4개 지표 기반 평가 (아래 참조) |
| **5단계: 상징 및 도구 선택** | 인지·생활 연령 고려; 현재 수행 수준·미래 상징 체계 고려 |

### 지도 효과 4가지 지표

| 지표 | 주요 내용 |
|------|----------|
| **조작적 지표** | 속도(의사소통 빠르기), 정확도(이해된 발화 비율), 융통성(사용 어휘·주제 다양성) |
| **표상적 지표** | 상징 인식 수, 수용 언어 능력, 문법적 표현 능력, 읽기/철자 능력 |
| **상호작용 지표** | 의사소통 양식 수·빈도, 의사소통 기능 수·빈도, 주제 개시·반응 빈도, 문제 개선 방법 |
| **심리사회적 지표** | 사용자 만족도, 대화 상대자 만족도, AAC에 대한 적응도 |

## 2. AAC 교수 모형 — 박은혜 8단계

| 단계 | 내용 |
|------|------|
| **1단계: 현재 수준 측정** | 일상 관찰; 대화 상대자·환경·활동 종류·시간 등 변인 고려 |
| **2단계: 지도 목표 수립** | 실생활 기능성 전제; 또래 상호작용·자기 요구·감정 표현 목표 |
| **3단계: 어휘 수집** | 환경 분석, 기존 어휘 목록, 일화 기록, 정보 제공자 브레인스토밍 |
| **4단계: 어휘 선정** | 발달적 관점(인지 수준·연령·흥미), 사회적 관점(문화·성별), 의사소통 맥락(환경·대화 상대) |
| **5단계: 상징 배열 및 구성** | 도구/비도구 체계 결정; 범주별 색깔 구분; 언어 발달에 따른 전략 수정 |
| **6단계: 지도 방법** | 환경 구조화, 메시지 확인, 기다리기(시간 지연), 모델링, 지적하기 촉진 |
| **7단계: 일반화와 유지 점검** | 학교·가정·지역사회 다양한 환경; 다른 사람과의 의사소통 관찰 |
| **8단계: 성과 측정** | 기술 습득 정도, 만족도, 사회성·또래 관계 영향, 기기 효용성 평가 |

### 어휘 목록 구성 3전략

> ⚠️ **기출 포인트**: 피츠제럴드 키 순서, 환경/활동 중심 방법 특징.

| 전략 | 특징 |
|------|------|
| **문법적 범주 방법** | 구어 어순(문법 기능)에 따라 배열 |
| → 피츠제럴드 키 | 좌→우: **사람·행위·수식어·명사·부사** 순서; 판 상하단에 자주 사용되는 어구 배열 |
| **의미론적 범주 방법** | 사람/장소/활동 등 의미론적 범주로 상징 배열 |
| **환경/활동 중심 방법** | 특정 환경·활동에 맞는 어휘로 의사소통판 구성; 초기 의사소통 방법으로 용이; 언어 발달 촉진 가능 |

## 3. 중도·중복 장애 학생 AAC 지도

### 3-1. 의사소통 특성

- 신체 움직임 제한 → 자발적 표현 어려움
- 수동적 상호작용 → 요구 능력 부족
- **문제 행동**으로 의사소통 대체 (자기 자극, 부적절 행동)
- 말하는 사람에게 주의 집중 어려움

### 3-2. 삼각 초점 중재

**정의**: 비상징적 의사소통 학생을 교수하기 위해 교사가 학습자·상대자·환경 3요소를 통합적으로 관리하는 전략

| 초점 | 내용 |
|------|------|
| **학습자 중심** | 문제 행동을 의사소통으로 해석; 각성·반응 촉진; 다양한 선택 형식; 일과 만들기 |
| **대화 상대자 중심** | 비상징 신호에 민감 반응; 의사소통 기회 확대; 학생보다 말 적게 하기; 긍정적 피드백 |
| **환경적 맥락** | 반복적 일과 구조화; 선택권 제공; 의사소통 기회 증가; 감각 질 수정 |

### 3-3. 의사소통 사전

**정의**: 비상징 의사소통 아동의 행동 관찰·면담·샘플 수집을 통해 만드는 개별 의사소통 목록

**목적**: 친숙하거나 낯선 상대방이 아동의 특정 행동이 나타내는 의사소통 의도를 일관성 있게 인식하도록 돕기

**3열 구조**:

| 어떻게 하는가?(형태) | 어떤 의미인가?(기능) | 어떻게 반응해야 하는가?(결과) |
|-------------------|-------------------|---------------------------|
| 머리를 돌려버림 | 이것을 하고 싶지 않다 | 하던 활동을 멈추고 "싫어요"라고 표현한 것에 반응 |
| 발성 소리를 크게 함 | 나는 이게 좋아요 | 활동을 지속함 |
| 눈 응시 | 이것을 원함 | 학생이 눈으로 본 것을 선택 행동으로 간주 |

### 3-4. 기초적 의사소통

비상징 시기의 중도·중복 장애 학생을 위한 의사소통 방법

| 방법 | 내용 |
|------|------|
| **호흡 리듬** | 파트너와 함께 숨을 내쉬면서 관계 맺음; 특정 음을 소리 내어 청각적 확인 제공 |
| **촉각적 접촉** | 신체 접촉이 의사소통 도구; 아동 신체를 위→아래, 안→밖으로 쓰다듬기 |

### 3-5. AAC 대화 상대자 훈련 목표

1. 현재 의사소통 수준·전략 인식
2. 언어·의사소통 발달 정보 제공
3. 반응적·성공적 의사소통 양식 개발 지원
4. 일과 수정 → 의사소통 능력 발달 촉진 활동 개발
5. 의사소통 발달 원리 이해 → 촉매자 역할 실행 지원
