## 절차의 흐름

병원에서 진료 → 검사 → 진단 → 처방 순서가 있듯, 특수교육 대상자 선정도 **신청 → 진단·평가 → 선정 → 배치**의 법적 절차를 따라야 한다. 절차를 생략하거나 임의로 배치하면 학생의 권리를 침해할 수 있다.

---

## 1. 조기발견 (제14조)

| 항목 | 내용 |
|------|------|
| **의무 주체** | 국가·지자체 |
| **대상** | 영유아 및 학생 |
| **방법** | 선별검사 실시, 장애 조기 발견 체계 구축 |
| **통보 의무** | 어린이집·유치원·학교장은 특수교육 대상 가능성 있는 아동 발견 시 보호자에게 통보 의무 |

> ⚠️ **기출 포인트**: 조기발견의 주체는 "국가·지자체"이며, 어린이집·유치원·학교장에게도 **통보 의무**가 있다.

---

## 2. 선정 신청 (제15조)

| 항목 | 내용 |
|------|------|
| **신청자** | 학생 본인 또는 보호자 |
| **신청 기관** | 특수교육지원센터 또는 각급 학교의 장 |
| **직권 의뢰** | 어린이집·유치원·학교장이 보호자 동의 후 직권 의뢰 가능 |

---

## 3. 진단·평가 (제16조)

### 평가 기관

| 기관 | 역할 |
|------|------|
| **특수교육지원센터** | 진단·평가 실시 |
| **복지기관·의료기관** | 연계 의뢰 가능 |

### 평가 기간

> 신청 접수 후 **30일 이내** 진단·평가 결과 통보 의무

> ⚠️ **기출 포인트**: 30일 이내 결과 통보는 법적 의무다. 빈출 숫자이므로 반드시 기억할 것.

### 평가 내용

| 항목 | 내용 |
|------|------|
| 장애 유무 및 정도 | 표준화 검사, 임상 관찰 |
| 교육적 요구 | 교육 지원 내용, 강도 |
| 적합한 교육 환경 | 일반학급, 특수학급, 특수학교 여부 |

### 보호자 권리

- 평가 과정 참여 요청 가능
- 평가 결과 열람·사본 요청 가능
- 재평가 요청 가능 (진단 결과에 이의 시)

## 4. 선정·배치 결정 (제17조)

### 의사결정 구조

```
특수교육지원센터 (진단·평가)
        ↓ 결과 통보
특수교육운영위원회 (심의)
        ↓ 심의 결과
교육장 (최종 결정)
        ↓
보호자에게 통보
```

### 배치 원칙

| 원칙 | 내용 |
|------|------|
| **최소제한환경** | 가능한 한 일반학교·일반학급 우선 |
| **보호자 의견 존중** | 보호자 의견을 최대한 반영 |
| **거주지 인근 학교** | 거주지에서 가장 가까운 학교 우선 |

> ⚠️ **기출 포인트**: 배치는 **교육장**이 결정한다 (교육감이 아님). 교육장 = 교육지원청의 장.

### 배치 유형별 우선순위

1. **일반학교 일반학급** (특수교육 서비스 제공)
2. **일반학교 특수학급**
3. **특수학교**
4. **순회교육 (가정·시설)**

## 5. 이의신청

| 항목 | 내용 |
|------|------|
| **이의 주체** | 보호자 |
| **이의 기관** | 시·도 특수교육운영위원회 |
| **이의 기간** | 배치 결정 통보받은 날로부터 **15일 이내** |

> ⚠️ **기출 포인트**: 이의신청 기간은 **15일 이내**. 30일(진단 기간)과 15일(이의신청)을 혼동하지 말 것.

---

## 6. 주요 기간 정리

| 기간 | 내용 |
|------|------|
| **30일** | 진단·평가 결과 통보 기한 |
| **15일** | 이의신청 기한 |
| **14일** | 이의신청 처리 기한 |
| **매년 3월 31일** | IEP 작성 완료 기한 |

---

## 7. 교육장 vs 교육감 권한 구분

| 권한 | 주체 |
|------|------|
| 특수교육 대상자 **선정·배치** | **교육장** |
| 특수학교 **설립·운영** | 교육감 |
| 특수교육 정책 수립 | 교육감 (시·도 수준) |
| 특수교육지원센터 **설치** | **교육장** (단, 교육감 위임 가능) |
