## 법이 IEP를 요구하는 이유

"모든 학생에게 동일한 교육"이 아닌 "모든 학생에게 **적절한** 교육"을 보장하기 위해 법은 IEP를 의무화했다. 쿠키커터로 모든 쿠키를 같은 모양으로 만드는 것이 아니라, 각 학생 맞춤의 틀을 만들어야 한다.

---

## 1. IEP 작성 의무 (제22조)

| 항목 | 내용 |
|------|------|
| **의무 주체** | 각급 학교의 장 |
| **대상** | 특수교육 대상자 전원 |
| **작성 시기** | 매 학년도 시작 전 (3월 31일 이전) |
| **검토 주기** | 매 학기 1회 이상 |

> ⚠️ **기출 포인트**: IEP는 학생이 아닌 **학교장**이 책임자다. 3월 31일까지 작성 완료해야 한다.

---

## 2. 개별화교육지원팀 구성 (제22조)

| 구성원 | 법령 규정 |
|--------|-----------|
| **보호자** | 필수 |
| **특수교사** | 필수 |
| **일반교사** | 통합학급 운영 시 필수 |
| **진로 및 직업교육 담당 교원** | 해당 시 포함 |
| **특수교육 관련 서비스 담당 인력** | 언어치료사, 물리치료사 등 해당 시 |

> ⚠️ **기출 포인트**: 보호자는 팀의 **필수 구성원**이다. 단순 서명자가 아닌 계획 수립에 참여하는 주체.

---

## 3. IEP 필수 포함 내용 (시행규칙 별표)

| 구성요소 | 내용 |
|----------|------|
| 학생 인적사항 | 이름, 생년월일, 학교 |
| 특수교육 대상자 선정 이유 | 장애 유형, 특수교육 필요 사유 |
| **현재 학습 수행 수준** | 교과별 현재 능력 (측정값 포함) |
| **교육 목표** | 연간 목표, 단기 목표 (측정 가능하게) |
| 교육 방법 | 교수 전략, 교수·학습 자료 |
| **특수교육 관련 서비스** | 종류, 제공 기관, 빈도, 기간 |
| 적용 교육과정 | 공통·선택·기본·특별 교육과정 |
| 평가 방법 및 기준 | 목표 달성 평가 계획 |

---

## 4. 통합교육 지원 (제21조)

각급 학교의 장은 통합교육을 실시하는 경우 특수교육 대상자가 차별 없이 교육받을 수 있도록 다음을 지원해야 한다:

| 지원 내용 | 세부 |
|-----------|------|
| **교수·학습 방법 개선** | 통합학급 교사 연수 |
| **시설·설비 개선** | 무장애 환경 조성 |
| **특수교육 담당 교원 배치** | 순회교사 포함 |
| **장애인식개선 교육** | 전교생 대상 의무 실시 |

> ⚠️ **기출 포인트**: **장애인식개선교육**은 학교장이 연 1회 이상 **전교생**을 대상으로 실시할 의무가 있다.

---

## 5. 특수교육 관련 서비스 상세 (제28조)

| 서비스 유형 | 제공 대상 | 제공 주체 |
|-------------|-----------|-----------|
| **상담 지원** | 학생·보호자 | 상담교사, 특수교사 |
| **가족 지원** | 보호자·가족 | 특수교육지원센터 |
| **치료 지원** | 치료 필요 학생 | 치료사 (언어·물리·작업) |
| **보조인력 지원** | 일상생활·학습 보조 필요 학생 | 특수교육 보조원 |
| **보조공학기기** | AT 필요 학생 | 무상 대여·지원 |
| **학습보조기기** | 학습 특수기기 필요 학생 | 무상 지원 |
| **통학 지원** | 통학 곤란 학생 | 통학 차량 또는 비용 지원 |
| **정보 접근 지원** | 의사소통 지원 필요 학생 | 수어 통역, 자막, 점자 |

---

## 6. 순회교육 (제25조)

| 항목 | 내용 |
|------|------|
| **대상** | 장기 입원, 시설 거주, 거동 불편 학생 |
| **방법** | 교사가 가정·시설·병원으로 방문 |
| **수업 시간** | 주 2시간 이상 (영아: 주 1회 이상) |
| **운영 주체** | 특수교육지원센터 또는 특수학교 |

> ⚠️ **기출 포인트**: 순회교육 수업 시간 기준은 **주 2시간 이상**이다.

---

## 7. 전환교육 (제23조)

| 적용 시기 | 내용 |
|-----------|------|
| **중학교 과정 이상** | 전환교육 의무 포함 |
| **IEP에 포함** | 전환계획을 IEP에 명시 |

### 전환 영역

| 영역 | 내용 |
|------|------|
| **직업** | 직업 탐색, 직업 훈련 |
| **자립생활** | 독립생활 기술 |
| **고등교육** | 대학 진학 준비 |
| **지역사회 참여** | 여가, 사회적 참여 |
