## 제도는 권리의 울타리다

아무리 좋은 권리도 제도적 보장 없이는 빈말이 된다. 특수교육법은 학교 설치 기준, 교원 자격, 그리고 권리 침해 시 구제 절차까지 규정하여 장애 학생의 교육권을 **현실적으로 보장**한다.

---

## 1. 특수교육기관 설치 기준

### 특수학교 설치 (제5조)

| 항목 | 내용 |
|------|------|
| **설치 주체** | 국가·지자체 또는 사인(법인) |
| **설립 인가** | 교육감의 인가 |
| **부설 기관** | 유치원·초·중·고·전공과 설치 가능 |

### 특수학급 설치 (제27조)

| 학교 급 | 특수학급 학생 수 기준 |
|--------|---------------------|
| 유치원 | 4명 이상 |
| 초등학교·중학교 | 6명 이상 |
| 고등학교 | 7명 이상 |

> ⚠️ **기출 포인트**: 특수학급 학생 수 기준은 급별로 다르다. 유치원(4명), 초중(6명), 고등(7명). 이 기준 이상이면 **의무적으로** 특수학급을 설치해야 한다.

### 학급당 학생 수 상한 (시행령)

| 학교 급 | 최대 학생 수 |
|--------|------------|
| 유치원 | 4명 |
| 초등학교 | 6명 |
| 중학교 | 6명 |
| 고등학교 | 7명 |
| 전공과 | 9명 |

## 2. 특수교육교원

### 정의 및 자격 (제3조, 교원자격검정령)

| 자격 | 요건 |
|------|------|
| **특수교육교원** | 특수학교 또는 특수학급 담당 자격증 소지자 |
| **자격 종류** | 특수학교 정교사 1·2급, 준교사 |
| **배치 의무** | 특수학급 1학급당 1인 이상 |

> ⚠️ **기출 포인트**: 일반학교 특수학급에는 **특수교육 자격증 소지자**가 담당해야 한다. 일반 교원이 겸임하는 것은 법적으로 불가.

### 특수교육지원센터 인력

| 직위 | 자격 |
|------|------|
| **센터장** | 특수교육교원 자격 소지자 |
| **순회교사** | 특수교육교원 |
| **치료사** | 언어치료사, 물리치료사, 작업치료사 등 |

## 3. 특수교육지원센터 기능 (제11조)

| 기능 | 내용 |
|------|------|
| 진단·평가 | 특수교육 대상자 진단 및 적합한 교육 판별 |
| 배치 | 학교·기관 배치 지원 |
| 순회교육 | 거동 불편 학생 방문 교육 |
| 가족 지원 | 가족 상담, 정보 제공 |
| 관련 서비스 | 치료 서비스 연계 |
| 보조공학 | 기기 대여 및 교육 |

## 4. 권리 구제 절차

### 침해 유형별 구제 기관

| 침해 유형 | 구제 기관 |
|-----------|-----------|
| IEP 미작성·미이행 | 교육청(교육장·교육감) 민원 |
| 배치 결정 불복 | 시·도 특수교육운영위원회 이의신청 |
| 통합교육 거부 | 교육청 또는 국가인권위원회 |
| 장애를 이유로 한 차별 | 국가인권위원회, 법원 |

### 이의신청 절차 (배치 관련)

```
배치 결정 통보
    ↓
15일 이내 이의신청 (시·도 특수교육운영위원회)
    ↓
14일 이내 처리
    ↓
결과 통보
    ↓ (불복 시)
행정심판 / 행정소송
```

## 5. 장애인차별금지 및 권리구제법 (장차법) 연계

| 항목 | 내용 |
|------|------|
| **교육 분야 차별 금지** | 입학 거부, 수업 참여 제한, 시험 편의 거부 등 |
| **정당한 편의 제공 의무** | 보조인력, 보조공학기기, 대체자료 등 |
| **차별 행위** | 불합리한 이유 없이 다르게 대우하거나 편의를 제공하지 않는 행위 |

> ⚠️ **기출 포인트**: "정당한 편의"를 제공하지 않는 것 자체가 **차별**에 해당한다. 적극적 조치를 거부하는 것도 차별임을 기억할 것.
