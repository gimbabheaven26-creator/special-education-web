## 1. 교수적 수정의 정의 및 필요성

교수적 수정(Instructional Modification)은 장애 학생이 일반교육 환경에 의미 있게 참여할 수 있도록 교수·학습의 조건을 조정하는 체계적 절차이다.

**교수적 수정이 필요한 이유**

- 특수교육 대상자의 개별적 특성(인지·신체·감각·행동)이 다양하여 단일 교수 방법으로는 효과적 학습 불가
- 일반교육과정에 접근하되 적절한 지원 없이는 참여가 제한됨
- IEP 목표를 일반교육 맥락에서 구현하기 위한 연결 고리 역할

> ⚠️ **기출 포인트**: 교수적 수정은 **학생이 달성해야 할 기대 수준을 낮추지 않는** 범위에서의 지원 조정이다. 기대 수준 자체를 변경하면 '교육과정 수정(Curriculum Modification)'이 된다. 이 경계선이 핵심 출제 포인트다.

## 2. 교수적 수정 5유형 비교

| 유형 | 정의 | 예시 |
|------|------|------|
| **① 교재·교구 수정** | 학습 자료의 형태·난이도·제시 방식 변경 | 확대 인쇄, 점자 자료, 오디오북, 그림 단서 카드 |
| **② 교수방법 수정** | 교사의 설명 방식·질문 방식·피드백 방식 변경 | 시각적 조직자 제공, 단계 세분화, 또래 교수 활용 |
| **③ 교수환경 수정** | 물리적·사회적 환경의 배열 및 조건 변경 | 자리 배치, 소음 차단, 집단 구성(소집단/1:1) 변경 |
| **④ 시간·분량 수정** | 과제 완성 시간, 과제 분량, 학습 속도 조정 | 추가 시간 허용, 과제 분량 50% 축소, 과제 분할 |
| **⑤ 평가방법 수정** | 평가 형식·조건·방법 변경 (평가 본질 유지) | 구술 시험, 그림으로 답 제출, 대안적 수행 평가 |

## 3. INCLUDE 모델 6단계

INCLUDE 모델은 일반학급 교사가 장애 학생의 요구를 체계적으로 파악하고 교수적 수정을 결정하는 의사결정 절차이다.

| 단계 | 영문 | 내용 |
|------|------|------|
| **1단계** | **I**dentify | 일반학급 환경의 요구(과제·행동·사회적 기술) 파악 |
| **2단계** | **N**ote | 학생의 강점과 요구 확인 (IEP, 평가 결과 활용) |
| **3단계** | **C**heck | 잠재적 문제(환경 요구와 학생 특성의 불일치) 점검 |
| **4단계** | **L**ook | 학생 참여를 방해하는 문제 중 우선순위 결정 |
| **5단계** | **U**se | 교수적 수정 방안 결정 및 적용 |
| **6단계** | **D**ecide | 효과 평가 후 특별 서비스 추가 여부 결정, **E**valuate |

> ⚠️ **기출 포인트**: INCLUDE 모델의 각 단계를 영문 첫 글자와 함께 기술하는 문제가 출제된다. 6단계가 단순한 평가에서 그치지 않고 '특별 서비스 필요 여부 결정'까지 포함함을 주목할 것.

## 4. 교육과정 수정 3차원

교육과정 수정(Curriculum Modification)은 기대 수준 자체를 조정하는 것으로, 내용·과정·결과 세 차원에서 이루어진다.

### 4-1. 내용(Content) 수정

무엇을 가르칠 것인가를 조정한다.

| 유형 | 정의 | 적용 대상 |
|------|------|---------|
| **동일 수준** | 수정 없이 일반 교육과정 그대로 | 경도 장애, IEP 목표 변경 불필요 학생 |
| **중복(중첩) 교육과정** | 같은 활동에 참여하되 **다른 교과 영역**의 다른 목표 추구 | 중등도 장애, 기능적 기술 필요 학생 |
| **대체 교육과정** | 일반 교육과정과 독립된 별도 내용 | 중도·중복 장애, 기능적 생활 중심 필요 학생 |

### 4-2. 과정(Process) 수정

어떻게 가르칠 것인가를 조정한다.

- 학습 내용을 더 작은 단계로 세분화
- 다중 감각 채널 활용 (시각+청각+촉각)
- 반응 형식 다양화 (구두·몸짓·그림·보완대체의사소통)
- 또래 지원·협동학습·개별 보조 활용

### 4-3. 결과(Product/Outcome) 수정

학습을 어떻게 표현·증명할 것인가를 조정한다.

- 서면 보고서 → 구두 발표 또는 프레젠테이션
- 필기 시험 → 포트폴리오 또는 수행 평가
- 개인 과제 → 협력 과제
- 결과물의 길이·복잡성·형식 조정

## 5. 교수적 수정 vs 교육과정 수정 구분

| 항목 | **교수적 수정** | **교육과정 수정** |
|------|--------------|----------------|
| **기대 수준** | 변화 없음 (동일 목표) | 변화 있음 (목표 자체 조정) |
| **초점** | 접근 방식·지원 조건 변경 | 학습 내용·목표·결과 변경 |
| **IEP 반영** | 선택적 (필요시 명시) | **IEP에 반드시 명시** |
| **적용 대상** | 경도~중등도 장애 학생 주로 | 중등도~중도 장애 학생 주로 |
| **예시** | 확대 글자 제공, 추가 시간 부여 | 학년 수준 이하 교과 내용 적용 |

> ⚠️ **기출 포인트**: "평가 본질을 유지하면서 형식만 바꾸는 것"은 교수적 수정(평가조정)이고, "기대 성취 수준 자체를 낮추는 것"은 교육과정 수정이다. 이 구분이 단골 출제 포인트다.

## 6. 적용 사례

### 자폐성장애 학생(민준, 초3) 사례

**환경**: 일반학급 사회 수업, 지도 읽기 활동

| 교수적 수정 유형 | 적용 내용 |
|---------------|---------|
| 교재·교구 수정 | 추상적 지도 → 실제 학교 건물 사진 지도 제공 |
| 교수방법 수정 | 단계별 과제 분석 카드 제공, 또래와 함께 수행 |
| 교수환경 수정 | 조명·소음 줄인 소집단(3명) 배치 |
| 시간·분량 수정 | 전체 5개 과제 중 3개로 축소, 시간 제한 없음 |
| 평가방법 수정 | 필기 대신 그림 카드 선택 또는 지목으로 응답 |

### 지체장애 학생(소연, 중2) 사례

**환경**: 과학 실험 수업 (pH 측정 실험)

| 교수적 수정 유형 | 적용 내용 |
|---------------|---------|
| 교재·교구 수정 | 손잡이 개조 스포이드, 확대 pH 색상표 제공 |
| 교수환경 수정 | 휠체어 접근 가능 실험대 높이 조정 |
| 과정 수정 | 실험 조작 파트너와 역할 분담(소연은 관찰·기록 담당) |
| 결과 수정 | 손글씨 보고서 → 태블릿 음성 입력 보고서 허용 |
