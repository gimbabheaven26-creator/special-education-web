## 6. 안내법

안내법은 시각장애인이 정안인의 팔을 잡고 이동하는 보행 방법으로, 안전하고 효율적인 이동을 위해 안내자와 시각장애인 모두의 정확한 기술 습득과 상호 신뢰가 중요하다.

> ⚠️ **기출 포인트**: 안내법의 **주체는 안내받는 시각장애인**이다. 안내인이 주체가 아님을 혼동하지 말 것.

### 6.1. 안내법의 기본 이해

| 구분 | 내용 |
|------|------|
| 정의 | 시각장애인이 정안인의 팔을 잡고 이동하는 기초적 보행 방법 (실내외 모두 사용 가능) |
| 주체 | **안내받는 시각장애인** |

**정안인(안내인)이 지켜야 할 기본 원리**

1. 행동을 거칠게 하지 않고 항상 시각장애인 앞에 서며, 걸음 속도를 맞추고 피곤해 하지 않는지 살핀다.
2. 보행 중인 길에 주의를 기울이고, 트인 공간에 시각장애인을 홀로 머물게 하지 않는다.
3. 보행에 대한 즐거움을 느끼게 하고 주변의 방향정위 정보를 제공하며, 안내 기술을 철저히 익혀야 한다.

**안내법 장단점 비교**

| 구분 | 내용 |
|------|------|
| 장점 | 보행이 안전하고 효율적; 안내자가 환경 정보를 대신 처리하여 시각장애인이 운동 감각적 인식·방향정위·개념 기술을 발전시킬 수 있음 |
| 단점 | 안내자가 방법을 모르면 불편; 시각장애인이 의존성을 갖게 됨; 환경 정보와 방향정위에 주목하지 않게 될 수 있음 |

### 6.2. 기본 안내법 절차 및 기술

> ⚠️ **기출 포인트**: 각 상황별 역할 분담(누가 무엇을 하는가)이 반복 출제된다. 특히 **문 열기/닫기** 역할 구분과 **좁은 길·방향전환** 동작을 정확히 암기할 것.

**단계 1 — 안내 시작**

- 안내인은 먼저 안내가 필요한지 시각장애 학생에게 묻는다.
- 수락 시: 안내인의 손등을 시각장애인의 손등에 가볍게 대어 팔 위치를 자연스럽게 알린다.

**단계 2 — 기본 자세**

- 시각장애인은 안내인의 **반보 뒤, 반보 옆**에 위치한다.
- 안내인의 팔꿈치 위를 **계란 쥐듯 가볍게** 감아 잡는다.

**상황별 기술 요약**

| 상황 | 핵심 동작 |
|------|----------|
| 계단 오르내리기 | 계단 시작·종료 전 잠시 멈추어 알림; **오름 계단 먼저** 지도 후 내림 계단 지도 |
| 문 열고나가기 | 문 여는 것 → **안내인**, 문 닫는 것 → **시각장애인**; 안내인은 열린 문 손잡이를 안내 손으로 바꿔 쥠 |
| 의자에 앉기 | 안내인이 시각장애인의 손을 의자 **등받이**에 대주면, 시각장애인이 스스로 확인 후 앉음 |
| 좁은 길 통과하기 | 안내인은 팔을 **뒤로 돌려**주고, 시각장애인은 안내인의 **팔목**을 잡고 이동 |
| 좁은 공간 방향전환 | 각자 90도 돌아 서로 마주봄 → 시각장애인이 반대 손으로 잡음 → 다시 90도 회전 |
| 안내의 일시 중지 | 상호 합의 후 중지; 안내인은 장소 설명 제공; 오픈 공간보다 **사물·벽 근처 안전한 곳**에서 대기 |
| 2명 이상 안내 | 날개형(양 팔 각각 잡기) / 체인형(연결해서 잡기) |
| 승용차·택시 | 시각장애인이 먼저 탑승; 안내인이 잡고 있는 손을 들어 **차 지붕 부분**에 대어 머리 숙여 탈 수 있도록 정보 제공 |
| 개인 배식 식당 | 시각장애인이 음식을 먼저 뜬 뒤, 쟁반 든 손등이 안내인 팔에 오도록 하고 안내 마침 |

## 7. 지팡이법

지팡이법은 시각장애인이 흰 지팡이를 사용하여 안전성 확보, 환경 정보 입수, 시각장애인 상징의 세 가지 목적으로 독립적 보행을 돕는 기술이다.

> ⚠️ **기출 포인트**: 지팡이 사용의 **3가지 목적**과 각 **터치법의 명칭·적용 장소**가 자주 출제된다. 2점 터치법 응용 기법(슬라이드/터치앤슬라이드/터치앤드래그/3점)의 차이를 구분할 것.

### 7.1. 지팡이 사용 개요 및 장단점

**지팡이 사용의 3가지 목적**

1. **안전성 확보** — 장애물 접촉하여 신체 안전 확보
2. **정보 입수** — 장애물 재질, 인도/차도 구분 등
3. **시각장애인 상징** — 보행자·운전자 주의 환기 및 도움 요청 시 효과적

**지팡이 구조**: 손잡이 → 손잡이 끈 → 자루 → 팁(tip)

**장단점 비교**

| 구분 | 내용 |
|------|------|
| 장점 | 물체와 보행 표면에 대한 정보 제공; 사용 간편; 이동 기동성 뛰어남 |
| 단점 | 움직이는 공간 전체 감지 어려움; 허리 위쪽 신체 부위 보호 불가; 정보는 **접촉 순간에만** 전달됨 |

### 7.2. 지팡이 사용 기법

**기본 자세 및 조작법**

| 항목 | 내용 |
|------|------|
| 잡는 법 | 손을 몸 중심에 두고, **검지는 뻗고** 나머지 손가락으로 감아 잡음; 팔을 최대한 곧게 뻗어 안전 공간 확보 |
| 팔의 위치 | 몸에서 약 **20~30cm** 떨어뜨려 전방 아래로 뻗고, 팔꿈치는 약간 구부리거나 펼 수 있음 |
| 손목 운동 | 손을 몸 중앙에 고정하고, **손목 중심**으로만 양쪽으로 움직여 동일한 넓이의 호 생성 |
| 호의 높이 | **2~5cm** |
| 호의 넓이 | 어깨 넓이보다 **약 5~6cm 넓게** |
| 터치 | 지팡이 끝으로 지면을 탐지할 수 있을 정도로 **가볍게** 접촉 |

**발과 리듬 맞추기**

| 지팡이 끝 위치 | 발 동작 |
|--------------|--------|
| 왼쪽 지점 | 오른발이 지면 |
| 오른쪽 지점 | 왼발이 지면 |

**터치법 종류 비교**

| 기법 | 주요 적용 장소 | 특징 |
|------|-------------|------|
| **2점 터치법 (촉타법)** | 주로 실외, 실내 가능 | 발을 내디딜 곳의 지면을 지팡이로 먼저 탐색 |
| **슬라이드 기법** | 지형 변화 탐지 | 지팡이 끝을 바닥에서 떼지 않고 양쪽으로 이동 |
| **터치 앤 슬라이드 (촉타 후 밀기법)** | 실외 | 2점 촉타 후, 양쪽 지점 도달 시 지팡이를 앞으로 10cm 밀어줌 |
| **터치 앤 드래그 (촉타 후 긋기법)** | 한적한 실내 직선 보행 | 장애물 피하며 걸어온 방향으로 경로 빠르게 설정 |
| **3점 터치법** | 실외 기준선 따라가기 | 난간·점자블록·지하철 스크린 도어 등 기준선 활용에 적합 |
| **한 번 바닥 치고 한 번 측면 치기** | 3점 터치법 대체 | 3점보다 배우기 쉬우나 바닥 정보 불충분 |
| **대각선법** | 주로 실내 | 벽을 이용 시 지팡이 끝을 벽에 붙이며 트레일링 기법과 병용 → 신속 이동 |

**계단 이동 절차**

오르기부터 지도 후 내리기를 지도한다 (위험을 덜 느끼는 순서).

- **계단 오르기**: 손잡이 바로 아랫부분을 연필 쥐듯 가볍게 잡고, 팔을 앞으로 뻗어 올라가야 할 계단 **바로 위 계단의 윗부분**을 지팡이 끝으로 가볍게 스치듯 치면서 올라간다.
- **계단 내리기**: 대각선법 유지; 지팡이 끝은 내려가야 할 계단의 **다음 계단 모서리 부분에서 약간 위**로 올려 공중에 띄운 상태로 내려온다.

**문 통과하기**

1. 지팡이 끝이 문에 닿으면 지팡이를 세워 문에 갖다 댄다.
2. 좌우로 문지르면 문 손잡이가 걸린다.
3. 지팡이를 잡지 않은 손으로 문손잡이를 잡고 문을 열고 나간다.

### 7.3. 프리케인

- 흰 지팡이 **사용 전 단계**의 이동 보조구
- 사각형 또는 원형으로 제작; 유아가 몸 앞에 위치시키고 양손으로 밀며 이동
- **목적**: 환경 탐색, 몸 보호 및 지탱

## 8. 안내견

안내견은 시각장애인의 안전한 보행을 돕는 보행 수단으로, 지적 불복종 능력과 사회적 상호작용 촉진 등 다양한 장점을 제공하지만 돌봄 노력과 일반인의 배려가 필요하다.

> ⚠️ **기출 포인트**: **지적 불복종**의 정의와 **16세 이상** 사용 자격 조건이 출제된다. 안내견 사용의 전제 조건(방향정위·이동 기술 습득)도 확인할 것.

### 8.1. 안내견의 기본 이해

| 항목 | 내용 |
|------|------|
| 기본 조건 | 시각장애인이 **방향정위와 이동 기술**을 잘 터득하고 있어야 함 |
| 사용 자격 | 안내견 돌봄 자격 심사 통과 + **16세 이상** |
| 역할 구분 | 시각장애인 → 방향 지시 및 이동 결정; 안내견 → 안전한 보행 보조 |

### 8.2. 안내견의 장단점

**장점**

| 장점 | 설명 |
|------|------|
| **지적 불복종** | 안전하지 못한 상황에서 사용자가 오판하여 명령해도 이를 **거부**하고 안전한 행동을 취함 |
| 상단 장애물 회피 | 허리 위쪽이나 통로의 장애물을 피할 수 있음 |
| 방향정위 집중 | 안전보다 방향정위에 집중할 수 있어 낯선 지역 보행에 편리 |
| 사회적 상호작용 | 안내견을 매개로 타인과의 접촉 기회 증가 |
| 잔존 감각 극대화 | 안전에 대한 노력을 줄이고 다른 감각 극대화 가능 |
| 독립성·자신감 | 타인 의존 감소, 자신감 증가 |
| 책임의식·정서적 안정 | 돌보는 과정에서 책임의식 향상; 유대감·애정으로 정서적 안정 |

**단점**

1. 안내견 돌봄을 위한 시간과 노력이 필요하다.
2. 보행하지 않을 때 안내견을 기다리게 하는 것이 어렵다.
3. 안내견이 주위의 주목을 더 끌어 시각장애인이 소외감을 느낄 수 있다.

### 8.3. 일반인의 고려사항

> ⚠️ **기출 포인트**: 일반인이 **절대 해서는 안 되는 행동** 목록이 출제된다.

| 금지 행동 | 이유 |
|----------|------|
| 주인 허락 없이 만지기·먹이 주기·부르기 | 집중력 저하 → 안내견 돌발 행동 → 시각장애인·안내견 모두에게 위험 |
| 하네스를 잡아당기거나 밀기 | 안내 기능 방해 |
| 손짓으로 불러 집중력 빼앗기 | 위험 유발 |

- 안내견과 인사를 나누고 싶다면 **주인의 허락을 구한 후** 동의 하에 다가가서 쓰다듬는다.

## 9. 전자보행보조구

전자보행보조구는 일정 범위 내 환경을 전파로 감지하고, 그 정보를 청각(소리)과 촉각(진동)으로 전달하여 이동을 돕는 기구다.

> ⚠️ **기출 포인트**: 전자보행보조구는 **2차적 수단**이며 흰 지팡이가 기본임을 명확히 할 것. **가위기법**과 **전방수직기법**의 차이와 사용 절차가 출제된다.

### 9.1. 개념 및 유형

| 항목 | 내용 |
|------|------|
| 정보 전달 방식 | 청각(소리) + 촉각(진동) |
| 함께 사용 가능 | 의자(휠체어), 스쿠터와 병용 가능 |
| 단점 | 항상 머리를 똑바로 들고 있어야 함; 무릎 부분 장애물 탐지 불가; 내리막 계단·연석 탐지 어려움 |

**우선순위 원칙 (APH 권장)**

흰 지팡이 또는 안내견 = **1차적** 이동 수단 → 전자보행보조구 = **2차적** 수단

### 9.2. 활용 방법 비교

| 기법 | 기본 설명 | 사용 절차 |
|------|----------|----------|
| **가위기법** | 흰 지팡이 손잡이 부분과 전자보행보조구를 **가위 모양으로 교차** | 1. 흰 지팡이로 2점 터치법 사용 2. 전자보행보조구는 지팡이 잡은 손의 **팔목 약간 위쪽**에 위치 3. 몸 중앙에서 수직으로 교차하여 머리 높이 장애물 탐지 4. 천천히 걷기 (빠르면 탐지 실패) |
| **전방수직기법** | 전자보행보조구를 **전방으로 똑바로** 가리킴 | 1. 흰 지팡이로 2점 터치법 사용 2. 몸 중앙에 보행보조구 고정 3. 상반신과 직각이 되도록 잡아 몸 앞에 위치 4. 걸으면서 위아래로 움직여 머리 위 장애물 탐지 |

**전방수직기법 사용 시점**: 흰 지팡이 단독으로는 **허리 위쪽 장애물 확인 불가** 상황에서 유용하다.

### 9.3. 주요 전자보행보조구 유형

| 기기 | 감지 방식 | 특징 |

## 10. 실외 훈련

실외 훈련은 한산한 지역(주택가)에서 시작하여 다양한 이동 기술과 상황회복 능력을 습득하는 훈련이다.

> ⚠️ **기출 포인트**: **드롭오프(낙하훈련)**의 목적, 절차, 난이도 순서가 자주 출제된다. 지도 거리(근접/중간/원거리) 수치를 암기할 것.

### 10.1. 주택가(한산한 지역) 훈련

**기본 지도 순서**

1. 대중교통 수단에 익숙해지기
2. 주택가 구조(보도·차도·횡단보도·가로수·가로등) 익히며 환경 정보를 이용한 방향정위 훈련
3. 지도할 핵심 기술:
   - **비직선 이동(veering) 방지** — 직선 이동하는 법
   - **차 소리 이용** — 차 소리를 이용하여 이동하는 법
   - **횡단보도 건너기** — 횡단보도 건너가는 법

**거리 단계별 지도 방법**

| 단계 | 거리 | 상황 |
|------|------|------|
| 근접 거리 | 팔걸이 정도 | 학생이 길을 익히는 단계 |
| 중간 거리 | **4m** | 연습 단계; 아직 전문가 도움 필요 |
| 원거리 | **6m** | 혼자 다닐 수 있는 단계 |

**상황회복(recovery) 요령 지도**

길을 잃었을 경우 스스로 길을 찾아내는 상황회복 요령을 지도한다.

**드롭오프(drop-off, 낙하훈련)**

> ⚠️ **기출 포인트**: 드롭오프는 방향정위·이동 능력 **평가** 도구다. 훈련이 아닌 평가라는 점과 아동이 타인에게 도움 요청 불가 원칙을 정확히 기억할 것.

| 항목 | 내용 |
|------|------|
| 목적 | 주택가 지역 훈련 단계에서 방향정위·이동 능력 **평가** |
| 절차 | 교사가 일체 정보 없이 아동을 한 장소로 데려감 → 아동이 스스로 환경 정보 수집 → 이동 경로 파악 → 목적지 이동 |
| 규칙 | 평가 중 **다른 사람에게 도움 요청 금지**; 교사는 멀리서 관찰하며 위험 상황 감시 |
| 장소 | 학생이 **전에 보행한 적 있는** 외부 넓은 공간 |
| 난이도 순서 | 비교적 익숙한 **실내 환경** → 낯설고 복잡한 **실외 환경** (점진적 향상) |
