## 11. 보조공학기기

보조공학기기는 시각장애인의 시각적·촉각적·청각적 학습을 지원하며, 확대법, 광학기구, 컴퓨터 프로그램, 비광학 보조기구, 맹인을 위한 특수 기기 등 다양한 유형으로 구성되어 개별 학생의 필요에 맞춰 활용되어야 한다.

### 보조공학기기 전체 분류

| 대분류 | 소분류 | 주요 기기 |
|--------|--------|-----------|
| 저시력용 | 확대법 | 상대적 거리·크기·각도·투사 확대법 |
| 저시력용 | 광학기구 | 확대경, 망원경, 시야 확대 보조구, 차광안경 |
| 저시력용 | 전자기기 | 확대 독서기(CCTV), 화면 확대 프로그램 |
| 저시력용 | 비광학 | 독서대, 타이포스코프, 쓰기 보조구, 확대 도서, 조명, 대비 |
| 맹인용 | 음성 | 음성합성장치(HW), 스크린 리더(SW) |
| 맹인용 | 문자 변환 | OCR 프로그램, 광학문자 판독기, 점역 프로그램 |
| 맹인용 | 촉각 | 옵타콘, 전자점자기, 점자정보단말기, 입체 복사기, 점자 타자기, 점자판 |
| 맹인용 | 디지털 도서 | 데이지(DAISY) |

---

## 11.1. 저시력을 위한 보조공학기기

### 1) 확대법

물체를 확대시켜 시각적 접근성을 높이는 방법이다.

| 종류 | 원리 |
|------|------|
| 상대적 거리 확대법 | 물체와 눈의 거리를 가깝게 하여 망막상의 크기를 확대 |
| 상대적 크기 확대법 | 물체의 실물 크기를 확대하여 망막상의 크기를 키움 |
| 각도 확대법 | 렌즈·광학기구를 사용하여 물체의 크기를 확대 |
| 투사 확대법 | 필름·슬라이드 등을 스크린에 투영하여 확대 |

### 2) 광학기구

렌즈를 사용하여 물체를 확대하거나 시야를 조절하는 기기이다.

#### 확대경 종류

| 종류 | 특징 |
|------|------|
| 손잡이형 확대경 | 가장 일반적, 배율에 따라 초점 거리를 조절해야 함 |
| 스탠드 확대경 | 자료와의 거리 일정 유지, 수전증·어린 학생에게 유용 |
| 플랫베드(집광형) 확대경 | 반구형·초점 고정, 빛을 모으는 성질 → 높은 조도 필요 학생에게 유용 |
| 막대 확대경 | 긴 막대형, 읽는 줄 위에 놓고 사용, 줄 놓침 방지 효과 |
| 안경 장착형 확대경 | 두 손이 자유로워 읽기·쓰기 동시 가능 |

#### 망원경 종류

| 종류 | 특징 |
|------|------|
| 손잡이형 망원경 | 단안용, 짧은 시간 원거리 물체 관찰에 적합 |
| 클립형 망원경 | 단안용, 안경에 걸이식 부착, 사용 훈련 필요 |
| 안경 장착형 망원경 | 양손 자유, 시야가 협소 |
| 전시야형 망원경 | 안경 중앙 장착, 조금 더 넓은 시야, 특정 작업 집중에 효과적 |
| 쌍안경 | 넓은 시야·높은 배율, 무거워 장시간 사용 어려움 |
| 역단안경 | 물체를 축소→상대적으로 넓은 시야, 시야 좁고 중심시력 좋은 저시각인에게 유용 |

#### 시야 확대 보조구

| 기구 | 원리 및 적용 |
|------|-------------|
| 리버스 망원경(역단안경) | 대상물 축소 → 시야 확장, 시야 협소 학생에게 적용 |
| 프레넬 프리즘 | 투명 플라스틱 막을 안경 렌즈에 부착, 빛을 꺾어 원하는 위치에 제공 → 반맹에 효과적 |
| 오목렌즈 | 물체를 축소 → 더 많은 물체를 볼 수 있어 시야 확장 |

> ⚠️ **기출 포인트**: 차광안경(흡광안경)은 심한 눈부심을 겪는 아동에게 적합하며, 눈 위쪽·옆쪽에서 들어오는 빛을 차단한다. 광학기구가 아닌 환경 조절 목적의 보조구로 분류된다.

### 3) 컴퓨터 프로그램 및 전자기기

| 기기 | 기능 |
|------|------|
| 확대 독서기(CCTV) | 비디오 카메라로 화면 확대, 컬러/흑백/역상 모드, 글자 크기·색상·밝기·명암 조절 |
| 화면 확대 프로그램 | 컴퓨터 화면의 문자·그림을 크게 확대·보정하여 보기 쉽게 변환 |

### 4) 비광학 보조기구

광학적 원리를 사용하지 않고 시각적 접근성을 높이는 도구이다.

| 기구 | 기능 및 특징 |
|------|-------------|
| 독서대 | 근거리 광학기구와 함께 사용, 높이·각도 조절로 피로 감소 |
| 독서보조판(타이포스코프) | 검은 책받침, 상단에 한 줄 개방 → 대비 향상, 눈부심 방지, 줄 놓침 방지 |
| 쓰기 보조구 | 쓰기 보조판 + 굵은 심 펜, 또는 줄 굵고 진하며 간격 넓은 공책 사용 |
| 확대 도서 | 문자 크기를 확대하여 제작한 도서 |
| 조명 | 밝기 조정, 눈부심 조절, 대비 활용 |
| 대비 | 물체와 배경 색의 차이 → 대비가 높을수록 시감도 증가 |

> ⚠️ **기출 포인트**: 타이포스코프는 '독서보조판'이라고도 하며, 단순히 줄을 가리는 것이 아니라 눈부심 방지와 대비 향상을 동시에 수행한다는 점이 자주 출제된다.

## 11.2. 맹인을 위한 보조공학기기

| 기기 | 유형 | 기능 및 특징 |
|------|------|-------------|
| 음성합성장치 | 하드웨어 | 문자·숫자·구두점 형태의 텍스트를 음성으로 출력 |
| 스크린 리더(화면 읽기 프로그램) | 소프트웨어 | 모니터 내용을 음성으로 읽어줌, 음성합성장치와 연계 |
| OCR 프로그램 | 소프트웨어 | 인쇄 문자를 이미지 스캐너로 입력받아 편집 가능한 텍스트로 변환 |
| 광학문자 판독기(음성 변환 출력기) | 하드웨어 | OCR 판독 문자를 문단·문장·줄·단어 단위로 선택하여 읽기 가능 |
| 옵타콘(촉각 장치) | 하드웨어 | 소형 촉지판의 핀이 문자 모양대로 돌출 → 맹학생이 일반 활자 촉독 가능 |
| 점역 프로그램 | 소프트웨어 | 컴퓨터 텍스트 파일을 점자 프린터 출력용 점역 파일로 변환 |
| 전자점자기(무지점자기) | 하드웨어 | 종이 없이 금속·나일론 핀이 표면으로 올라와 점자 구성, 소음 없음 |
| 점자정보단말기 | 하드웨어 | 문서를 음성·점자로 출력 가능한 휴대용 기기, 인터넷 이용 가능 |
| 입체 복사기 | 하드웨어 | 플랙시 페이퍼에 인쇄된 그림을 통과시켜 그림 부분을 부풀려 촉각 자료 생성 |
| 점자 타자기 | 하드웨어 | 점자 6점에 해당하는 6개 키 구성, 쓰면서 동시에 읽기 가능 → 점자 학습에 효과적 |
| 점자판 | 하드웨어 | 점첩·점판·점간·점필로 구성, 점자를 직접 찍는 방식 (좌우 반전 주의) |
| 데이지(DAISY) | 디지털 포맷 | 시각장애인을 위한 디지털 도서 형식 |

> ⚠️ **기출 포인트**: 전자점자기와 점자 타자기의 차이를 구분해야 한다. 전자점자기는 종이 없이 핀이 올라와 점자를 표시하며 소음이 없다. 점자 타자기는 종이에 점자를 찍으며, 쓰면서 동시에 읽을 수 있어 초기 점자 학습에 특히 효과적이다.

> ⚠️ **기출 포인트**: 점자판으로 점자를 쓸 때는 점형의 좌우를 바꾸어 써야 하므로 오류를 범하기 쉽다. 이는 종이를 뒤집어 읽기 때문이다.

## 12. 시각장애아동교육

시각장애 아동 교육은 시각기술, 시기능, 시효율, 시지각 등 시각 관련 기본 개념을 바탕으로 Corn의 시기능 모형과 Lowenfeld의 특수 방법 원리를 적용하여 구체적이고 통합적인 경험 학습을 제공해야 한다.

---

## 12.1. 시각과 관련된 기본 개념

| 개념 | 정의 |
|------|------|
| 시각기술 | 시력, 시야, 대비감도, 색각 등을 실제로 활용하는 기술 |
| 시기능 | 생활 속에서 학업 과제 또는 일상 활동을 수행할 때 시각을 어느 정도 사용하는가 |
| 시효율 | 특정 시각 과제를 쉽고 편안하게 시간을 적게 들여 수행하는 능력 |
| 시지각 | 눈을 통해 들어온 감각 정보를 분석·이해하는 과정 (이전 학습 지식이 기초) |

> ⚠️ **기출 포인트**: 시기능과 시효율은 혼동하기 쉽다. **시기능**은 시각 사용의 정도(얼마나 사용하는가), **시효율**은 시각 과제 수행의 질과 효율(얼마나 잘 수행하는가)을 의미한다.

---

## 12.2. Corn의 시기능 모형

Corn의 시기능 모형은 학생 능력, 시각 능력, 환경 요인을 고려하여 시각장애 아동의 시기능 발달을 돕는 세 가지 수준의 접근 방법을 제시한다.

| 수준 | 명칭 | 목표 및 내용 |
|------|------|-------------|
| 1수준 | 시각 자극 접근 | 질 높은 시각 자극 경험을 단계적으로 제공 → 뇌에서 시각 자극을 인식하도록 유도 |
| 2수준 | 시각 효율 접근 | 시기능 발달·시지각과 관련된 환경 요소를 고려 → 시각 자극 해석 지원, 시각적 잠재력 최대화 |
| 3수준 | 시각 활용 접근 | 광학·비광학 보조구 사용, 시각 환경 수정 등으로 시각 사용 극대화 → 아동의 적극적 참여 유도 |

> ⚠️ **기출 포인트**: 3수준은 보조공학기기 활용을 포함하는 가장 능동적인 단계이다. "저시각을 향상시키는 데 아동이 적극적으로 참여"한다는 서술이 핵심이다.

## 12.3. Lowenfeld의 특수방법의 원리

### 실명에 따른 세 가지 제한성

실명은 다음 세 가지를 제한한다.
1. 경험의 질과 양
2. 보행 능력
3. 환경과 상호작용하는 기회

### 특수방법의 원리(교육의 원리)

| 원리 | 내용 |
|------|------|
| 구체적 경험 학습 | 주위 환경 지식 획득과 의미 있는 개념 발달을 위해 사물에 대한 풍부하고 다양하며 일관성 있는 구체적 경험 제공 |
| 실천 경험 학습 | 시각장애 아동이 스스로 직접 실행하는 경험을 통해 학습 |
| 통합적 경험 학습 | 사물과 상황을 총체적으로 경험하고, 촉각으로 확인되는 부분마다 정보를 전체로 통합하는 기회 제공 |

> ⚠️ **기출 포인트**: Lowenfeld의 세 원리는 '구체→실천→통합'으로 기억할 수 있다. 특히 **통합적 경험 학습**에서 잔존 감각, 그중에서도 촉각의 역할이 핵심이다. "부분 정보를 전체로 통합"하는 과정이 시각 없이 세계를 이해하는 맹아동의 핵심 학습 원리이다.
