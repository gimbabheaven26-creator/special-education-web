## 13. 시각장애학생의 교육과정

시각장애 학생 교육과정은 **확대 핵심 교육과정**을 통해 보상 기술, 기능적 기술, 방향정위와 이동 기술, 사회 기술, 일상생활 기술, 레크리에이션/여가 기술, 공학, 시기능 훈련, 진로 교육, 전환 교육 등 독립적인 삶에 필요한 지식과 기술을 습득하도록 지도해야 한다.

> ⚠️ **기출 포인트**: 확대 핵심 교육과정의 10가지 구성 영역은 단순 암기 대상이 아니라, 각 영역이 "왜 시각장애 학생에게 필수적인가"를 설명할 수 있어야 한다. 특히 사회 기술·진로 교육은 "시각을 통한 자연스러운 습득이 어렵기 때문에 계획적 지도가 필요하다"는 이유와 함께 출제된다.

### 13.1. 시각장애 확대 핵심 교육과정

시각장애 확대 핵심 교육과정은 시각장애인이 사회의 구성원으로 **독립적으로 살아가기 위해 필수적으로 습득**해야 하는 지식과 기술로 구성된 교육과정이다.

- **교사의 역할**: 시각장애 학생들이 확대 핵심 교육과정에 접근할 수 있도록 하고, 독립과 삶의 질 측면에서 필요한 기술을 습득하도록 지도한다.

#### 확대 핵심 교육과정 10개 영역

| 영역 | 핵심 내용 | 시각장애 관련 이유 |
|------|-----------|------------------|
| 보상 기술 | 교육에 쉽게 접근할 수 있도록 돕는 기술 | 시각 결손으로 인한 정보 접근 한계 보완 |
| 기능적 기술 | 사회에서 개인적 요구를 해결하기 위한 기술 | 일상생활·직업 환경에서의 독립 기능 확보 |
| 방향정위와 이동 기술 | 독립적 이동 및 환경 탐색 | 독립 이동의 기본 권리 충족 |
| 사회 기술 | 인간관계 기술 | 시각을 통한 자연스러운 습득이 어려워 계획적 지도 필요 |
| 일상생활 기술 | 개인 관리·일상생활 지식 및 기법 | 스스로 수행 가능한 수준 습득 → 사회 통합 촉진 |
| 레크리에이션/여가 기술 | 성인기에도 즐길 수 있는 여가 활동 개발 | 삶의 질 향상 |
| 공학 | 화면 확대·음성 지원·점자 출력 소프트웨어 및 하드웨어 | 컴퓨터 기반 학습·업무 접근 |
| 시기능 훈련 | 시기능 평가 + 효과적 시각 활용 학습 계획 | 잔존 시기능의 기능적 활용 극대화 |
| 진로 교육 | 직업 인식·탐색·준비·배치 활동 | 직업 정보를 자연스럽게 수집하기 어려워 부모 협조 아래 계획적 지도 필요 |
| 전환 | 태도·가치관·직업 경험·독립 생활 기술 | 성인기 독립 생활 및 사회 참여 준비 |

## 14. 시각 활용 기술 훈련

시각 활용 기술 훈련은 **중심시야 상실**과 **주변시야 상실**에 따른 시각 기술을 각각 다르게 지도하며, 시각장애 학생의 **시각 효율성을 극대화**하는 데 중점을 둔다.

> ⚠️ **기출 포인트**: 중심시야 상실 vs. 주변시야 상실에 따른 거리 조절 방향이 반대이다. 중심시야 상실 → **가깝게** 다가가기, 주변시야 상실 → **멀리** 뒤로 물러나기. 이 대비 구조가 자주 출제된다.

### 14.1. 중심시야 상실에 따른 시각기술

| 기술 | 설명 | 세부 방법 |
|------|------|----------|
| 중심 외 보기 훈련 | 시야 중심 암점을 대체할 망막 부위 사용 훈련 | 암점에서 가장 가까운(가장 양호한) 시야 부위를 찾아 활용 |
| A4 용지 훈련법 | 중심 외 보기 훈련의 간단하고 효과적인 방법 | A4 용지 가운데 숫자 '8'을 적고, 고개·안구를 돌리지 않고 가장 선명하게 보이는 지점을 찾아 훈련 |
| 거리 조절 (가깝게 다가가기) | 중심 암점의 영향 최소화 전략 | 물체에 가까이 다가가면 목표물이 암점보다 커져 확인이 쉬워지는 원리 활용 |

- 진단 도구: 시계보기 검사, A4 용지 활용법

### 14.2. 주변시야 상실에 따른 시각기술

망막색소변성으로 주변시야가 상실되거나 시로장애로 반맹이 되면 보행과 일상 활동에서 보이는 범위가 좁아진다.

#### 잔존 시야를 사용하는 3가지 시각기술

| 기술 | 정의 |
|------|------|
| 추시하기 | 환경 속에 **정지해 있는** 대상물의 윤곽을 눈으로 따라가는 기술 |
| 추적하기 | 환경에서 **움직이는** 대상물을 따라가는 능력 |
| 주사하기 | 특정 공간(환경, 사물)을 **훑어보는** 기술 |

#### 거리 조절 및 보조구

| 방법 | 대상 | 내용 |
|------|------|------|
| 거리 조절 (멀리 뒤로 물러나기) | 주변시야 상실 + 중심시력 양호한 경우 | 대상물과의 거리를 늘려 시야 범위 내에 목표물이 들어오도록 조절 |
| 시야 확대 보조구 | 시야 범위가 지나치게 좁은 아동 | 시야를 넓혀 주는 보조 도구 활용 |

## 15. 일반교과 교육

시각장애 학생을 위한 일반교과 교육은 학습 자료의 수정, 선명도와 대비 증진, 눈부심 감소, 산만한 시각 자극 감소 등을 통해 **시각적 접근성을 높이고 학습 효율을 극대화**해야 한다.

> ⚠️ **기출 포인트**: 눈부심 감소 전략(모자·색안경·커튼·가리개)과 산만한 시각 자극 감소 전략(독서보조판·투명 용지·자료 단순화)은 구체적인 방법이 함께 출제된다. 단순화 자료가 "최후의 수단"이라는 표현도 중요 포인트.

### 15.1. 시각 자료 수정

#### 1단계: 확대

| 방법 | 내용 |
|------|------|
| 복사하여 확대본 만들기 | 원본 자료를 복사·확대하여 제공 |
| 확대 보조구 사용 | 확대경, 망원경 등 활용 |
| 자료를 눈 가까이로 이동 | 거리 단축으로 확대 효과 획득 |

#### 2단계: 선명도와 대비 증진

- 가능한 한 복사된 자료 사용, 심이 굵고 짙은 색의 사인펜 활용
- 시기능 평가 시 **아세테이트 필터지**를 사용하여 자료와 대비 효과 형성

#### 3단계: 눈부심 감소

| 방법 | 세부 내용 |
|------|----------|
| 챙 달린 모자 착용 | 실내·실외에서 빛의 양 조절 (실내 금지 시 허락 또는 대체 방안 마련) |
| 색안경 착용 | 실내·실외 모두 적용 |
| 좌석 선택 | 햇빛이 들어오는 창가·문 쪽에서 먼 자리에 스스로 앉도록 유도 |
| 커튼·블라인드 설치 | 빛 차단 |
| 자료 위 가리개 사용 | 빛과 눈부심 조절 |

#### 4단계: 산만한 시각 자극 감소

- **독서보조판, 투명 용지, 시각 자료의 단순화** 활용
- 단순화된 자료가 최후의 수단인 경우, 교사는 동일한 과제를 완수할 수 있는 **대체 전략**을 찾아야 한다.
- 교사는 시각 자료에서 복잡성에 대한 지식을 길러 주고, 학생은 정보 획득을 위한 적절한 시각 기술 또는 대체 기술을 배우도록 한다.

### 15.2. 교과서

| 구분 | 정의 | 특징 및 유의사항 |
|------|------|-----------------|
| 확대 교과서 | 문자 크기를 확대하여 제작한 도서 | 운반·취급 불편; 그림은 흑백으로 원본보다 선명도 낮음 |
| 점자 교과서 | 일반 글자를 점자로 변환한 교과서 | 일반 글자 1권 → 여러 권의 점자책으로 제작 |

#### 점자 교과서 세부 규정

| 항목 | 내용 |
|------|------|
| 쪽수 표시 위치 | 각 쪽 오른쪽 윗부분 |
| 쪽수 기준 | 일반 교과서의 쪽수 그대로 사용 |
| 페이지 분할 표시 | 한 페이지가 여러 장으로 나뉠 경우 a, b, c 순으로 덧붙여 표시 |
| 그림·지도·표·그래픽 | 생략 가능; 교과서 용어 목록은 별도 제작 |
| 보관 방법 | 책을 쌓지 않고 **선반에 세워서 보관** (범이 눌려 지워질 위험 방지) |
| 일반 글자 사본 | 점자를 사용하지 않는 사람의 도움을 받기 위해 제공 |

#### 점자 교과서 사용 지침 (일반 교사 대상)

1. 점자 번호 체계를 가르친다.
2. 일반 교과서의 쪽수를 사용하는 방법을 가르친다.
3. 학생이 해당 쪽수가 있는 권(冊)과 쪽을 찾을 수 있도록 돕는다.
4. 용어 목록은 바로 사용할 수 있도록 책상 위에 준비한다.
5. 점역사가 생략한 그림·도표에 대한 설명 기록을 학생이 참조하도록 한다.
