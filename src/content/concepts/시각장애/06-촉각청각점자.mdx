## 16. 촉각 자료

촉각 자료는 시각장애 학생의 공간 이해를 돕기 위해 도형, 그래프, 도표, 그림 등을 손으로 만져 볼 수 있도록 제작된 자료이다. 약도, 입체 모형, 촉각 지도, 확대 지도 등 다양한 형태로 제공되어야 한다.

### 16.1. 촉각 자료 제작 원칙

> ⚠️ **기출 포인트**: 촉각 자료 제작 원칙은 "촉각 위주"와 "정보 최소화"가 핵심이다. "시각 위주 → 촉각 위주", "있는 그대로 → 도식화"라는 방향 전환을 묻는 선택지에 주의할 것.

| 번호 | 원칙 |
|------|------|
| 1 | 볼록한 선·점·면을 사용하여 촉각으로 명확하게 식별할 수 있도록 제작 |
| 2 | 복잡한 지도나 도표의 불필요한 부분을 생략하고 단순화하여 촉각 식별이 용이하도록 함 |
| 3 | 자나 각도기를 사용하여 선의 길이와 각도를 정확하게 측정 |
| 4 | 제작 후 교사가 눈을 가리고 손으로 직접 만져 완성도를 평가하고 수정 |
| 5 | 학생의 연령, 인지, 언어, 소운동 기술 수준, 교과 지식을 고려하여 개인 수준에 맞게 제작 |
| 6 | 크기: 두 손을 나란히 붙이고 열 손가락을 펼칠 때의 넓이(약 16~18인치)가 적절 |
| 7 | 척도(scale)는 필수 정보를 보여줄 만큼으로 최소화하고, 있는 그대로보다는 도식화 |
| 8 | **시각 위주가 아니라 촉각 위주**로 기호를 선택하고, 정보 밀도(density)는 가능한 한 최소화 |
| 9 | 기호에 대한 설명은 필요한 것만 제시 |

### 16.2. APH에서 제시한 촉각자료 제작 지침

> ⚠️ **기출 포인트**: APH 지침의 6개 항목명(유도선·일반 디자인·기호·명칭·수치 및 축척·예비 정보)과 각 내용을 매칭하는 문제가 출제된다. 특히 "유도선은 최후의 수단"이라는 표현이 자주 등장한다.

| 항목 | 핵심 내용 |
|------|----------|
| **유도선** | 최후의 수단으로만 사용. 유도선을 화살표로 사용하지 말 것 |
| **일반 디자인** | 촉각 그래픽이 전부 필요한지 먼저 결정. 촉각적으로 분명해야 하며 교수·학습과 관계된 정보를 담을 것 |
| **기호** | 선·점·기호는 만져서 서로 쉽게 구별되도록 제한. 같은 내용 내 같은 유형의 그래픽에서는 일관성 유지 |
| **명칭** | 모든 그래픽 기호를 같은 쪽이나 마주보는 쪽 등 특정 기호 쪽에 설명하고 정의 |
| **수치 및 축척** | 지도 한 페이지에 북쪽을 표시할 때 서언에 명기하면 각 지도마다 반복 표시 불필요 |
| **예비 정보** | 모든 제목·유의점·설명을 그래픽 앞에 위치. 번안자의 주안점보다 저자의 것을 우선 |

### 16.3. 방향정위 보조도구

> ⚠️ **기출 포인트**: 약도·입체모형·촉각지도·확대지도의 대상(누구에게 유용한가)과 장단점이 비교 문제로 출제된다. 입체모형이 "공간 개념 부족" 대상에게 가장 유용하다는 점을 기억할 것.

| 유형 | 대상 및 특성 | 장점 | 단점 |
|------|-------------|------|------|
| **약도** | 공간 이해력이 낮은 시각장애 아동, 시각중복장애인 | 경로만 간단히 기술하여 인지지도 형성에 효과적 | 상세 정보 부족 |
| **입체모형** | 환경 개념이 부족한 시각장애인 | 삼차원 공간을 구체적으로 나타내 방향정위 촉진에 가장 유용 | 실물 축소 필요, 제작 어려움 |
| **촉각지도** | 생소한 공간 배치 학습, 새 경로 발견 | 새로운 보행 경로 발견 및 공간 갱신에 유용 | 휴대 불편, 제작·읽기에 시간·노력 필요 |
| **확대지도** | 저시각인 | 크기·대비·배열 조정 가능 | 고대비 필요, 불필요한 정보 삭제 필요 |

## 17. 청각 자료

청각 자료는 묵자나 점자를 읽는 데 어려움이 있는 시각장애 학생에게 중요한 학습 수단이다. 녹음 교과서 형태로 제공되어 학습 자료를 효과적으로 전달하지만, 그림·차트·그래프 등 일부 내용은 전달하기 어렵다는 한계가 있다.

### 17.1. 청각 자료의 기능

1. 묵자나 점자를 읽는 데 어려움이 있는 학생에게 중요한 학습 수단이다.
2. 듣기는 말하기·읽기·쓰기의 발달에 도움을 준다.
3. 듣기는 점독에 비해 속도가 빠르며, 자료를 구하고 처리하는 데 점자보다 더 효과적이다.

### 17.2. 녹음 교과서

#### 자료 녹음 시 지침

> ⚠️ **기출 포인트**: 녹음 교과서 제작 지침은 세부 절차 순서가 선택지로 출제된다. "두 명이 한 조", "음절 단위", "장·절 위계 녹음" 등 특이 사항에 주의할 것.

| 번호 | 지침 내용 |
|------|----------|
| 1 | 그림·표·차트의 주요 사항 확인 및 단어 정확 발음을 위해 사전에 자료를 읽어볼 것 |
| 2 | 품질 좋은 녹음기·테이프 사용, 조용한 장소에서 녹음, 녹음 전 자료 출처 확인 |
| 3 | 책 전체의 위계(장·절·순서 숫자)를 알 수 있도록 함께 녹음 |
| 4 | 이해에 필요한 그림·표·차트에 대한 설명 포함 |
| 5 | 새로운 쪽·부분·장 시작 시 소리 신호 녹음, 영어 용어·이름은 발음과 철자 함께 녹음 |
| 6 | 한문 표기 단어는 글자의 토를 읽어주거나 낱말의 뜻을 녹음 |
| 7 | 음절 단위로 제시. 내용과 직접 관계없어도 필요한 부분 설명 함께 녹음 |

#### 녹음도서 제작의 실제

| 번호 | 실제 제작 사항 |
|------|--------------|
| 1 | 음절 단위로 제시. 내용과 무관해도 필요한 설명 함께 녹음 |
| 2 | 외국어 용어·이름: 정확한 발음 + 철자 읽기. 한문: "일 사", "형상 상" 방식으로 녹음 |
| 3 | 희귀·어려운 낱말, 혼동 가능 낱말은 반드시 뜻도 함께 읽기 |
| 4 | 도표·차트·그래프·그림은 낭독자가 완전히 이해한 뒤 뜻을 풀어 간결하게 설명 |
| 5 | 부·장·절·순서 숫자뿐 아니라 제목·출판사·출판 연월일·트랙 수 녹음 |
| 6 | 괄호·따옴표 등 중요 부호도 녹음. 페이지 전환 시 읽던 문장 완전히 다 읽은 후 전환 |
| 7 | **두 명이 한 조**가 되어 낭독하는 것이 원칙 |

#### 표·차트·그래프 설명 지침

| 번호 | 지침 내용 |
|------|----------|
| 1 | 표·차트의 구성 설명 → 칸의 수와 명칭, 칸별 하위 칸 수와 명칭을 왼쪽→오른쪽으로 읽기 |
| 2 | 칸별 또는 줄별로 읽을지 설명. 줄이 많은 경우 **5번째 줄마다** 칸별 명칭 반복 |
| 3 | 마지막 줄임을 명시하여 표·차트 끝남을 알릴 것 |
| 4 | 그래프 종류(선·막대·원그래프), 제목, 출처, 표제 먼저 언급 |
| 5 | 선·막대그래프: X축·Y축이 나타내는 것과 최대 한정치 알려주기 |
| 6 | 원그래프: 색깔에 대해 언급. 설명 끝났음을 명시 |

### 17.3. 청독의 장단점

> ⚠️ **기출 포인트**: 청독(녹음 교과서 듣기)의 단점으로 "그림·차트·그래프는 정확하게 전달 불가"와 "자료 참조·통제 어려움"이 자주 출제된다.

| 구분 | 내용 |
|------|------|
| **장점** | 컴퓨터보다 휴대 편리, 값이 저렴하고 경제적 |
| **장점** | 속도 조절 장치로 테이프 속도 조절 → 독서 시간 단축 가능 |
| **장점** | 묵자·점자를 읽기 어려운 학생에게 중요한 수단. 점독보다 속도 빠름 |
| **단점** | 재생 속도를 빨리 하면 재생음이 높아짐 |
| **단점** | 그림·차트·그래프·도형은 듣기로 정확하게 전달 불가 |
| **단점** | 앞 내용 다시 듣기, 건너뛰기, 원하는 장·페이지 찾기 어려움 (자료 참조 어려움) |
| **단점** | 속도·억양·고저·간격 등을 낭독자가 결정 (자료 통제 어려움) |
| **단점** | 청취는 수동적이며, 듣기 자료를 구하기 어려움 |

## 18. 점자

점자 교육은 시각장애 학생의 문해력 발달에 필수적이다. 음소 접근, 음소-단어 접근, 단어 접근 등 다양한 지도 방법을 통해 개별 학생의 특성과 필요에 맞춰 효과적으로 이루어져야 한다.

### 18.1. 점자의 기본 이해 및 점자 찍기 순서

> ⚠️ **기출 포인트**: 점자지를 접는 이유와 앞면/뒷면을 읽을 때 접힌 부분의 방향이 반대임을 묻는 문제가 출제된다.

| 상황 | 접힌 부분 위치 |
|------|--------------|
| **앞면 읽기** | 접힌 부분이 **왼쪽**, 접은 부분이 종이 **뒤쪽** |
| **뒷면 읽기** | 접힌 부분이 **오른쪽**, 접힌 부분이 종이 **위** |

- 점자지를 접는 이유: 촉각으로 종이의 앞뒤와 위아래를 구분하기 위해

### 18.2. 점자지도의 실제

#### 점자지도 접근 방법 비교

> ⚠️ **기출 포인트**: 세 가지 접근 방법의 기본 단위(음소 vs 단어)와 손가락 이동 방식 차이를 비교하는 문제가 출제된다. 단어 접근의 "일촉단어 접근" 용어를 기억할 것.

| 접근 방법 | 기본 단위 | 특징 |
|----------|----------|------|
| **음소접근 방법** | 음소 | 한 번에 단 하나의 음소를 만짐. 읽기의 기본 단위인 음소를 중시 |
| **음소-단어 접근 방법** | 음소 | 손가락이 움직이면서 빠르게 연속적으로 여러 음소를 만져 지각. 음소 접근과 유사하나 속도가 빠름 |
| **단어 접근 방법** | 단어·절·문장 | 개별 음소 학습 전에 단어·절·짧은 문장을 먼저 학습. 손끝으로 단어를 전체로 인식(**일촉단어 접근**) |

#### 중복장애학생을 위한 점자지도

> ⚠️ **기출 포인트**: 점보점자와 표준점자의 차이(점 크기·모양은 동일, 가로 간격만 다름)를 묻는 문제와 개별화 의미 중심 접근의 어휘 선정 방식을 묻는 문제가 출제된다.

| 방법 | 대상 | 특징 |
|------|------|------|
| **점보점자** | 중복장애 학생 | 점 크기·모양은 표준점자와 동일. 점 및 셀 사이의 **가로 간격만** 약간 넓음 |
| **개별화 의미 중심 접근** | 중복장애 학생 | 의미를 중심으로 지도. 학생 개인별 실제 생활 기능에 필요한 어휘를 선택하여 점자로 제시 |

### 18.3. 악보 점자(Braille Music Notation)

시각장애 학생이 음악을 배우기 위한 점자 악보 체계.

| 항목 | 내용 |
|-----|------|
| **기본 단위** | 일반 문자 점자와 동일한 **6점 점자 셀** 사용 |
| **기호 체계** | 음표·쉼표·박자·조표·셈여림 등 고유 점 조합 — 문자 점자와 **다른 의미** |
| **표준** | 국제점자악보위원회(BANA: Braille Authority of North America) 기준 |
| **표기 방식** | 선형(linear) 순서 표기 — 위·아래 성부를 **동시에 좌우 병렬 표기하지 않음** |
| **연주 전략** | '기억-연주(read-memorize-play)' — 먼저 손으로 읽고 구절을 암기한 후 연주 |
| **학습 순서** | 한 성부를 먼저 암기 → 다른 성부 학습 (단계적 접근) |

> ⚠️ **기출 포인트**: 악보 점자는 위·아래 성부 **동시 병렬 표기 불가** → 순차 암기 후 연주. 6점 셀 사용이지만 문자 점자와 **점 조합 의미가 다름**.
