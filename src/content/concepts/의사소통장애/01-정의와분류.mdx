## 1. 정의

### 1-1. 특수교육법상 정의

다음 중 특별한 교육적 조치가 필요한 경우:

| 항목 | 내용 |
|-----|------|
| ① | 언어의 수용 및 표현 능력이 인지 능력에 비해 현저히 부족한 사람 |
| ② | 조음 능력이 현저히 부족하여 의사소통이 어려운 사람 |
| ③ | 말 유창성이 현저히 부족하여 의사소통이 어려운 사람 |
| ④ | 기능적 음성장애로 의사소통이 어려운 사람 |

### 1-2. ASHA(미국 언어-청각 협회) 정의

개념·구어·비구어·그래픽 상징체계를 **수용·전달·처리하는 능력의 손상**

- 청각·언어·말 처리 과정에서 나타날 수 있음
- 경도~최중도까지 다양; 발달적 또는 후천적
- **단일 또는 복합 장애, 다른 장애의 이차적 장애로 나타날 수 있음**

> ⚠️ **기출 포인트**: ASHA 정의에서 "다른 장애의 이차적 장애로 나타날 수 있음" — KICE 오지선다 단골.

## 2. 분류

### 2-1. 말장애 (Speech Disorders)

말소리 산출 자체의 문제 → **음성·흐름·명료도** 손상

| 유형 | 세부 분류 | 정의 |
|-----|---------|------|
| **조음음운장애** | 조음장애 | 조음기관을 통한 말소리 구성 결함 (생략·첨가·대치·왜곡) |
| | 음운장애 | 연령에 맞는 음운 지식·능력 부족, 음운 패턴 부정확 |
| **유창성장애** | 말더듬 | 말소리 반복·연장·막힘, 부수행동 동반 |
| | 말빠름증(속화) | 말 속도 지나치게 빠름, 비조직적·불규칙한 말 흐름 |
| **운동말장애** | 말실행증(AOS) | 근육 마비 없이 조음 운동 프로그래밍 어려움 (중추신경계) |
| | 마비말장애 | 중추·말초신경계 손상 → 근육 조정 장애 |
| **음성장애** | 기능적 | 성대 병변 없이 오용·남용 (성대결절, 성대폴립 등) |
| | 기질적 | 발성기관 구조적 손상 (성대 유두종, 후두암 등) |
| | 신경학적 | 성대 마비, 경련성 부전실성증 |

### 2-2. 언어장애 (Language Disorders)

언어의 형태·내용·사용의 이해 및 활동 손상

| 유형 | 정의 |
|-----|------|
| **언어발달장애** | 영유아기 언어 이해·표현 어려움; 인지·동작성 지능은 정상 |
| **실어증** | 후천적 신경계 손상 → 말하기·듣기·쓰기·읽기 모두 손상 |
| **단순언어장애(SLI)** | 감각·신경·정서·인지 장애 없이 언어 발달만 문제 |

## 3. 언어의 구성 영역 5가지

> ⚠️ **기출 포인트**: 형태 / 내용 / 사용 구별.

| 영역 | 분류 | 정의 |
|-----|------|------|
| **음운론** | 형태(form) | 언어의 소리 체계와 합성 규칙 |
| **형태론** | 형태(form) | 단어의 구조와 형태 구성 규칙 |
| **구문론** | 형태(form) | 문장 내 단어 순서·조합·요소 간 관계 |
| **의미론** | 내용(content) | 단어와 문장의 의미 |
| **화용론** | 사용(use) | 기능적·사회적으로 적절한 언어 사용 체계 |

## 4. 언어 습득 이론

### 4-1. 행동주의 이론 (Skinner)

언어 발달은 환경에 의해 결정; 강화가 핵심

| 개념 | 정의 |
|-----|------|
| **맨드(Mand)** | 요구 → 부모 충족 → 언어 형성; **가장 먼저 사용** |
| **텍트(Tact)** | 사물·사건에 이름 붙임; 사물과 접촉하며 낱말 습득 |
| **모방** | 말소리 듣고 비슷하게 소리 냄; 부모가 초기에 많이 활용 |
| **오토클래티스** | 다른 언어 행동 기능에 영향 주는 꾸밈어 |
| **상호적 언어** | 다른 언어 행동에서 생겨나는 언어 (질문·대화 반응) |

### 4-2. 생득주의 이론 (Chomsky)

언어 습득은 타고난 능력에 의함

- **언어습득장치(LAD)**: 인간이 언어 학습을 위해 준비된 장치를 가지고 태어남
- 최소한의 언어 환경 노출만으로 모국어 습득 가능
- **언어 습득의 결정적 시기** 존재

### 4-3. 구성주의 이론

환경과의 상호작용을 통해 지식을 재구성

| 유형 | 학자 | 핵심 |
|-----|------|------|
| **인지적 상호작용주의** | 피아제 | 인지 발달이 언어 발달에 영향 |
| **사회적 상호작용주의** | 비고츠키 | 사회문화적 배경·상호작용이 언어 발달에 중요 |

## 5. 언어발달 단계와 어휘 발달 특징

### 5-1. 언어발달 단계

| 단계 | 내용 |
|-----|------|
| **한 단어 시기** | 단어 하나로 의사소통 시작 |
| **두 단어 시기** | 두 단어 연결 → 초보 문장 |
| **전보식 문장** | 내용어(명사·동사) 사용, 기능어(조사·접속사) 생략 |

### 5-2. 어휘 발달 특이 현상

| 현상 | 정의 | 예시 |
|-----|------|------|
| **과잉확대** | 단어의 의미를 실제보다 광범위하게 사용 | 모든 남자를 '아빠'라고 부름 |
| **과잉축소** | 단어를 실제 의미보다 좁게 사용 | 아는 특정 의자만 '의자'라고 부름 |
| **과잉일반화** | 문법 규칙을 지나치게 일반화한 문법 오류 | 과거형을 모든 동사에 일괄 적용 |
| **주축문법** | 주축어(고정 위치, 단독 불가) + 개방어(합쳐지는 단어) 조합 | — |

### 5-3. 의미의 발달 방향

| 방향 | 설명 |
|-----|------|
| **수평적 발달** | 어휘 의미에 새로운 속성을 덧붙여 나감 |
| **수직적 발달** | 관련 단어들을 습득하며 어휘 개념 확장 |
