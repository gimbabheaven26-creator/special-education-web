## 1. 조음·음운장애

### 1-1. 정의 비교

| 구분 | 조음장애 | 음운장애 |
|-----|---------|---------|
| **문제 수준** | 조음기관의 말소리 구성 결함 | 음운 지식·능력 부족 |
| **오류 특성** | 특정 음소에서 일관된 오류 | 단어 내 음소 결합 시 오류, 비일관적 |
| **독립 음소 산출** | 어려울 수 있음 | 가능, 하지만 단어에서 오류 |

### 1-2. 주요 오류 형태 4가지

> ⚠️ **기출 포인트**: 4가지 오류 형태 정의와 예시 암기.

| 오류 형태 | 정의 | 예시 |
|---------|------|------|
| **생략** | 단어에서 특정 음소·음절이 빠짐 | '사자' → '아자' (초성 생략) |
| **첨가** | 필요 없는 음소·음절 삽입 | '기차' → '기으차' |
| **대치** | 목표 음소 대신 다른 음소로 발음 | '버스' → '버트' |
| **왜곡** | 목표 음을 변이음으로 발음 (음성학적 오류) | 알아듣기 어려우나 뚜렷한 대치 음소 없음 |

### 1-3. 치료 접근법

| 접근 유형 | 방법 | 특징 |
|---------|------|------|
| **전통적 접근법** | Van Riper 5단계 (확인→비교→변화→수정→안정) | 지각+산출 훈련, 단일 음소 접근 |
| **음운변동 접근법** | 비정상 음운 변동 패턴 제거 | 여러 오류음 동시 수정 |
| └ **주기법** | 주기별로 목표 변동·음소 교체 | 심한 조음·음운장애에 효과적 |
| **변별자질 접근법** | 오류 자질 패턴 분석 → 미습득 변별 자질 훈련 | — |
| └ 최소 대립자질 | 최소 대조 음소 비교 훈련 | — |
| └ 최대 대립자질 | 자질 차이 극대화 쌍 활용 | 심한 조음장애에 효과적 |
| **짝자극 기법** | 핵심단어(9/10 정조음) + 훈련단어(2/3↑ 오조음) 쌍 | 토큰 강화 활용 |
| **점근법** | 목표 음소 유사 중간 발음도 강화 → 점진적 접근 | 고착 위험(단점) |
| **의사소통 중심법** | 실생활 상황 속 자동적 조음 훈련 | 일반화 촉진 |

## 2. 유창성 장애

### 2-1. 말더듬

#### 핵심 행동 (1차·공통 행동)

> ⚠️ **기출 포인트**: 반복·연장·막힘 정의 구별.

| 핵심 행동 | 정의 |
|---------|------|
| **반복** | 말소리·음절·낱말을 1회 이상 되풀이 |
| **연장** | 한 번의 날숨에서 하나의 소리를 길게 지속 (0.5초~수 초) |
| **막힘** | 말 흐름이 부적절하게 중단, 조음 기관 고착 |

#### 부수 행동 (2차 행동)

말더듬에서 벗어나려는 과도한 노력으로 나타나는 신체적 행동

| 유형 | 정의 | 예시 |
|-----|------|------|
| **탈출 행동** | 말더듬 도중 빠져나오려는 행동 | 발 구르기, 눈 깜빡임 |
| **회피 행동** | 말더듬 가능성 있는 상황 회피 | 돌려 말하기, 순서 바꾸기, 대용어, 상황 회피 |

#### 말더듬 진행 4단계

| 단계 | 특성 |
|-----|------|
| **경계선** | 10% 이하 비유창성; 반복 2회↑; 2차 증상 없음 |
| **초기** | 반복 속도 빠르고 불규칙, 막힘 나타남 |
| **중간급** | 막힘 > 반복·연장; 탈출·회피 뚜렷 |
| **진전된** | 막힘 길게 지속, 부정적 자아 개념 |

#### Van Riper MIDVAS 6단계 (말더듬 수정법)

> ⚠️ **기출 포인트**: MIDVAS 두문자 기억.

| 단계 | 명칭 | 내용 |
|-----|------|------|
| 1 | **동기(Motivation)** | 치료사 신뢰 + 말더듬 직시·수용 |
| 2 | **확인(Identification)** | 1차·2차 증상·느낌·태도 스스로 확인 |
| 3 | **둔감(Desensitization)** | 공포·부정적 감정 감소; 청자 반응에 무감각 |
| 4 | **변형(Variation)** | 고착된 말더듬 형태 변형 |
| 5 | **접근(Approximation)** | 취소·이끌어내기·준비하기로 쉬운 말더듬 형태 접근 |
| 6 | **안정(Stabilization)** | 치료실 밖 검증, 두려운 상황에서 의도적 말하기 |

**접근 단계의 세 기법:**
- **취소**: 더듬은 후 잠시 쉬고 다시 편안하게 시도
- **이끌어내기**: 막히면 멈추고 천천히 부드럽게 이어감
- **준비하기**: 더듬을 것 예상되는 단어에서 의식적으로 조절

#### 유창성 완성법

최종 목표: **자발적 유창성** ('수용적 말더듬' 인정하지 않음)
- 호흡 훈련, 천천히 말하기(DAF 활용), 휴지와 분절화 기법

### 2-2. 말빠름증(속화)

자신의 구어에 문제가 있다는 것을 **인식하지 못함**

| 특이 현상 | 정의 |
|---------|------|
| **스코핑** | 두 개 이상의 단어를 축약해서 발화 |
| **스프러니즘** | 두 개 이상 단어에서 어두음을 무의식적으로 교환 |
| **프로포리즘** | 문장에서 단어를 우스꽝스럽게 잘못 사용 |

## 3. 운동말장애

### 3-1. 말실행증(AOS) vs 마비말장애 비교

> ⚠️ **기출 포인트**: 핵심 차이 — **오류 일관성**.

| 항목 | 말실행증(AOS) | 마비말장애 |
|-----|------------|---------|
| **손상 기제** | 조음 운동 프로그래밍 어려움 | 말 기제 근육 조정 장애 |
| **근육 마비·약화** | **없음** | 있음 |
| **오류 일관성** | **비일관적** | **일관된** 조음 오류 |
| **모색 현상** | **있음** (입술을 계속 움직이며 위치 탐색) | 없음 |
| **언어 능력** | 언어적 측면 정상 | 언어적 측면 손상 없음 |
| **자동 발화** | 의도적 발화 < 자동 발화 | — |

**말실행증 치료**: 단순한 자극 → 복잡·긴 자극 점진적 확장; 속도 늦추기; 시각적 단서

## 4. 음성장애

### 4-1. 원인에 따른 분류

| 유형 | 원인 | 대표 질환 |
|-----|------|---------|
| **기능적 음성장애** | 성대 오용·남용 | 성대 비대증, 성대 결절, 성대 폴립, 변성기 음성장애 |
| **기질적 음성장애** | 발성기관 구조적 손상 | 성대 유두종, 접촉성 궤양, 후두암, 라인케 부종 |
| **신경학적 음성장애** | 신경계 손상 | 성대 마비, 경련성 부전실성증, 파킨슨병 |

### 4-2. 기능적 음성장애 주요 유형

| 질환 | 원인 | 음성 특성 |
|-----|------|---------|
| **성대 비대증** | 지속적 음성 남용·오용 | 성대 가장자리 확장 → 숨소리 |
| **성대 결절** | 성대 비대 장기 방치 → 굳은살 | 거친 소리; **양측성**, 지속적 남용·오용 |
| **성대 폴립** | 순간적 음성 남용 | 목소리 쉼; **편측성**, 30~50대 남성 多 |
| **변성기 음성장애** | 16세↑ 변성기 미완료 | 후두 근육 협응 문제 |

### 4-3. 음성 위생 교육 (간접 치료)

- 가습기, 마스크, 충분한 수분 섭취
- 소음 심한 곳 발성 자제, 귓속말·고성 금지
- 흡연·음주·카페인·유제품 자제
- 목 가다듬기·헛기침 금지
