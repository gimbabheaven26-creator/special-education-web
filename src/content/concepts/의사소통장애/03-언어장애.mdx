## 1. 화용론과 담화 기술

### 1-1. 화용론 핵심 개념

| 개념 | 정의 |
|-----|------|
| **직시** | 인칭대명사·지시대명사·장소부사·시간부사 등 가리키고 지시하는 것 |
| **전제** | 발화에 부수적으로 전달되는 의미; 상대방 배경 지식 파악 필요 |
| **함축** | 명시적 표현 외에 추가로 전달되는 의미 |
| **화행** | 발화를 통해 행동이 수반 (직접 화행·간접 화행) |
| **대화체 구성** | 제3자의 말을 인용하거나 행동 전달 시 새 대화체 구성 |

### 1-2. 담화 기술 3가지

> ⚠️ **기출 포인트**: 3가지 담화 기술 + 발화 수정 전략 vs 명료화 요구 전략 구별.

| 담화 기술 | 정의 |
|---------|------|
| **말차례 주고받기** | 화자·청자가 차례대로 말하는 대화의 기본 |
| **대화 주제 관리** | 주제를 시작·유지·전환하는 능력 |
| **의사소통 실패 해결** | 실패 원인 파악 후 수정 시도 |

- **발화 수정 전략(화자)**: 자신이 잘못 말한 것을 분석·수정
- **명료화 요구 전략(청자)**: 이해 못한 부분에 다시 말해달라 요구

## 2. 실어증

### 2-1. 정의

후천적 뇌 손상(신경계 손상)으로 생기는 언어 장애 — 언어 능력이 완전히 상실된 것이 아니라 **사용의 효율성 저하**

- 말하기·듣기·쓰기·읽기 4가지 영역 모두 손상

### 2-2. 유형 비교 (보스턴 분류)

> ⚠️ **기출 포인트**: 브로카 vs 베르니케 비교표 — 유창성·청각적 이해력·따라 말하기·특이 증상.

| 유형 | 손상 부위 | 유창성 | 청각적 이해력 | 따라 말하기 | 특이 증상 |
|-----|---------|------|------------|---------|---------|
| **베르니케** | 측두엽 | **유창** | **손상** | 손상 | 의미착어, 자곤, 신조어, 언어 상동증 |
| **브로카** | 전두엽 | **비유창** | **보존** | 손상 | 비문법성(전보식), 음소착어, 자가수정 |
| **전도** | 베르니케→브로카 연결로 | 유창 | 보존 | **두드러진 손상** | 음소착어 |
| **이름** | — | 유창 | 보존 | 보존 | **이름 대기 유독 저하** |
| **전반** | 광범위 | 비유창 | 심한 손상 | 심한 손상 | 가장 심한 실어증 |
| **혼합 초피질** | 브로카+베르니케 주변 | 비유창 | 심한 손상 | **가능** | 두 영역 특성 모두 |

### 2-3. 실어증 치료

| 방법 | 대상 | 핵심 |
|-----|------|------|
| **멜로디 억양 치료법(MIT)** | 브로카 실어증 (우반구 손상 없음) | 노래 형식으로 발화 길이 점진적 확장 |
| **시각적 동작 치료(VAT)** | 전반 실어증 | 제스처 최대 활용, 기능적 의사소통 접근 |

## 3. 단순언어장애(SLI)

### 3-1. 정의

감각·신경·정서·인지 장애 없이 **언어 발달에만 문제**를 보이는 장애

### 3-2. 판별 기준 6가지 (Leonard, 1998)

> ⚠️ **기출 포인트**: 6가지 모두 충족해야 SLI 진단.

| 기준 | 내용 |
|-----|------|
| ① | 표준화 언어 검사에서 **–1.25 표준편차 이하** |
| ② | 비언어성(동작성) 지능 검사 **85 이상** |
| ③ | **청력 이상 없음** |
| ④ | **신경학적 이상 없음** |
| ⑤ | 구강 구조·기능 이상 없음 |
| ⑥ | 사회적 상호작용 능력에 심각한 이상·장애 없음 |

### 3-3. 언어적 특성

| 영역 | 특성 |
|-----|------|
| **음운론** | 말소리 습득 지체, 오류 형태 음운론 더 오래 지속 |
| **형태론·구문론** | 정상 아동과 가장 큰 차이; 문법 형태소 이해·오류 인지 어려움 |
| **의미론** | 첫 낱말 출현 지체, 어휘 습득 속도 느림 |
| **화용론** | 말 대신 제스처 사용 |

## 4. 교사 발화 전략

### 4-1. 발화 유도 전략

| 기법 | 정의 |
|-----|------|
| **혼잣말 기법** | 교사가 자기 자신의 행위를 말함 |
| **평행적 발화 기법** | 교사가 아동 입장에서 아동의 행동을 말로 묘사 |

### 4-2. 발화 후 언어 자극 전략

> ⚠️ **기출 포인트**: 확장·확대·재구성 구별.

| 기법 | 정의 |
|-----|------|
| **확장** | 문법 오류 있는 아동 표현 → 문법적 완전 형태로 교정 |
| **확대** | 아동 발화의 **의미**를 보완 |
| **교정적 피드백** | 잘못된 표현을 긍정적 방법으로 교정 |
| **재구성** | 아동 표현을 **다른 문장 구조**로 바꿔 말함 |
| **수정** | 잘못된 발화를 직접적으로 고침 |
| **자기 수정 요청** | 교사가 잘못 말한 부분을 그대로 따라 → 아동 스스로 수정 유도 |
