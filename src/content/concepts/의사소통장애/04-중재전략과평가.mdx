## 1. 환경중심 언어중재 (MT / EMT)

**목적**: 기능적 언어 습득 — 의사소통 빈도 증가, 더 길고 복잡한 발화 산출

### 1-1. 기본 가정

- 자연적인 환경에서 수행
- 아동에게 영향을 미치는 여러 사람이 수행
- 기능적 언어를 가르쳐야 함
- **아동 중심**, 언어의 형태·기능·전략 동시 교수

### 1-2. 환경 조성 전략 (물리적 환경 구조화)

아동의 의사소통 가능성을 높이기 위한 환경 배치:
- 흥미 있는 사물 활용, 손에 닿지 않는 곳에 두기
- 조금 부족하게 주기, 선택 상황 만들기
- 중요 요소가 빠진 상황, 도움 필요 상황, 엉뚱한 상황 만들기

### 1-3. 환경 교수 절차 4가지

> ⚠️ **기출 포인트**: 4가지 절차 — 적용 조건이 각각 다름.

| 절차 | 적용 조건 | 방법 |
|-----|---------|------|
| **우발교수** | 아동이 자발적으로 의사소통 시 | 아동 흥미·요구로 시작 → 자발 표현 시 칭찬+강화+언어적 확장 |
| **시범** | 언어 습득 **결함** 시 | 아동이 관심 보인 것에 언어적 시범 → 모방 유도 |
| **요구-시범** | 언어 습득 완료, **수행 결함** 시 | 관심 보인 것에 언어적 요구 또는 지도 |
| **시간 지연** | **유창성 결함** 시 | 시범/요구-시범 전 자발 반응 시간 제공 |

**성공 전략**: 유인력(선행 자극 매력적), 정확한 모델링, **최소 3초 반응 대기**, 반응 후 **2~3초 내 강화** 제공

## 2. 반응적 대화양식 (반응적 상호작용)

### 2-1. 요소 4가지

| 요소 | 정의 |
|-----|------|
| **공동 관심** | 교사·학생이 같은 사물·활동에 집중 |
| **상호 관심** | 서로를 바라보며 대화 |
| **차례 주고받기** | 대화 순서 교환 |
| **긍정적 감정 표현** | 상호작용에서 긍정 정서 유지 |

### 2-2. 대화 전략

- 의사소통 차례 주고받기
- 학생의 의사소통에 반응하기
- 학생의 목표 수준에서 이야기하기
- 학생의 행동 반영하기 + 학생의 주도 따르기
- 학생의 의사소통·활동 확장하기

## 3. 스크립트 문맥 언어치료

**정의**: 특정 상황의 대본(스크립트)을 활용하여 그 상황에서의 적절한 언어 사용 지도

| 항목 | 내용 |
|-----|------|
| **장점** | 상황에 맞는 언어를 가장 **일반화된 형태**로 지도 가능 |
| **단점** | 최소한의 **구어적 능력** 필요 (AAC와 대비) |
| **AAC와 차이** | AAC = 비구어적 의사소통 목표; 스크립트 = **구어 사용** 초점 |
| **전통 언어지도와 차이** | 전통 = 구문·어휘 강조; 스크립트 = 실제 상호작용 속 자연스러운 습득 |

## 4. 참조적 의사소통

**정의**: 특정 참조물에 대해 다른 사람과 정보를 주고받는 화용적 기술 (화자·청자가 공유하지 않는 정보 전달)

| 필요 능력 | 정의 |
|---------|------|
| **전제** | 상대방이 어떤 배경 지식을 가지고 있는지 판단 후 말하는 방식 조절 |
| **결속 표지(직시)** | 가리킴말·접속사·연결어미를 사용하여 문장 간 관계 명확화 |
| **피드백** | 반응이 적절치 못하면 전제를 바꾸어 다시 표현 |

- 화자 기술이 청자 기술보다 **먼저 발달**

## 5. 자발화 분석

비표준화 검사 — 아동의 평소 언어를 수집·분석

### 5-1. 언어 표본 수집 기준

- **크기**: 50~200 발화, 녹화 30분
- **수집 방법**: 개방형 질문 > 폐쇄형; 아동 주도 따르기; 질문 자제

### 5-2. 발화 제외 기준

| 제외 상황 |
|---------|
| 아동이 스스로 수정하거나 의미 첨가 없이 반복한 낱말·구 → 괄호로 구분하고 제외 |
| 노래하기, 숫자 세기 등 자동 구어 |
| 습관적 간투사, 불명료 발화, 단순 반응 ('예', '아니요', '음') |
| 앞부분 5~10개 발화 (라포 형성 전) |

### 5-3. 구문론적 분석 — MLU

> ⚠️ **기출 포인트**: MLU-m과 MLU-w 계산 공식 차이.

| 측정치 | 계산법 | 의미 |
|------|------|------|
| **MLU-m** (평균 형태소 길이) | 전체 형태소 수 ÷ 총 발화 수 | 값 증가 = 문장 길이↑ + 구조적 복잡↑ |
| **MLU-w** (평균 낱말 길이) | 총 낱말 수 ÷ 총 발화 수 | 한 발화 내 단어 수 많을수록 높음 |
| **MSL** (평균 구문 길이) | 2개↑ 형태소 발화만 분석; 총 형태소 ÷ 총 발화 수 | 1형태소 발화 제외 |

### 5-4. 의미론적 분석 — TTR (어휘 다양도)

**TTR = NDW(다른 낱말 수) ÷ NTW(총 낱말 수)**

얼마나 다양한 낱말을 사용하는가를 측정 (품사별로 기록, 활용 형태는 1낱말로 계산)

### 5-5. 자발화 분석 의미론 분류

| 의미 유형 | 예시 |
|---------|------|
| **체언** | 명사·대명사·수사; 어미 활용 없음 (조사의 도움으로 기능) |
| **용언** | 동사·형용사; 행위·상태·부정 서술 |
| **수식언** | 체언·용언·수식언 수식; 때·장소·도구·이유 등 |
| **기능적 구성요소(CD)** | 의미 관계 없이 독립 기능; 주의 끌기·인사·접속 등 |
