## 1. 마음이론 (Theory of Mind) 개요

### 1-1. 정의와 Baron-Cohen의 주장

마음이론(ToM)은 **자신과 타인이 서로 다른 믿음·의도·욕구·감정을 가질 수 있음**을 이해하는 능력이다.
Baron-Cohen(1985)은 ASD 아동이 타인의 정신 상태를 표상하지 못하는 **마음맹(mindblindness)** 상태에 있다고 주장했다.

| 구분 | 내용 |
|------|------|
| **마음이론 정상 발달 시기** | 만 3~4세 (1차 믿음), 만 6~7세 (2차 믿음) |
| **ASD에서의 양상** | 대다수 ASD 아동은 정신 연령에 비해 현저히 지연 또는 결함 |
| **마음맹** | ToM 결함으로 타인의 의도·감정을 읽지 못하는 상태 |

> ⚠️ **기출 포인트**: Baron-Cohen은 ASD의 핵심 결함을 마음이론 결함으로 설명했다. Sally-Anne 과제에서 ASD 아동의 약 80%가 실패한 반면, 발달장애(지적장애) 통제집단은 약 86%가 통과했다.

## 2. 틀린믿음 과제 (False Belief Task) 유형

### 2-1. 1차 믿음 과제 (First-Order False Belief)

타인이 현실과 다른 **잘못된 믿음**을 가지고 있음을 이해하는가를 측정.

**Sally-Anne 과제:**

| 장면 | 내용 |
|------|------|
| ① Sally가 공을 바구니에 넣고 자리를 비운다 | — |
| ② Sally가 없는 동안 Anne이 공을 상자로 옮긴다 | — |
| ③ Sally가 돌아온다 | — |
| **질문** | "Sally는 공을 어디서 찾겠는가?" |
| **정답 (ToM 통과)** | 바구니 (Sally의 잘못된 믿음을 이해) |
| **ASD 아동 오답** | 상자 (공의 실제 위치를 대답 — 타인의 믿음을 자신의 지식과 분리 못 함) |

**스마티 과제 (Smarties Task):**

| 장면 | 내용 |
|------|------|
| ① 아이에게 스마티(과자) 통을 보여준다 | — |
| ② 열어보면 연필이 들어있다 | — |
| **질문** | "처음에 이 통 안에 뭐가 있다고 생각했어?" / "친구가 이 통을 보면 뭐가 있다고 할까?" |
| **정답 (ToM 통과)** | 스마티 (자신의 사전 믿음/타인의 믿음 이해) |
| **ASD 아동 오답** | 연필 (현재 지식을 과거 믿음/타인 믿음에 적용) |

### 2-2. 2차 믿음 과제 (Second-Order False Belief)

"A는 B가 ~라고 생각한다고 알고 있다"와 같이 **믿음에 대한 믿음**을 이해하는가를 측정.

| 수준 | 예시 질문 | 통과 연령(일반 아동) |
|------|---------|----------------|
| **1차 믿음** | "Sally는 공이 어디 있다고 생각할까?" | 만 3~4세 |
| **2차 믿음** | "Maxi는 엄마가 초콜릿을 어디 있다고 생각한다고 생각할까?" | 만 6~7세 |

> ⚠️ **기출 포인트**: 고기능 자폐(HFA) 또는 아스퍼거 증후군 학생은 1차 믿음 과제를 통과하는 경우도 있으나, 2차 믿음 과제에서 실패하거나 일상적 사회적 상황에서 여전히 어려움을 보인다.

## 3. 마음맹 (Mindblindness)

Baron-Cohen(1995)이 제시한 개념으로, ToM 결함으로 인해 **사람들의 마음 상태를 "볼 수 없는" 상태**.

| 마음맹의 사회적 결과 | 설명 |
|------------------|------|
| 타인의 의도 오해 | 농담·비꼬기·거짓말을 문자 그대로 해석 |
| 공감 어려움 | 타인의 감정 상태를 추론하지 못함 |
| 화용론적 실패 | 대화의 맥락·의미·상대 의도를 고려하지 못함 |
| 사회적 규범 위반 | 상황에 맞는 행동 선택 어려움 |

**교육적 함의:**
- 사회적 단서를 명시적·체계적으로 교수
- "그 사람은 어떤 마음일까?" 를 반복 연습하는 관점 취하기 훈련
- 사회적 상황 이야기(Social Story), 코믹북 대화(Comic Strip Conversation) 활용

## 4. 공동주의 (Joint Attention)

### 4-1. 정의와 발달 시기

공동주의란 **두 사람이 같은 대상이나 사건에 주의를 함께 향하는 것**으로, 사회적 의사소통의 기초가 된다.

| 유형 | 정의 | 목적 | 발달 시기 |
|------|------|------|---------|
| **원시 명령적 공동주의** | 원하는 물건을 얻기 위해 타인의 주의를 끄는 것 | 수단적 (도구로서의 타인) | 약 9~12개월 |
| **원시 선언적 공동주의** | 흥미로운 것을 함께 공유하기 위해 타인의 주의를 끄는 것 | 공유적 (사회적 참조) | 약 11~14개월 |

> ⚠️ **기출 포인트**: ASD 아동은 원시 명령적 공동주의는 상대적으로 보존되는 경우가 있으나, **원시 선언적 공동주의가 현저히 결핍**된다. 선언적 공동주의의 결핍이 ToM 발달 부재와 직접 연결된다.

### 4-2. ASD에서의 공동주의 특성

| 발달 영역 | 일반 발달 | ASD |
|---------|---------|-----|
| 시선 추적 | 12개월경 자연스럽게 발달 | 결핍 또는 지연 |
| 손가락 가리키기 (선언적) | 12~14개월 | 현저히 감소 또는 부재 |
| 손가락 가리키기 (명령적) | 9~12개월 | 상대적으로 보존 |
| 사회적 참조 | 10~12개월 | 부재 또는 감소 |

**교육적 지원:**
- 공동주의 기술을 개별화교육프로그램(IEP) 목표로 설정
- 자연스러운 활동(그림책 읽기, 놀이) 중 공동주의 기회 확보
- 교사가 학생의 시선을 따르며 언어적으로 명명: "맞아, 그거 보고 있구나. 기차네!"

## 5. 정서 인식 어려움

### 5-1. 얼굴 표정 해석

ASD 아동은 얼굴 표정에서 정서를 인식할 때 **비전형적인 시각 처리 전략**을 사용한다.

| 특성 | 내용 |
|------|------|
| 눈 영역 회피 | 눈보다 입·코 영역에 더 많이 주목 |
| 얼굴 전체보다 부분 처리 | 전체적 얼굴 게슈탈트보다 세부 특징으로 처리 |
| 역전된 얼굴 효과 감소 | 얼굴 처리의 특수성 결핍 |

### 5-2. 맥락 기반 정서 이해

기본 정서(기쁨, 슬픔, 분노, 공포)는 인식할 수 있어도, **복합 정서**(당혹, 질투, 죄책감)나 **맥락에 따른 정서 변화**는 이해하기 어려움.

| 정서 유형 | ASD 인식 수준 | 교수 접근 |
|---------|-----------|---------|
| 기본 정서 (6가지) | 상대적으로 가능 | 그림·사진 카드 매칭 |
| 복합 정서 | 어려움 | Baron-Cohen의 Mind Reading 소프트웨어 |
| 맥락 기반 정서 | 매우 어려움 | 역할극·비디오 모델링 |

> ⚠️ **기출 포인트**: Baron-Cohen의 "눈으로 마음 읽기(Reading the Mind in the Eyes)" 검사는 눈 사진만 보고 정서를 추론하는 과제로, ASD 성인도 일반 성인에 비해 유의하게 낮은 수행을 보인다.

## 6. ESDM (Early Start Denver Model, 얼리 스타트 덴버 모델)

### 6-1. 개요 및 배경

Rogers와 Dawson이 개발한 **관계 중심 발달 접근 + ABA 원리**를 통합한 초기 집중 중재 모델. 12~48개월 ASD 영·유아를 대상으로 한다.

| 구분 | 내용 |
|------|------|
| **이론적 기반** | 발달심리학 + ABA (행동주의) + 관계 중심 접근 |
| **중재 강도** | 주 20~25시간 이상 권장 |
| **환경** | 자연적 환경 (가정, 놀이터, 어린이집) |
| **핵심 원리** | 긍정적 정서·관계 형성이 학습의 기반 |

### 6-2. ESDM의 핵심 구성 요소

| 구성 요소 | 설명 |
|---------|------|
| **공동 활동 루틴** | 아동과 성인이 함께 하는 즐거운 활동을 반복 구조화 |
| **공동주의 집중 촉진** | 놀이 내에서 공동주의 기회를 체계적으로 삽입 |
| **모방 교수** | 아동의 행동 모방 → 아동도 성인을 모방하도록 유도 |
| **의사소통 기회 확대** | 모든 상호작용에서 언어·비언어 의사소통 기회 내장 |
| **부모 코칭** | 부모가 ESDM 전략을 일상에서 실행할 수 있도록 집중 훈련 |

### 6-3. DTT와의 비교

| 구분 | DTT | ESDM |
|------|-----|------|
| **환경** | 구조화된 테이블 환경 | 자연적 놀이 환경 |
| **주도** | 교사 주도 | 아동 주도 + 교사 삽입 |
| **목표** | 개별 기술 습득 | 사회적 참여·관계 + 기술 발달 |
| **관계** | 교사-학생 역할 명확 | 공동 파트너 관계 강조 |
| **부모 참여** | 선택적 | 핵심 구성 요소 |

> ⚠️ **기출 포인트**: ESDM은 ABA와 발달 접근을 통합한 모델로, 2010년 Dawson 등의 RCT 연구에서 IQ, 언어, 사회성 영역에서 유의한 향상을 보고했다. 임용시험에서 ESDM의 두 이론적 기반(ABA + 관계 중심 발달)을 묻는 문항이 출제된다.

## 7. 사회성 발달 지원 전략 요약

| 영역 | 전략 | 근거 이론 |
|------|------|---------|
| ToM 교수 | 관점 취하기 훈련, 코믹북 대화, 마음 읽기 교육 | Baron-Cohen ToM 결함 |
| 공동주의 | 일상 루틴 내 공동주의 삽입, 시선 따르기 지원 | 사회적 의사소통 발달 |
| 정서 인식 | 정서 카드 매칭, 비디오 모델링, Mind Reading SW | 얼굴 처리 비전형성 |
| 초기 집중 | ESDM (12~48개월), 관계 중심 + ABA 통합 | 신경가소성, 조기 중재 |
