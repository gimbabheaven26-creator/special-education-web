## 1. ASD 도전행동 유형

도전행동(challenging behavior)이란 **자신이나 타인에게 해를 끼치거나 학습·사회 참여를 방해하는 행동**이다.

| 도전행동 유형 | 예시 | ASD에서의 빈도 |
|-----------|------|-------------|
| **자해행동 (SIB)** | 머리 박기, 손 물기, 눈 찌르기 | 상대적으로 높음 |
| **공격행동** | 때리기, 물기, 할퀴기 | 상황적으로 발생 |
| **상동행동** | 손 흔들기, 반복적 발성, 물건 돌리기 | 매우 흔함 |
| **탈출·도주행동** | 갑작스럽게 교실/활동 이탈 | 회피 기능과 연관 |
| **기물 파손** | 물건 던지기, 환경 파괴 | 분노·좌절 상황 |

## 2. 도전행동의 4가지 기능

기능행동평가(FBA: Functional Behavior Assessment)에서 확인하는 **행동이 유지되는 이유(기능)**는 크게 4가지로 분류된다.

| 기능 | 설명 | 강화 유형 | 대표 예시 |
|------|------|---------|---------|
| **① 주의 획득 (Attention)** | 타인의 관심·반응을 얻기 위한 행동 | 사회적 강화 (정적) | 교사가 달래주는 반응으로 유지 |
| **② 회피/도피 (Escape/Avoid)** | 과제·상황·사람을 피하거나 끝내기 위한 행동 | 사회적 강화 (부적) | 어려운 과제 제시 시 자해 → 과제 중단 |
| **③ 자동강화 (Automatic/Sensory)** | 외부 결과 없이 행동 자체의 감각적 자극이 강화 | 자동 강화 | 손 흔들기, 특정 소리 반복 |
| **④ 유형적 결과 (Tangible)** | 원하는 물건·활동을 얻기 위한 행동 | 사회적 강화 (정적) | 원하는 물건이 거부되자 공격행동 |

> ⚠️ **기출 포인트**: 동일한 행동(예: 자해)이 아동마다 다른 기능을 가질 수 있다. 반드시 FBA를 통해 기능을 파악한 후 중재를 설계해야 한다. 기능을 잘못 파악하면 중재가 역효과를 낳는다.

### 기능별 중재 방향 개요

| 기능 | 잘못된 중재 (오류 예) | 올바른 방향 |
|------|------------------|---------|
| 주의 획득 | 행동 발생 후 주의를 줌 (강화) | 적절한 주의 요청 방법 교수 + 행동 소거 |
| 회피 | 행동 발생 후 과제 중단 (강화) | 과제 수정 + 적절한 휴식 요청 교수 |
| 자동강화 | 사회적 무관심 (효과 없음) | 감각 기능 대체 행동 제공 |
| 유형적 | 행동 후 원하는 물건 제공 (강화) | 적절한 요청 방법 교수 |

## 3. FCT (기능적 의사소통 훈련) 심화 절차

03번 파일에서 FCT 기본 절차를 다루었다. 여기서는 **FCT의 단계별 심화 절차와 유지·일반화 전략**에 집중한다.

### 3-1. FCT 단계별 절차

| 단계 | 내용 | 핵심 고려사항 |
|------|------|------------|
| **1단계: FBA 실시** | 직접·간접 관찰, ABC 기록, 기능 가설 수립 | 기능 파악 없이 다음 단계 진행 불가 |
| **2단계: 대체 의사소통 수단 선정** | 기능과 동등한 의사소통 반응 선택 | 아동의 현재 의사소통 능력에 맞는 수단 (구어/PECS/AAC) |
| **3단계: 대체 반응 교수** | 최소 촉진으로 대체 반응 교수, 즉각 강화 | 도전행동과 동등하거나 더 즉각적인 강화 제공 |
| **4단계: 도전행동 소거** | 도전행동에 대한 강화 중단 | 소거 발작(extinction burst) 대비 필요 |
| **5단계: 변별 훈련** | 의사소통이 가능한 맥락 vs 불가한 맥락 구분 교수 | 모든 상황에서 대체 반응이 강화될 수 없음을 점진적으로 학습 |
| **6단계: 유지·일반화** | 다양한 환경·사람·상황으로 확장 | 부모·보조인력 등 다수 중재자 훈련 |

### 3-2. 동등 기능 대체행동 선정 기준

| 기준 | 설명 |
|------|------|
| **기능 동등성** | 도전행동과 동일한 기능(강화)을 획득할 수 있어야 함 |
| **효율성** | 도전행동보다 더 빠르고 쉽게 강화를 얻을 수 있어야 함 |
| **사회적 수용성** | 다양한 환경에서 적절하게 인정되는 행동 |
| **현재 능력 적합** | 아동의 현재 의사소통 레퍼토리 내에 있거나 쉽게 학습 가능 |

> ⚠️ **기출 포인트**: FCT에서 대체 의사소통이 도전행동보다 **비효율적**이면(더 많은 노력, 더 느린 강화 획득) 아동은 기존 도전행동으로 돌아간다. 대체 반응이 더 '쉽고 빠른 길'이 되도록 설계하는 것이 성공의 핵심이다.

## 4. 상동행동 기능 분석과 중재

### 4-1. 상동행동의 기능 분석

상동행동은 단순 제거 대상이 아니라, **기능을 파악한 후 맞춤 중재**가 필요하다.

| 상동행동 예시 | 추정 기능 | 판단 근거 |
|-----------|---------|---------|
| 손 흔들기 (자동적) | 자동강화 (감각 자극 추구) | 혼자 있어도 발생, 사회적 맥락 무관 |
| 반복적 발성 (주의) | 주의 획득 | 타인 존재 시 증가, 반응 시 소멸 |
| 앞뒤 흔들기 (불안) | 자동강화 (자기 진정) | 전이·변화 상황에서 증가 |
| 물건 돌리기 (탐색) | 자동강화 (시각 자극 추구) | 혼자 있어도 동일, 강화물 제거 후 지속 |

### 4-2. 기능에 따른 상동행동 중재

| 기능 | 중재 접근 | 구체 전략 |
|------|---------|---------|
| **자동강화 (감각 추구)** | 동일 감각의 대체 활동 제공 | 피젯 도구, 감각식이 내 대체 활동 시간 배정 |
| **자동강화 (자기 진정)** | 불안 원인 제거 + 대체 진정 전략 | 시각적 일정표로 예측가능성 제공, 심호흡 교수 |
| **주의 획득** | FCT — 적절한 주의 요청 방법 교수 | "봐주세요" 카드, 탭하기 |
| **환경 자극 회피** | 과자극 환경 수정 | 감각식이, 감각 코너 접근 허용 |

## 5. 선행사건 중재 3유형

선행사건 중재는 **도전행동이 발생하기 전에 배경 또는 선행 조건을 수정**하여 행동 발생을 예방하는 접근이다.

### 5-1. 배경사건 수정 (Setting Event Modification)

도전행동과 직접 연결되지 않지만 **행동 발생 가능성을 높이는 배경 조건**을 제거·수정한다.

| 배경사건 예 | 수정 전략 |
|---------|---------|
| 수면 부족 | 수면 위생 지원, 가정 연계 |
| 배고픔 | 수업 전 간식 제공 |
| 전날의 부정적 경험 | 등교 시 안정화 루틴 제공 |
| 약물 효과 변화 | 의료팀과 협력, 복약 시간 확인 |

### 5-2. 선행사건 수정 (Antecedent Modification)

도전행동을 유발하는 **직접 선행자극을 제거·변경**한다.

| 도전행동 유발 선행자극 | 수정 전략 |
|------------------|---------|
| 어려운 과제 제시 | 과제 난이도 조정, 과제 분할 |
| 전이 (활동 전환) | 사전 경고 신호 제공, 시각적 타이머 |
| 선호 활동 종료 | "먼저-그 다음" 보드로 후속 활동 예고 |
| 요구 과부하 | 선택 기회 제공, 요구 수 감소 |

### 5-3. 변별 자극 조작 (Discriminative Stimulus Manipulation)

도전행동이 **허용되지 않는 맥락의 단서를 명확히** 하거나, 적절한 행동에 대한 단서를 강화한다.

| 전략 | 설명 | 예시 |
|------|------|-----|
| 토큰 경제 조건 명확화 | 어떤 상황에서 어떤 행동으로 강화를 얻는지 명시 | 시각적 토큰 보드 |
| 허용·금지 단서 시각화 | 허용 맥락과 금지 맥락의 시각적 단서 제공 | 녹색/빨간색 카드 |
| 촉진 전략 (prompting) | 적절한 행동 수행을 위한 단서 제공 후 점진적 소거 | 신체·제스처·시각 촉진 위계 |

> ⚠️ **기출 포인트**: 선행사건 중재는 "반응 중재(도전행동 발생 후 처리)"와 구분된다. 임용에서 배경사건(setting event)과 선행사건(antecedent)의 개념적 차이를 묻는 문항이 자주 출제된다. 배경사건은 선행사건보다 시간적으로 앞서거나 더 원거리에서 행동에 영향을 미친다.

## 6. 위기지원계획 (Crisis Support Plan)

### 6-1. 위기지원계획의 필요성

행동지원계획(BSP)만으로 즉각 행동 변화가 이루어지지 않는 경우, **위기(자해·타해 위험) 상황에서의 단계별 대응** 절차를 사전에 계획해야 한다.

### 6-2. 위기지원계획 구성 요소

| 단계 | 명칭 | 내용 |
|------|------|------|
| **1단계** | 위기 예방 | 도전행동 유발 배경사건·선행사건 목록 + 예방 전략 |
| **2단계** | 경고 신호 파악 | 위기 행동 전 나타나는 초기 신호 (발성 변화, 안절부절, 상동 증가) |
| **3단계** | 초기 대응 | 경고 신호 감지 시 즉각 적용할 de-escalation 전략 |
| **4단계** | 위기 대응 | 자해/타해 발생 시 안전 확보 절차 (다른 학생 이동, 안전 보조 기법) |
| **5단계** | 사후 지원 | 위기 후 아동·교사·학급의 회복 지원, 사건 기록 및 BSP 검토 |

### 6-3. De-escalation 전략 (초기 대응)

| 전략 | 설명 |
|------|------|
| **언어 최소화** | 언어 자극 줄이기 (단어 수 최소화, 차분한 톤) |
| **공간 확보** | 아동과의 물리적 거리 유지, 탈출 경로 확보 |
| **선택 제공** | 제한된 선택으로 통제감 회복 ("의자에 앉을래, 바닥에 앉을래?") |
| **감각 지원** | 진정 효과의 감각 자극 제공 (웨이티드 담요, 조용한 공간 이동) |
| **요구 일시 중단** | 학업·과제 요구를 일시 중단하고 안전 확보 우선 |

### 6-4. 선행중재 vs 결과중재 비교

| 구분 | 선행중재 (Antecedent Intervention) | 결과중재 (Consequence Intervention) |
|------|--------------------------------|----------------------------------|
| **적용 시점** | 행동 발생 전 | 행동 발생 후 |
| **목적** | 도전행동 예방 | 행동 감소 또는 대체행동 강화 |
| **예시** | 과제 수정, 선택 제공, 배경사건 수정 | FCT, 소거, 차별강화 (DRO/DRI/DRA) |
| **효과** | 장기적 예방 효과 | 즉각적 행동 수정 |
| **권장 비율** | 긍정적 행동지원(PBS) 강조 | 선행중재 보완으로 사용 |

> ⚠️ **기출 포인트**: 긍정적 행동지원(PBS: Positive Behavior Support)은 **선행중재를 우선**으로 하며, 처벌적 결과중재를 최소화하는 것을 원칙으로 한다. PBS의 3단계 예방 모델(1차: 보편적 지원, 2차: 선별적 지원, 3차: 집중적 개별 지원)과 연결하여 이해할 것.

## 7. ABA 차별강화 절차 요약

도전행동 감소를 위한 결과중재 시 사용하는 차별강화 유형.

| 유형 | 명칭 | 강화 조건 | 적용 상황 |
|------|------|---------|---------|
| **DRO** | 다른 행동 차별강화 | 일정 시간 동안 도전행동이 **발생하지 않으면** 강화 | 행동 대체 없이 감소가 우선인 경우 |
| **DRI** | 양립불가행동 차별강화 | 도전행동과 **동시에 할 수 없는** 행동 강화 | 예: 손 흔들기 대신 손으로 물건 잡기 |
| **DRA** | 대체행동 차별강화 | 도전행동과 **동일 기능의 적절한 대체행동** 강화 | FCT와 가장 직접 연결 |
| **DRL** | 저율행동 차별강화 | 행동 발생 **빈도가 일정 수준 이하일 때** 강화 | 완전 소거가 목표가 아닌 경우 |
