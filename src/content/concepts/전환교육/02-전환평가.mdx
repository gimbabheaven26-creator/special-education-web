## 전환평가란?

장애 학생의 성공적인 사회 통합을 위해 **개인의 능력과 미래 환경의 요구를 종합적으로 분석**하고, 이를 바탕으로 개별화된 전환 계획을 수립하는 과정이다.

> ⚠️ **기출 포인트**: 전환평가는 단순한 직업평가가 아니라 IEP·ITP 수립의 **전제 조건**으로 출제된다. 평가 방법의 명칭·특징·장단점 비교, 특히 **작업표본 vs 상황평가 vs 직무현장평가의 현실성 순서**는 단골 문항이다.

---

## 1. 전환평가의 정의 및 목적

### 1.1 정의

| 핵심 요소 | 내용 |
|---|---|
| 주된 평가 요소 | 작업(work)을 중심으로 개인의 직업 잠재성 측정 |
| 평가 범위 | 교육·고용 영역에 집중 (모든 성인 생활 영역의 포괄적 평가보다 **집중성·특수성** 강조) |
| 대상 요소 | 능력, 태도, 적성, 흥미, 작업 행동, 대인관계 기술, 학업 기술, 자립생활 기술 |
| 추가 포함 내용 | 자기결정·자기옹호 기술, 학습 스타일, 직업 흥미·적성·능력 |
| 목적 문서 | IEP 계획 수립을 위한 자료 제공 |

### 1.2 목적

1. 내담자 개개인에 적합한 **결과 판별** (경쟁 고용 ~ 완전한 타인 의존까지)
2. 내담자의 **기능적 능력과 장애 판별** (종합적·이차적 서비스 포함)
3. 최적의 결과 획득에 장애가 되는 요인을 극복하는 데 필요한 **서비스 판별**
4. 내담자의 기능적 장애 최소화 → **치료적 기능** (태도·행동·수행에 긍정적 변화 유발)
5. 효율적인 전환평가 → 개인적 성장과 **삶의 질 향상**으로 이어지는 전환 실현

## 2. 평가 정보 수집 방법

### 2.1 개인에 관한 정보 수집

#### ① 배경 정보의 분석

- 공식·비공식 기록 문서를 첫 번째 자료로 활용
- 주의사항: 학생이 **다른 환경에서 다르게 반응**할 수 있다는 점 고려, 직접 관찰·학생 자기 보고 병행

#### ② 면접 / 설문지

- 대상: 학생, 가족 구성원, 이전 담임교사, 친구, 상담자 등 실제 생활 밀접 관련자
- 주의사항: 설문 작성자가 내용을 명확히 이해하고 답변할 수 있도록 구성

#### ③ 심리측정 도구 (공식 평가)

- 표준화된 점수 제공 → **유일한 방법으로 사용 금지**, 타 평가와 병행
- **관심 목록(Interest Inventory, 흥미평가)**

| 흥미 접근 방식 | 설명 |
|---|---|
| **표현된 흥미** | 언어로 흥미를 표현 (초기 면접에서 직접 질문) |
| **행동화된 흥미** | 과제·직업·활동 참여로 관찰되는 흥미 |
| **검사된 흥미** | 심리 검사 도구를 통해 객관적으로 측정된 흥미 |

> ⚠️ **기출 포인트**: 흥미의 3가지 접근 방식(표현된·행동화된·검사된)은 단독 서술형 문항으로 출제된 바 있다. 각 방식의 정의를 정확히 구분할 것.

- 직업 흥미: 적성보다도 **직업 만족·직업 적응**에서 중요하게 고려됨

#### ④ 작업표본

- 실제 직업/직업군과 **유사한 재료·연장·기구**를 이용해 평가실에서 실시
- 직업 현장 배치 전 기술 훈련 병행 가능

| 구분 | 내용 |
|---|---|
| **장점** | 직접 평가 → 동기 유발 촉진, 문화·교육·언어적 편견 감소 |
| **단점** | 표준화 어려움, 흥미·태도의 심층적 면 제시에 한계 |

#### ⑤ 교육과정 기반 평가(CBA)

- 교육과정 내에서 학생이 배운 것을 바탕으로 평가

**Salvia와 Hughes(1990) CBA 8단계**:
평가 이유 구체화 → 교육과정 분석 → 행동적 목표 진술 → 평가 절차 개발 → 자료 수집 → 자료 요약 → 자료 제시 → **자료 해석 및 의사 결정**

**Albright와 Cobb의 교육과정 기반 직업평가(CBVA) 3단계**:

| 단계 | 시기 | 주요 활동 |
|---|---|---|
| **프로그램 배치·계획 수립 동안** | 직업 프로그램 실시 이전·이후 몇 주 | 선정, 프로그램 배치, 계획 수립 지원 |
| **직업 프로그램 참가 동안** | 프로그램 진행 중 | 모니터링, 서비스 전달 계획 적합성 결정, 성공 여부 평가 |
| **프로그램 종료 동안** | 프로그램 종료 시점 | 고용·중등 이후 교육 전환에 필요한 서비스 확인 |

#### ⑥ 상황평가

- 지역사회 내 산업 현장과 **유사한 모의 작업장**에서 체계화된 관찰 기법으로 평가
- 작업 활동, 감독, 임금, 근로 시간 등 전체 작업 환경과 최대한 유사한 조건 구성

| 구분 | 내용 |
|---|---|
| **장점** | 고용 환경 속 기능에 대한 정확한 평가 가능, 직업 의욕·흥미·기술 파악 |
| **단점** | 비표준화 → 주관적 평가·신뢰도 문제, 개인차 표현 어려움 |

#### ⑦ 직무현장평가(On-the-Job Evaluation)

- **경쟁적인 실제 작업 환경**에 배치 후 직무 수행 중 평가
- 훈련과 평가가 동시에 이루어짐 → **직무 기능 일반화가 어려운 학생에게 적합**

| 구분 | 내용 |
|---|---|
| **장점** | 해당 직종 요구 능력의 정확한 평가·관찰 가능, 현실적·직접적·효과적 |
| **단점** | 인원 제한, 평가 시간 소요 큼, 직종·장소 선정 어려움 |

> ⚠️ **기출 포인트**: **현실성 순서** — 작업표본 < 상황평가 < 직무현장평가 순으로 현실성이 높다. 이 순서는 반드시 암기할 것.

### 2.2 환경에 대한 정보 수집

| 방법 | 내용 |
|---|---|
| **잠재적 환경 평가** | 미래의 직업·교육·생활 환경 분석 → 필요한 훈련·지원 확인 |
| **지역사회 환경 분석** | 학생이 장차 살아갈 환경(현재·미래 가정 환경 모두)의 요구 사항 확인 |
| **직무분석** | 특정 직무의 업무 내용 + 작업자에게 요구되는 역량을 체계적으로 밝히는 것 |

**직무분석 4단계**:
1. 각 작업 완수를 위해 개인이 어떻게 반응해야 할지 확인
2. 과제 완성에 영향을 미치는 **환경적 단서** 확인
3. 작업 속도 관련 사항 (평균 시간, 주어진 시간 내 수행 과제 수) 확인
4. 각 작업의 **질적 요구 조건** 및 **예외 사항** 확인

### 2.3 MAGIC 평가 모델

공식 평가(표준화·비교 가능한 자료)와 비공식 평가(특정 환경에서의 개별 능력 정보)를 **통합**하는 접근 방법으로, 학생의 미래에 대한 예측을 돕는다.

| 구성 요소 | 전체 명칭 | 내용 |
|---|---|---|
| **M** | 평가하기 | 흥미, 적성, 학습 스타일, 직업적 일치, 초기 도입 수준 기술 평가 |
| **A** | 문서에 제시된 기술 검토 | 교사 면담 실시, 프로그램 검사 도구 작성 |
| **G** | 불일치 분석 실시 | IEP를 위한 불일치 기술 목표화, 필요 교육과정·교수 지원 파악 |
| **I** | 일반화 교수 | 기술 시연·적응 기술·기술 실행 방법 가르치기, 유지 검사 실행 |
| **C** | 면담하기 | 교사, 가족, 학생 면담 |

## 3. 전환평가와 전환계획

### 3.1 전환평가의 중요성

> ⚠️ **기출 포인트**: 전환평가가 ITP 구성의 **전제 조건**임을 명시한 서술형 문항이 출제된다. 전환평가는 지속적·상시 진행되어야 하며, IEP팀이 필요 정보 영역을 구체적으로 제시해야 한다는 점을 기억할 것.

1. 전환평가의 궁극적 목적: 학생·가족 참여 → 정보 수집·활용 → **개별화 전환교육계획(ITP)** 구성·실행
2. 전환평가는 **지속적이고 항상 진행 중**이어야 한다
3. 전환평가 → IEP팀이 전환 계획 구성 시 **필요 정보 영역을 구체적으로 제시**
4. 전환평가는 **학생 개별적**으로 이루어져야 하며, 그 결과로 적절·타당한 전환 계획 수립
5. 전환 계획 수립 시 포괄적 전환평가가 **선행**되어야 함

### 3.2 개별화 전환교육계획(ITP)

| 항목 | 내용 |
|---|---|
| **수립 주체** | IEP팀 (평가 결과를 IEP에 포함시켜 ITP 수립) |
| **포함 내용** | 고용, 주거 생활, 지역사회 기능화 관련 목적·목표 |
| **책임 명시** | 서비스를 제공할 전문가·기관 확정 |
| **시간 계획** | 서비스 준비에 따른 시간 계획 포함 |
| **훈련 시작** | 적합한 직업 부여 후 → 학령기 이후부터 **지속적** 전환 훈련 |
| **훈련 내용 초점** | 학업 기술, 직업 기능, 진로 발달, 생활 기능 |

> ⚠️ **기출 포인트**: ITP는 IEP의 **일부**로 포함된다는 점, 결과 중심 전환교육에 초점을 맞춘 목표 달성을 위한 체계적 훈련이 ITP 수립 이후 시작된다는 점을 구분하여 기억할 것.
