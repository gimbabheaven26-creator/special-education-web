## 1. 정서행동장애의 정의

### 진단 기준 (4가지 종합 판단)
문제행동의 **빈도, 강도, 지속성, 복합성(교육적 성취 어려움)** 을 종합적으로 고려한다.

> ⚠️ **기출 포인트**: 정상 아동에게도 나타나는 행동이 장애가 되는 기준 — 빈도·강도·지속시간·상황·문화·연령을 고려해야 함.

### 특수교육법상 정의
지적·감각적·건강상의 이유로 설명할 수 없는 다음 특성으로 학습에 어려움을 겪는 자:
- 또래나 교사와의 대인관계 어려움
- 부적절한 행동이나 감정
- 불행감이나 우울증
- 신체적 통증이나 공포

**비판**: '학습에 어려움'이라는 조건 때문에 심한 정서행동장애가 있어도 학습 문제가 없으면 특수교육 대상에서 제외될 수 있다.

### 정의의 포함 범위 (좁은 순)
| 정의 유형 | 설명 | 범위 |
|---------|------|------|
| **법적 정의** | 특수교육 대상자 선정용 (정부 명시) | 가장 좁음 |
| **임상적 정의** | 정신건강 전문가가 진단에 사용 | 중간 |
| **교육적 정의** | 장애 위험 아동까지 포함, 교육 전문가 사용 | 가장 넓음 |

## 2. 장애의 분류

### 의학적 분류 (임상적/범주적)
각 하위 유형을 식별하는 데 초점. **문제점**: 표찰(낙인)로 인한 편견, 교육 기회 박탈.

### 교육적 분류 (행동적·차원적)
문제행동을 두 차원으로 분류 — 한 아동이 두 특성을 동시에 또는 번갈아 보일 수 있음.

**K-CBCL (아동·청소년 행동 검목표)** — 만 6~18세 평가 도구

| 분류 | 구성 | 특징 |
|------|------|------|
| **내재화 (과잉 통제)** | 불안/우울, 위축/우울, 신체 증상 | 눈에 띄지 않아 심화 위험 ↑ |
| **외현화 (통제 결여)** | 규칙 위반, 공격 행동 | 타인·환경을 향해 표출, 분류 용이 |
| 문제행동 총점 | 사회적 미성숙, 사고 문제, 주의집중 문제 + 내재화 + 외현화 | — |

**교육적 분류 장점**: 낙인 ↓, 세분화된 중재 가능, 내재화에도 관심.
**단점**: 두 특성을 동시에 보이는 학생 분류 어려움.

### 장애의 공존 (복합적 분류)
두 가지 이상 장애가 동시에 나타나거나, 정서행동장애 하위 범주들이 공존. 외현화 행동 이면의 내재적 문제에도 주목해야 함.

## 3. 선별 절차

### SSBD (체계적 선별)
교사 추천 + 리커트 척도 + 직접 관찰을 통합한 3단계 선별. 특수교사는 모든 단계에서 정보원 역할.

| 단계 | 담당 | 내용 |
|------|------|------|
| 1단계 | 담임교사 | 외현화·내재화 각 3명씩 추천 |
| 2단계 | 담임교사 | CEI(위기 사건 척도) + CFI(총빈도 척도) 작성 |
| 3단계 | 특수교사 | 학업 기술·사회적 행동 직접 관찰 → 개별화 교육 의뢰 |

**장점**: 조기 선별 가능, 내재화 아동 포함, 교사 판단 활용.

### 의뢰 전 중재
특수교육 의뢰 전 일반 학급에서 증거 기반 중재를 먼저 실시했음을 증명.
**생략 가능한 경우**: 문제가 너무 명확하거나 중재 방법이 없는 경우.

## 4. 관련 요인

### 생물학적 요인

**기질 (Temperament)** — 생득적 경향성 + 환경적 영향으로 표출되는 행동 스타일

기질의 9가지 특성: 규칙성, 활동 수준, 접근/회피, 적응성, 기분의 질, 반응 역치, 반응 강도, 주의 산만, 지속성

| 유형 | 특성 |
|------|------|
| **순한 기질** | 규칙적, 잘 적응, 쾌활 |
| **까다로운 기질** | 부정적 정서, 불규칙, 고집, 적대적 |
| **느린 기질** | 변화에 느리지만 결국 긍정 적응 |
| **행동 억제 기질** | 새 자극에 위축·수줍음 → 내재화 장애 위험 ↑ |

### 환경적 요인

**애착 유형** (Ainsworth)

| 유형 | 양육자 특성 | 아동 특성 |
|------|-----------|---------|
| **안정** | 민감하고 일관된 반응 | 신뢰 형성, 탐색 적극적 |
| **회피** | 무감각, 신체 접촉 회피 | 양육자 무시·회피 |
| **저항** | 일관성 없는 양육 | 불안, 분리 시 강하게 저항 |
| **혼란** | 심한 스트레스·우울 | 가장 불안정, 위안/불안 혼동 |

**훈육 유형** (반응성 × 요구)

| 유형 | 반응성 | 요구 | 아동 결과 |
|------|--------|------|---------|
| **권위적 (신뢰적)** | 높음 | 높음 | 최적 행동 발달 |
| **권위주의적 (독재적)** | 낮음 | 높음 | 공격적·낮은 자존감 |
| **관대한/허용적** | 높음 | 낮음 | 충동적·의존적 |
| **무관심한** | 낮음 | 낮음 | 반사회적, 정서행동 위험 ↑ |

> ⚠️ **부적 강화의 덫**: 공격적 아동의 문제 행동 → 부모 양보 (부적 강화) → 아동 문제 행동 강화 → 악순환. 비효율적 훈육이 원인.

## 5. 개념적 모델 (5가지)

> **모델의 목적**: ① 사정·평가 지표 ② 중재 방법 제공 ③ 전문가 간 의사소통

### 신체생리학적 모델

| 항목 | 내용 |
|------|------|
| **관점** | 행동이 신경생리학적 기제와 관련; 원인이 개인 내적에 존재 (의료 모델) |
| **원인** | 유전, 뇌 기능 이상, 신경전달물질(세로토닌·도파민), 기질 |
| **평가** | 발달력 평가, 신체 기능 평가, FBA |
| **중재** | 약물치료, 영양치료, 교사는 관찰·점검 역할 |

### 심리역동적 모델

| 항목 | 내용 |
|------|------|
| **관점** | 정신이 행동의 기초; 정서적 미성숙, 충족 안 된 욕구가 원인 |
| **평가** | 투사법 (로르샤흐, CAT, 집-나무-사람), 자기 보고식 |
| **중재** | 심리치료, 현실치료, 환경치료, **생활 공간 면접 (LSI)** |

**생활 공간 면접 (LSI)**: 즉각적 정서 응급치료 + 생활 사건에 대한 임상적 개발.

### 행동주의 모델

| 항목 | 내용 |
|------|------|
| **관점** | 모든 행동은 학습된 것; 관찰 가능한 외현적 행동만 연구 대상 |
| **원인** | 잘못된 학습; 환경적 변인 파악 후 조작 |
| **평가** | 체크리스트, 행동 평정 척도, 면접, 행동 관찰, **FBA** |
| **중재** | 행동 수정 기법, SST, 토큰 체계, 반응 대가 |

### 인지주의적 모델

| 항목 | 내용 |
|------|------|
| **관점** | 손상된 사고·신념이 직접 원인; 인지 변화로 행동 변화 가능 |
| **원인** | 인지 결함(충동·공격) + 인지 왜곡(불안·우울) |
| **중재** | 모델링, 자기 교수, 분노 대처 프로그램, 귀인 재훈련, **REBT** |

**REBT (합리적 정서 행동 치료)**: 비합리적 신념을 논박하여 합리적 신념으로 재구조화.
**ABCDE 체계**: A(선행사건) → B(신념) → C(결과) → D(논박) → E(효과)

### 생태학적 모델

| 항목 | 내용 |
|------|------|
| **관점** | 원인이 개인과 환경의 **상호작용**에 기인; 여러 위험 요인의 복합적 작용 |
| **평가** | 생태학적 사정, FBA, BRP-2 |
| **중재** | 가족 체계 + 교육 체계(PBS, 재통합) + 사회적 체계 |

**사회적 체계 중재 3가지**:
- **랩어라운드 (wraparound)**: 지역사회 전문가·가족·학교 공동 협력
- **RE-ED 프로젝트**: 모든 환경에 걸쳐 일관된 중재
- **CASSP**: 사례 관리 서비스 중심 지역사회 연계

> ⚠️ **브론펜브레너의 생태학적 체계**: 미시체계 → 중간체계 → 외부체계 → 거시체계 → 시간체계

### 통합 모델

- **취약성-스트레스 모델**: 개인적 취약성 + 생활 스트레스의 상호작용으로 장애 유발
- **생물심리사회적 모델**: 생물학적 + 심리적 + 사회적 요인을 종합적으로 고려
