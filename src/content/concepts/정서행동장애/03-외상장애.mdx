## 1. 반응성 애착장애 (Reactive Attachment Disorder)

### 정의
아동과 주양육자 사이에 애착 관계가 형성되지 않아, 아동이 위안·지원·보호·돌봄을 얻기 위해 애착 대상에게 **거의 다가가지 않는** 특성을 보이는 장애.

### 진단 기준

| 기준 | 내용 |
|------|------|
| A | 성인 양육자에 대해 **정서적으로 억제되고 위축된 행동**을 일관성 있게 보임; 괴로울 때 위안을 구하지 않거나 제공되는 위안에 반응하지 않음 |
| B | 지속적인 사회적·정서적 장해 (다른 사람에 대한 최소한의 반응, 제한된 긍정 정서, 비위협적 상황에서도 과민함·슬픔·두려움의 삽화) |
| C | 극단적으로 불충분한 양육 경험 (사회적 방치/박탈, 주 양육자 반복 교체, 비정상적인 양육 환경) |
| D | 적절한 돌봄 없이 양육되어 기준 A의 행동 양상을 보임 |
| E | 자폐 스펙트럼 장애에 해당하지 않음; **만 5세 이전**에 나타남; 아동의 발달 연령 **9개월 이상** |

## 2. 외상 후 스트레스 장애 (PTSD)

### 정의
경험한 치명적인 사건을 **재경험**하며 지속적으로 강한 불안 증상을 나타내는 장애.

### 진단 기준 (성인·청소년·6세 초과 아동)

| 기준 | 내용 |
|------|------|
| A | 죽음, 심각한 상해, 성폭행에 실제 노출되었거나 위협을 당한 경험 |
| B | 외상성 사건과 관련된 반복적·무의식적 고통스러운 **회상, 꿈, 분열적 반응, 심리적 고통, 생리적 반응** 중 **1가지 이상** |
| C | 외상성 사건 관련 자극을 **지속적으로 회피** |
| D | 사건 관련 **인지와 기분이 부정적으로 변화**: 기억 상실, 부정적 생각·기대, 왜곡된 인지, 부정적 감정, 흥미 감소, 소외감, 긍정 정서 경험 불가 중 **2가지 이상** |
| E | **각성 반응 현저하게 반복**: 공격성, 자기 파괴적 행동, 지나친 경계, 과장된 놀람 반응, 집중 어려움, 수면 장애 중 **2가지 이상** |
| F | 기준 B~E의 증상이 **최소 1개월 이상** 지속; 사회적·학업적·직업적 기능에 임상적으로 중요한 손상 |
| G | 약물이나 다른 의학적 상태의 생리적 효과에 기인하지 않음 |

> ⚠️ **기출 포인트**: PTSD 진단 기준 B(침습), C(회피), D(인지·기분 변화), E(각성)의 4가지 클러스터와 각 필요 증상 수 자주 출제. D와 E는 각각 **2가지 이상** 필요.
