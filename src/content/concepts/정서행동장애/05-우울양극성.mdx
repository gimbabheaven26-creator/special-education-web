## 1. 기초신체장애

### 섭식 관련 장애

| 유형 | 정의 | 진단 기준 |
|------|------|---------|
| **이식증** | 먹을 수 없는 종이·머리카락·벌레·먼지 등을 먹음 | **1개월 이상** 지속 |
| **반추장애** | 소화된 음식을 의도적으로 입으로 역류시켜 토하거나 다시 씹음 | **1개월 이상** 증상 |
| **거식증** | 체중 증가 공포로 음식 거절; 심각한 저체중임에도 체중 증가 강하게 두려워함 | 저체중 + 체중 증가 공포 |
| **폭식증** | 반복적 폭식 + 부적절한 보상 행동 (자기 유발 구토·설사); 저체중 아님 | 거식증과 구분: **저체중 아닌 경우** |

### 배변장애

| 유형 | 정의 | 진단 기준 |
|------|------|---------|
| **유뇨증** | 옷이나 침구에 소변을 반복하여 배설 | **5세 이상** 아동, **3개월 동안 최소 주 2회** 이상 |
| **유분증** | 부적절한 곳에 대변을 봄 | **만 4세 이상** 아동, **최소 3개월 동안 월 1회** 이상 |

### 수면장애

| 유형 | 내용 |
|------|------|
| **잠들기 문제** | 잠자리 거부, 오래도록 잠 못 자거나 밤에 깨면 다시 자기 어려움 |
| **수면 각성 문제** | 야경증, 수면 중 보행(몽유병), 악몽 포함 |

## 2. 양극성 장애

| 유형 | 핵심 특성 | 진단 기준 |
|------|---------|---------|
| **양극성 장애 I** | 비정상적으로 들뜬 기분·민감, 자존감 고조, 과다 활동·말·생각 | 최소 **한 번의 조증 삽화**가 진단 기준 A~D에 부합 |
| **양극성 장애 II** | 경조증 삽화 + 주요 우울장애 반드시 포함; **조증 삽화 없음** | 경조증 기준 A~F + 주요 우울장애 A~C 부합; 조증 삽화 경험 없어야 함 |
| **순환기질 장애** | 경조증·우울 증상이 반복되지만 각 진단 기준에는 미치지 못함 | 최소 **2년** (아동·청소년은 **1년**) 중 반 이상의 기간 동안 증상 |

## 3. 우울장애

### 주요 우울장애

- **진단 기준**: DSM-5 우울 기준 **9가지 중 5가지 이상**, **우울한 기분** 또는 **흥미와 즐거움의 상실** 중 하나는 반드시 포함
- **증상 지속**: **2주 이상** 지속; 임상적으로 의미 있는 기능 손상

### 지속성 우울장애

- DSM-IV-TR의 만성적 주요 우울장애 + 기분 부전 장애를 **통합**
- 주요 우울장애 진단 기준을 **2년 이상** 나타낸 경우 두 가지 진단명 부여

### 파괴적 기분조절장애 (DSM-5 신규 추가)

- **대상**: 12세 이전 아동
- **특성**: 지속적 초조·극단적 행동 조절 불가 삽화가 자주 반복
- **기대 효과**: 청소년기 이전 아동에 대한 **과다한 양극성 장애 진단 예방**
- **진단 기준**: 심한 울화 폭발 반복; 발달 단계와 일관성 없음; **주 평균 3회 이상**; **12개월 이상** 지속

> ⚠️ **기출 포인트**: 양극성 장애 I vs II 차이 — II는 조증 삽화가 **없고** 경조증 + 주요 우울장애 조합이 핵심.

## 4. 우울장애 중재

| 중재 | 내용 |
|------|------|
| **약물치료** | 삼환계 항우울제, 선택적 세로토닌 재흡수 억제제 |
| **인지행동중재** | 인지치료, 구조화된 행동 프로그램, 자기 조절·자기 통제 기술 |
| **문제 해결 기술** | 문제 해결 기술이 부족한 아동·청소년에게 적용 |
| **대인 간 심리치료** | 멘토 지정, 사회적 관계·교우 관계 확대 |
| **이완 훈련** | 충동 조절 지도 |
| **팀 접근** | 일반 교사·상담 교사·부모 팀 → 다양한 인지적 접근 |
