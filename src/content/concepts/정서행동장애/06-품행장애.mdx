## 1. 적대적 반항장애 vs 품행장애 비교

| 항목 | **적대적 반항장애 (ODD)** | **품행장애 (CD)** |
|------|----------------------|----------------|
| **핵심 특성** | 반항적·불복종적·도발적 행동 | 타인 기본 권리 침해 + 사회 규범 위반 |
| **사회 규범 위반** | ❌ 없음 | ✅ 있음 |
| **타인 권리 침해** | ❌ 없음 | ✅ 있음 |
| **심각도** | ODD < CD (경도의 품행장애로 간주) |
| **진단 기준 지속** | **6개월** 이상 | 12개월 중 **3가지 이상**, 그 중 **1개는 6개월** 이상 |

> ⚠️ **기출 포인트**: ODD는 "사회 규범 위반 없음"이 핵심 변별 기준.

---

## 2. 적대적 반항장애 (ODD) 진단 기준

화난 민감한 기분 + 시비 걸거나 반항하는 행동 + 보복적 행동이 **최소 6개월** 지속되며, 형제가 아닌 다른 사람 **1인 이상**과의 상호작용에서 적어도 **4가지 증후**를 보임.

## 3. 품행장애 (CD) 세부 사항

### 품행장애 행동 유형

| 유형 | 예시 |
|------|------|
| **공격 행동** | 타인의 기본 권리를 침해하는 반사회적 행동 |
| **규칙 위반 행동** | 연령에 적합한 사회적 규범을 어기는 반사회적 행동 |

### 아동기 vs 청소년기 구분

| 유형 | 기준 |
|------|------|
| **아동기 품행장애** | 만 **10세 이전**에 나타남 |
| **청소년기 품행장애** | 만 **10세 이후**에 나타남 |

### 심각도

| 정도 | 내용 |
|------|------|
| 가벼운 | 진단 기준 초과 문제가 매우 적음; 타인에게 가벼운 해만 끼침 |
| 중간 | 문제 수와 해의 정도가 가벼움과 심함의 중간 |
| 심한 | 문제 많거나 타인에게 **심각한 해** 끼침 |

## 4. 품행장애 원인

### 문제 행동의 발달 경로 (3가지)

| 경로 | 시작 | 발전 과정 |
|------|------|---------|
| **외현 경로** | 사소한 공격 | 신체적 싸움 → 폭력 |
| **내재 경로** | 사소한 내재화 행동 | 기물 파괴 → 심각한 비행 |
| **권위 갈등 경로** | 고집스러운 행동 | 반항·불복종 → 권위 회피 |

### 위험 요인 vs 보호 요인

| 구분 | 내용 |
|------|------|
| **위험 요인** | 미성숙, 뇌 손상, 낮은 지능, 까다로운 기질, 가정 폭력·불화, 학대, 학교생활 실패 |
| **보호 요인** | 적응 유연성 (사회적 능력, 공감 능력, 다루기 쉬운 기질, 부모와 친밀한 관계 등) |

## 5. 품행장애 중재

| 중재 | 정의 |
|------|------|
| **부모 훈련** | 부모가 아동 행동·환경 관찰·자료 수집, 적절한 행동 증가·부적절한 행동 감소 전략 사용 |
| **기능적 가족 중재 (FFT)** | 가족 내 의사소통 향상; 인식·기대·태도·정서 반응 수정 → 가족 기능 향상 |
| **다중 체계 중재 (MST)** | 가족·학교·또래·지역사회 체계 수정; 부모에게 기술·자료 제공 |
| **학교 중심 프로그램** | 학교 차원의 긍정적 행동 지원 (SWPBS) |
| **지역사회 기반 프로그램** | 수용 시설 대신 지역사회 참여 → 비행 행동 추가 학습 예방 |
| **사회적 기술 훈련 (SST)** | 적절한 사회적 상호작용에 필요한 특정 기술 체계적 지도 |
