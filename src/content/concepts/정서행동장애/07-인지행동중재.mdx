## 1. 인지행동중재 (CBI) 개요

학생이 자신의 행동을 점검·평가·관리하거나, 특정한 인지적 단계에 따라 자극에 반응하도록 하는 **교수적 절차**.

---

## 2. 주요 CBI 기법

### 2-1. 문제 해결 훈련 (Problem Solving Training)

5단계 문제 해결 절차를 학습:

| 단계 | 내용 |
|------|------|
| 1 | 문제 **인식** |
| 2 | 문제의 **정의** |
| 3 | 대안적 **해결 방법** 탐색 |
| 4 | **의사 결정** |
| 5 | 문제 **해결 및 평가** |

---

### 2-2. 분노 조절 훈련 (Anger Control Training)

자기 교수를 통해 분노와 공격 행동을 자제·조절하는 방법.

| 단계 | 내용 |
|------|------|
| 1 | **인지 준비** |
| 2 | **기술 습득** |
| 3 | **적용 훈련** |

---

### 2-3. 자기 관리 훈련 — 4가지 기법

**장점**: 학생 주도적 접근, 외부 관리 감독 없이도 적절한 행동 학습, **행동의 일반화** 가능.

| 기법 | 정의 | 특징 |
|------|------|------|
| **자기 교수** | 자신의 행동을 규제하기 위해 **내적 언어**를 사용하여 스스로에게 이야기하는 과정 | 교사 주도 → 학생 주도 전환 목표 |
| **자기 기록 (자기 점검)** | 자기 행동의 양이나 질을 측정하여 **스스로 기록** | 자기가 주는 보상/벌, 환경 단서로 작용 |
| **자기 평가** | 사전에 선정된 준거와 자신의 행동을 **비교** | 이전 자기 행동 / 집단 기준 / 교사 기준 사용 가능 |
| **자기 강화** | 정해진 목표 달성 시 **스스로 선택한 강화제** 제공 | 강화 기준을 지나치게 낮게 설정하지 않도록 지도 필요 |

> ⚠️ **기출 포인트**: 자기 교수 5단계 순서 출제 빈도 높음.

### 자기 교수 5단계

| 단계 | 내용 |
|------|------|
| 1 | **인지적 모델링** — 교사가 큰 소리로 자기 교수 시범 |
| 2 | **모델의 외현적 지도** — 아동이 교사의 언어 지도를 들으며 수행 |
| 3 | **아동의 외현적 자기 교수** — 아동이 스스로 큰 소리로 자기 교수하며 수행 |
| 4 | **외현적 자기 교수의 용암** — 속삭이는 목소리로 점차 줄임 |
| 5 | **내면적 자기 교수** — 내적 언어(속마음)로 수행 |

---

### 2-4. 대안 반응 훈련

- **정의**: 바람직하지 않은 반응의 기회를 차단하는 **대안적 반응**을 지도
- **조건**: 대안적 반응이 학생의 레퍼토리에 이미 있으면 강화; 학생은 반드시 **자기 점검** 가능해야 함

---

### 2-5. 귀인 재훈련

- **부정적 귀인**을 **긍정적 귀인**으로 대체하여 과제 수행의 지속성을 높임

---

### 2-6. 합리적 정서 행동 치료 (REBT)

정서행동장애 학생의 **비합리적 신념**을 논리적 반박을 통해 **합리적 신념**으로 바꿔줌.

**ABCDE 체계**:

| 요소 | 내용 |
|------|------|
| **A** (Activating event) | 선행 사건 |
| **B** (Belief) | 신념 |
| **C** (Consequence) | 결과 (정서·행동) |
| **D** (Disputing) | 비합리적 신념에 대한 **논박** |
| **E** (Effect) | 논박 효과 (합리적 신념으로 재구조화) |
