## 1. ADHD (주의력 결핍 과잉행동장애)

### 진단 기준

| 항목 | 내용 |
|------|------|
| 증상 수 | 부주의 9가지 중 **6가지** 그리고/또는 과잉행동·충동성 9가지 중 **6가지** (17세 이상은 **5가지**) |
| 지속 기간 | 적어도 **6개월** 동안 지속 |
| 발생 연령 | **12세 이전** 발생 |
| 상황 | **2가지 이상** 상황에서 광범위하게 나타남 |
| 기능 손상 | 사회적·학업적·직업적 기능 문제 분명한 증거 있어야 함 |

### ADHD 유형

| 유형 | 기준 |
|------|------|
| **부주의 우세형** | 지난 6개월간 부주의 기준 충족, 과잉행동·충동성 기준 미충족 |
| **과잉행동·충동 우세형** | 지난 6개월간 과잉행동·충동성 기준 충족, 부주의 기준 미충족 |
| **복합형** | 부주의 + 과잉행동·충동성 기준 **모두** 충족 |

### 핵심적 특성

| 특성 | 내용 |
|------|------|
| **주의력 결핍 (부주의)** | 선택적 주의, 지속적 주의, 주의 용량 중 하나 이상에서 결함 |
| **과잉행동·충동성** | 자기 조절력 또는 억제력에 어려움; 자기 대화 전략 활용 필요 |

---

### Barkley의 다면 모형

> **과잉행동-충동성 우세형** 아동에게만 적용; 반응 억제가 핵심 요소.

**행동 억제를 구성하는 3가지 능력:**

| 능력 | 내용 |
|------|------|
| ① | 일어날 수 있는 반응에 대한 **억제** |
| ② | 현재 진행 중인 반응에 대한 **억제** |
| ③ | **간섭에 대한 통제** |

**집행 기능의 4가지 어려움:**

| 기능 | 어려움 |
|------|------|
| 작업 기억 (비언어적) | — |
| 언어의 내재화 | — |
| 정서·동기·각성에 대한 자기 조절 | — |
| 재구성 | — |

---

### ADHD 원인

| 원인 | 내용 |
|------|------|
| **각성 수준** | 과잉 각성 또는 **과소 각성** (과소 각성 증거가 더 많음) |
| **보상 민감성** | 지연 보상보다 **즉시 보상** 비정상적으로 강하게 선호 |
| **지연 혐오** | 지연 상황 회피·탈출 시도 |
| **억제·실행 기능** | ADHD와 관련하는 비교적 강력한 증거 있음 |
| **이중 경로 모형** | 실행 기능 결여 경로 (주의력 결핍) + 지연 혐오 경로 (과잉행동-충동성) — **두 가지 독립된 경로** |

---

### ADHD 중재

| 중재 | 내용 |
|------|------|
| **약물치료** | 효과와 부작용 관계 고려 후 결정 |
| **행동 중재** | 바람직한 행동 증가·바람직하지 않은 행동 감소 |
| **인지행동 중재** | 인지적 충동성 통제, 상황에 맞는 행동 조절 |
| **교실 지도** | 눈 맞춤, 목소리 변화, 학습량 줄이기, 단순 이동 허용 |

**ADHD 특성별 교수 전략:**

| 결여 특성 | 전략 |
|---------|------|
| 능력 결여 | 자기 교수, 자기 점검, 자기 강화 |
| 동기 결여 | 과제 분석, 좋아하는 과제 먼저 제시 |
| 억제력 결여 | 자기 대화 전략 |
| 작업 기억 결여 | 장기 기억 향상 전략 |
| 자기 조절력 결여 | 인지적 행동 수정 전략 |

## 2. 틱장애

### 틱의 특징
- 반복적으로 갑작스럽고 빠르게 나타나는 **근육의 움직임이나 소리**
- 스트레스 상황에서 **악화**, 특정 활동 몰입 시 **약화**
- 잠시 동안 **참을 수 있음**; 수면 중에는 나타나지 않음

### 틱 장애 유형

| 유형 | 기준 |
|------|------|
| **잠정적 틱 장애** | 운동 또는 음성 틱이 **1년 이내**로 지속 |
| **만성 운동 또는 음성 틱 장애** | **1년 이상** 지속; 운동·음성 **둘 다 있는 것은 아님** |
| **뚜렛 장애** | 여러 개의 운동 틱 + **하나 이상의 음성 틱**이 **1년 이상** 지속 |

### 운동/음성 틱 유형

| 종류 | 예시 |
|------|------|
| **단순한 운동 틱** | 눈 깜박거림, 찌푸리는 얼굴, 어깨 으쓱거림 |
| **복잡한 운동 틱** | 손짓, 뛰기, 만지기, 사물 냄새 맡기, 무릎 구부리기 |
| **단순한 음성 틱** | 코 킁킁거림, 컹컹대기, 헛기침 |
| **복잡한 음성 틱** | 단어·문장 소리치기, 반향어, 음담패설 |

---

### 틱 장애 중재

#### 습관 반전 훈련 (4가지 구성요소)

| 구성요소 | 내용 |
|--------|------|
| **틱 알기 훈련** | 틱의 본질, 빈도, 선행 사건·후속 결과를 알도록 교육 |
| **경쟁 반응 훈련** | 틱과 관련된 근육의 **반대 방향 긴장** 유지; 틱 발생 후 또는 경고 사인 있을 때 **2분** 동안 경쟁 반응 |
| **이완 훈련** | 스트레스 상황에서 자각 수준 낮춰 틱 발생 빈도 감소 |
| **후속 사건 중심 중재법** | 틱으로 인한 불편·당황 결과 목록 작성 + 틱 감소 이점 기록 → 동기 부여 |

#### 상황 역실행
아동에게 틱이 발생할 때마다 **30초 동안 스스로 계속 틱**을 하도록 지도 → 틱이 일어나지 않도록 노력하게 가르침.

#### 틱 발생 시 주의사항
- 틱 증상을 지적하거나 쳐다보지 않음
- 일시적인 악화에 민감하게 반응하지 않음
- 적절한 운동·취미 생활은 도움이 됨
- 컴퓨터·TV 등 전자 제품 사용 제한

> ⚠️ **기출 포인트**: 뚜렛 장애 = 운동 틱 + 음성 틱 **동시 존재**가 핵심. 만성 틱 장애와의 차이 구분.
