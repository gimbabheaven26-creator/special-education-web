## 1. 기능적 생활중심 교육과정

### 1-1. 교육과정 구성·운영의 기본 전제

| 전제 | 내용 |
|------|------|
| **연령 적합성** | 현재 생활 연령 기준으로 기술 선택; 준비도 충족 여부와 무관 |
| **궁극적 기능성** | 성인이 되어 통합 환경에서 일반인과 독립·생산적으로 기능하기 위한 기술 목표 |
| **최소위험 가정** | 결정적 근거가 없는 한 학생·가족에게 위험이 가장 적은 방향으로 교육 결정 |
| **영 수준 추측** | 기술이 자연스럽게 습득될 것이라고 추측하지 않음; 직접 교수 + 일반화 확인 필수 |
| **자기결정 증진** | 교육과정 전반에 자기결정 기회 포함 |

> ⚠️ **기출 포인트**: **준비도 함정** = 발달적 접근으로 인해 기능적 기술 교수 기회를 놓치는 현상. **기능적 배제** = 상위 단계 이동이 늦어 기능적 기술 격차가 벌어지는 현상. 두 개념 구분 필수.

---

### 1-2. 기능적 생활중심 교육과정 정의 및 특징

**정의**: 개인의 현재와 미래 생활에 필수적으로 요구되는 기술 — 다양한 환경에서 즉시 사용 가능하고 의미 있는 기술들을 포함하는 교육과정

**핵심 특징**

- 생활 연령을 기준으로 현재·미래 환경에서 필요한 기술을 그 환경에서 직접 가르침
- 하향식(top-down) 접근 — 선수 기술 습득과 무관하게 필요 기술을 조사하고 교수
- 학생별 개별화 — 한 학생에게 기능적인 기술이 다른 학생에게는 아닐 수 있음

| 장점 | 단점 |
|------|------|
| 현재·미래에 사용하지 않을 기술 학습에 시간 낭비 없음 | 통합 기회와 일반 교육과정 참여 감소 가능 |
| 일반화 어려움 감소 | 학업 중심 교육을 원하는 부모와의 갈등 |
| 성공감 경험 → 유지 향상 | 이동·안전·비용·시간 문제 |
| 지역사회 접촉 → 사회적 상호작용 기회 증가 | 통합 환경과 분리될 수 있음 |

## 2. 생태학적 목록

### 2-1. 정의 및 목적

**정의**: 학생이 현재와 미래 생활에서 필요한 개별 기술을 찾기 위한 환경 조사 도구 (관찰지·평가 도구)

- 하향식 접근 — 아동의 생활 연령을 기준으로 현재·졸업 후 독립 생활 기술 선정
- **영역**: 가정(주거)·지역사회·여가·교육/직업 환경

### 2-2. 교육과정 개발 5단계

| 단계 | 내용 |
|------|------|
| **1단계: 교육과정 영역 정하기** | 기술을 가르칠 상황·맥락이 될 교육과정 영역 결정 |
| **2단계: 현재·미래 환경 확인** | 현재 주거·미래 주거(그룹홈, 시설 등) 각각 파악 |
| **3단계: 하위 환경으로 나누기** | 활동이 일어날 수 있는 환경을 세분화 |
| **4단계: 활동 목록 만들기** | 하위 환경에서 학생의 삶에 필요한 활동 목록 작성 |
| **5단계: 필요 기술 결정** | 각 활동을 수행하기 위해 필요한 기술을 과제 분석 수준으로 세분화 |

> ⚠️ **기출 포인트**: 5단계 순서와 내용이 자주 출제. **1→환경 영역 / 2→환경 / 3→하위 환경 / 4→활동 / 5→기술**로 점점 구체화된다.

### 2-3. 격차 분석

| 질문 | 내용 |
|------|------|
| ① 비장애 또래는 이 상황에서 **어떻게** 수행하는가? | 기준 수행 확인 |
| ② 장애 학생은 현재 **어떻게** 수행하는가? | 현재 수행 확인 |
| ③ 또래와 같이 하려면 **어떻게** 해야 하는가? | 지원·수정 방안 도출 |

기술 선택 우선순위 기준: **경험적 타당도**(독립 생활에 중요도) + **사회적 타당도**(주변인이 중요하게 여기는 기술)

## 3. 지역사회 중심 교수 (Community-Based Instruction)

### 3-1. 세 가지 교수 유형 비교

| 유형 | 정의 | 특징 |
|------|------|------|
| **CS (지역사회 모의교수)** | 지역사회 장면·과제를 교실로 가져와 모의 활동 | 실제 환경 접근 어려울 때; 위험 부담 높은 기술 안전 훈련 |
| **CRI (지역사회 참조교수)** | 학교 공간 내에서 지역사회 기술을 간접 연습 | 모의교수보다 실제적 맥락 강화; 준비 없는 외부 활동 위험 줄임 |
| **CBI (지역사회 중심교수)** | 교수·학습 활동이 실제 지역사회에서 이루어짐 | 단순 일회성 행사 아님 — 분명한 목표·전개 과정 전제 필수 |

> ⚠️ **기출 포인트**: CS·CRI는 교실/학교 안, CBI는 **실제 지역사회**. 세 유형의 '장소' 구분이 핵심.

### 3-2. CBI 교수 절차 (4단계)

| 단계 | 내용 |
|------|------|
| **1단계: 목표 설정** | 효과성·연령 적합성·자기결정·통합·안전 고려 |
| **2단계: 과제 분석 (일반사례 분석)** | 전통 과제 분석 + 자극 변인·반응 유형 포함; 긍정적 예·부정적 예 모두 포함 |
| **3단계: 교수 전략 선정·적용** | 촉진·소거·용암법·강화 병행; 유사한 긍정/부정 예를 연이어 교수 |
| **4단계: 자료 수집·평가** | 독립성·일반화·유지 평가; 최소 월 1회 유지, 2주 1회 일반화 |

## 4. 일반사례교수법 (General Case Instruction)

**목표**: 학습한 기술이 어떤 상황·조건에서도 요구될 때 수행 가능해야 함

**핵심 원리**

- 교수 사례의 **선택과 계열화**를 강조
- 모든 관련 자극·반응 유형을 포함한 충분한 예를 사용
- 일반화 평가를 위해 **교수 사례**(훈련)와 **검사 사례**(비교수)를 분리하여 계획

| 개념 | 내용 |
|------|------|
| **교수 사례** | 자극 상황과 요구 반응을 대표하는 최소한의 예; 긍정·부정 예 모두 포함 |
| **검사 사례** | 교수하지 않은 새로운 자극 상황 — 일반화 여부 확인에 사용 |
| **계열화 원칙** | 한 회기에 모든 구성 요소 교수; 유사한 긍정/부정 예를 연이어 제시 |
