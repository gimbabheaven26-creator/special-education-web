## 1. 뇌성마비 아동의 특성 개요

뇌성마비 아동은 구어 산출, 심리사회적, 신체 및 운동, 생리조절, 지각 등 다섯 영역에서 복합적인 특성을 보인다.

---

## 2. 구어 산출 특성

### 호흡장애

- 역호흡, 불규칙하고 유연하지 않은 호흡 주기
- 분당 호흡수 비장애아동보다 8~10회 더 많고 폐활량 적음 → 말이 한 낱말/짧은 구로 제한
- **지도**: 입·코로 부드럽게 숨쉬기, 날숨 및 발성 지속 시간 연장, 복식호흡으로 날숨 근육 강화

### 발성장애

- **원인**: 비정상적 근긴장도·비협응 운동 → 후두근육·성대 움직임 이상

### 조음장애

| 항목 | 내용 |
|------|------|
| **원인** | 잘못된 자세, 비정상적 근육긴장·반사, 안면 근육 불수의 운동, 혀·입술·턱 운동장애 |
| **문제** | 강도·음도·운율 문제 → 말 명료도 저하 |
| **경직형 특이사항** | 연인두 폐쇄 기능 결함 → 파열음·마찰음·파찰음 산출 어려움, **과대비성** |

> ⚠️ **기출 포인트**: 경직형 뇌성마비의 연인두 폐쇄 기능 결함 → 과대비성, 파열음·마찰음·파찰음 산출 곤란이 자주 출제됨.

### 자세 조정 훈련 (조음 보조)

| 목표 음 | 자세 |
|---------|------|
| 양순음 | 머리를 앞으로 숙여 양 입술 폐쇄 보조 |
| 경구개음/치조음 | 머리를 앞으로 숙여 설첨이 경구개·치조에 가깝게 |
| 연구개음 | 목을 뒤로 젖혀 혀뿌리를 구강 뒤쪽으로 |

## 3. 심리·사회적 특성

| 특성 | 내용 |
|------|------|
| **산만성(피전도성)** | 무관련 자극에 쉽게 반응, 특정 과제 주의 유지 어려움 |
| **과잉행동(다동성)** | 주의 산만·침착성 결여, 학습장애·충동성 동반 가능 |
| **고집성** | 한 자극에 집착, 다른 것으로 전환 어려움 |
| **충동성** | 목적 없이 반응하는 경향 |
| **통합곤란** | 대상을 통일된 형태로 인식하기 어려움 |

## 4. 신체 및 운동 특성 (정형외과적 손상)

### 고관절 탈구

- 고관절 주변 근육 경직 → 대퇴골이 고관절에서 이탈
- 심한 사지마비 아동에게 빈발, 척추측만 유발

### 관절 구축

- 과긴장 지속 → 관절막·인대·근육 길이 단축, 영구적 단축
- **예방**: 능동적 팔·다리 사용 권장, 기능 과제로 통합된 관절가동범위 운동

| 관련 변형 | 설명 |
|-----------|------|
| **내반족** | 발바닥이 몸 중앙을 향해 휘어진 상태 |
| **외반족** | 발 안쪽이 내려앉고 바깥쪽이 솟아오른 변형 |
| **첨족** | 발가락 관절이 아래로 굽어 발끝으로 걷는 형태 |

### 척추 만곡증

| 유형 | 정의 |
|------|------|
| **척추후만증** | 요추부가 꼿꼿하게 서 있는 상태 |
| **척추전만증** | 요추부가 앞으로 돌출, 허리 뒤로 굽고 배가 나온 것처럼 보임 |
| **척추측만증** | 근육 비대칭 긴장 → 척추가 S자·C자형으로 만곡 |

**척추측만증 원인별 분류**

| 유형 | 특징 |
|------|------|
| 특발성 | 원인 불명, 장애 없는 청소년에게도 발견 |
| 선천성 | 출생 시 추골 구조 비정상 |
| 신경근성 | 뇌성마비·이분척추·소아마비·근이영양증 합병증 |
| 비구조적(보상적) | 고정 기형 아닌 단순 만곡, 영구 변화 없음 |

> ⚠️ **기출 포인트**: 신경근성 척추측만증 원인 질환(뇌성마비·이분척추·소아마비·근이영양증), 보바스 운동요법·TLSO 치료법도 확인.

## 5. 생리조절 특성

### 위식도 역류

- 조임근 이완 → 위 내용물 식도 역류
- **예방**: 소량 자주 식사, 걸쭉한 음식 제공, 식후 약 1시간 비스듬히 앉힘

### 요로감염

- **예방**: 항생제 치료 + 충분한 수분 섭취, 청결 지도, 잦은 기저귀 교체

## 6. 지각 특성

| 지각 특성 | 내용 |
|-----------|------|
| **공간 위치 지각** | 자신 기준 물체 위치 파악 어려움 (안/밖, 좌/우 혼동) |
| **공간 관계 지각** | 복수 물체 간 위치 관계 파악 어려움, 도형 베끼기 곤란 |
| **지각의 항상성** | 조명·거리 변화에도 색채·크기를 동일하게 지각하는 경향 파악 어려움 |
| **고유수용성 감각** | 보지 않고 신체 위치·자세·평형 파악하는 감각 결함 → 감각통합치료 |
| **도형-배경 지각장애** | 도형보다 배경에 반응, 페이지에서 학습 위치 찾기 어려움 |
| **형태 지각** | 넓이·길이 차이 식별 곤란 |
| **시-운동 협응장애** | 시각과 신체 운동 조정 빈약 → 쓰기·읽기·일상생활 곤란 |
| **시지각 변별력** | 시각 정보 차이 구별 능력 저하 |
| **지각의 체계화** | 근접성·유사성·연속성·공통성·완결성 적용 어려움 |

> ⚠️ **기출 포인트**: 도형-배경 지각장애(배경에 반응), 시-운동 협응장애(구슬 꿰기로 지도), 고유수용성 감각(감각통합치료)이 자주 출제됨.

## 7. 근이영양증 (Muscular Dystrophy, MD)

### 정의

근디스트로핀이 점진적으로 상실되어 근육이 지방질로 바뀌며 퇴화하는 **불치성·유전성·진행성** 질병.

### 유형 비교

| 유형 | 유전 방식 | 주 발생 | 특징 |
|------|-----------|---------|------|
| **듀센형** | X염색체 결함, 반성 열성 | 남아 | 디스트로핀 부족·결핍, 진행 빠름 |
| **베커형** | X염색체 결함, 반성 열성 | 남아 | 소량의 비효과적 디스트로핀 존재, 듀센형보다 경미 |
| **안면견갑상완형** | 상동염색체 우성 | 남녀 10~40대 | 얼굴·어깨 상지부터 근력 퇴화, 익상견갑 |
| **지대형** | 상동염색체 열성 | 남녀 | 견관절·고관절부 근육 약화부터 시작, 진행 느림 |

> ⚠️ **기출 포인트**: 듀센형 유전 방식(반성 열성), 가우어스 징후, 가성비대 정의가 자주 출제됨.

### 듀센형 진행 과정

| 시기 | 주요 증상 |
|------|-----------|
| 영유아 | 첨족 보행, **가우어스 징후**, **가성비대증**, 다리·고관절 약화 |
| 5~10세 | 걷기 어려움 심화, 척추측만증 시작 |
| 10~12세 | 보행 기능 상실, 수동 휠체어 필요 |
| 청소년기 이후 | 전동 휠체어, 폐 감염·심근증으로 10대 후반~20대 초반 사망 |

**주요 징후**

- **가성비대(false growth)**: 근육처럼 보이는 종아리 비대 → 실제로는 지방세포·결합조직이 근육 조직 대체
- **가우어스 징후(Gower's sign)**: 바닥에서 일어날 때 양손으로 무릎·허벅지를 밀며 일어서는 모습
- **트렌델렌버그(오리걸음)**: 중둔근 약화로 반대쪽 골반이 내려가는 현상

### 교육적 지원

| 지원 영역 | 내용 |
|-----------|------|
| 심리사회적 | 상담·격려, 또래 격리 예방 |
| 신체적 | 근력보다 **근지구력** 중점, 적당한 운동 (과도한 휴식 금지) |
| 일상생활 | 일정 간격 체위 변경, 에어 매트리스, 저항 낮은 운동 |
| 자율성 | 높이 조절 가능한 책상, 보조공학 기기 제공 |

## 8. 이분척추 (Spina Bifida)

### 정의

임신 초기(28일 무렵) 척수가 완전히 닫히지 못한 채 태어나는 **선천적 이상**. 엽산 부족·유전자 이상이 원인.

### 유형 비교

| 유형 | 특징 | 마비/감각 |
|------|------|-----------|
| **척수수막류** | 척수·신경근이 척추 밖으로 탈출, 가장 심각 | 하지마비·방광장애, 70~90% 뇌수종 |
| **수막류** | 뇌척수막만 돌출, 척수 영향 없음 | 마비·감각 손상 없음, 외과 수술로 완치 가능 |
| **잠재이분척추** | 뼈 결손만, 외관상 눈에 안 띔 | 마비·감각 손상 없음, 피부 변색·털 등 |

> ⚠️ **기출 포인트**: 척수수막류의 뇌수종 동반(70~90%), 션트 수술 목적, 간헐적 청결 도뇨법 절차가 자주 출제됨.

### 의료적 처치

| 처치 | 내용 |
|------|------|
| **외과수술** | 출생 후 돌출 낭 제거, 척추 열린 부분 봉합 |
| **션트** | 뇌실 내 뇌척수액을 다른 신체 부위로 배출 → 뇌압 상승 방지 |
| **간헐적 청결 도뇨법** | 도뇨관을 요도→방광 삽입, 소변 배출 → 신장 역류·요도감염 예방 |
| **정형외과적 처치** | 교정기·부목으로 척추·다리·엉덩이 변형 교정 |

### 교육적 지원

- **션트 삽입 학생**: 추상 개념 이해 어려움, 주의집중·기억력 어려움 → 구체적 사례 제시, 그래픽 조직자 활용
- 물리치료(보행 훈련), 작업치료(자조 기술·소근육 운동) 협력 필요

## 9. 경련장애 (발작, Seizure)

### 정의

뇌에서 전기에너지가 **과도하게 방출**되어 나타나는 의식·감각·행동의 혼란. 만성 반복 시 **간질(epilepsy)**.

**전조/전구 증상**
- **전조(aura)**: 발작 직전 나타나는 부분발작 형태 (시각·감각 이상)
- **전구 증상(prodromal)**: 불안·두통·피로·메스꺼움 등 발작 전 수시간~수일 전부터 나타남

### 발작 유형

| 분류 | 유형 | 특징 |
|------|------|------|
| **전신발작** | 전신 긴장성·간대성 발작(대발작) | 의식 상실·경직·수축·이완 반복, 2~3분, 각성 후 방향감각 상실 |
| **전신발작** | 부재발작(소발작) | 순간 의식 상실 1초~90초, 어린아이 다발, 하루 100번까지 |
| **부분발작** | 단순 부분발작 | 의식 있음, 일부 근육 떨림, 감각 이상 |
| **부분발작** | 잭슨형 발작 | 의식 변화 없음, 경직성 마비가 한 부위→전신으로 확산 |
| **부분발작** | 자율신경 발작 | 창백, 땀, 동공 확대, 맥박 증가 |
| **부분발작** | 복합 부분발작(정신운동 발작) | 의식 저하·행동 변화, 2~5분, 발작 중 기억 없음 |

> ⚠️ **기출 포인트**: 부재발작(소발작) 특징(의식 순간 상실, 하루 100번), 잭슨형 발작(한 부위→전신 확산), 발작 대처 시 입에 아무것도 넣지 않음이 자주 출제됨.

### 치료 방법

| 치료 | 내용 |
|------|------|
| **항경련제** | 발작 억제, 부작용: 과다행동·주의집중 어려움·수면장애 |
| **케톤 식이요법** | 탄수화물·단백질 제한, 지방 중심 식이 → 대사 전환으로 발작 조절 |
| **미주신경 자극(VNS)** | 왼쪽 목 미주신경에 미세 전기 자극 → 약으로 통제 안 되는 경우 |
| **수술적 치료** | 비정상 전기 방출 뇌 부분 절제 (병소 제거술·대뇌반구 절제술·뇌량 절제술) |

### 발작 시 대처

**경련 중**
1. 주변 위험물 제거, 머리 밑에 쿠션·담요
2. 목 주위 옷 느슨하게, 옆으로 눕혀 기도 유지
3. 입을 강제로 열거나 입 속에 아무것도 넣지 않음
4. 발작 멈추게 하려 흔들거나 억압하지 않음

**경련 후**
1. 완전히 깰 때까지 옆에서 지켜봄
2. 의식 회복 전 음식·음료 제공 금지
3. 학생의 자존감 보호

**구급차 출동 기준**
- 발작 후 1분 이상 호흡 없을 때
- 전신 긴장성·간대성 발작이 **5분 이상** 지속될 때
- 의식 회복 없이 연이어 발작
- 발작이 처음 발생한 경우
- 부상 발생 또는 당뇨병이 있을 때

## 10. 골형성 부전증 (Osteogenesis Imperfecta)

### 정의 및 특징

뼈가 약하여 작은 충격에도 쉽게 골절되는 **유전성 질환**.

| 증상 | 내용 |
|------|------|
| 주요 증상 | 골절, 난청, **청색공막** (공막이 얇아 맥락막 색이 비침) |
| 이차적 증상 | 왜소함, 치아 결손, 척추 만곡 |
| 기타 | 유스타키오관 문제로 귀 감염 빈번, 정상 지능, 골절 빈도는 나이 들수록 감소 |

### 교육적 지원

- 척추측만 예방 자세 교정, 교실을 **1층 배치**
- 적당한 신체 활동으로 잔존 뼈 강화 (골 손상 가능 활동 금지)
- 청각 재활 필요, 심리적 위축 예방을 위한 사회·심리적 지원

## 11. 외상성 뇌손상 (Traumatic Brain Injury, TBI)

### 정의 및 분류

외부 물리적 충격에 의한 **후천적** 두뇌 손상.

| 유형 | 특징 |
|------|------|
| **개방성** | 날카로운 물체가 두개골·뇌 관통 → 뇌 특정 부분에 **제한적** 손상 |
| **폐쇄성** | 머리가 맞거나 눌리거나 흔들림 → **분산된(넓게 퍼진)** 손상 |

> ⚠️ **기출 포인트**: 개방성(제한적 손상) vs 폐쇄성(분산된 손상) 대비, 교육 복귀 시점(10~15분 집중 가능 시), 만성 피로 예방이 출제됨.

### 특징

- 경도 뇌손상도 수개월 지속 시 심한 인지·행동장애 초래 가능
- 심각한 뇌손상의 경우 대부분 영구적 장애

### 교육적 지원

| 지원 영역 | 내용 |
|-----------|------|
| **일반적 지원** | 수업 10~15분 집중 가능할 때 복귀 권장, 만성 피로 예방 위해 학습 부담 감소·자주 휴식 |
| **인지적 지원** | 불필요한 자극 차단, 출입문·창문가 배치 금지, 반복 수업으로 기억력 결함 보완 |
| **공격 성향 지원** | 선행 사건 수정, 감정 조절 역할 모델 시연 |
| **심리사회적 지원** | 성취 가능한 과제 제시, 장점 부각, 구조화된 또래 협력 활동으로 통제력 습득 |
