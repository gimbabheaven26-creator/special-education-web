## 1. 자세 및 앉기 지도

지체장애 학생의 좋은 자세는 신체적 안정성과 기능성을 극대화하고 이차적 장애를 예방하는 데 필수적이며, 이를 위해 근 긴장도, 신체 정렬, 자세 반사 등을 고려한 개별화된 자세 및 앉기 지도 전략과 보조기기 활용이 중요하다.

> ⚠️ **기출 포인트**: **운동 조절점**(목·어깨·척추·골반)을 정상 위치로 잡아주는 것이 비정상적 근 긴장도·자세 교정의 핵심 원리이다. 단순 보조기기 사용과 구분할 것.

### 1.1. 좋은 자세

| 항목 | 내용 |
|------|------|
| 정의 | 에너지 소비 최소화, 어느 방향으로든 잘 반응할 수 있으며 환경과 상호작용이 잘 되는 자세 |
| 필수 조건 | ① 무게 중심이 신체에 균등하게 분배 ② 체중 지지면 안정 ③ 정중선을 중심으로 대칭 |
| 효과 | 기형 예방, 신체 손상 감소, 학습·일상생활 기능 향상 |
| 양손 사용 중요성 | 척추측만 등 신체 불균형 보완; 머리 조절·안정성 확보; 원시반사 예방 |

**좋은 자세 유지 방법**

1. 적어도 2~3시간마다 자세를 바꿔 혈액 순환 유지
2. 신체적 요인, 활동 종류, 생활 연령, 기술 수준 등을 고려하여 자세 결정
3. 체중 지지면 안정 + 무게 중심 균등 분배
4. 신체 정렬이 중앙선을 중심으로 대칭 유지

### 1.2. 자세의 특징

| 특징 요소 | 정의 | 문제점 |
|-----------|------|--------|
| **근 긴장도** | 중추신경계가 근육에 지속적으로 영향을 미치는 긴장 상태; 신전·굴곡의 양과 정도 | 비정상적 근 긴장도 → 자세·움직임·호흡·언어·학습·경련 등 다양한 이차 문제 유발 |
| **신체 정렬** | 신체 각 부위의 상대적 관계; 공간·중력 방향에 대한 신체 전체의 관계 | 자세·움직임 문제가 일상생활 어려움 유발; 척추 기형·요로 감염·호흡 장애 초래 |
| **자세반사** | 신체가 중력에 대항하여 스스로를 유지하려는 무의식적 대응 과정 | 비정상적인 자세반사 → 비정상적 협응; 눕기·앉기·서기 자세 유지 어려움 |

**자세반사 발달 순서**: 앙와위(배와위) → 복와위 → 좌위 → 네발 기기 → 서기 → 걷기 → 달리기

### 1.3. 자세 및 앉기 지도

**자세 지도의 목적**

1. 신체 정렬·안정성 제공, 적절한 근 긴장도 유지
2. 기형·이차 근육 장애 예방, 심리적 두려움 감소
3. 안정된 자세 → 상지 사용 기능 극대화

**자세 지도 6대 원칙**

| 원칙 | 내용 |
|------|------|
| 최소 지원·용암 | 필요 이상의 신체적 보조 금지; 점차 지원 줄여나가기 |
| 비정상적 자세 패턴 소거 | 비정상적 반사·자세에 반대 자세를 취해 줌으로써 이탈 경험 제공 |
| **운동 조절점 사용** | 목·어깨·척추·골반의 정상 위치 잡기 → 비정상적 근 긴장·자세 교정 극대화 |
| 균형 자율 운동 촉진 | 정위반사·평형반사·보호반사 등 자율 운동 촉진 |
| 최대한의 움직임 기회 제공 | 관절 가동 범위(ROM) 유지를 위해 하루 일과 내 꾸준한 운동 |
| 자세 바꾸어 주기 | 한 자세 장시간 유지 금지; 서 있기·옆으로 눕기·엎드리기·누운 자세 등 대안 제공 |

> ⚠️ **기출 포인트**: **운동 조절점**(key points of control)은 목·어깨·척추·골반이다. 이 부위의 정상 위치를 잡아주면 비정상적 근 긴장 및 자세를 전신적으로 교정할 수 있다.

**자세 유형별 지도 방법**

| 자세 유형 | 적용 조건·방법 |
|-----------|--------------|
| 앉기 | 목 가누기 가능 + 등 신전 가능한 경우 시작 |
| 옆으로 누운 자세 | 머리 가누기 어렵거나 엎드리기 어려운 경우; 주위를 둘러볼 수 있게 함 |
| 엎드린 자세 | 체중 좌우 균형 분산; 어려운 경우 수건을 말아 앞가슴 밑에 넣어 팔을 앞으로 유도 |
| 누운 자세 | 쿠션으로 좌우 균형 유지; 근 긴장도 높은 경우 목받침·베개로 긴장 낮춤 |
| 서기 자세 | 엉덩이·무릎 신전; 팔 자유 사용 위해 책상 높이 조절 |

**앉기 자세 지원 요소**

| 지지 부위 | 방법 |
|-----------|------|
| 골반·고관절 | 좌석 벨트·팔걸이·엉덩이 받침 활용 |
| 하지 | 허벅지가 좌석에 닿도록; 무릎·발목 직각 유지 |
| 다리 정렬 | **외전대**(다리 바깥쪽 배치, 외반 방지) / **내전대**(무릎 사이 배치, 내반 방지) |

### 1.4. 자세 보조기기

> ⚠️ **기출 포인트**: 프론 스탠더(엎드린 자세로 기립 보조)와 수파인 스탠더(등을 대고 누운 자세로 기립 보조)의 차이를 구분할 것.

**누운 자세 보조기기**

| 항목 | 내용 |
|------|------|
| 목적 | 수면 중 질식 방지, 자세 변형 예방, 다양한 체위 변환 |
| 유형 | 자세 교정용 쿠션(positioning wedge), 매트, 경사형 보조대(Wedge), 에어매트 |

**앉기 자세 보조기기**

| 유형 | 특징 | 주요 대상 |
|------|------|-----------|
| **피더 시트(feeder seat)** | 머리 조절 능력 촉진, 균형 감각 향상 | 근 긴장도가 낮은 아동 |
| **학습용 의자** | 일반 의자에 쿠션·벨트·팔걸이·발받침 부착 수정 | 일반적 자세 지원 필요 학생 |
| **맞춤형 착석 시스템(inner system)** | 개인 신체 특성·용도에 맞게 맞춤 제작 | 머리·몸통 조절 어렵거나 경직 심한 경우 |
| **코너 체어** | 각진 등받이로 어깨를 앞으로 모아 머리 뒤로 꺾임 방지 | 머리 조절 지원 필요 학생 |

**서기 자세 보조기기**

| 유형 | 특징 |
|------|------|
| **프론 스탠더(prone stander)** | 엎드린 자세에서 다리·몸통 고정 후 각도 세워 기립 보조 |
| **수파인 스탠더(supine stander)** | 등을 대고 누운 자세에서 다리·몸통 고정 후 각도 세워 기립 보조 (상체·하체 조절 저조한 경우) |
| **기립대** | 균형감 없어도 선 자세 유지·지지 |
| **스탠딩 테이블** | 몸통·다리 근육 제한으로 스스로 서기 어려운 학생 지원 |

**기타 자세 보조기기**

| 보조기기 | 용도 |
|----------|------|
| 머리 조절용 보조기기 | 머리 움직임·정렬 유지 불가 시 |
| 등받이·의자 내 삽입장치 | 체형에 맞지 않는 의자·휠체어로 인한 척추 만곡 보완 |
| 몸통·상체 지지용 | 정중앙 자세 유지, 몸통 굴곡 방지 |
| 하지 지지용 | 무릎 굴곡 구축 90도 이상 방지; 발 받침대·고정끈 제공 |
| 치료용 공·롤 | 자세 교정·감각 운동 활동 도구 |

## 2. 보행 및 이동 기술

지체장애 학생의 독립적인 이동성 확보는 사회적 자립의 핵심 목표이며, 보행 및 이동용 보조기기 선택 시 학생의 기능, 환경, 신체 변형 예방 등을 종합적으로 고려해야 한다.

> ⚠️ **기출 포인트**: 목발(크러치) 계단 이동 원칙 — **내려갈 때**: 목발+불편한 발 먼저 → 건강한 발 / **올라갈 때**: 건강한 발 먼저 → 목발+불편한 발. "좋은 발이 먼저 천국(올라갈 때)으로 간다"로 기억.

### 2.1. 보행·이동 보조기기 선택 기준

연령, 선호도, 기기 사용 환경, 이동의 자립성·효율성, 앉기 및 자세 상태 등 종합 고려.

### 2.2. 보장구

| 유형 | 특징 |
|------|------|
| **보조기** | 근육 지지·고정 → 바른 자세 유지, 구축 예방 |
| **부목(스플린트)** | 구축·변형 최소화; 수술 후 변형 재발 방지 |
| **석고붕대(Cast/깁스)** | 과도한 근긴장도 감소; 관절 구축 완화 및 근육 펴기 |

유의점: 맞춤 제작이므로 학생 성장 시 수정 필요; 피부에 직접 닿지 않도록 면 셔츠·양말 착용

### 2.3. 이동용 보조기기

| 유형 | 장점 | 단점 | 주요 고려 요소 |
|------|------|------|--------------|
| **일반 수동 휠체어** | 핸들 조작 쉬움; 보관·수리 수월; 학교생활 편리 | 자기 조작 근력 필요 또는 타인 도움 필요 | 무게, 안정성, 내구성, 조작 능력 |
| **전동 휠체어** | 중증 장애인에 적합; 타인 도움 불필요 | 가격 비쌈; 보관·수리 어려움; 무거움 | 기능 상태, 인지 상태, 시각, 접근성 |
| **스포츠용 휠체어** | 레저·스포츠 활동 참여 | — | 몸통 조절 능력 양호한 경우 |
| **전동 스쿠터** | 조작 능력 우수한 경우 | — | 착석 기술·이동 기술이 더 나은 학생 |
| **스트롤러** | 초기 이동 편의 | — | 발달 초기 진단 받은 학생 |

### 2.4. 보행용 보조기기

| 유형 | 특징 |
|------|------|
| **지팡이(cane)** | 보행 중 균형 유지·안정성 확보; 보행 능력 있는 학생 |
| **목발(crutch/크러치)** | 나무·알루미늄 재질; 끝에 고무 부착; 키에 따라 높이 조절 |
| **워커(walker)** | 독립 보행 가능 학생의 수직 이동 지원; 전방지지형·후방지지형·몸통팔지지형 |
| **게이트 트레이너(gait trainer)** | 균형 잡기·근육 통제 문제 학생의 걷기 훈련 |
| **리프트(lift)** | 스스로 이동 불가 학생의 자리 이동(휠체어↔침대·의자 등) |

**목발 계단 이동 방법**

| 방향 | 순서 |
|------|------|
| 계단 내려갈 때 | 목발 + 불편한 발 동시 먼저 → 건강한 발 |
| 계단 올라갈 때 | 건강한 발 먼저 → 목발 + 불편한 발 동시 |

**워커 지도**: 초기 → 몸통·팔 지지형 워커 → 걷기 능력 향상 시 → 일반형 워커로 교체

**보행 벨트**: 과도한 체중 부하, 균형 상실, 넘어짐 예방을 위해 허리에 착용

### 2.5. 안아서 이동하기

| 상황 | 원칙 |
|------|------|
| 아동 체중이 성인 체중의 35% 초과 | 혼자 들어올리기 시도 금지 |
| 한 사람이 바닥에서 들어올리기 | 장비를 가까이 가져오거나 아동을 장비 근처로 이동; 허리 곧게 유지한 채 다리 근육 사용 |

**들어올리기 6단계 과정**

| 단계 | 내용 |
|------|------|
| 1. 접촉 | 팔·어깨에 손을 얹고 이동 목적지 안내 |
| 2. 의사소통 | 목적지 관련 사진·사물 제시 |
| 3. 준비 | 근육 긴장 여부 확인 |
| 4. 들어올리기 | 서지 못할 경우 앉은 채로 이동 |
| 5. 이동하기 | 아동이 이동 방향을 볼 수 있도록 거리 유지 |
| 6. 다시 자세 잡기 | 다음 활동 참여 자세 확보 |

**바르게 안는 법**: 머리·체간 수직 유지 + 팔 안으로 안기 + 허리 걸치는 자세 → 이상 자세·병적 반사 억제, 정상 반사 발달 촉진

## 3. 일상생활 기술

지체장애 학생의 일상생활 기술 지도는 섭식, 용변, 착탈의, 몸단장, 치아 관리 등 다양한 영역에서 개별화된 접근과 보조기기 활용을 통해 독립성을 증진하고 삶의 질을 향상시키는 것을 목표로 한다.

> ⚠️ **기출 포인트**: 섭식 지도 시 **음식 제시 위치는 얼굴 아래**가 원칙이고, 먹이는 사람은 **아동의 옆 또는 뒤**에서 보조한다. 또한 바른 자세는 **45도 체간 각도**가 핵심이다.

### 3.1. 식사 기술

**섭식 기능 3요소**

| 기능 | 내용 |
|------|------|
| 저작(씹기) | 음식물을 입에서 잘게 씹는 기능 |
| 빨기 | 컵·숟가락 등을 이용해 구강으로 보내는 기능 |
| 삼키기 | 저작된 음식물을 식도를 통해 위장으로 보내는 기능 |

섭식과 구어 산출은 구강에서 이루어지는 공통점이 있으며, 올바른 식사 패턴이 잘 형성되면 언어 발달에도 긍정적 영향을 미친다.

**섭식 기술의 어려움 유형**

| 유형 | 내용 |
|------|------|
| 근 긴장도 이상 | 저긴장성(씹기 시 머리·턱·입술 움직임 저하) / 과긴장성(구강 근육 과도 긴장으로 움직임 제한) |
| **정향반사(설근반사)** | 입 주위 자극 시 입술이 자극 쪽으로 따라가는 반사 → 식사 시 자발적 머리 조절 방해 |
| **강직성 씹기 반사** | 입 안 음식 제공 시 의도하지 않게 갑자기 입을 다무는 강직성 |
| **혀 밀기/돌출 행동** | 씹기·삼키기 대신 치아 사이로 혀를 밀어내는 비자발적 행동 |
| 빨고 삼키기 | 고형 음식 섭취 방해; 씹지 않고 빨다가 삼키는 행동 |
| **구역반사** | 구개·혀에 물체 접촉 시 구역질 유발 |
| **삼킴장애** | 인두 단계에서 음식물을 인두로 밀기 위한 압력 생성 불가 |
| **흡인** | 액체나 작은 음식 조각이 폐로 유입 |

**흡인 예방**: 머리 약간 앞으로 구부린 똑바른 자세 + 고체·반고체 또는 진한 액체 섭취 + 변비 예방

**기도폐쇄 응급처치**

| 방법 | 내용 |
|------|------|
| **하임리히 구명법(횡경막하 복부밀기)** | 뒤에서 안아 명치 끝 복부를 아래쪽으로 강하게 밀쳐 올림 |
| **흉곽밀기** | 뒤에서 잡을 수 없을 때 대안; 반듯이 눕히고 흉골 바로 아래 손 대어 빠르고 강하게 앞으로 밀기 |

**섭식 기술 지도 원칙**

| 항목 | 내용 |
|------|------|
| 자세 교정 | 체간 45도까지 일으켜 → 능동적 음식물 처리·연하 용이 |
| 음식 제시 위치 | **아동 얼굴 아래** (위치에 따라 근 긴장도 변화) |
| 신체적 지원 위치 | 먹이는 사람은 **아동의 옆 또는 뒤**에서 최소 보조 |
| 음식 투입 위치 | **혀의 중앙 부분**; 턱 움직임 제한 시 치아 사이에 직접 |
| 급식 속도 | 학생의 숨쉬기에 맞추어 충분히 느리게 |

**음식 형태 수정 단계**

| 형태 | 설명 |
|------|------|
| 유동식 | 걸쭉한 액체 |
| 퓨레 | 야채·곡류 삶아 걸쭉하게 만든 것 |
| 연식 | 부드러운 음식 |
| 고형식 | 일반식 |

지도 원칙: 작은 알갱이보다 **덩어리가 큰 음식**부터 시작

**식사 도구별 지도 유의사항**

| 도구 | 유의사항 |
|------|---------|
| 컵 | 컵 가장자리를 아랫입술에 놓아 깨무는 자극 감소; 충분히 기울여 음료 유입 |
| 숟가락 | 입 크기·구강 민감도·경직성 물기 반사·쥐기 특성 고려하여 선택 |
| 빨대 | 혀 위에 얹고 혀 전후 운동으로 마시지 않도록 지도 |

**구강 운동**: 입술·뺨 두드리기, 잇몸·입천장 마사지, 씹기·삼키기 관련 피부 문지르기; **번게이드법**(수동·능동·저항 운동)

**구강 외 섭식(위관 영양/튜브 섭식)**

- 정의: 입으로 음식 섭취 불가 시 위·장에 관 삽입하는 방법
- 방법: 집중식, 간헐적 중력 점적식, 연속법
- 유의사항: 관 배치·개방성 점검 필수; 자연스러운 식사 시간에 교실에서 실시

### 3.2. 용변 기술

**용변 훈련 선행 조건**: 배설 시간이 비교적 정기적·예측 가능 + 옷에 실수 없이 1~2시간 버틸 수 있는 능력

**용변 기술 3단계**

| 단계 | 목표 |
|------|------|
| 1단계 습관 만들기 | 규칙적인 계획표에 따라 변기에 앉는 경험 |
| 2단계 자기주도적 용변 | 화장실 가야 할 필요성 인식·징후 나타내기 |
| 3단계 독립적 화장실 사용 | 화장실 이용 전 과정 스스로 수행 |

**훈련 장소 단계**: 교실 내 화장실 → 교실 내 칸막이 → 일반 화장실

**용변 훈련 규칙**: 성공 시 강화; 실수 시 지체 없이 청결 처리; 혼내지 않기; 깨끗이 할 때까지 학급 활동 일시 제외

> ⚠️ **기출 포인트**: 용변 훈련 **선행 조건**(예측 가능한 배설 + 1~2시간 억제 능력)과 3단계 목적 서술형 출제.

**자세 교정**: 골반·엉덩이·몸통 근육의 안정성 확보가 화장실 훈련의 필수 요소.

### 3.3. 착탈의 및 몸단장 기술

**착탈의 기술 발달 원칙**

| 원칙 | 내용 |
|------|------|
| 순서 | 옷 **벗기** 기술이 **입기**보다 먼저 발달 |
| 운동 순서 | 대근육 운동 필요 기술 → 소근육 운동 필요 기술 순 습득 |

**지체장애 유형별 옷 입기**

| 유형 | 방법 |
|------|------|
| **편마비** | 입을 때: **마비 쪽 소매 먼저** 어깨까지 → 비마비 쪽 소매 끼움 |
| **근이영양증** | 좌우로 몸을 흔들며 체중 이동 시 넘어지지 않으면 시간이 걸려도 옷 입기 가능 |
| **뇌성마비(불수의 운동형)** | 불수의 운동·변화하는 근 긴장 문제로 일정한 자세 유지 어려움 |

> ⚠️ **기출 포인트**: 편마비 학생의 착의 순서 — **마비 쪽 먼저 입고, 비마비 쪽 먼저 벗는다.** ("마비 측 먼저, 건강 측 나중"으로 기억)

**착탈의 훈련 5단계**: 성인 협력 요구 단계 → 감독 없는 독립 단계 (점진적 독립심 증진)

### 3.4. 치아관리

**이 닦기 지도 시 유의사항**

| 특성 | 지도 시 유의사항 |
|------|----------------|
| 감각적 측면 | 입 주변에 사물이 닿으면 깜짝 놀라므로 구강 주변 충분히 마사지 후 지도 |
| 도구적 측면 | 거친 질감 거부 반응 있으므로 부드러운 칫솔모 사용 |

- 충치 예방: 식사 후 이 닦기 + 당분 많은 음식 피하기 + 칼슘·비타민 섭취
- 치아 검진: **연 3~4회** 정기 검진 필수
- 독립 수행 어려운 아동: 입 벌리기·헹구기·고개 젖히기·수건으로 입 닦기 등 **부분 참여** 지도

### 3.5. 몸단장 및 개인위생

| 항목 | 내용 |
|------|------|
| 목적 | 개인위생 기초 기술 습득; 자기 표현; 외모 관리 |
| 편마비 학생 | 마비된 손의 아랫부분을 비마비 손으로 받쳐 **양손 협응** 세면 지도 |
| 환경 수정 | 세면대 수도 위치까지 손이 닿지 않는 학생 → 연장 탭 설치 |
| 지원 원칙 | 잔존 능력 파악 + **최소한의 지원**으로 최대 자기 참여 유도 |

### 3.6. 영양관리 및 의료관리

**영양관리 유의사항**

- 또래 대비 영양 결핍 발생 쉬움; 의사 표현 부족 시 수분 섭취 상태 주의 깊게 관찰
- 항경련제 장기 복용 → 비타민 D·칼슘 대사 방해 + 체중 영향

**욕창(압박궤양)**

| 항목 | 내용 |
|------|------|
| 정의 | 뼈의 돌출부에 지속적·반복적 압박 → 혈액 순환 불량 → 조직 괴사로 인한 궤양 |
| 주요 원인 | 외부 압력의 지속적 작용; **변실금**은 요실금보다 더 중요한 위험 요인 |
| 예방 | 자주 자세·무게 중심 변환; 적절한 영양·수분 섭취 (저단백질증·빈혈·비타민 부족 시 저항력 저하) |
| 교육 | 중복·지체장애 학생은 통증 표현 어려움 → **몸짓 등 비구어적 신호** 개발을 의사소통 지도 목표에 포함 |

> ⚠️ **기출 포인트**: 욕창 위험 요인에서 **변실금이 요실금보다 더 중요한 위험 요인**임을 구분할 것.
