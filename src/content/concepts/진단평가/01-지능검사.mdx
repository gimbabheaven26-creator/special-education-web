## 1. 한국판 웩슬러 아동용 지능 검사 (K-WISC-IV)

> ⚠️ **기출 포인트**: 4개 지표와 각 지표에 속하는 주요·보충 소검사 구분, 처리점수의 특성(합산점수 미포함), 장애학생 적용 시 다양한 검사 조정 가능 여부가 반복 출제된다.

### 기본 정보

| 항목 | 내용 |
|------|------|
| 대상 | 만 6세 0개월 ~ 만 16세 11개월 |
| 목적 | 아동의 소검사 수행에 기여하는 인지적 능력에 대한 자세한 정보 제공 |
| 구성 | 4개 지표, 10개 주요 소검사 + 5개 보충 소검사 |

---

### 4개 지표 및 소검사 구성

| 지표 | 주요 소검사 (3개 기본) | 보충 소검사 |
|------|----------------------|------------|
| **언어이해(VCI)** | 공통성, 어휘, 이해 | 상식, 단어추리 |
| **지각추론(PRI)** | 토막짜기, 공통그림찾기, 행렬추리 | 빠진곳찾기 |
| **작업기억(WMI)** | 숫자, 순차연결 | 산수 |
| **처리속도(PSI)** | 기호쓰기, 동형찾기 | 선택 |

> 💡 **암기 팁**: 언지이해(공어이) / 지각추론(토공행) / 작업기억(숫순) / 처리속도(기동)

---

### 주요 특징

1. **규준 상향**: 기존 검사 대비 지능이 낮게 산출될 수 있음 → 검사 결과 해석 시 **부모의 교육수준** 참조 필요
2. **장애학생 적용**: 지적장애·영재 판별 및 신체적·언어적·감각적 제한이 있는 아동에게 **다양한 검사 조정** 가능
3. **보충검사 활용 조건**: 장애학생 또는 1년 이내 반복 검사 시 5개 보충검사 활용
4. **산수·선택 소검사**: 작업기억·처리속도 주요 소검사가 무효가 될 경우를 대비해 **항상 실시**

---

### 실시 방법

| 단계 | 내용 |
|------|------|
| 순서 | 주요 소검사 먼저 → 보충 소검사 이어서 실시 |
| 시작점 | 생활연령에 맞는 시작점(검사 시간 단축 + 흥미 유지 목적) |
| 중지 규칙 | **연속 0점** 시 검사 중지 |
| 대체 가능 | 상황에 따라 주요 소검사 → 보충 소검사로 대체 가능 |

---

### 결과 산출

| 점수 유형 | 설명 |
|----------|------|
| **전체 IQ** | 전체적 인지능력 (4개 지표 합산) |
| **합산점수** | 연령집단 평균 수행능력을 나타냄 |
| **환산점수** | 또래 연령집단 내 상대적 수행 결과 |
| **처리점수** | 토막짜기·숫자·선택 3개 소검사에서 7개 산출 → **합산점수에 미포함**, 다른 소검사로 대체 불가 |

## 2. 한국판 카우프만 지능 검사 (K-ABC-II)

> ⚠️ **기출 포인트**: 비언어성 척도의 적용 대상(청각장애·언어장애·다문화), 5개 척도 명칭과 2개 지표(인지처리지표·유동성지표) 구분, 문화적 변인 영향 최소화 특성이 자주 출제된다.

### 기본 정보

| 항목 | 내용 |
|------|------|
| 대상 | 만 3세 ~ 만 18세 |
| 목적 | 정보처리능력 및 인지능력 측정 (심리·임상·심리교육·신경심리 평가) |
| 구성 | 2개 지표, 5개 척도, 18개 하위검사 |

---

### 2개 지표 vs. 5개 척도

| 구분 | 내용 |
|------|------|
| **인지처리지표** | 순차처리 + 동시처리 + 학습력 + 계획력 |
| **유동성지표** | 순차처리 + 동시처리 + 학습력 + 계획력 + **지식** 포함 |

| 척도 | 측정 내용 |
|------|----------|
| **순차처리** | 자극을 순서대로 처리하는 능력 |
| **동시처리** | 여러 자극을 동시에 통합·처리하는 능력 |
| **학습력** | 새로운 정보를 학습·저장하는 능력 |
| **계획력** | 문제해결을 위한 계획·실행 능력 (유동적 추론) |
| **지식** | 누적된 지식·결정화된 지능 |

---

### 주요 특징

| 특징 | 상세 |
|------|------|
| **비언어성 척도** | 청각 상실·언어 장애·다문화 가정 아동 등 제한된 언어 능력을 가진 아동을 **타당하게 평가** |
| **문화적 공정성** | 검사 내용이 문화적 변인의 영향을 받지 않도록 구성 |
| **질적 지표** | 각 하위검사마다 검사 중 관찰된 피검사자의 특별한 행동을 기록하여 결과 해석에 활용 |
| **적용 범위** | 또래 지능 평균 범위를 벗어나는 아동에게도 실시 가능 |

---

### 하위검사 유형

| 유형 | 기능 |
|------|------|
| **핵심하위검사** | 전체척도 지수 및 각 하위척도 지수 산출 |
| **보충하위검사** | 핵심하위검사로 측정된 능력·처리과정을 심도 있게 탐색할 때 보충 실시 |

---

## K-WISC-IV vs. K-ABC-II 비교

| 비교 항목 | K-WISC-IV | K-ABC-II |
|----------|----------|---------|
| 대상 연령 | 6세 0개월 ~ 16세 11개월 | 3세 ~ 18세 |
| 지표 수 | 4개 (언어이해·지각추론·작업기억·처리속도) | 2개 (인지처리·유동성) |
| 척도/소검사 | 15개 소검사 (10주요+5보충) | 5개 척도, 18개 하위검사 |
| 언어 제한 아동 | 검사 조정 가능 | **비언어성 척도** 제공 |
| 문화적 영향 | 언어 기반 문항 포함 | 문화적 변인 영향 최소화 |
| 처리점수 | 있음 (합산점수 미포함) | 질적 지표 기록 |
