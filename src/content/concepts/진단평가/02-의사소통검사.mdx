## 개요

아동의 구문 이해력, 어휘력, 언어 문제 해결 능력, 조음 및 음운 발달 수준을 종합적으로 평가하여 의사소통 장애를 **진단**하고 **개입 계획**을 수립하는 데 활용한다.

> ⚠️ **기출 포인트**: 5개 도구 각각의 **대상 연령·목적·결과 지표**가 자주 출제된다. 특히 구문이해력(문법초점/의미초점), 언어문제해결력(3범주), U-TAP(자음 정확도·모음 정확도)은 세부 구성까지 암기할 것.

---

## 5개 도구 한눈에 비교

| 검사명 | 대상 연령 | 핵심 목적 | 결과 지표 |
|--------|-----------|-----------|-----------|
| 구문이해력 검사 | 만 4~9세 | 구문의미 **이해력** 측정 | 연령·학년별 백분위 |
| 그림 어휘력 검사 | 만 2~8세 | **수용 어휘력** 검사 | 등가연령, 백분위 |
| 언어문제해결력 검사 | 만 5~12세 | 언어를 통한 **문제해결능력** 측정 | 3범주 + 총점 백분위 |
| 한국 표준 수용어휘력 검사 | 만 2~12세 (일반) / 만 3~8세 (지적장애) | **수용어휘력** 측정 | 백분율, 수용언어연령 |
| U-TAP (우리말 조음음운평가) | 만 20개월~12세 | **조음·음운** 발달 평가 | 자음 정확도, 모음 정확도 |

## 1. 구문이해력 검사

| 항목 | 내용 |
|------|------|
| **대상** | 만 4~9세 일반아동 및 장애아동 |
| **목적** | 구문의미 이해력 측정 |
| **구성** | 문법초점 38문항 + 의미초점 19문항 (총 57문항) |
| **실시 방법** | 검사자가 읽어주는 문장을 듣고 해당 그림을 지적 |
| **중단 기준** | 연습문항 2개 모두 실패 시 검사 중단 |
| **결과** | 연령별 백분위, 학년별 백분위 |

> ⚠️ **기출 포인트**: **문법초점(38)과 의미초점(19)** 문항 수 구분이 출제된다. 연습문항 2개 모두 실패 시 중단한다는 중단 기준도 확인할 것.

---

## 2. 그림 어휘력 검사

| 항목 | 내용 |
|------|------|
| **대상** | 만 2~8세 |
| **목적** | 수용 어휘력 검사 |
| **구성** | 범주영역 + 품사영역 |
| **실시 방법** | 검사자가 읽어주는 문장을 듣고 해당 그림을 지적 |
| **결과** | **등가연령**, 백분위 |

> ⚠️ **기출 포인트**: 결과 지표가 **등가연령**이라는 점이 다른 검사와 구별되는 특징이다.

---

## 3. 언어문제해결력 검사

| 항목 | 내용 |
|------|------|
| **대상** | 만 5~12세 (일반아동, 학습장애, 단순언어장애, 언어장애 아동) |
| **목적** | 언어를 통한 문제해결능력 측정 |
| **구성** | 17개 그림판 + 50개 문항 |
| **실시 방법** | 그림판을 보여주고 질문한 뒤 대답을 기록 |
| **결과** | 3범주 + 총점 백분위 |

### 결과 3범주

| 범주 | 설명 |
|------|------|
| **원인이유** | 현상의 원인을 언어로 설명하는 능력 |
| **해결추론** | 문제 상황의 해결책을 추론하는 능력 |
| **단서추측** | 단서를 바탕으로 결과를 추측하는 능력 |

> ⚠️ **기출 포인트**: **원인이유 / 해결추론 / 단서추측** 3범주 명칭을 정확히 암기할 것. 총점과 범주별로 각각 백분위를 제공한다.

---

## 4. 한국 표준 수용어휘력 검사

| 항목 | 내용 |
|------|------|
| **대상** | 일반아동 만 2~12세 / 지적장애 만 3~8세 |
| **목적** | 수용어휘력 측정 |
| **구성** | 500개 문항 (100개씩 5조) + 100개의 그림판 |
| **결과** | 백분율, **수용언어연령** |

> ⚠️ **기출 포인트**: **지적장애 대상 연령(만 3~8세)**이 일반아동(만 2~12세)과 다르다는 점을 구분할 것. 500문항을 100개씩 5조로 나눈 구성도 출제된 바 있다.

---

## 5. 우리말 조음음운평가 (U-TAP)

| 항목 | 내용 |
|------|------|
| **대상** | 만 20개월~12세 |
| **목적** | 조음·음운장애 아동의 단어·문장 수준 발음 평가 |
| **구성** | 그림낱말검사 + 그림문장검사 |
| **실시 순서** | 그림낱말검사 → 그림문장검사 |

### 검사 구성 상세

| 하위검사 | 검사 수준 | 내용 |
|----------|-----------|------|
| **그림낱말검사** | 낱말 수준 | 자음 및 단모음 검사 |
| **그림문장검사** | 문장 수준 | 자음 및 단모음 검사 |

### 결과 지표

| 지표 | 설명 |
|------|------|
| **자음 정확도** | 낱말/문장 수준별로 산출 |
| **모음 정확도** | 낱말/문장 수준별로 산출 |
| **음소 습득 연령 비교** | 정상 발달과의 비교 가능 |

> ⚠️ **기출 포인트**: U-TAP은 **낱말 수준과 문장 수준을 모두** 평가한다는 점, 결과가 **자음 정확도**와 **모음 정확도**로 각각 산출된다는 점이 핵심이다. 정상 발달 기준과 비교할 수 있는 규준 참조 검사이다.

## 검사 선택 가이드

```
의사소통 장애 의심 아동
        │
        ├─ 문장 이해에 어려움? → 구문이해력 검사 (만 4~9세)
        │
        ├─ 어휘 수준 확인 필요?
        │       ├─ 만 2~8세 → 그림 어휘력 검사 (등가연령 제공)
        │       └─ 만 2~12세 → 한국 표준 수용어휘력 검사
        │
        ├─ 언어 추론·문제해결 능력? → 언어문제해결력 검사 (만 5~12세)
        │
        └─ 발음·조음 문제? → U-TAP (만 20개월~12세)
```
