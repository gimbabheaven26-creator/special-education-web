## 개요

아동 및 성인의 적응행동 수준을 평가하여 지적장애 및 발달장애를 구별하고, 지역사회 통합을 위한 지원 계획 수립에 필요한 정보를 제공하는 검사 도구들이다.

적응행동 관련 검사는 크게 5가지로 구분된다.

| 검사명 | 약칭 | 대상 연령 | 핵심 특징 |
|--------|------|-----------|-----------|
| 사회성숙도검사 | SMS | 0~30세 | 사회연령(SA)·사회지수(SQ) 산출 |
| 지역사회적응검사 | CIS-A | 제한 없음 | 지원점수·지원강도 결정 |
| 지역사회적응검사 2판 | CISA-2 | 만 5세 이상 | 피검사자 직접 답변 방식 |
| KNISE 적응행동검사 | KNISE-SAB | 5~17세 | 적응행동지수(평균 100, SD 15) |
| 한국판 적응행동검사 | K-SIB-R | 0~18세 | 적응+부적응행동 조합 → 지원점수 |

## 1. 사회성숙도검사 (SMS)

> ⚠️ **기출 포인트**: SMS는 **사회적으로 무력한 지적장애**와 **사회적으로 무력하지 않은 지적장애**를 구별하는 도구로 출제된다. 결과 지표인 **사회연령(SA)**과 **사회지수(SQ)**도 자주 출제된다.

### 기본 정보

| 항목 | 내용 |
|------|------|
| 대상 | 0세 ~ 30세 |
| 문항 수 | 총 117문항 (난이도 순 배열) |
| 실시 방법 | 피검사자를 잘 아는 사람과의 **면접** |
| 산출 지표 | 사회연령(SA), 사회지수(SQ) |

### 실시 절차

1. 피검사자를 잘 아는 사람(부모·양육자)과의 면접으로 실시한다.
2. 정보제공자의 대답이 믿기 어려운 경우, 피검사자를 직접 만나 **행동을 관찰**하여 판단한다.
3. **6개월 이상** 아동을 관찰하여 행동 특성을 파악한 정보제공자와 면담한다.
4. 모든 소검사의 **1번 문항부터** 질문을 시작한다.
5. 피검사자의 연령과 능력을 고려하여 시작 질문 번호를 정한다.

### 채점 기호

| 기호 | 의미 |
|------|------|
| `+` | 수행함 |
| `+F` | 도움을 받아 수행함 |
| `+NO` | 기회가 없었으나 수행 가능 |
| `±` | 부분 수행 |
| `-` | 수행 못함 |

### 활용

- 개인의 성장 변화 측정 및 개인차 측정 도구
- 생활지도·이동 훈련의 기초 자료 수집
- **사회적으로 무력한 지적장애**와 **사회적으로 무력하지 않은 지적장애** 구별

## 2. 지역사회적응검사 (CIS-A)

> ⚠️ **기출 포인트**: CIS-A와 K-SIB-R은 **적응행동+부적응행동 점수를 조합하여 지원점수 산출 → 지원강도 결정**이라는 공통 활용법이 핵심 출제 포인트이다.

### 기본 정보

| 항목 | 내용 |
|------|------|
| 대상 | 나이 제한 없음 (지적·발달장애인) |
| 목적 | 지역사회 통합에 필수적인 적응기술 포괄적 평가 |
| 산출 지표 | 영역별 지수, 지역사회 적응지수(평균 100, SD 15) |

### 영역 구성

| 영역 | 내용 |
|------|------|
| 기본생활 영역 | 일상생활에 필요한 기본 적응기술 |
| 사회자립 영역 | 지역사회 내 독립생활 관련 기술 |
| 직업생활 영역 | 직업 적응 및 유지 관련 기술 |

### 실시 방법

1. 피검사자를 잘 아는 사람(부모·양육자)이 체크리스트 형식으로 기입하거나 검사자의 질문에 응답한다.
2. 검사자와 피검사자 **외에 타인이 없는** 검사실에서 실시한다.
3. 모든 하위척도 **1번 문항부터** 질문을 시작한다.

### 활용

적응행동과 부적응행동 점수를 **조합하여 지원점수**를 제공하며, 지원의 강도를 결정하는 데 활용한다.

## 3. 지역사회적응검사 2판 (CISA-2)

> ⚠️ **기출 포인트**: CISA-2는 CIS-A와 달리 **검사자가 질문 → 피검사자가 직접 답변**하는 방식이다. 대상 장애 유형(지적+자폐성)과 최소 연령(만 5세)을 구별해야 한다.

### 기본 정보

| 항목 | CISA-2 | CIS-A |
|------|--------|-------|
| 대상 | 만 5세 이상, 지적·자폐성장애인 | 나이 제한 없음, 지적·발달장애인 |
| 실시 방식 | **피검사자 직접 답변** | 정보제공자(부모·양육자) 응답 |
| 영역 구성 | 기본생활·사회자립·직업생활 | 동일 |

### 실시 방법

1. 검사자가 질문하면 **피검사자가 직접 답변**하는 형식으로 실시한다.
2. 요인의 순서는 바꿔서 질문할 수 있다.

## 4. KNISE 적응행동검사 (KNISE-SAB)

> ⚠️ **기출 포인트**: KNISE-SAB의 3가지 적응행동 영역(**개념적·사회적·실제적**)은 AAIDD 적응행동 분류와 연계되어 출제된다. 24개 영역 242문항이라는 구체적 수치도 확인해 두자.

### 기본 정보

| 항목 | 내용 |
|------|------|
| 개발 기관 | 국립특수교육원(KNISE) |
| 대상 | 지적장애 학생 5~17세, 일반학생 21개월~17세 |
| 목적 | 지적장애 및 발달장애 구별 |
| 문항 수 | 24개 영역, 총 242문항 |
| 산출 지표 | 원점수, 환산점수, 적응행동지수(평균 100, SD 15) |

### 영역 구성

**행동영역 검사 (6개 영역)**

| 영역 | 내용 |
|------|------|
| 자조 | 식사·위생·착탈의 등 자기돌봄 |
| 이동 | 이동 및 이동수단 활용 |
| 작업 | 과제 수행·작업 유지 |
| 의사소통 | 언어적·비언어적 의사소통 |
| 자기관리 | 자기조절 및 행동 통제 |
| 사회화 | 대인관계·사회적 상호작용 |

**3축 적응행동 분류**

| 분류 | 내용 |
|------|------|
| 개념적 적응행동 | 언어·읽기·쓰기·수 개념 등 |
| 사회적 적응행동 | 대인관계·책임감·자존감 등 |
| 실제적 적응행동 | 일상생활·직업·건강·안전 등 |

### 실시 절차

SMS와 동일한 절차를 따른다.

1. 피검사자를 잘 아는 사람과의 면접으로 실시한다.
2. 정보제공자의 대답이 믿기 어려운 경우, 피검사자를 직접 **관찰**하여 판단한다.
3. **6개월 이상** 아동을 관찰한 정보제공자와 면담한다.
4. 모든 소검사의 **1번 문항부터** 시작한다.
5. 피검사자의 연령·능력을 고려하여 시작 번호를 정한다.
6. `+` / `+F` / `+NO` / `±` / `-` 로 채점한다.

## 5. 한국판 적응행동검사 (K-SIB-R)

> ⚠️ **기출 포인트**: K-SIB-R은 **적응행동 + 부적응행동 점수 조합 → 지원점수 산출**이 핵심이다. 부적응행동의 3가지 유형(내적·외적·반사회적)을 구분할 수 있어야 한다.

### 기본 정보

| 항목 | 내용 |
|------|------|
| 대상 | 0세 ~ 18세 |
| 실시 방식 | 정보제공자(부모·양육자) 체크리스트 기입 또는 면담 |
| 산출 지표 | 영역별 지수, 지역사회 적응지수(평균 100, SD 15) |

### 영역 구성

**적응행동 (4개 영역)**

| 영역 | 내용 |
|------|------|
| 운동기술 | 대·소근육 운동 기술 |
| 사회적 상호작용 및 의사소통 기술 | 대인관계·언어·비언어 의사소통 |
| 개인생활기술 | 자조·일상생활 기술 |
| 지역사회생활기술 | 지역사회 내 독립생활 기술 |

**부적응행동 (3가지 유형)**

| 유형 | 예시 |
|------|------|
| 내적 부적응행동 | 자해·강박·위축 등 자신에게 향하는 행동 |
| 외적 부적응행동 | 파괴·공격·분노폭발 등 외부로 향하는 행동 |
| 반사회적 부적응행동 | 거짓말·도벽·무단결석 등 사회규범 위반 |

### 실시 방법

1. 피검사자를 잘 아는 사람(부모·양육자)이 체크리스트 형식으로 기입하거나 검사자의 질문에 응답한다.
2. 검사자와 피검자 **외에 타인이 없는** 검사실에서 실시한다.
3. 모든 하위척도 **1번 문항부터** 질문을 시작한다.

### 활용

- 훈련 프로그램 적용 전 피검사자의 현 상태 파악 기초 자료
- 평가 이후 **추후 지도를 위한 프로그램 제공**
- 적응행동과 부적응행동 점수를 **조합하여 지원점수** 산출 → 지원강도 결정

## 채점 기호 (SMS·KNISE-SAB 공통)

| 기호 | 의미 |
|------|------|
| `+` | 수행함 |
| `+F` | 도움을 받아(Facilitated) 수행함 |
| `+NO` | 기회가 없었으나(No Opportunity) 수행 가능 |
| `±` | 부분 수행 |
| `-` | 수행 못함 |

> ⚠️ **기출 포인트**: `+NO`는 "기회가 없어서 못 한 것이지, 능력이 없는 것이 아님"을 의미한다. `±`와 구분하는 문항이 출제된다.
