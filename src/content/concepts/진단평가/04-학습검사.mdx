## 개요

아동의 기초학습기능과 학력을 측정하여 학습 부진 아동을 선별하고, 개별화된 교육 계획을 수립하며, 학습 효과를 지속적으로 모니터링하는 4개 검사 도구를 다룬다.

| 검사 도구 | 대상 연령/학년 | 핵심 목적 |
|---|---|---|
| 기초학습기능검사 | 만 5세 ~ 12세 | 기초학습기능 측정 |
| KISE-BAAT | 만 5세 ~ 14세 | 기초학력 측정 + 학습장애 진단 |
| 학습준비도 검사 | 유치원 졸업생, 초1 | 학습 문제 예상 아동 선별 |
| BASA | 초1 ~ 초3 | 교육과정중심측정, 형성평가 |

---

## 1. 기초학습기능검사

### 기본 정보

- **대상**: 만 5세 ~ 12세
- **결과**: 학년별(연령별) 백분위 점수, 학년(연령) 점수 제공

### 구성 및 실시 순서

| 영역 | 소검사 | 실시 순서 |
|---|---|---|
| 정보처리기능 | 정보처리기능 | 1순위 |
| 언어기능 | 읽기Ⅰ, 읽기Ⅱ, 쓰기 | 3·4·5순위 |
| 수 기능 | 셈하기 | 2순위 |

> **실시 순서**: 정보처리 → 셈하기 → 읽기Ⅰ → 읽기Ⅱ → 쓰기

각 소검사는 아동의 학년에 따라 검사 설명서에 제시된 **시작 문항 번호**에서 시작한다.

## 2. KISE 기초학력검사(KISE-BAAT)

### 기본 정보

- **대상**: 만 5세 ~ 14세
- **유형**: 개인용 기초학력검사

### 목적

1. 아동의 **읽기·쓰기·수학** 기초학력 측정
2. 국어·수학 부진 아동 선별·진단 → 부진 영역과 수준 파악 → 교육 계획 수립에 필요한 정보 제공

### 실시 규칙

| 상황 | 처리 방법 |
|---|---|
| 일반적인 경우 | 구성 영역 순서대로 실시 |
| 특정 영역 기피·지루함 | 다른 영역 모두 실시 후 빠진 영역 실시 |
| 부적절한 동기·피로 누적 | 영역별 분리 실시 가능 (단, 회기 간격 **1주일 이내**) |

### 결과 및 판단 기준

소검사별로 **백분위 점수**, **학력지수**(평균 100, 표준편차 15), **학년 규준** 제공

> ⚠️ **기출 포인트**: 학력지수나 학년 규준 점수 둘 중 어느 한 점수가 **–2 표준편차 이하**이거나 **2년 이상 지체**된 것으로 나타났을 때 학습장애로 진단한다.

## 3. 학습준비도 검사

### 기본 정보

- **대상**: 유치원 졸업생, 초등학교 1학년
- **유형**: **집단 검사**
- **구성**: 8개 하위요인
- **결과**: 백분위 점수(남/여/전체), 표준점수(도시/지방/전국)

### 목적

특별한 조력이 제공되지 않으면 **초등학교 2학년 학습을 수행할 만한 준비성이 없을 것**으로 생각되는 아동들을 미리 선별한다.

> ⚠️ **기출 포인트**: 학습준비도 검사는 **집단 검사**이며, 초1 진입 시점에서 학습 문제 예상 아동을 **사전 선별**하는 것이 핵심 목적이다.

## 4. 기초학습기능 수행평가(BASA)

### 핵심 특징

BASA는 **교육과정중심측정(CBM)** 절차에 근거하여 개발된 검사로, 일반 표준화 검사와 달리 **반복 실시**를 통한 진전도 모니터링에 초점을 둔다.

| 특징 | 내용 |
|---|---|
| 이론적 기반 | 교육과정중심측정(CBM) |
| 활용 형태 | 기초평가 + **형성평가** |
| 실시 빈도 | 간편·저비용 → 자주 실시 가능 |
| 규준 유형 | 지역 규준의 개인용 표준화 검사 |
| 핵심 기능 | 학습 효과 확인 → 진도·교수 계획·개입 계획 수립 → 아동 변화 모니터링 |

> ⚠️ **기출 포인트**: BASA는 **형성평가**로 활용 가능한 유일한 학습 검사이며, **교육과정중심측정** 절차 기반이라는 점이 다른 검사와의 핵심 차별점이다.

### 읽기 BASA

| 항목 | 내용 |
|---|---|
| 대상 | 초등학교 1학년 ~ 3학년 |
| 목적 | 읽기 곤란·읽기 장애 진단 + 형성평가 도구 |
| 구성 | 기초평가 + 형성평가 / 읽기검사자료1(읽기검사 1·2) + 읽기검사자료2(빈칸채우기검사) |

### 수학 BASA

| 항목 | 내용 |
|---|---|
| 대상 | 초등학교 1학년 ~ 3학년 |
| 목적 | 학습 부진 아동·특수교육 대상자의 수학 수행 수준 진단·평가 |
| 구성 | Ⅰ·Ⅱ·Ⅲ 수준(학년 수준 검사) + 통합 수준 검사(Ⅳ) |
