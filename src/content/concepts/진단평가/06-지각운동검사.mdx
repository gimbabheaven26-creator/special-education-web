## 지각 및 운동능력 검사 개관

아동의 소근육·대근육 운동 기술 수준과 **시지각 및 시각-운동 통합 능력**을 측정하여 운동 기능 평가 및 시지각 장애 조기 발견에 기여한다.

| 검사명 | 약칭 | 핵심 측정 영역 |
|--------|------|--------------|
| 한국판 오세레츠키 운동능력검사 | 오세레츠키 | 소근육·대근육 운동 기술 |
| 한국판 시지각 발달 검사-2 | K-DTVP-2 | 시지각·시각-운동 통합 |
| 아동기 자폐증 평정척도 | CARS | 자폐 발달·행동 수준 |

## 1. 한국판 오세레츠키 운동능력검사

> ⚠️ **기출 포인트**: 결과로 **운동연령**을 제공한다는 점이 핵심 — 다른 검사들이 표준점수·백분위를 주는 것과 대비된다.

### 기본 정보

| 항목 | 내용 |
|------|------|
| **대상** | 만 4세 ~ 16세 아동 및 청소년 |
| **목적** | 소근육·대근육 운동 기술 수준 측정, 운동 기능 평가 |
| **실시 방법** | 아동의 반응을 정확히 파악할 수 있는 검사자가 실시 |
| **결과** | **운동연령** 제공 |

### 하위 검사 구성

| 하위 검사 | 측정 내용 |
|-----------|----------|
| 일반적 정적 협응검사 | 정적 자세에서의 신체 균형·협응 |
| 손동작 협응검사 | 손과 손가락의 정밀 동작 |
| 운동속도검사 | 일정 시간 내 운동 수행 속도 |
| 동시적 자발동작검사 | 양측 신체를 동시에 사용하는 협응 |
| 단일동작 수행능력검사 | 목적 있는 단일 동작의 정확성 |

## 2. 한국판 시지각 발달 검사-2 (K-DTVP-2)

> ⚠️ **기출 포인트**: **8개 하위 검사 이름**이 자주 출제된다. "시간제한 없음"과 "개인·집단 검사 모두 가능"도 반드시 기억할 것.

### 기본 정보

| 항목 | 내용 |
|------|------|
| **대상** | 만 4세 ~ 8세 |
| **목적** | 시지각 및 시각-운동 통합 능력 측정, 시지각 장애 **조기 발견** |
| **실시 방법** | 개인 및 집단 검사 가능, **시간제한 없음** |
| **결과** | 연령 수준, 백분위 점수, 표준점수(평균 100, 표준편차 15) |

### 8개 하위 검사

| 하위 검사 | 측정 내용 |
|-----------|----------|
| **눈-손 협응** | 선을 따라 그리기 등 시각-운동 조절 |
| **공간 위치** | 도형의 위치 변별 (회전·반전 등) |
| **따라 그리기** | 주어진 도형을 모방하여 그리기 |
| **도형-배경** | 복잡한 배경에서 목표 도형 구분 |
| **공간관계** | 점 배열을 보고 도형 재현 |
| **시각통합** | 부분 자극으로 전체 도형 인식 |
| **시각-운동 통합** | 지각한 도형을 손으로 재현 |
| **형태 향상성** | 크기·위치가 달라진 도형을 동일 형태로 인식 |

## 3. 아동기 자폐증 평정척도 (CARS)

> ⚠️ **기출 포인트**: CARS는 **15문항**으로 구성되며, 척도가 **발달척도·행동척도** 두 가지로 구분된다는 점 주의. 감각 처리(시각·청각·미각·후각) 문항이 포함된다는 것도 출제된다.

### 기본 정보

| 항목 | 내용 |
|------|------|
| **대상** | 자폐장애아동, 유사발달장애아동 |
| **실시 방법** | 관찰, 사례적 검토, 부모와의 면담 |
| **문항 수** | **15문항** |

### 목적

1. 자폐장애·유사발달장애아동의 발달 수준 및 특이 학습 행동 패턴 평가
2. 개별 치료 프로그램에 활용
3. 자폐장애와 기타 장애 **구별**, 장애 정도 **판별**

### 구성 및 채점 결과

| 척도 | 특징 | 채점 결과 |
|------|------|-----------|
| **발달척도** | 발달 영역별 수준 평가 | 합격 / 싹트기 반응 / 실패 |
| **행동척도** | 행동 특성 평가 (감각 처리 포함) | 정상 / 경증 / 중증 |

### 제공 결과물

| 결과물 | 내용 |
|--------|------|
| 발달척도 결과 | 하위별 발달연령 + 전체 발달연령 |
| 행동척도 결과 | 행동척도결과표 |

### 감각 처리 문항

CARS 행동척도에는 **시각, 청각, 미각, 후각** 등 다양한 감각 처리 관련 문항이 포함되어 있어, 감각 처리 이상이 자폐 판별의 주요 지표로 활용된다.
