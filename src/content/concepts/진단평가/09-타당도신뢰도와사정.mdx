## 1. 타당도

**정의**: 검사 도구가 측정하고자 하는 능력·특성을 실제로 측정하는 정도

### 1-1. 타당도 3종

> ⚠️ **기출 포인트**: 구인타당도는 가장 입증하기 어렵고 시간이 오래 걸림.

| 종류 | 정의 | 방법 |
|------|------|------|
| **내용타당도** | 검사 문항이 측정 영역을 얼마나 충실히 대표하는가 | 전문가 논리적 분석 (주관적) |
| **준거관련타당도** | 검사 결과와 준거 변인 간의 관련 정도 | — |
| → 공인타당도 | 거의 동일 시기의 다른 검사 결과와 일치하는 정도 | 상관계수 (.80 이상) |
| → 예언타당도 | 미래 행동을 정확히 예측하는 정도 | 상관계수 (.60 이상) |
| **구인타당도** | 눈에 보이지 않는 추상적·가설적 심리 특성(구인)을 측정하는 정도 | 가장 어렵고 오래 걸림 |

## 2. 신뢰도

**정의**: 반복된 측정에 대한 일관성

### 2-1. 신뢰도 5종

> ⚠️ **기출 포인트**: 중재충실도는 신뢰도의 한 유형.

| 종류 | 정의 |
|------|------|
| **검사-재검사 신뢰도** | 동일 검사를 동일 집단에 일정 간격 두고 2회 실시; 점수 간 상관계수 |
| **동형검사 신뢰도** | 두 동형검사(다른 문항, 동일 내용·형태·난이도)를 일정 간격 실시; 유사성 지표 |
| **내적일관성 신뢰도** | 검사 구성 문항들 간의 일관성 정도 |
| → 반분 신뢰도 | 검사를 2분할 후 상관계수 산출+보정 |
| → 문항내적일관성 신뢰도 | 개별 문항들 간의 일관성 추정 |
| **채점자간 신뢰도** | 두 검사자가 동일 피검자에게 부여한 점수 간 상관계수 |
| **중재충실도** | 계획한 대로 교수·중재 전략을 정확·일관성 있게 적용하는 정도 |

**중재충실도 핵심**: 중재가 비효과적이라고 결정하기 **전에** 프로그램 실행 정확도 먼저 평가

**신뢰도 계수 기준**: .80 이상 (중요한 의사결정: .90 이상)

### 2-2. 타당도와 신뢰도의 관계

> ⚠️ **기출 포인트**: 신뢰도 → 타당도 방향성.

- 신뢰도는 타당도의 **필요조건**이자 선행 조건 (충분조건 아님)
- 높은 신뢰도가 반드시 높은 타당도를 보장하지는 않음
- 타당도 > 신뢰도 불가 (타당도 ≤ 신뢰도)
- 검사 도구 양호성 판단: **타당도 > 신뢰도** (타당도 우선)

## 3. 측정의 표준오차(SEM)와 신뢰구간

**SEM 정의**: 획득점수로 진점수(이론적 능력 점수)를 추정할 때의 오차 정도

| 관계 | 내용 |
|------|------|
| **신뢰도 ↑** → SEM ↓ | 신뢰도가 높을수록 오차 작아짐 |
| **표준편차 ↑** → SEM ↑ | 표준편차가 클수록 오차 커짐 |

**신뢰구간 설정**: 획득점수 ± z × SEM

## 4. 사정 방법

### 4-1. 공식적 vs 비공식적 사정

| 구분 | 공식적 사정 | 비공식적 사정 |
|------|-----------|------------|
| **특징** | 실시·채점·해석 지침 명백 | 지침 없음; 교사 재량 |
| **주 활용** | 총괄평가 | 형성평가 |
| **예** | 규준참조검사, 표준화 준거참조검사 | 관찰, 면접, CBM |

### 4-2. CBA / CBM / CR-CBA 비교

> ⚠️ **기출 포인트**: CBM은 규준참조의 대안; CR-CBA는 준거참조의 대안.

**교육과정 중심 사정(CBA)**: 아동에게 가르치는 교육과정 기반 수행 자료 수집

| 구분 | **CBM** | **CR-CBA** |
|------|---------|-----------|
| **대안 방법** | 규준참조검사의 대안 | 준거참조검사의 대안 |
| **표준화 여부** | 표준화 (공식적) | 비표준화 (비공식적) |
| **타당도·신뢰도** | 입증 가능 | 입증 어려움 |
| **목표 초점** | 장기 목적 | 단기 목표 |
| **사용 영역** | 기초학습기술(읽기·철자·쓰기·셈하기) | 다양한 영역 |
| **대상 연령** | 학령기(학령전 제한) | 학령기 + 학령전 |
| **활용 단계** | 모든 단계 | 프로그램 계획·형성·총괄 |

**CBM 기초선 점수**: 3회 점수 중 **중앙값**으로 결정

**CBM 표적선**: 기초선 점수와 목적 점수를 연결; 주 2회 검사 후 약 1개월(7회) 기준 점검

### 4-3. 대안적 사정

| 종류 | 특징 |
|------|------|
| **수행사정** | 행위 수행·결과 산출 기술 관찰; 복잡한 학습 결과 평가 가능; 신뢰도 낮음 |
| **과정 포트폴리오** | 학습 과정 초점; 교수 프로그램 실시 과정에서 사용 |
| **결과 포트폴리오** | 최상의 작품 초점; 교수 프로그램 종료 시점에서 사용 |

**포트폴리오 특성**: 다양한 자료(사진·비디오·결과물) 활용; 발달지체 아동 변화 파악 적합

**포트폴리오 단점**: 시간·노력 소요; 채점 주관적; 신뢰도 확보 어려움
