## 1. CBM 정의 및 4원칙

**교육과정중심측정(Curriculum-Based Measurement, CBM)**: Deno(1985)가 개발한, 학생이 직접 배우고 있는 교육과정 자료를 사용하여 학업 수행을 반복·체계적으로 측정하는 표준화된 방법

> ⚠️ **기출 포인트**: CBM은 교육과정중심사정(CBA)의 하위 유형이자 **표준화된** 형태로, 규준참조검사의 대안으로 개발되었다.

CBM의 4원칙:

| 원칙 | 내용 |
|------|------|
| **간편성** | 짧은 시간(1~3분) 안에 실시 가능 |
| **반복성** | 동일 학생에게 주 2회 이상 반복 실시 |
| **체계성** | 표준화된 실시·채점 절차 사용 |
| **기준참조** | 교육과정 숙달 기준 대비 수행 평가 |

## 2. CBM 유형

### 2-1. 읽기 CBM

| 유형 | 약어 | 측정 내용 | 측정 단위 |
|------|------|----------|----------|
| **구어읽기유창성** | ORF | 1분간 소리 내어 읽기 | 정확하게 읽은 단어 수(WRCM) |
| **미로과제** | Maze | 빈칸에 적절한 단어 고르기 | 정확하게 선택한 단어 수 |

**ORF 실시 절차**: 학년 수준 읽기 지문 → 1분 소리 내어 읽기 → 오류(대체·생략·삽입·머뭇거림 3초 이상) 표시 → WRCM 산출

### 2-2. 수학 CBM

| 유형 | 약어 | 측정 내용 |
|------|------|----------|
| **수학 계산** | COMP | 2~3분 안에 혼합 사칙연산 문항 풀기 |
| **수학 개념과 적용** | CAP | 수학 개념·응용 문제 8분 해결 |

### 2-3. 쓰기 CBM

| 유형 | 약어 | 측정 내용 | 채점 단위 |
|------|------|----------|----------|
| **쓰기 총 단어 수** | TW | 3분 이야기 쓰기 | 작성한 모든 단어 수 |
| **바른 단어 순서** | CWS | 철자·문법·의미 모두 맞는 연속 2단어 쌍 수 | CWS 수 |

## 3. CBM 활용 2단계

### 3-1. 1단계: 탐색수준사정(SLA)

**탐색수준사정(Survey-Level Assessment, SLA)**: 학생의 현재 기능 수준을 파악하여 적절한 교수 수준(instructional level)을 결정하는 과정

| 절차 | 내용 |
|------|------|
| ① 학년 수준 검사 | 현재 학년 수준 CBM 자료 실시 |
| ② 기준 미충족 확인 | 또래 규준 하회 시 하위 학년으로 이동 |
| ③ 교수 수준 결정 | 학생이 80~90% 수행 가능한 수준 탐색 |
| ④ 기초선 산출 | 동일 수준 3회 실시 → **중앙값**으로 기초선 설정 |

> ⚠️ **기출 포인트**: 기초선 점수는 3회 측정값의 **중앙값**으로 결정한다. 평균이 아님.

### 3-2. 2단계: 진전도 모니터링

**진전도 모니터링(Progress Monitoring)**: 교수 기간 동안 학생의 학업 성장을 주기적으로(주 2회) 측정하여 교수의 효과를 평가하는 과정

| 항목 | 내용 |
|------|------|
| **실시 주기** | 주 2회 이상 (최소 주 1회) |
| **기간** | 최소 6~8주 (의사결정에 충분한 데이터 점 확보) |
| **비교 기준** | 목표선(aimline) 대비 실제 수행 비교 |

---

## 4. 목표선 설정

**목표선(Aimline)**: 기초선 점수에서 목표 점수까지 선형으로 연결한 예상 성장선

**목표 설정 공식**:

> 목표 점수 = 기초선 점수 + (주차 수 × ROI)

| 용어 | 정의 |
|------|------|
| **ROI(진전 비율, Rate of Improvement)** | 주당 예상 성장 정도 (규준 집단 자료 또는 연구 기반) |
| **포부 수준** | 또래와 동일 비율로 성장 vs 차이 좁히기 (더 높은 ROI 설정) |

**읽기 ROI 기준 예시**:

| 학년 | 주당 ORF 성장 기준(WRCM) |
|------|------------------------|
| 1학년 | 2.0~3.0 |
| 2학년 | 1.5~2.0 |
| 3학년 | 1.0~1.5 |
| 4~6학년 | 0.85~1.1 |

## 5. 의사결정 규칙: 이중차이선

**이중차이선(Dual Discrepancy)**: 학생이 또래 집단과 비교하여 (1) 수행 수준과 (2) 성장 비율 모두에서 뒤처질 때 중재 변경 또는 특수교육 의뢰를 결정하는 기준

> ⚠️ **기출 포인트**: 이중차이선은 단순 수행 수준 차이만이 아니라 **성장 비율(ROI)** 까지 함께 고려하는 의사결정 준거다.

| 의사결정 기준 | 행동 |
|-------------|------|
| 4개 연속 데이터 점이 목표선 **아래** | 교수 수정 검토 |
| 4개 연속 데이터 점이 목표선 **위** | 목표 상향 조정 |
| 수행 수준 + 성장 비율 **모두** 또래 평균 -1SD 이하 | 특수교육 적격성 의뢰 검토 |

**목표선 기반 그래프 분석**:

| 추세선 위치 | 해석 |
|-----------|------|
| 데이터 추세선 > 목표선 | 현재 교수 효과적; 목표 상향 고려 |
| 데이터 추세선 = 목표선 | 현재 교수 유지 |
| 데이터 추세선 < 목표선 | 교수 수정 필요 |

## 6. CBM의 기술적 특성

| 특성 | 내용 |
|------|------|
| **신뢰도** | 검사-재검사·채점자간 신뢰도 높음 (.90 이상) |
| **타당도** | 구인타당도·예언타당도 연구로 입증 |
| **민감도** | 단기간 성장에도 변화 감지 가능 (표준화 검사보다 민감) |
| **표준화** | 공식적 사정 도구; 규준 참조 해석 가능 |
| **적용 범위** | 학령기 기초학습기술(읽기·수학·쓰기); 학령전 적용 제한 |
