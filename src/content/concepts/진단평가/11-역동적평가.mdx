## 1. 정적 평가 vs 역동적 평가

**정적 평가(Static Assessment)**: 표준화된 조건 하에 학생의 현재 독립적 수행 수준을 일회성으로 측정하는 전통적 평가 방식

**역동적 평가(Dynamic Assessment)**: 평가 과정에 교수·보조·단서를 포함하여 학습잠재력과 변화 가능성을 측정하는 평가 방식

> ⚠️ **기출 포인트**: 역동적 평가는 현재 능력이 아니라 **보조를 받았을 때 도달 가능한 수준(ZPD)** 을 측정한다.

| 비교 항목 | 정적 평가 | 역동적 평가 |
|----------|----------|-----------|
| **측정 시점** | 사전 또는 사후 일회성 | 평가 과정 전체 |
| **평가자 역할** | 중립적 관찰자 | 능동적 교수자·촉진자 |
| **목표** | 현재 독립적 수행 수준 확인 | 학습잠재력·변화 가능성 측정 |
| **표준화** | 높음 (실시·채점 표준화) | 낮음 (상호작용적·유연한 절차) |
| **결과 해석** | 점수 비교 (규준·준거 참조) | 보조 유형·양·학습 변화 기술 |
| **문화적 편향** | 높음 (배경 지식 의존) | 낮음 (보조를 통해 학습 반응 평가) |
| **활용 목적** | 진단·분류 | 교수 계획·잠재력 파악 |

## 2. 검사-교수-재검사 절차

**검사-교수-재검사(Test-Teach-Test)**: 역동적 평가의 핵심 절차로, 사전 검사 → 교수 → 재검사의 3단계로 구성

| 단계 | 내용 |
|------|------|
| **1단계: 사전 검사** | 표준화 조건(보조 없음)에서 독립적 수행 수준 파악; 기저선 확인 |
| **2단계: 교수(보조)** | 단서 제공·모델링·피드백·힌트 등 보조를 통해 과제 수행 지도 |
| **3단계: 재검사** | 동일 또는 유사 과제를 다시 독립적으로 수행; 학습 변화량 측정 |

**학습잠재력 산출**: 재검사 점수 - 사전 검사 점수 = 학습잠재력 지수

**교수 보조 위계(덜 침습적 → 더 침습적)**:

| 보조 수준 | 예시 |
|----------|------|
| 1수준 | 일반적 격려 ("잘 생각해봐") |
| 2수준 | 메타인지 단서 ("어떤 전략을 쓸 수 있을까?") |
| 3수준 | 내용 단서 ("첫 글자는 ㅂ이야") |
| 4수준 | 모델링 (평가자가 직접 시연) |

> ⚠️ **기출 포인트**: 검사-교수-재검사에서 보조는 최소 침습적 수준부터 시작하며, 학생이 올바른 반응을 보일 때까지 단계적으로 증가한다.

## 3. ZPD와 학습잠재력

**Vygotsky의 근접발달영역(ZPD, Zone of Proximal Development)**: 학생이 혼자서 해결할 수 있는 수준(현재 독립적 수행)과 유능한 타인의 도움으로 해결할 수 있는 수준(잠재적 발달 수준) 사이의 영역

| 수준 | 정의 |
|------|------|
| **현재 독립적 수행 수준** | 보조 없이 혼자 해결 가능한 과제 수준 |
| **ZPD** | 혼자서는 불가능하지만 보조를 받으면 해결 가능한 영역 |
| **잠재적 발달 수준** | 보조를 받았을 때 도달 가능한 최고 수행 수준 |

**역동적 평가와 ZPD의 연결**:
- 정적 평가: 현재 독립적 수행 수준만 측정
- 역동적 평가: ZPD 내의 **보조를 받은 수행(assisted performance)** 측정 → 교수 방향 결정

**교육적 시사점**: ZPD 내에서 보조를 제공할 때 가장 효과적인 학습 발생; 독립적 수행 수준 과제는 연습 용도

## 4. 매개학습경험(MLE): Feuerstein

**매개학습경험(Mediated Learning Experience, MLE)**: Feuerstein이 제안한 개념으로, 의미 있는 성인(부모·교사)이 아동과 환경 사이에서 의도적으로 개입하여 경험을 해석하고 구조화해주는 상호작용

**MLE의 3가지 핵심 특성**:

| 특성 | 정의 | 예시 |
|------|------|------|
| **의도성(Intentionality)** | 매개자가 특정 자극에 아동의 주의를 집중시키려는 의도를 명확히 전달 | "이 부분을 잘 봐, 여기가 중요해" |
| **초월성(Transcendence)** | 눈앞의 과제를 넘어 더 넓은 원리·맥락과 연결 | "이 규칙은 다른 수학 문제에서도 쓸 수 있어" |
| **의미화(Meaning)** | 활동·자극의 가치·중요성을 설명하여 동기 부여 | "이걸 배우면 실생활에서 이렇게 쓸 수 있어" |

> ⚠️ **기출 포인트**: MLE의 3특성(의도성·초월성·의미화)은 단순 교수와 구분되는 매개학습의 핵심이다. 초월성은 **일반화·전이**와 직결된다.

**Feuerstein의 학습잠재력 평가 도구**: LPAD(Learning Propensity Assessment Device)
- 표준화된 검사가 아닌, 상호작용적·역동적 평가 도구
- 인지 변화 가능성(modifiability) 평가에 초점

## 5. 역동적 평가의 장단점

### 5-1. 장점

| 장점 | 내용 |
|------|------|
| **학습잠재력 파악** | 현재 수준이 낮아도 잠재 능력 확인 가능 |
| **문화적 공정성** | 배경 지식보다 학습 반응 능력을 측정해 편향 감소 |
| **교수 계획 연계** | 어떤 유형의 보조가 효과적인지 직접 정보 제공 |
| **CLD 학생 적합** | 문화·언어적으로 다양한 학생, ELL 학생에게 유용 |

### 5-2. 단점

| 단점 | 내용 |
|------|------|
| **시간·비용** | 개별 실시 필수; 표준화 검사보다 훨씬 많은 시간 소요 |
| **표준화 부족** | 절차·보조의 일관성 확보 어려움; 채점자간 신뢰도 낮음 |
| **훈련 요건** | 평가자의 높은 전문성·훈련 필요 |
| **결과 해석의 주관성** | 학습 변화를 수치화하기 어려움 |
