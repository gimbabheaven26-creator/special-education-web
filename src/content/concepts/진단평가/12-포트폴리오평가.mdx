## 1. 포트폴리오 정의 및 특징

**포트폴리오(Portfolio)**: 특정 기간에 걸쳐 학생의 학습 과정·노력·성취를 보여주는 작품·산출물·기록을 의도적으로 수집한 자료 모음

**포트폴리오 평가 vs 표준화 검사 비교**:

| 항목 | 표준화 검사 | 포트폴리오 평가 |
|------|-----------|--------------|
| **측정 내용** | 특정 시점의 수행 | 시간에 따른 성장 과정 |
| **맥락** | 인위적 검사 상황 | 실제 수업·생활 맥락 |
| **자료 유형** | 단일 (점수) | 다양 (작품·관찰·자기평가) |
| **학생 참여** | 수동적 (검사 받음) | 능동적 (수집·선정·성찰) |
| **결과 형식** | 수치 점수 | 서술적 기술 + 실제 작품 |

> ⚠️ **기출 포인트**: 포트폴리오는 결과보다 **과정과 성장**을 강조하며, 학생이 작품 선정에 능동적으로 참여한다는 점이 표준화 검사와 근본적으로 다르다.

## 2. 포트폴리오 3유형

| 유형 | 정의 | 주요 내용 | 평가 주체 |
|------|------|----------|----------|
| **작품집 포트폴리오** | 완성된 작품·결과물 수집 | 최상의 작품, 과제 완성본 | 교사·학생 |
| **교사관찰 포트폴리오** | 교사의 체계적 관찰 기록 | 체크리스트, 일화기록, 수행 평가 결과 | 교사 |
| **성찰 포트폴리오** | 학생의 자기평가·메타인지 중심 | 성찰일지, 자기평가 양식, 학습 목표 기록 | 학생 |

**전시회 포트폴리오 vs 작업 포트폴리오**:

| 유형 | 특징 | 용도 |
|------|------|------|
| **전시회(showcase) 포트폴리오** | 최상의 작품만 선정 | 성취 수준 보여주기; 부모·학교 공유 |
| **작업(working) 포트폴리오** | 과정 중인 작업·초안 포함 | 교수 계획 수립; 진행 중인 학습 파악 |

**과정 포트폴리오 vs 결과 포트폴리오**:

| 유형 | 시점 | 초점 |
|------|------|------|
| **과정 포트폴리오** | 교수 프로그램 실시 과정 중 | 학습 과정·노력·발달 |
| **결과 포트폴리오** | 교수 프로그램 종료 시점 | 최상의 성취·완성도 |

## 3. 작품 수집·선정·평가 절차

### 3-1. 수집 단계

- 작품 수집 기간·범위·유형을 사전에 학생과 협의
- 다양한 형태 포함: 글쓰기·그림·수학 과제·사진·녹음·녹화·교사 관찰 기록

### 3-2. 선정 기준

| 선정 기준 | 설명 |
|---------|------|
| **최적 작품** | 학생이 가장 잘한 작품; 현재 최고 수행 수준 대표 |
| **성장 증거** | 초기 → 후기 작품 비교; 학습 진전 보여주는 자료 |
| **노력 증거** | 어렵지만 열심히 시도한 작품 |
| **자기선택 작품** | 학생 본인이 의미 있다고 생각하는 작품 |

> ⚠️ **기출 포인트**: 포트폴리오 작품 선정에서 학생은 단순 수집자가 아니라 **선정 이유를 설명하는 능동적 참여자**여야 한다.

### 3-3. 성찰일지

**성찰일지**: 포트폴리오에 포함된 작품을 선정한 이유, 학습 과정에서 느낀 점, 앞으로의 학습 목표를 학생이 직접 기록하는 서면 자료

성찰일지 구성 요소:
- 이 작품을 선택한 이유
- 이 작품에서 잘한 점
- 더 개선하고 싶은 점
- 다음 목표

## 4. 루브릭 작성 방법

**루브릭(Rubric)**: 과제·수행에 대한 기대 수준과 채점 기준을 명시한 평가 도구

### 4-1. 분석적 루브릭(Analytic Rubric)

각 평가 준거를 개별적으로 채점

| 특징 | 내용 |
|------|------|
| **구조** | 여러 준거(내용·조직·언어·기계적 규칙 등)를 각각 독립적으로 채점 |
| **장점** | 구체적 피드백 제공; 학생이 강점·약점 파악 가능 |
| **단점** | 채점 시간 많음; 준거 수 많을 경우 복잡 |
| **적용** | 형성평가, 학습 개선 목적 |

### 4-2. 전체론적 루브릭(Holistic Rubric)

전체 수행을 종합하여 하나의 점수 부여

| 특징 | 내용 |
|------|------|
| **구조** | 수행 전체를 하나의 수준으로 판단 |
| **장점** | 채점 빠름; 전체적 인상 반영 |
| **단점** | 구체적 피드백 어려움; 개선 방향 불명확 |
| **적용** | 총괄평가, 빠른 판단 필요 시 |

**루브릭 작성 단계**:
1. 평가 준거 결정 (무엇을 평가할 것인가)
2. 수행 수준 수 결정 (보통 3~4단계)
3. 각 준거 × 수준 조합에 행동 기술어 작성
4. 학생·동료와 루브릭 공유 및 수정

## 5. 포트폴리오 장단점 및 신뢰도 한계

### 5-1. 장점

| 장점 | 설명 |
|------|------|
| **성장 과정 포착** | 단일 시점 검사로는 볼 수 없는 발달 변화 확인 |
| **다양한 능력 평가** | 창의성·문제해결·자기조절 등 복합적 능력 평가 |
| **학생 동기 향상** | 자신의 작품을 스스로 선정하고 성찰하며 학습 주인의식 증가 |
| **문화적 반응성** | 표준화 검사보다 문화·언어적 다양성 수용 가능 |
| **장애 아동 적합** | 발달지체 아동의 미세한 변화를 다양한 자료로 파악 가능 |

### 5-2. 단점

| 단점 | 설명 |
|------|------|
| **시간·노력** | 수집·선정·평가 전 과정에 교사·학생 모두 상당한 시간 투입 |
| **채점 주관성** | 루브릭 없이는 평가자마다 결과 달라짐 |
| **신뢰도 확보 어려움** | 채점자간 신뢰도 낮음; 표준화 어려움 |
| **비교 어려움** | 학생 간 비교, 규준 참조 해석 어려움 |

> ⚠️ **기출 포인트**: 포트폴리오의 가장 큰 단점은 **채점자간 신뢰도** 확보의 어려움이다. 루브릭 사용과 채점자 훈련이 신뢰도 향상의 핵심 전략이다.

### 5-3. 신뢰도 향상 전략

| 전략 | 내용 |
|------|------|
| **루브릭 사전 개발** | 명확한 채점 기준 사전 작성·공유 |
| **채점자 훈련** | 기준점 작품(anchor work) 사용 채점 연습 |
| **이중 채점** | 2인 이상이 독립적으로 채점 후 비교 |
| **채점 프로토콜** | 일관된 채점 절차·순서 유지 |
