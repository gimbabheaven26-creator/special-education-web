## 1. 형성평가 개념 및 교수적 기능

**형성평가(Formative Assessment)**: 교수·학습이 진행되는 **과정 중에** 학생의 이해 수준을 파악하여 교수를 조정하기 위해 실시하는 평가

> ⚠️ **기출 포인트**: 형성평가의 핵심 목적은 **성적 부여가 아니라 교수 조정(instructional adjustment)** 이다. 교사가 어떻게 다음 교수를 바꿀지 결정하기 위한 정보 수집이다.

**형성평가의 교수적 기능**:

| 기능 | 설명 |
|------|------|
| **교수 방향 결정** | 학생이 이해한 내용과 그렇지 않은 내용 구별 |
| **학습 곤란 조기 발견** | 단원 종료 전에 오개념·학습 장애물 확인 |
| **피드백 제공** | 학생에게 즉각적이고 구체적인 피드백 전달 |
| **교수 집중도 향상** | 교사가 학생 이해 수준에 맞게 교수 속도·깊이 조절 |

## 2. 형성평가 유형

### 2-1. 구두 형성평가

| 유형 | 방법 |
|------|------|
| **질문·응답** | 수업 중 개방형·닫힌형 질문으로 이해 확인 |
| **토의·토론 관찰** | 학생 간 대화에서 이해 수준 파악 |
| **Think-Pair-Share** | 개인 사고 → 짝 공유 → 전체 발표 구조 |

### 2-2. 서면 형성평가

| 유형 | 특징 |
|------|------|
| **출구 티켓(Exit Ticket)** | 수업 종료 전 1~3문항 작성 후 제출; 교사가 다음 수업 전 검토 |
| **3-2-1 카드** | 배운 것 3, 흥미로운 점 2, 질문 1 기록 |
| **오개념 진단 퀴즈** | 흔한 오개념을 선택지에 포함한 짧은 퀴즈 |
| **학습 일지** | 수업 내용·이해도·질문을 학생이 직접 기록 |

**출구 티켓 활용 사이클**:

| 단계 | 내용 |
|------|------|
| ① 설계 | 수업 핵심 목표에 맞는 1~3문항 제작 |
| ② 실시 | 수업 마지막 5분 활용 |
| ③ 분석 | 교사가 3그룹으로 분류 (이해·부분이해·미이해) |
| ④ 조정 | 다음 수업 내용·집단 구성 조정 |

### 2-3. 수행 기반 형성평가

| 유형 | 방법 |
|------|------|
| **관찰** | 교사가 학생 수행 과정 체계적 관찰·기록 |
| **화이트보드 응답** | 개별 화이트보드에 답 작성 → 동시 제시 |
| **수행 체크리스트** | 핵심 기술 습득 여부 즉시 체크 |

## 3. 형성평가 vs 총괄평가

| 항목 | 형성평가 | 총괄평가 |
|------|---------|---------|
| **실시 시점** | 교수·학습 진행 **과정 중** | 교수 단위 **종료 후** |
| **주요 목적** | 교수 방향 조정; 학습 개선 | 성취 수준 판정; 성적 부여 |
| **피드백 방향** | 학생·교사 모두에게 즉각적 | 주로 학생에게 최종 결과 |
| **결과 활용** | 다음 교수 계획 수립 | 특수교육 계속 여부 결정 |
| **채점 방식** | 비공식적; 점수 부여 불필요 | 공식적; 점수·등급 부여 |
| **예시** | 출구 티켓, CBM, 관찰 | 단원 평가, 기말 검사 |

> ⚠️ **기출 포인트**: 총괄평가는 특수교육의 **지속 여부**를 결정하는 근거가 되는 반면, 형성평가는 **현재 프로그램을 수정**하기 위한 자료다.

## 4. 진전도 모니터링과 데이터 기반 의사결정

### 4-1. 진전도 모니터링 vs 일반적 성과측정

| 항목 | 진전도 모니터링 | 일반적 성과측정 |
|------|--------------|--------------|
| **측정 도구** | CBM (표준화) | 다양한 비공식 평가 |
| **실시 주기** | 주 2회 이상 | 분기·학기별 |
| **목적** | 개별 성장 추적; 의사결정 | 전반적 학습 결과 확인 |
| **결과 형식** | 그래프·차트 (추세 시각화) | 점수·서술 |
| **특수교육 적용** | IEP 목표 달성 모니터링 | 프로그램 전체 평가 |

### 4-2. 4-데이터 점 규칙 (의사결정)

**데이터 기반 의사결정 규칙**: 연속 4개의 데이터 점을 근거로 교수 수정 여부를 결정

| 패턴 | 해석 | 행동 |
|------|------|------|
| **4개 점이 모두 목표선 아래** | 현재 교수 불충분 | 교수 전략·강도 수정 |
| **4개 점이 모두 목표선 위** | 현재 교수 과도하게 효과적 | 목표 상향 조정 |
| **불규칙 패턴** | 교수 일관성 또는 측정 문제 | 중재 충실도·측정 절차 재검토 |

> ⚠️ **기출 포인트**: **연속 4개** 데이터 점이 기준이다. 1~2개 점이 목표선 아래라고 바로 수정하지 않는다. 충분한 데이터 축적 후 의사결정.

**중재 수정 유형**:

| 수정 방향 | 예시 |
|----------|------|
| **강도 증가** | 소집단 → 개별 지도, 주 3회 → 5회 |
| **방법 변경** | 파닉스 접근 → 전체언어 접근 |
| **내용 조정** | 선수 기술로 후퇴 |
| **평가 의뢰** | 진전도 없으면 특수교육 적격성 재평가 |

## 5. 특수교육에서의 형성평가 적용

### 5-1. IEP와 형성평가 연계

| 단계 | 형성평가 역할 |
|------|------------|
| **IEP 목표 설정** | 현행 수준 파악 (형성평가 기초 자료 활용) |
| **교수 실행 중** | 단기 목표 도달 여부 주기적 확인 |
| **IEP 검토** | 진전도 자료 기반 목표 수정·유지·상향 결정 |

### 5-2. RTI/MTSS와 형성평가

**RTI(중재-반응 모델)**: 3단계 지원 구조에서 형성평가·진전도 모니터링은 각 단계 효과 판단의 핵심

| 단계 | 모니터링 주기 | 도구 |
|------|------------|------|
| **1단계 (일반교육)** | 학기당 3회 (benchmark) | CBM |
| **2단계 (보충 지원)** | 주 2회 | CBM |
| **3단계 (집중 지원)** | 주 2~3회 | CBM + 기타 진단 |

### 5-3. 특수교육 학생 대상 형성평가 유의사항

| 유의사항 | 설명 |
|---------|------|
| **편의 제공** | 확대 문자, 시간 연장, 대안 반응 방식 제공 |
| **개별화 기준** | 또래 비교보다 개인 목표 대비 성장에 초점 |
| **빈도 증가** | 일반 학생보다 더 자주 측정하여 조기 반응 포착 |
| **다양한 방법** | 구두·수행·서면 방법을 혼합하여 학생의 강점 활용 |
