## 1. 비공식 평가의 종류 및 특징

**비공식 평가(Informal Assessment)**: 표준화된 실시·채점·해석 지침 없이 교사가 학생의 수행을 측정·관찰하는 평가 방법; 형성평가에 주로 활용

> ⚠️ **기출 포인트**: 비공식 평가는 표준화가 되어 있지 않아 타당도·신뢰도 입증이 어렵지만, 교수 맥락에서 즉각적인 교수 결정에 유용하다.

### 1-1. 교사제작검사

| 유형 | 특징 |
|------|------|
| **성취도 검사** | 교수 내용 대비 학생 습득 수준 측정 |
| **진단 검사** | 특정 기술의 강점·약점 파악 |
| **오류 분석(error analysis)** | 학생의 오류 패턴 분석 → 오개념·절차 문제 파악 |

**오류 분석 절차**:
1. 학생 반응 수집 (검사·과제 결과)
2. 오류 유형 분류 (무선 오류 vs 체계적 오류)
3. 체계적 오류의 패턴 기술
4. 오류 원인 가설 설정
5. 교수 수정

### 1-2. 관찰

| 유형 | 특징 | 적용 |
|------|------|------|
| **일화기록** | 특정 행동·사건을 서술적으로 기록 | 행동·사회적 상호작용 관찰 |
| **체계적 관찰** | 사전 정의된 행동을 빈도·지속시간·간격 기록 | 목표 행동의 양적 측정 |
| **체크리스트** | 기술 습득 여부(있음/없음)를 목록으로 확인 | 발달 이정표, 기술 습득 |
| **평정척도** | 행동·기술의 질·빈도를 척도(1~5점)로 평가 | 수행의 질적 수준 평가 |

### 1-3. 면담

| 면담 대상 | 수집 정보 |
|----------|----------|
| **학생 면담** | 자기인식, 학습 전략, 선호도, 흥미 |
| **부모 면담** | 가정 행동, 발달 역사, 가족 우선순위 |
| **교사 면담** | 교실 행동, 또래 관계, 학업 수행 패턴 |

### 1-4. 작품 분석

- 학생이 생산한 과제·그림·쓰기·만들기 결과물 분석
- 기술 수준, 발달 단계, 오류 패턴 파악
- 포트폴리오 구성 자료로 활용

## 2. 교육과정중심사정(CBA) 정의 및 4가지 유형

**교육과정중심사정(Curriculum-Based Assessment, CBA)**: 학생에게 실제로 가르치고 있는 교육과정 자료를 직접 사용하여 학생의 현재 수행 수준을 측정하는 사정 방법의 총칭

> ⚠️ **기출 포인트**: CBA는 하나의 특정 검사 도구가 아니라 **교육과정을 측정 재료로 사용하는 사정 접근법의 총칭**이다. CBM은 CBA의 한 유형이다.

**CBA의 4가지 유형**:

| 유형 | 개발자 | 특징 |
|------|--------|------|
| **CBM (교육과정중심측정)** | Deno(1985) | 표준화; 규준참조; 반복 측정 가능 |
| **준거참조 CBA (CR-CBA)** | Blankenship(1985) | 비표준화; 준거참조; 숙달기준 대비 평가 |
| **교육과정중심 언어 사정 (CBA-LA)** | Nelson(1994) | 언어 기술에 특화; 읽기·쓰기·구어 통합 |
| **읽기 CBA (CBA-R)** | Hintze et al. | 읽기 특화; 음운인식·해독·유창성·이해 포함 |

## 3. CBA vs CBM 비교

| 비교 항목 | **CBA (일반)** | **CBM** |
|----------|--------------|---------|
| **표준화 여부** | 비표준화 (비공식적) | 표준화 (공식적) |
| **대안 방법** | 준거참조검사의 대안 | 규준참조검사의 대안 |
| **타당도·신뢰도** | 입증 어려움 | 연구로 입증 가능 |
| **목표 초점** | 단기 목표 (교수 단원 수준) | 장기 목적 (연간 IEP 목적) |
| **사용 영역** | 다양한 영역 | 기초학습기술(읽기·수학·쓰기) |
| **대상 연령** | 학령기 + 학령전 | 학령기 (학령전 제한) |
| **점수 해석** | 준거 참조 (숙달/미숙달) | 규준 참조 + 준거 참조 |
| **시간** | 가변적 | 고정 (1~5분) |
| **반복 측정** | 제한적 | 주 2회 이상 가능 (동형 자료 다수) |

> ⚠️ **기출 포인트**: CBM은 **규준참조검사의 대안**, CR-CBA는 **준거참조검사의 대안**이다. 이 구분이 핵심이다.

## 4. 준거참조 CBA 절차 (5단계)

**준거참조 CBA (Criterion-Referenced CBA, CR-CBA)**: Blankenship(1985)이 개발한 방법으로, 교육과정 목표에서 직접 검사를 제작하여 학생의 숙달 여부를 판단하는 절차

| 단계 | 내용 |
|------|------|
| **1단계: 교육과정 목표 파악** | 교수 단원의 핵심 목표·기술 목록 작성 |
| **2단계: 검사 문항 제작** | 각 목표당 최소 3~5개 문항 제작 (교육과정 직접 출제) |
| **3단계: 숙달기준 설정** | 각 기술에 대한 정확도·속도 기준 결정 (예: 80% 이상) |
| **4단계: 검사 실시 및 채점** | 표준화 조건에서 실시; 정확도·오류 유형 채점 |
| **5단계: 결과 해석 및 교수 계획** | 숙달 여부 판단 → 교수 순서·내용 결정 |

**숙달기준 설정 방법**:

| 방법 | 설명 |
|------|------|
| **경험적 기준** | 교사 판단·교육과정 지침 기반 (예: 80%, 90%) |
| **또래 수행 기준** | 일반 또래의 평균 수행을 기준으로 설정 |
| **교육과정 지침 기준** | 교과서·교육과정 문서에 명시된 기준 사용 |

## 5. 비공식 평가 신뢰도 향상 전략

**비공식 평가의 신뢰도 한계**: 표준화 절차 없음 → 실시·채점·해석의 일관성 부족 → 채점자·시간에 따른 오차 큼

| 전략 | 방법 |
|------|------|
| **관찰 행동 명확히 정의** | 측정할 행동을 조작적으로 정의 (모호한 용어 배제) |
| **구조화된 기록 양식 사용** | 체크리스트·평정척도로 주관성 최소화 |
| **채점자간 합치도 확인** | 2명 이상이 독립 관찰 후 비교; 80% 이상 합치 목표 |
| **복수 관찰 실시** | 단일 시점 의존 금지; 다양한 상황·시간대에 반복 관찰 |
| **명확한 채점 기준 사전 작성** | 루브릭·예시 답안 사전 준비 |
| **절차 표준화** | 동일한 지시문·과제·채점 순서 유지 |

> ⚠️ **기출 포인트**: 비공식 평가의 신뢰도를 높이는 핵심은 **행동의 조작적 정의**와 **채점자간 합치도 확인**이다. 80% 이상의 채점자간 합치도를 목표로 한다.

### 5-1. 관찰 신뢰도 계산

**채점자간 합치도 계산 공식**:

| 공식 | 내용 |
|------|------|
| **간격기록** | 합의 간격 수 ÷ 전체 간격 수 × 100 |
| **빈도기록** | 작은 빈도 ÷ 큰 빈도 × 100 |
| **기준** | 80% 이상 = 수용 가능한 신뢰도 |

### 5-2. 면담 신뢰도 향상

| 전략 | 내용 |
|------|------|
| **반구조화 면담** | 핵심 질문은 표준화; 추가 질문은 유연하게 |
| **면담 기록** | 녹음 또는 정확한 서면 기록 |
| **반응 확인** | 면담 후 핵심 내용 요약 확인 ("이렇게 이해했는데 맞나요?") |
