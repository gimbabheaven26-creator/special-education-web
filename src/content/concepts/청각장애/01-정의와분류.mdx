## 1. 정의

### 특수교육법상 정의
보청기를 착용해도 청각을 통한 의사소통이 어렵거나, 보청기 착용 시에만 의사소통이 가능하여 **교육적 성취가 어려운 사람**.

### 장애인 복지법상 청력 손실 등급

| 등급 | 기준 | 일상 청취 수준 |
|-----|------|-------------|
| **2급** | 양쪽 귀 청력 각각 **90dB 이상** 손실 | 완전히 들리지 않음 |
| **3급** | 양쪽 귀 청력 각각 **80dB 이상** 손실 | 큰 소리로 말해도 듣지 못함 |
| **4급** | 양쪽 귀 청력 각각 **70dB 이상** 손실, 또는 양쪽 보통 말소리 명료도 50% 이하 | 귀에 대고 말하면 들을 수 있음 |
| **5급** | 양쪽 귀 청력 각각 **60dB 이상** 손실, 또는 한쪽 80dB↑·다른쪽 40dB↑ 손실 | 40cm 이상 거리에서 말소리 못 들음 |

## 2. 소리의 물리적 특성

### 2-1. Hz (헤르츠) — 주파수·높낮이

| 범위 | 설명 |
|-----|------|
| **가청 주파수** | 16Hz ~ 20,000Hz |
| **가장 민감한 범위** | 500Hz ~ 8,000Hz |
| **인간 말소리 포함 범위** | 80Hz ~ 8,000Hz |
| **주요 회화 음역대** | **500Hz ~ 2,000Hz** |

> ⚠️ **기출 포인트**: 순음청력검사가 **1,000Hz에서 시작**하는 이유 — 인간 청력이 가장 민감하고 말소리가 주로 분포하기 때문.

### 2-2. dB (데시벨) — 강도·크기

| 단위 | 정의 |
|-----|------|
| **dB SPL** (Sound Pressure Level) | 소리의 절대 물리적 압력 수준 |
| **dB HL** (Hearing Level) | 정상 성인 평균 가청역치 기준 (0dB HL = 정상 최소 가청역치) |
| **dB SL** (Sensation Level) | 개인의 역치 기준값 (예: 역치 40dB HL에서 70dB HL 제시 = 30dB SL) |

### 2-3. 소리의 전달 경로

| 경로 | 구성 | 특징 |
|-----|------|------|
| **기도 전도** | 외이 → 중이 → 내이 → 청신경 | 공기를 매체로 전달 |
| **골도 전도** | 두개골 → 내이 → 청신경 | 중이 우회; 내이 직접 자극 |

## 3. 청각기관의 구조

| 부위 | 구성 요소 | 기능 |
|-----|---------|------|
| **외이** | 이개(귓바퀴), 외이도 | 소리 수집·방향 분별, 이물질 차단, 공명 |
| **중이** | 고막, 고실, 이소골(추골·침골·등골), 이관(유스타키오관) | 진동 증폭·내이 전달, 압력 조절 |
| **내이** | 세반고리관·전정기관(평형), 와우(코르티기관 유모세포) | 기계적 에너지 → 전기적 에너지 변환 |
| **청신경** | 청각 피질·청각 경추 | 소리 정보를 뇌로 전달 |

## 4. 청력 손실 부위별 분류

> ⚠️ **기출 포인트**: 기도-골도 청력 차이(기도-골도 갭)가 유형 구분의 핵심.

| 유형 | 손상 부위 | 기도 청력 | 골도 청력 | 청력도 형태 | 의료 처치 | 보청기 효과 |
|-----|---------|---------|---------|-----------|---------|-----------|
| **감음성** | 내이(와우)·청신경 | 저하 | **저하** (기도≈골도) | **고음점경형** | 어려움 | 효과 제한 |
| **전음성** | 외이·중이 | 저하 | **정상** | 수평형·저음장애형 | 가능 | 효과적 |
| **혼합성** | 외이+중이+내이 | 저하 | 저하 (기도>골도) | 혼합형 | 부분 가능 | 일부 효과 |
| **중추성(CAPD)** | 청각 중추 | 정상 | 정상 | 정상 (순음) | — | — |
| **심인성** | 정신적 원인 | — | — | — | 심리치료 | — |

### 감음성 청각장애 세부 특징

**누가현상(음의 보충현상):**
- 음압에 비례하여 들리는 것이 아니라 갑자기 음량이 커지는 현상
- 역동범위가 좁아 불쾌역치에 쉽게 도달
- → 보청기 활용이 효과적이지 않을 수 있음

**인공와우 이식 대상 기준 (5가지 모두 충족):**

| 기준 | 내용 |
|-----|------|
| ① | 양측 모두 **고도 이상** 감음성 청각장애 |
| ② | 보청기 효과 없음 |
| ③ | 청신경이 일부라도 기능함 |
| ④ | 내과적·신경적·정신과적 문제 없음 |
| ⑤ | 청능 재활을 위한 강한 동기 |

**언어·청각적 특성:**
- 전보식 문장(내용어 사용, 기능어 생략), 어휘 제한
- 과비음, 자음 산출 어려움
- 추상 명사보다 구체 명사 우세, 시간·양 개념 어려움

### 전음성 청각장애 세부 특징
- 60dB 이하 청력 손실이 대부분
- 모든 주파수에서 청력 수준 동일 (수평형)
- 의료적 처치 가능 → 의료 우선, 보청기 효과적

### 중추성 청각정보처리 장애 (CAPD) 특징

> ⚠️ **기출 포인트**: 순음청력검사 결과 **정상**이나 청력 손실처럼 행동함.

| 특징 | 내용 |
|-----|------|
| 청력 | 정상 (순음청력검사 정상) |
| 행동 | 청력 손실처럼 행동, 말을 자주 다시 말해달라 요청 |
| 주의 집중 | 짧고 쉽게 피곤 |
| 기억 | 장·단기 기억 장애 |
| 학습 | 정상 지능이나 읽기·쓰기 어려움, 소음 속 듣기 어려움 |
| 중재 | 교실 수정 전략, FM 보청기, 직접 기술 훈련 |
