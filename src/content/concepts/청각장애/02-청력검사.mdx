## 1. 청력검사 개요

| 분류 | 검사 방법 | 특징 |
|-----|---------|------|
| **객관적 검사** | 이미턴스, OAE, ABR/AEP | 피검자 반응 불필요, 영·유아 가능 |
| **주관적 검사** | 순음, 어음, Ling 6개음 | 피검자 반응 필요 |

---

## 2. 객관적 검사

### 2-1. 이미턴스(Immittance) 검사

중이 질병 유무·유형 파악 + 청력 손실 정도 보충 진단

| 세부 검사 | 원리 | 목적 |
|---------|-----|------|
| **고막운동성 검사** | 고막 반사 에너지를 외이도에서 측정 | 고막 운동 상태·이소골 연쇄 상태 예측 |
| **등골근반사 검사(ART)** | 강한 소리 → 등골근 수축 원리 | 내이·청신경 상태 파악, 미로성/후미로성 판별 |
| **반사피로 검사(Acoustic Reflex Decay)** | 등골근반사 소실 측정 | 청신경 이상 유무, 후미로성 청각장애 발견 |

### 2-2. 이음향방사 검사 (OAE)

달팽이관 **외유모세포** 움직임에 의해 생성된 음향 반응 측정 → 정상 청각 기능 여부 파악

| 항목 | 내용 |
|-----|------|
| **유형** | 자발이음향방사, 유발이음향방사, 변조이음향방사 |
| **장점** | 비침습적, 저렴, 간편, 빠름, 주관적 검사 어려운 대상자에게 유용 |

### 2-3. 청성유발전위(AEP) 검사

소리 자극 후 청각 중추에서 일어나는 전기적 변화 측정

| 세부 검사 | 특징 |
|---------|------|
| **전기와우도 검사** | 가장 먼저 나타나는 반응, 와우·와우 신경 병태 상세 정보 — **침습적** |
| **뇌간유발반응 검사(ABR)** | 청신경→뇌간 전달로 전기 신호 기록; 신생아 청력 선별에 활용 — **비침습적**, 마취·신경안정제 영향 적음 |

> ⚠️ **기출 포인트**: ABR은 가장 안정적으로 발현하며 비침습적, 마취에 영향을 덜 받아 신생아·중증장애인 검사에 적합.

## 3. 주관적 검사

### 3-1. 순음청력검사

주파수별 순음을 들려주어 각 주파수대 **최소 가청 역치** 측정

#### 검사 목적

청력 손실의 유무·편측성/양측성·정도·종류·병변 부위·청력형·청능 재활 정보 파악

#### 기도청력검사

이어폰으로 음을 들려주어 최소 가청 역치 측정

| 항목 | 내용 |
|-----|------|
| **시작 주파수** | **1,000Hz** 부터 시작 → 높은 Hz → 낮은 Hz → 다시 1,000Hz |
| **이유** | 인간 청력이 가장 민감하게 반응하고 말소리가 주로 분포하는 주파수이기 때문 |

> ⚠️ **기출 포인트**: 1,000Hz 시작 이유는 KICE 단골 출제.

#### 골도청력검사

골도 진동기를 유양돌기에 고정 → 내이를 직접 자극하여 내이 이상의 경로 기능 확인

- 골도 자극음 이간감약: **0~5dB** → 반드시 차폐 필요

#### 결과 해석

| 용어 | 정의 |
|-----|------|
| **평균청력손실도(PTA)** | 500Hz + 1,000Hz + 2,000Hz 최소 가청 역치 평균 |
| **사회적응청력** | 40dB 이내 — 정상 사회생활 최저 청력 |
| **청력형** | 청력도 모양 → 보청기 조절·언어 지도의 기초 자료 |

### 3-2. 차폐 (Masking)

**정의**: 양쪽 귀 청력 차이가 심할 때 비검사 귀에 잡음 제공 → 음영 청취(교차청취) 차단

| 용어 | 정의 |
|-----|------|
| **반대청취(교차청취)** | 나쁜 귀 검사음이 두개골 진동을 통해 좋은 귀로 전달되는 현상 |
| **이간감약** | 한쪽 귀 자극음이 반대측 와우에 도달 시 감소하는 강도 |

> ⚠️ **기출 포인트**: 차폐 필요 조건

| 검사 방법 | 차폐 필요 조건 |
|---------|------------|
| **골도청력검사** | **항상** 실시 |
| **기도청력검사** | 검사 귀와 비검사 귀 골도청력 차이가 **40dB 이상**일 때 |

- 차폐 곤란 조건: 양측 전음성 난청이 심하여 과잉/과소 차폐로 정확한 차폐 불가능한 경우

### 3-3. 어음청력검사

일상 대화 어음으로 청취·이해 능력 측정

| 검사 | 정의 | 특이점 |
|-----|------|-------|
| **어음청취역치(SRT)** | 50% 이상 정확히 알아들을 수 있는 최소 어음 강도 | 순음 PTA와 10dB 이내 차이 → 신뢰 |
| **어음명료도 검사(SDT)** | 쾌적 수준에서 단음절어를 정확히 이해하는 비율(%) | — |
| **어음탐지역치** | 50% 탐지 가능한 최소 강도 | SRT보다 약 10dB 낮음 |
| **어음 쾌적역치(MCL)** | 가장 편안하게 들을 수 있는 수준 | — |
| **어음 불쾌역치(UCL)** | 불편함·통증을 느끼는 강도 | — |
| **역동범위** | UCL − SRT | 보청기 음량 증폭 정도 결정 |

> ⚠️ **기출 포인트**: **말림현상(Rollover)** — 어음 강도를 높여도 명료도가 더 이상 증가하지 않거나 오히려 낮아지는 현상. **후미로성 난청**에서 특징적으로 나타남.

### 3-4. Ling 6개음 검사

말소리를 구성하는 대표 음소 6개의 청취 능력을 간편 진단

| 음소 | 주파수 특성 |
|-----|-----------|
| [m] | 저주파 (~250Hz) |
| [a] | 중저주파 (~1,000Hz) |
| [u] | 저주파 (~250~500Hz) |
| [i] | 고주파 (~3,000Hz↑) |
| [∫] sh | 고주파 (~3,000~5,000Hz) |
| [s] | 최고주파 (~4,000~8,000Hz) |

- **스피치 바나나**: 말소리가 250~8,000Hz 범위에 위치하며 오디오그램에 표시하면 바나나 모양
- **활용**: 보청기 적절성 신속 확인, 청능 수준·어음 능력 예측 단서

## 4. 유소아 청력검사 (6개월~5세)

| 검사 방법 | 대상 | 방법 |
|---------|-----|------|
| **행동관찰 청력검사(BOA)** | 신생아~6개월 | 소리 → 행동 반응(눈 깜빡임, 울음 등) 관찰 |
| **시각강화 조작조건검사(VROCA)** | 6개월~2.5세 | 소리 들을 때 시각 강화물(움직이는 장난감 등) 제공 |
| **물질강화 조작조건검사(TROCA)** | 2~5세 | 소리 들을 때 물질(과자 등) 강화 제공 |
| **유희조건 조작조건검사(CPA)** | 2.5~5세 | 소리에 반응하여 놀이 활동 수행 |

## 5. 기타 검사

| 검사 | 방법 | 정보 |
|-----|-----|------|
| **음장청력검사** | 헤드폰 없이 스피커로 순음 검사 | 보청기 착용 상태로 검사 시 활용 |
| **린넨(Rinne)검사** | 음차로 기도·골도 비교 | 전음성 난청 유무 판별 |
| **웨버(Weber)검사** | 음차를 두개 중앙에 대어 어느 귀에서 들리는지 | 편측 난청 시 전음성 vs 감각신경성 구별 |
| **슈바바(Schwabach)검사** | 골도 전도 청각 민감도 상대 비교 | — |
| **빙(Bing)검사** | 귀 막음 시 저주파 더 잘 들리는 폐쇄효과 확인 | 전음성 난청 유무 |
