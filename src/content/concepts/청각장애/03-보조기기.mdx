## 1. 보청기의 이해

소리 강도를 증폭하여 잔존 청력을 최대한 활용하도록 돕는 장치. **전음성 청각장애 아동에게 효과적**.

### 1-1. 기본 구조

| 부품 | 기능 |
|-----|------|
| **송화기(마이크)** | 음향 에너지 → 전기 신호 변환 |
| **증폭기** | 전기 신호 증폭 |
| **수화기(리시버)** | 전기 신호 → 음향 신호 변환 |

## 2. 보청기 유형

### 2-1. 착용 위치에 따른 분류

| 유형 | 특징 |
|-----|------|
| **상자형(포켓형)** | 본체를 주머니에 넣고 선으로 귀에 연결; 조작 쉬움, 크기 큼 |
| **안경형** | 안경 다리에 장착; 미용상 유리, 수리 불편 |
| **귀걸이형(BTE)** | 귀 뒤에 착용; 가장 보편적, 모든 청력 손실 적용 가능 |
| **귀속형(ITE/ITC/CIC)** | 외이도 내부 착용; 미용상 유리, 소형, 조작 어려울 수 있음 |

### 2-2. 신호 처리 방식에 따른 분류

| 유형 | 특징 |
|-----|------|
| **아날로그** | 연속적 전기 신호 처리, 저렴 |
| **디지털 프로그램형** | 디지털 프로그래밍 + 아날로그 신호 처리 |
| **디지털** | 고품질 신호 처리, 소음 억제, 프로그래밍 정밀 |

### 2-3. 특수 보청기

| 유형 | 대상 | 특징 |
|-----|------|------|
| **FM 보청기** | 교실 환경, CAPD | 무선 마이크 → 특정 주파수 수신; **SNR 개선**, 거리 무관 명료음 전달 |
| **크로스(CROS) 보청기** | 편측성·비대칭형 난청 | 나쁜 귀 소리 → 좋은 귀로 전달 (외부: 유·무선, 내부: 골도 진동) |
| **바이크로스(BiCROS)** | 양쪽 비대칭 난청 | 양쪽 입력 증폭 → 방향 감각 개선, 음향 되울림 가능 |
| **주파수압축 보청기** | 고도·최고도, 저주파 잔존 청력 有 | 고주파 에너지 → 저주파 대역으로 변환·압축·증폭 |
| **골도 보청기** | 전음성 청각장애, 외이 착용 불가 | 중이 우회 → 와우 직접 자극 |
| **촉각 보청기** | 일반 보청기 효과 없는 경우 | 소리 → 진동·전기 → 피부 자극으로 소리 인식 |

> ⚠️ **기출 포인트**: FM 보청기 단점 — 일반화 어려움, 주파수 혼선 소음 가능, 차폐물에 약함.

### 2-4. 증폭 방식에 따른 분류

| 유형 | 원리 | 적합 대상 |
|-----|------|---------|
| **선형 보청기** | 작은~큰 소리 1:1 비율 증폭 → 출력 제한 | 전음성·혼합성 난청 |
| **비선형 보청기** | 작은 소리는 이득 크게, 큰 소리는 이득 작게 (압축) | 역동범위 좁은 **감각신경성 난청** |

> ⚠️ **기출 포인트**: 비선형 보청기 → 역동범위 좁은 감음성 난청에 적합. 음의 왜곡 방지.

**그림자 효과**: 소리와 청취자 사이에 물체가 있을 때 소리 강도가 약해지고 고주파 특성이 결여되는 현상.

## 3. 인공와우(Cochlear Implant: CI)

외이·중이 우회 → 전극이 청신경을 직접 전기 자극하여 소리 인식. 보청기 효과가 없는 **양측 고도 감음성 난청**에 유용.

### 3-1. 구조

| 부위 | 부품 | 기능 |
|-----|------|------|
| **체외부** | 마이크(송화기) | 외부 소리 감지·전기 에너지 변환 |
| **체외부** | 어음처리기(speech processor) | 소리 분석·부호화, 증폭기·신호변환기 역할 |
| **체외부** | 송신기(transmitter) | 어음처리기 신호 → 체내 수신기로 전달 |
| **체내부** | 수신기(receiver-stimulator) | 송신기 신호 수신 → 전극부로 전달 |
| **체내부** | 전극부(electrode) | 와우 내 삽입 → **청신경 직접 자극** |

### 3-2. 이식 대상자 기준 (5가지 모두 충족)

| 번호 | 기준 |
|-----|------|
| ① | 양측 모두 **고도 이상** 감음성 청각장애 |
| ② | 보청기로 적절한 기간 재활해도 효과 없음 |
| ③ | 청신경이 일부라도 기능함 |
| ④ | 내과적·신경학적·정신과적 문제 없음 |
| ⑤ | 환자·보호자가 청능 재활에 강한 동기 |

> ⚠️ **기출 포인트**: 5가지 기준 모두 암기 필수. 수술 전 최소 3~6개월 보청기 청능 학습 필요.

### 3-3. 매핑(Mapping)

수술 후 어음처리기를 개별 프로그래밍하는 과정

| 항목 | 내용 |
|-----|------|
| **목적** | 아동에게 최대 음향 정보 + 알아들을 수 있는 말소리 제공 |
| **T-level(자극역치)** | 전기 자극을 감지하는 최소 전류 |
| **C-level(쾌적레벨)** | 불편 없이 가장 편안하게 듣는 전류 |
| **역동범위 설정** | T-level ~ C-level 범위 |

**재매핑 필요 상황**:
- 말소리 반응이 줄어들 때
- 어음 명료도가 나빠질 때
- 말소리 모방이 부정확할 때

### 3-4. 수술 후 재활

- 첫 매핑 후 1개월부터 정기 청력 검사
- Ling 6개음 검사로 회화음 전 주파수 청취 여부 확인
- 청능 훈련 + 독화 병행, 기억력 향상 반복 훈련

### 3-5. 인공와우 주의사항

| 항목 | 주의 내용 |
|-----|---------|
| **마이크·어음처리기** | 수분·땀에 약함 |
| **전극부** | 전자파·자기에 영향 받음 |
| **정전기** | 어음처리기 프로그램 손상 가능 |
| **신체 충격** | 머리 충격 주의; 헬멧 착용 권장 |
| **매일 확인** | 착용 여부 + 정상 작동 확인 |
