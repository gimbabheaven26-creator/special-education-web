## 1. 수화(Sign Language)

농인의 자연 발생적·비구어적·규칙 지배적 **시각-운동 언어 체계**

### 1-1. 수화소 (5가지)

> ⚠️ **기출 포인트**: 수화소 5가지 각각의 정의 정확히 구별.

| 수화소 | 정의 |
|------|------|
| **수형** | 의미 있는 위치에서 의미 있는 운동을 하는 한 손/두 손의 결합 구도 |
| **수위** | 수화 어휘를 구성하기 위해 손이 작용하는 **위치** |
| **수동** | 수형의 운동·방향·양손 관계·동작의 질적 측면 |
| **수향** | 손가락 또는 손바닥이 향하는 **방향** |
| **비수지 신호** | 표정, 몸 움직임 등 **손을 사용하지 않는** 신호 |

### 1-2. 수화의 표현상 특징

| 특징 | 설명 |
|-----|------|
| **공간성과 동시성** | 수화소를 공간적으로 배열, 동시적으로 단어 형성 |
| **가역성** | 반의어에서 나타나는 수화 고유 특성 |
| **도상성(사상성)** | 형태를 보고 뜻을 유추할 수 있는 정도가 강함 |
| **자의성(규약성)** | 사회적 약속으로 형성, 추상적 표현에 주로 사용 |
| **반복성** | 수화소를 반복하여 의미 표현 |
| **축약성** | 메시지 손상 없이 간결하게 표현 |

### 1-3. 수화 유형 비교

| 유형 | 문법 체계 | 특징 |
|-----|---------|------|
| **자연수화** | 농사회 독자적 문법 (국어 문법과 다름) | 농문화의 주된 의사소통 수단 |
| **문법수화** | 국어 문법 체계 | 음성 언어 문법을 손 운동으로 표현 |
| **지화** | 철자 1:1 대응 | 문어의 철자와 유사, 고유명사·새 단어 표기 |

### 1-4. 수화교육의 필요성

- 농학생의 언어 능력 개선
- 음성 언어 습득에 긍정적 영향
- 인지 발달 촉진
- 읽기·쓰기 능력 향상에 기여

## 2. 구화(Oralism)

청각장애 아동이 일반 사회의 언어를 배워야 한다는 교육 철학. 수화 등 다른 의사소통 방법 허용하지 않음.

### 2-1. 청능훈련 (수용언어)

음성 언어의 청각적 수용과 잔존 청력을 최대한 일상에서 활용하는 기술 학습

> ⚠️ **기출 포인트**: 4단계 순서와 각 단계 정의 정확히 암기.

| 단계 | 명칭 | 내용 |
|-----|------|------|
| 1단계 | **음의 감지(인식)** | 소리의 존재를 탐지하고 주의를 기울이는 것 학습 |
| 2단계 | **음의 변별** | 특정 소리와 다른 소리가 같은지 다른지 구분 |
| 3단계 | **음의 확인(말의 변별)** | 청각 정보를 기지(旣知) 정보와 비교하여 인식; 따라 말하기·가리키기·쓰기·명명하기로 반응 |
| 4단계 | **음의 이해** | 음성 언어 자극을 의미 있게 이해; 질문에 대답하거나 지시를 따르는 등 대화 속 의미 이해 |

#### 청능훈련 설계 시 고려사항

| 구분 | 내용 |
|-----|------|
| **자극 단위** | 분석적 훈련법(음소·음절 초점) vs 종합적 훈련법(발화 전체 의미 초점) |
| **활동 유형** | 형식적 활동(구조화된 1:1 또는 소그룹) vs 비형식적 활동(일상생활 속 청능 과제) |

### 2-2. 독화(Speechreading)

말의 시각적 정보를 통해 화자의 의미를 이해하는 '말의 시각적 수용 기술'

#### 독화 관련 변인

| 변인 | 요소 |
|-----|------|
| **독화자** | 시력, 청력 수준, 지적 수준, 시지각, 종합 능력 등 |
| **화자** | 언어 특성, 표정, 동작, 손짓, 독화자에 대한 태도 |
| **환경** | 조명, 각도, 화자의 수, 거리, 노출 등 |

#### 독화의 한계

| 한계 | 설명 |
|-----|------|
| **낮은 가시도** | 말소리 형성 근육 운동 대부분이 입 안에서 발생 → 시각 확인 어려움 |
| **동형이음어(동구형이음어)** | 소리·철자 달라도 입 모양이 비슷 → 독화만으로 구별 곤란 |
| **빠른 구어 속도** | 일반적 말 속도 > 독화 속도 |
| **전이 효과** | 음운 환경에 따라 한 단어 발음이 다른 단어에 영향 |
| **조음 운동 개인차** | 화자마다, 같은 화자도 발화마다 차이 |
| **환경적 제약** | 시야 방해, 차폐물, 계속적 주시의 어려움 |

#### 독화 지도 방법

| 방법 | 특징 |
|-----|------|
| **분석법** | 음운 단위 → 음절 → 단어 → 문장 순서 |
| **종합법** | 문장·문단 → 단어 순서 |
| **절충법** | 분석법 + 종합법 병용 |
| **종합적 접근법** | 한 가지 기법만 고집하지 않음, 동기·인내력·책임감 강조 |
| **말 추적법** | 대화 맥락에서 청각적으로 이해한 단어/구/문장 반복하며 전체 완성 |

### 2-3. 큐드스피치(발음암시법 / 단서언어)

독화 외에 음성 언어의 시각적 단서를 추가하는 방법

- **방법**: 뺨 근처에서 자·모음 말소리를 나타내는 **수신호** 추가
- **목적**: 독화로 구별 어려운 음소 인식 → 구형 변별 보완

## 3. 복합 의사소통 방법

### 3-1. 종합적 의사소통법 (Total Communication: TC)

의사소통 유형에 제한 없이 **가능한 모든 의사소통 체계 활용**

| 항목 | 내용 |
|-----|------|
| **특징** | 구어를 1차 언어로 지지; 교사가 구어 + 문법수화를 **동시에** 사용 |
| **한계** | 문해·학업 성취 여전히 낮음; 구어·수화 모두 훈련된 교사 부족 |

### 3-2. 로체스터법

**지문자** + **구화** 병용 → 청각장애 아동에게 구화를 시각적으로 보충

- 지문자로 모든 단어를 철자별로 표현하면서 동시에 말함
- 구화의 제한을 지문자로 보완

### 3-3. 이중언어-이중문화 접근 (Bilingual-Bicultural: 2Bi)

> ⚠️ **기출 포인트**: L1·L2 설정과 목적.

| 항목 | 내용 |
|-----|------|
| **L1 (1차 언어)** | **자연수화** |
| **L2 (2차 언어)** | 국어 (음성 언어 + 문자 언어) |
| **목적** | 농아동이 농문화 수용 → 자아 정체감 형성; 자연수화를 모국어로 발달 → 학습 방법 개선 |
| **농문화 핵심 요소** | 수화, 집단 정체감, 족내혼, 집단 응집력과 네트워크 |

## 4. 통합교육 지원

### 4-1. 교사의 배려 사항

| 영역 | 내용 |
|-----|------|
| **좌석 배치** | 소음원 멀리; 교사·교육 활동 가까이; 교사 정면 바라보기 가능한 자리 |
| **의사소통** | 이름 부르기 전 청각·시각적 주의 먼저; 얼굴 보며 자연스럽게 말하기 |
| **어휘 지원** | 새로운 어휘 학습 자료 사전 제공 + 칠판에 판서 |
| **수업 진행** | 강의식 수업 시 독화·필기 동시 불가 → 노트 지원; 시각 자료와 교사 입 동시 보기 어려움 주의 |
| **환경** | 교실 소음 최소화; 카펫·커튼 소음 흡수; FM 보청기 활용 |

> ⚠️ **기출 포인트**: 독화와 필기는 동시에 하기 어려우므로 최소 두 명에게 노트 빌릴 수 있게 지원.

### 4-2. 수화통역사 활용 시 유의사항

| 시기 | 유의 사항 |
|-----|---------|
| **수업 전** | 교안·주요 단어·교재 사전 제공; 청각장애 학생이 교사·통역사·시각 자료 모두 보기 편한 자리 배치 |
| **수업 중** | 학생 행동 지도·학습 이해 책임은 **교사**에게; 질문은 통역사가 아닌 **학생에게 직접**; 통역사 위치: 교사 약간 뒤·약간 옆, 학생과 마주봄 |

### 4-3. 학습 환경 조성

- 바닥 카펫, 창문 커튼, 냉·난방기 소음 제거
- 책걸상 소음 점검; 천장·벽 방음 재료 부착

### 4-4. 농학생 단기기억·읽기·쓰기 특성

| 영역 | 특성 |
|-----|------|
| **단기 기억** | 시각적 부호화(수화 등)는 단기 계열 기억 용량 저하 가능 |
| **읽기·쓰기** | 내적 언어 결손 → 읽기 발달 지체; 비유적 표현 어려움; 독특한 통사 구조 사용 |

## 5. 농·맹 중복장애 의사소통

| 방법 | 설명 |
|-----|------|
| **손가락 점자** | 왼손·오른손 각 3개 손가락을 상대방 손가락 위에 접촉하여 점자로 의사소통 |
| **촉수화** | 상대방의 손 위에 자신의 손을 얹어 수화를 촉각으로 이해 |
