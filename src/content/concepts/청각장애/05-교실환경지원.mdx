## 1. 교실 음향 환경 관리

청각장애 학생은 배경 소음과 잔향이 어음 명료도에 결정적 영향을 미친다.

### 1-1. 핵심 음향 지표

| 지표 | 정의 | 청각장애 학생 권장 기준 |
|-----|------|----------------------|
| **SNR (신호대잡음비)** | 말소리 강도 − 배경소음 강도 (dB) | **+15 dB 이상** (일반 교실 +6~+10 dB 수준) |
| **RT60 (잔향 시간)** | 소리가 60 dB 감쇠하는 데 걸리는 시간 | **0.4초 이하** (일반 교실 0.6~0.9초) |
| **배경 소음 수준** | 공조·외부 소음 | **35 dBSPL 이하** |

> ⚠️ **기출 포인트**: SNR은 절댓값이 아닌 **차이(dB)**임에 주의. RT60이 길면 말소리 겹침 → 명료도 저하.

### 1-2. 교실 음향 개선 방법

| 방법 | 적용 예 |
|-----|---------|
| 카펫·커튼 설치 | 바닥·벽면 흡음 처리 |
| 방음 천장재 | 반사음 감소 |
| 책상·의자 발 테니스공 부착 | 이동 소음 차단 |
| 창문 이중유리 | 외부 소음 차단 |
| 학생 좌석 배치 | 교사 정면 가까운 앞쪽, 보청기·CI 마이크 방향 고려 |

## 2. FM 시스템 교실 적용

### 2-1. 기본 원리

교사가 **송신기(마이크)**를 착용 → 특정 FM 주파수로 전송 → 학생의 **수신기(보청기 연결)**가 수신.

- **SNR 10~15 dB 개선** 효과 → 거리·잡음 영향 최소화
- 교사의 음성이 학생 귀 바로 옆 15 cm 거리와 동일한 효과

### 2-2. 교실 FM 적용 시 유의사항

| 항목 | 내용 |
|-----|------|
| **주파수 설정** | 교실 간 혼선 방지 → 학교별 채널 배정 필요 |
| **배터리 관리** | 매일 완충 확인, 방전 시 무음 |
| **마이크 위치** | 교사 입에서 15~20 cm 거리 (가슴 중앙) |
| **교사 이동 시** | 마이크는 항상 교사 따라가야 함 |
| **단점** | 차폐물(벽, 가구)에 신호 약화; 교사 목소리만 증폭(학생 간 토론 전달 안 됨) |

> ⚠️ **기출 포인트**: FM 시스템은 교사 목소리만 전달 → 그룹 토론 시 추가 마이크 패스 전략 필요.

### 2-3. FM 시스템 유형

| 유형 | 특징 |
|-----|------|
| **개인용 FM** | 학생 개인 수신기 (보청기/CI에 직접 연결, DAI 방식) |
| **사운드 필드 FM** | 교실 전체 스피커 시스템 (경도~중도 난청, 비장애 학생도 혜택) |

## 3. 보완 지원 서비스

### 3-1. CART (Computer-Aided Real-time Translation)

**속기사가 교사·학생 말을 실시간으로 자막 변환**, 화면에 표시.

| 장점 | 단점 |
|-----|------|
| 음성을 즉시 텍스트로 제공 | 속기사 전문 인력·비용 필요 |
| 청각 수준 무관하게 활용 | 음성 → 자막 0.5~1초 지연 |
| 기록 파일 저장 가능 | 말이 빠르면 요약될 수 있음 |

### 3-2. 노트테이커(Note-taker)

수업 내용을 대신 필기하여 제공. **청각장애 학생이 독화·자막 집중할 수 있도록 필기 부담 경감.**

- 동료 노트테이커: 같은 반 학생이 카본지 등으로 복사
- 전문 노트테이커: 별도 훈련된 인력

### 3-3. 통합학급 청각장애 지원 전략

| 상황 | 전략 |
|-----|------|
| 교사 발화 | 항상 칠판 등 보조물과 함께 → 학생 쪽으로 얼굴 향하기 |
| 그룹 토론 | 발언자 지목 → 청각장애 학생이 화자 확인 가능하게 |
| 동영상 자료 | 자막 필수 |
| 질의응답 | 학생 질문 **재진술(재방송)** → 전체가 들을 수 있도록 |
| 조명 | 독화를 위해 교사 얼굴에 역광 없도록 |

## 4. 인공와우 신호처리 방식 심화

### 4-1. CIS vs ACE

| 항목 | **CIS** (Continuous Interleaved Sampling) | **ACE** (Advanced Combination Encoder) |
|-----|------------------------------------------|---------------------------------------|
| 개발 | 1991 Wilson 등 | Cochlear사 |
| 원리 | 모든 채널에 연속·교대로 자극 | **활성 채널 선택** (에너지 큰 채널만 자극) |
| 자극 채널 수 | **전체 채널** | 가장 큰 n개 채널 선택 (maxima) |
| 장점 | 구현 간단, 안정적 | 소음하 어음 인지 개선, 배터리 효율 |
| 임상 적용 | 광범위 사용 | Cochlear Nucleus 기기에 주로 사용 |

> ⚠️ **기출 포인트**: CIS는 전 채널, ACE는 선택 채널. ACE가 소음 환경에서 유리.

### 4-2. 양측 인공와우 (Bilateral CI)

| 항목 | 내용 |
|-----|------|
| **정의** | 양쪽 귀 모두에 인공와우 이식 |
| **장점** | 방향 감각(localization) 개선, 소음 속 어음 인지 향상, 그림자 효과 감소 |
| **단점** | 수술 위험 2배, 비용 증가, 한쪽 이미 이식한 경우 추가 수술 필요 |
| **동시 vs 순차** | 동시 이식이 재활 기간 단축; 순차는 한쪽 회복 후 반대쪽 |

### 4-3. 인공와우 착용 후 음악 지각의 한계

- **주파수 분해능 제한**: 전극 수(~22개) < 와우 유모세포 수(약 3,500개)
- **음색(timbre) 구별 어려움**: 악기 음색 변별 곤란
- **멜로디 인지 > 리듬 인지**: 리듬은 비교적 잘 인식, 멜로디는 어려움
- **음악 선호도 개인차 큼**: 이식 전 음악 경험·이식 나이에 따라 차이

## 5. 교실 지원 통합 적용 사례

**상황**: 중학교 통합학급, 양측 인공와우 착용 학생 A

| 환경 요소 | 적용 |
|---------|------|
| 좌석 | 교사 정면 1~2열, 창문 쪽(역광 방지) |
| FM 시스템 | 교사 송신기 착용 → A의 CI 수신기 연결 |
| 소음 관리 | 카펫 설치, 창문 방음, 의자 발 패드 |
| 자료 | 동영상 자막 필수, 판서 함께 제공 |
| 노트테이커 | 동료 노트테이커 지정 |
| 그룹 활동 | 발언자 지목 후 마이크 패스 |
