## 1. 한국수어(KSL) vs 수지한국어

### 1-1. 핵심 비교

| 항목 | **한국수어(KSL)** | **수지한국어** |
|-----|-----------------|--------------|
| **성격** | 농인 공동체 고유의 **자연 언어** | 한국어 어순·구조에 수화 어휘를 얹은 **인공 체계** |
| **어순** | **SOV + 공간문법** (토픽-주석 구조) | 한국어 어순 그대로 (SOV, 조사 포함) |
| **문법 수단** | 공간 활용, 비수지 신호(표정·몸동작) | 한국어 조사·어미를 수화 단어로 표현 |
| **사용자** | 농 성인, 농학교 출신 | 청인이 농인에게 말하거나 통역할 때 주로 사용 |
| **의사소통** | 농인 간 자연스러운 의사소통 | 농인 → 청인 방향에서 자연스러움 떨어질 수 있음 |
| **법적 지위** | **한국수화언어법(2016)**: 국어와 동등한 언어로 인정 | 별도 법적 지위 없음 |

> ⚠️ **기출 포인트**: 한국수어는 **한국어의 시각적 표현이 아니라 독립된 언어**. 어순도 다르고 문법도 다르다.

## 2. 한국수어의 공간 문법

청인의 언어가 **시간 축**에서 순차적 단어로 이루어지듯, 한국수어는 **3차원 공간**을 문법 자원으로 사용한다.

### 2-1. 세 가지 핵심 문법 장치

| 문법 장치 | 정의 | 예시 |
|---------|------|------|
| **배치(Placement / Referent Locus)** | 공간의 특정 위치에 **명사를 할당**해 지시 대상을 확립 | "철수"를 왼쪽 공간에, "영희"를 오른쪽 공간에 설정 |
| **이동(Movement)** | 동사가 행위자(A) 위치에서 대상(B) 위치로 **이동하며 행위 관계 표현** | '주다'를 철수 쪽 → 영희 쪽으로 이동 = "철수가 영희에게 준다" |
| **방향(Directionality)** | 동사 손바닥·손목의 **방향이 주어·목적어를 지시** | 1인칭(나)→2인칭(당신): 손이 앞으로 / 역방향: 당신→나 |

> ⚠️ **기출 포인트**: 이동 동사는 공간 내 A→B 경로로 주어·목적어를 동시 표현 → 별도 대명사 없이 문법 관계 전달.

### 2-2. 분류사(Classifier)

수화 공간에서 **손 형태가 대상의 범주를 나타내는** 형태소.

| 분류사 유형 | 설명 |
|-----------|------|
| 형태 분류사 | 대상의 모양 묘사 (납작한 것: 손바닥 수평, 긴 것: 검지 세움) |
| 크기-형태 분류사 | 사물 크기·형태를 손 형태로 묘사 |
| 행위자 분류사 | 사람·동물의 이동 경로 표현 |

## 3. 비수지 신호 (NMS: Non-Manual Signals)

손 움직임(수지 신호) 이외에 **얼굴 표정·입 모양·눈썹·머리·몸 기울기**로 전달되는 문법적·어휘적 신호.

### 3-1. 문법적 NMS 기능

| NMS 요소 | 문법 기능 | 예시 |
|---------|---------|------|
| **눈썹 올림** | 예/아니오 의문문 | "밥 먹었어?" |
| **눈썹 내림 + 머리 앞으로** | WH 의문문 | "어디 가?" |
| **고개 끄덕임** | 긍정·확인 | 진술 강조 |
| **고개 젓기** | 부정 | 동사와 동시에 → 부정 동사 |
| **입 모양** | 어휘적 변별 또는 강도 수식 | '많다-엄청' 등 강도 구분 |
| **눈 크게 뜸** | 강조 | 부사 기능 |
| **몸 기울기** | 조건절·화제 표시 | 조건절 시작 시 몸 기울임 |

> ⚠️ **기출 포인트**: NMS는 단순 감정 표현이 아니라 **문법 형태소**다. 눈썹·고개로 의문문 유형을 구분.

### 3-2. 구화(spoken language) 성분과의 구분

일부 농인이 수화와 동시에 한국어 입 모양을 보이는 것은 **수지한국어 동시적 사용**이며, 한국수어 NMS와는 구별된다.

## 4. 지문자(Fingerspelling)

**한글 자음·모음을 손 형태로 표현**하는 체계. 고유명사·전문용어처럼 수화 단어가 없는 경우 보조적으로 사용.

| 특징 | 내용 |
|-----|------|
| 역할 | 수화 어휘 보완 (주로 이름, 외래어, 전문 용어) |
| 속도 | 숙달자는 빠르게 사용; 초보자는 음절 단위로 느리게 |
| 한국 지문자 | 자음 14개 + 모음 10개 손 형태 |
| 주의 | 지문자 ≠ 수화. 지문자만으로 의사소통하는 것은 비효율적 |

## 5. 청각장애 의사소통 교육 접근법

### 5-1. 주요 접근법 비교

| 접근법 | 핵심 원리 | 수화 사용 | 강조점 |
|-------|---------|---------|-------|
| **구화법(Oral Method)** | 말하기·독화 집중 훈련 | 사용 금지 | 청인 사회 통합 |
| **수화법(Manual Method)** | 수화를 주된 언어로 사용 | 적극 사용 | 농 정체성 |
| **총체적 의사소통(TC)** | 구화+수화+몸짓 등 **모든 수단 동시 활용** | 허용 | 의사소통 극대화 |
| **동시적 의사소통(SimCom)** | 말하기 + 수지한국어 **동시에** 사용 | 수지한국어 | 음성·수화 병행 |
| **이중언어-이중문화(Bi-Bi)** | 수어(L1)→문자언어(L2) 이중언어 교육 | 한국수어(L1) | 농 정체성+리터러시 |

> ⚠️ **기출 포인트**: TC는 모든 의사소통 수단 허용. Bi-Bi는 **수어를 제1언어**로, 문자(한국어 읽기·쓰기)를 제2언어로 배움.

### 5-2. 총체적 의사소통(Total Communication) 심화

- 1970년대 Roy Holcomb이 제창
- **어떤 단일 방법도 모든 아동에게 최적이 아니다**는 철학
- 포함 수단: 구화, 독화, 수화, 지문자, 몸짓, 그림, 문자
- 비판: 구화·수화를 동시 사용 시 두 언어 모두 **퇴화**될 수 있다는 지적

### 5-3. 이중언어-이중문화(Bi-Bi) 접근 심화

| 요소 | 내용 |
|-----|------|
| **L1 (제1언어)** | 한국수어 — 자연언어로서 완전 습득 |
| **L2 (제2언어)** | 한국어 — 주로 **읽기·쓰기** 형태로 학습 |
| **문화적 정체성** | 농 문화를 긍정적 정체성으로 수용 |
| **교육적 목표** | 이중언어자(bilingual) 육성, 문식성(literacy) 향상 |
| **교사 요건** | 유창한 한국수어 능력 필수 |

## 6. 독화(Lip Reading / Speech Reading)

**입 모양, 얼굴 표정, 몸짓을 시각적으로 해독**하여 말을 이해하는 기술.

| 항목 | 내용 |
|-----|------|
| **독화 가능 한계** | 한국어 음소 중 **시각적으로 구별 가능한 것은 약 30~40%** |
| **혼동 음소** | 양순음(ㅁ/ㅂ/ㅍ) — 입 모양 유사 → 문맥으로 추론 |
| **독화 용이 조건** | 화자 정면 응시, 조명 충분, 입 가리지 않음, 말 속도 적당 |
| **독화 어려운 조건** | 어두운 조명, 역광, 수염, 마스크 착용, 빠른 말, 방언 |
| **독화 보완 수단** | 큐드 스피치(Cued Speech): 손 모양 단서로 독화 보완 |

> ⚠️ **기출 포인트**: 독화 독립 사용은 한계가 크다 → 잔존 청력 활용 + 문맥 의존 필수. 역광 없는 좌석 배치 중요.

## 7. 청각장애 의사소통 통합 사례

**상황**: 고등학교 통합학급, 한국수어 우세 청각장애 학생 B (구화 가능, 보청기 착용)

| 상황 | 지원 전략 |
|-----|---------|
| 강의 수업 | CART 자막 + 독화 위해 교사 얼굴 향하기 |
| 조별 토론 | 수화통역사 배치 또는 발언자 지목 후 자막 표시 |
| 영상 자료 | 한국어 자막 필수 (KSL 자막은 별도) |
| 고유명사 전달 | 교사가 칠판·화면에 텍스트 병행 제시 |
| 쉬는 시간 | 농 정체성 존중 — 구화 강요 지양 |
