## 1. 통합교육의 개념

### 1-1. 핵심 개념 비교

| 개념 | 정의 |
|------|------|
| **완전통합** | 비분리 원칙; 물리적·학문적·사회적 통합 모두 포함; 동일 교실·동시·같은 프로그램 |
| **주류화(Mainstreaming)** | 장애 학생이 학교생활 일부를 일반학급에서 또래와 함께 보내는 것 |
| **최소제한환경(LRE)** | 장애 아동을 또래·가정·지역사회에서 가능한 한 최소로 분리; 절대적 필요 이상 자유 침해 금지 |

> ⚠️ **기출 포인트**: 완전통합은 분리를 전제로 하지 **않음**. 불가능한 경우에만 분리.

### 1-2. 통합의 수준 분류

**통합 수준 6단계**: 물리적 → 용어적 → 행정적 → 사회적 → 교육과정 → 심리적 통합

**Kauffman 분류**: 시간적 통합 / 교수활동적 통합 / 사회적 통합

### 1-3. 통합교육의 목적

| 목적 | 내용 |
|------|------|
| **다양성 인정** | 개인차는 차별의 근거가 아닌 독특한 교육적 요구 |
| **평등성** | 학습 능력과 요구에 적합한 서비스 제공 |
| **수월성** | 개인의 잠재력을 최대한 계발 |
| **조화** | 단 한 사람도 소외되지 않는 집단사회 |

## 2. 협력적 팀 접근 — 전문영역 간 협력 3유형

> ⚠️ **기출 포인트**: 3유형 각각의 특징과 장단점 정확히 구별.

| 항목 | **다학문적** | **간학문적** | **초학문적** |
|------|------------|------------|------------|
| **진단 방식** | 각 전문가 독립 평가 | 각 전문가 독립 평가 + 공식 의사소통 | 동시 평가 (원형진단) |
| **중재 방식** | 독립적(직접) 서비스 | 독립적(직접) 서비스 | 간접/통합적; 주 서비스 제공자 1인 |
| **장점** | 다양한 전문성 반영, 실수·편견 감소 | 통일된 중재 계획, 전문성 강화 | 시간 절약, 즉각 전문성 접근, 일관성 |
| **단점** | 서비스 조각화 → 부모 혼란 | 조정자 역할 불분명, 의사소통 문제 | 업무 과중 (주 서비스 제공자) |

## 3. 초학문적 팀 접근의 주요 요소

### 3-1. 역할 방출(Role Release)과 주 서비스 제공자

> ⚠️ **기출 포인트**: 역할 전이 5단계 순서 암기.

**역할 전이 단계** (순서 있음):

| 단계 | 명칭 | 내용 |
|------|------|------|
| 1 | **역할 확대** | 자신 전문 영역의 전문성·기술 향상 |
| 2 | **역할 확장** | 타 영역 배우기 |
| 3 | **역할 교환** | 타 전문가 감독 하에 타 영역 중재 실시 |
| 4 | **역할 방출** | 감독 없이 독립적으로 타 영역 연습 |
| 5 | **역할 지원** | 주 서비스 제공자가 동료 전문가에 의존해 모든 영역 중재 진행 |

- **주 서비스 제공자**: 대개 교사가 담당; 다른 팀원은 자문 역할
- 가족의 중심적 역할 강조

### 3-2. 원형진단(Arena Assessment)

**정의**: 다양한 영역 전문가들이 **동시에** 대상 유아를 진단하는 방법

**실행 절차**: 인사 및 분위기 조성 → 공식적 과제 중심 진단 → 간식·재충전 → 이야기/교수활동 → 자유 놀이 → 간단 회의·반응하기

**장점**: 동일한 행동 샘플에 대해 즉시 전문성 교환 가능

### 3-3. 통합된 치료

- 기본 가정: 학생 능력 진단은 **자연적 환경**에서 가장 잘 이루어짐
- 일상적·자연적 상황의 기능적 활동을 통해 학생이 가장 잘 배움

## 4. 협력적 접근

### 4-1. 협력의 원칙 (6가지)

1. 상호 합의된 공동 목표
2. 참여자 동등성
3. 책임감 공유
4. 수행 결과 공동 책임
5. 자원 공유
6. **자발적 관계성** (참여 강요 금지)

### 4-2. 협력팀 구성원 역할

| 구성원 | 주요 역할 |
|--------|----------|
| **통합학급 교사** | 장애 학생 교수 기술·평가·성적 수정; 다른 전문가 지식 활용 |
| **특수교사** | 통합 촉진자·사례 관리자; 자문·협력 제공 |
| **행정가** | 교과 과정·자료·자원 수정·제시 |
| **부모** | 가족 우선순위 제시; 다양한 환경에서 학생 수행 의견 제시 |

### 4-3. 협력적 자문 5단계

자문 시작하기 → 문제 확인하기 → 중재 목표 설정 → 중재 계획 수립 → 진도·과정 평가

> 협력적 자문: 자문가가 **직접 학생과 상호작용하지 않고** 교사에게 전문 정보 제공 (간접 지원)

### 4-4. 교사 간 공동 협력 형태

| 유형 | 특징 |
|------|------|
| **협력적 교화** | 경력 무관, 정보·지식 자유 교환 |
| **협력적 모델링** | 경험 많은 교사가 교수 실제 시범 |
| **협력적 코칭** | 새 교수 전략 습득·적용에 피드백 제공 |
| **협력적 감독** | 교수 실제에 평가적 피드백 |
| **협력적 조언** | 문제 발생 시 경험 많은 교사가 조언 |
