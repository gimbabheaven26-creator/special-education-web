## 1. 협력교수(Co-teaching) 5유형

**정의**: 두 명 이상의 전문가가 지속적으로 함께 가르치며, 일반학급 내 모든 학생을 대상으로 함

> ⚠️ **기출 포인트**: 협력교수는 특정 집단이 아닌 **전체 아동**을 함께 가르침.

### 유형별 비교

| 유형 | 구조 | 장점 | 단점 | 주의점 |
|------|------|------|------|--------|
| **교수-지원** | 1교사 전체 교수 + 1교사 순회 지원 | 전문성 발휘, 1:1 지도 가능 | 지원 교사가 보조원처럼 보임; 학생 의존적 학습자화 | 역할 고정 금지 → 내용에 따른 역할 교대 |
| **대안교수** | 대집단 + 소집단 (심화·보충) | 심화·보충 기회; 결석생 보충 | 분리 환경 조성; 성취 낮은 학생만 소집단 고착 가능 | 소집단에 취약 학생만 배치 금지 |
| **평행교수** | 학급 2분할; 같은 내용 각각 지도 | 개별화 가능; 복습에 효과적 | 집단 간 난이도·속도 불일치 우려 | 충분한 공동 계획 필요; 소음 관리 |
| **스테이션 교수** | 수업 내용 모둠화; 학생이 스테이션 이동 | 능동적 학습; 협동·독립성 증진 | 많은 준비; 감독 어려움; 넓은 공간 필요 | 연계성 없는 스테이션은 효과 저하 |
| **상호교수(팀티칭)** | 두 교사 동등 책임·역할; 5~10분 교대 | 역할·내용 공유; 체계적 관찰 가능 | 많은 공동 준비 시간; 업무 과중 | 교수 전달 방식 차이 → 학생 혼란 |

> ⚠️ **기출 포인트**: 상호교수(팀티칭)는 협력교수 중 **가장 높은 수준**의 협력.

## 2. 협동학습

### 2-1. 협동학습의 4가지 원리

| 원리 | 내용 |
|------|------|
| **긍정적 상호의존성** | 타인의 성과가 나에게, 내 성과가 타인에게 도움 |
| **개별적 책무성** | 집단 속에 자신을 감추지 못하도록 개인 역할·책임 부여 |
| **동등한 참여** | 독점·소외 없이 모두 적극 참여 |
| **동시다발적 상호작용** | 여러 명이 동시에 활동하는 학습 구조 |

### 2-2. 협동학습 유형 비교

| 유형 | 핵심 특징 | 보상 구조 |
|------|----------|----------|
| **STAD** | 4명 이질 팀; 팀 학습→개인 퀴즈; 향상 점수로 팀 기여 | 개인 향상+팀 보상 |
| **TGT** | 게임 형식; 같은 수준 학생끼리 타 팀과 경쟁 | 집단 내 협동, 집단 외 경쟁 |
| **LT** | 2~6명 이질 그룹; 상호작용 기술 학습 후 과제 수행 | 집단 응집력 강조 |
| **직소 I** | 전문가 활동 후 원 집단에 가르침; 과제 상호의존성↑ | 개인 보상 (집단 보상 없음) |
| **직소 II** | 직소 I + 집단 보상 추가 | 개인+집단 보상 |
| **직소 III** | 직소 II + 평가 유예기 (정리 시간) + 원 집단 평가 준비 | 개인+집단+복습 |
| **직소 IV** | 퀴즈로 정확성·이해도 점검; 선택적 재교수 | 개인+집단+점검 |
| **TAI** | 협동학습 + 개별 맞춤 학습 결합 | 개인 수준별 진도 |
| **GI** | 5~6명; 주제 선정→하위 주제→집단 발표 | 개방적 탐구 |
| **Co-op Co-op** | GI를 정교화; 학생이 과제 선택; 자기·동료 평가 참여 | 흥미 기반 소주제 |

> ⚠️ **기출 포인트**: TAI는 대부분의 협동학습과 달리 개인 수준(속도)에 따라 진행.

## 3. 또래교수

### 3-1. 또래교수 효과

| 대상 | 효과 |
|------|------|
| **튜티(학습자)** | 개별화 교육; 또래와 의사소통; 동기 유발 |
| **튜터(교수자)** | 정보 처리 이해 향상; 자아 개념 향상; 책임감·사회성 증가 |

### 3-2. 또래교수 유형

| 유형 | 특징 |
|------|------|
| **동급학생 또래교수** | 같은 연령; 역할 교대 가능 |
| **상급학생 또래교수** | 연장자 → 연하 교수 |
| **전학급 또래교수(CWPT)** | 모든 학생이 짝 지어 역할 교대; 통합학급에 효과적 |
| **전학급 또래교수팀(CSTT)** | 4명(1튜터+3튜티) 소그룹; 팀별 경쟁 |
| **PALS** | 읽기 기술 향상 특화; 교정 피드백·비계 활동 |

#### PALS 3단계 구성

| 단계 | 내용 |
|------|------|
| **파트너 읽기** | 고성취 학생이 먼저 읽고 저성취 학생이 따라 읽기 |
| **단락 요약** | 단락 읽기 후 핵심 내용 요약 |
| **예측 릴레이** | 다음 내용 예측 후 확인하며 읽기 |

> **교사 역할**: 긍정적 피드백·총감독 수행; 교수 역할이 단순히 또래에게 전가되지 않도록 주의
