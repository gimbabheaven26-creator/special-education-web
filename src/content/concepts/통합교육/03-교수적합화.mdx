## 1. 교수 적합화의 두 가지 기제

> ⚠️ **기출 포인트**: 수정과 조절의 차이 — 기대 수준 변화 여부.

| 항목 | **수정(Modification)** | **조절(Accommodation)** |
|------|----------------------|------------------------|
| **정의** | 기대 수준을 변화시킴 | 목표 도달을 돕는 전략 제공; 내용·난이도 변화 없음 |
| **가정** | 학생이 표준 학년 목표 도달 불가 | 어렵지만 도달 가능 |
| **적용 대상** | 중등도·중도 장애 학생 | 경도 지체, 학습 부진 아동 |
| **부담** | 일반학급 교사에게 다소 부담 | 상대적으로 적용 용이 |

---

## 2. 교수 적합화 절차

1. IEP 장단기 교수 목표 검토
2. 일반학급 참여를 위한 교과 선택
3. 일반학급 환경 정보 수집
4. 일반 교과 수업에서 학생의 수행·행동 평가
5. 한 학기 개별화된 단원별 학습 목표 결정
6. **교수적 수정 유형 결정** (환경·집단화·방법·내용·평가)
7. 수정 적용 및 참여의 양·질 평가

## 3. 교육과정 내용 수정 4유형

> ⚠️ **기출 포인트**: 4가지 유형의 정의와 특징 구별.

| 유형 | 정의 | 핵심 특징 |
|------|------|----------|
| **동일 수준 교육과정** | 차별화 필요 없는 학생; IEP 목표 수정 없이 일반 교육과정 그대로 | 별도 조정 없음 |
| **중다수준 교육과정** | 같은 내용; 도달 수준·개념을 달리하여 차별화 | 2개 이상의 수준별 목표; 교육 목표 위계에 기초 |
| **중복(중첩) 교육과정** | 같은 활동에 참여; **다른 교과 영역**에서 다른 목표 추구 | 학생들은 동일 활동; 교과 목표는 상이 |
| **대체 교육과정** | 대다수 학생과 완전히 독립된 교육과정 | 기능성·일상생활 초점; IEP와 일반 교육과정 직접 연관 없음 |

**중다수준 교육과정 원칙**: 수준 무관하게 모든 학생이 **핵심 개념**을 포함한 학습 목표 설정; 표현 방법 학습자가 선택

## 4. 교수환경·집단화 수정

### 4-1. 물리적 환경

- 조명·소음·시청각 자극 정도·가구 배열·자료 접근성 수정
- 자리 배치: 소음·장애물에서 멀리, 교사·활동 가까이

### 4-2. 교수적 집단화 종류

| 유형 | 설명 |
|------|------|
| 대집단 | 전체 학급 동일 내용 |
| 교사 주도 소집단 | 5~6명 소집단 교사 교수 |
| 협동학급 집단 | 2~6명 학생 함께 학습 |
| 학생 주도 소집단 | 또래 파트너·상급 학생 교사 주도 |
| 1:1 교수 | 직접 교수·지도 |
| 자습 | 혼자 학습 |

## 5. 평가방법의 수정

### 5-1. 평가 유형

| 유형 | 정의 |
|------|------|
| **표준평가** | 일반 학생과 동일 조건 |
| **평가조정(시험조정)** | 평가 본질 유지; 자료·절차만 변경 (제시 형식·환경·답안·시간 조정) |
| **대안평가** | 기본 교육과정 적용; 표준 평가 수정 필요한 학생 |

> ⚠️ **기출 포인트**: 평가조정은 **평가의 본질(목적)을 훼손하지 않는 범위** 내에서만 실시.

### 5-2. 루브릭(Rubric)

**정의**: 수행 수준을 기술적으로 진술한 평가 척도

**루브릭의 3가지 구성 요소**:
1. 과제·학습 활동 관련 **중요한 영역/특성**
2. 각 영역을 평정하는 **수치/척도**
3. 각 척도에 대한 **구체적인 지표**

**장점**: 학습자 도달 상태·발달 가능성·방향 제공; 자기 조절 학습 촉진; 교사·학생 모두 기준으로 분석 가능

## 6. 차별화 교수

학습자의 특성을 3가지로 분석 → 각 특성에 맞는 학습 경험 제공

| 특성 | 내용 |
|------|------|
| **준비도** | 현재 학습 수준·선수 지식 |
| **관심사 및 흥미** | 학습자가 흥미를 갖는 주제 |
| **학습 프로파일** | 선호하는 학습 방식·속도 |

**사회적 환경**: 소속감·평등감·존중감·협동심·상호의존감·참여 보상을 느낄 수 있는 교실 환경 조성
