## 1. 삽입교수(Embedded Instruction) 정의와 특징

**정의**: 일과 및 활동 내에서 발생하는 자연스러운 기회를 이용하여 장애 학생의 IEP 목표를 교수하는 방법

> ⚠️ **기출 포인트**: 삽입교수는 **별도 분리된 시간·장소**가 아닌 자연적 일과 안에서 이루어진다. 교수 기회가 일과에 '삽입'된다는 것이 핵심.

### 1-1. 삽입교수의 주요 특징

| 특징 | 내용 |
|------|------|
| **자연적 맥락** | 일반학급의 일상적 활동·일과 속에서 교수 실시 |
| **분산 연습** | 하루 중 여러 시간에 걸쳐 목표 기술을 반복 연습 |
| **IEP 목표 연계** | 개별 학생의 IEP 목표를 일반교육 활동에 연결 |
| **자연적 강화** | 활동 자체에서 발생하는 자연적 결과를 강화로 활용 |
| **최소 방해** | 다른 학생의 수업 흐름을 최대한 방해하지 않음 |

### 1-2. 삽입교수가 필요한 이유

- 전통적 개별 교수(pull-out)는 일반학급 참여 시간을 감소시킴
- 분리된 환경에서 습득한 기술은 자연적 맥락으로의 일반화가 어려움
- 통합교육 환경에서 IEP 목표와 일반교육 참여를 동시에 달성

## 2. 삽입교수 절차 4단계

> ⚠️ **기출 포인트**: 4단계 순서와 각 단계의 핵심 활동 구분.

| 단계 | 명칭 | 주요 활동 |
|------|------|----------|
| **1단계** | **목표 기술 선정** | IEP에서 삽입교수로 다룰 목표 기술 확인; 기능성·일반화 가능성 고려 |
| **2단계** | **교수 기회 확인** | 일과 및 활동 분석; 목표 기술을 자연스럽게 연습할 수 있는 시간·상황 파악 |
| **3단계** | **교수 계획 수립** | 삽입 방법(촉구 유형·오류 수정)·교수 횟수·담당자(일반교사/특수교사/보조원) 결정 |
| **4단계** | **자료 수집 및 평가** | 교수 기회별 반응 기록; 목표 달성 여부 모니터링; 필요 시 수정 |

### 2-1. 교수 기회 확인 예시

| 일과 | 삽입 가능한 IEP 목표 |
|------|---------------------|
| 아침 인사 | 이름 대답하기, 눈 맞춤, 인사말 표현 |
| 소집단 활동 | 지시 따르기, 물건 요청하기 |
| 전이 시간 | 목적지 찾아가기, 소지품 챙기기 |
| 점심 식사 | 식사 도구 사용, 선택하기, 사회적 의사소통 |
| 정리 시간 | 분류하기, 지시 이행, 물건 위치 기억 |

## 3. 삽입교수 vs 전통적 개별 교수 비교

| 항목 | **삽입교수** | **전통적 개별 교수(Pull-out)** |
|------|------------|-------------------------------|
| **교수 장소** | 일반학급 내 자연적 맥락 | 별도 분리된 공간(특수학급·치료실) |
| **교수 시간** | 일과 전반에 분산 | 지정된 시간에 집중 |
| **교수자** | 일반교사·특수교사·보조원 협력 | 특수교사 또는 치료사 단독 |
| **자극 조건** | 자연적 환경 자극 | 통제된 인공적 자극 |
| **일반화** | 자연적 맥락으로 일반화 용이 | 별도 일반화 훈련 필요 |
| **또래 관계** | 또래와 함께 유지 | 또래와 분리 |
| **강화** | 자연적 강화 활용 | 인위적·외재적 강화 의존 |

> ⚠️ **기출 포인트**: 삽입교수는 일반화(generalization) 문제를 구조적으로 해결하는 접근이다. 분리된 환경에서 습득한 기술이 자연 환경으로 전이되지 않는 '훈련-전이 불일치' 문제를 예방한다.

## 4. 중복 교육과정(Overlapping Curriculum)

**정의**: 장애 학생이 일반학생과 **동일한 활동**에 참여하되, **다른 교과 영역의 다른 목표**를 학습하는 교육과정 수정 방식

### 4-1. 중복 교육과정의 핵심 구조

| 구성 요소 | 내용 |
|----------|------|
| **동일한 활동** | 일반학생과 장애학생이 같은 수업·과제에 참여 |
| **다른 목표** | 장애학생은 해당 교과가 아닌 별도 영역(IEP 목표)을 학습 |
| **다른 교과 영역** | 예: 과학 수업에서 장애학생은 의사소통·사회적 기술 목표 |

### 4-2. 중복 교육과정 vs 중다수준 교육과정 비교

| 항목 | **중복(중첩) 교육과정** | **중다수준 교육과정** |
|------|------------------------|---------------------|
| **활동** | 동일 | 동일 |
| **교과 목표** | 다른 교과 영역의 목표 | 같은 교과의 다른 수준 목표 |
| **목표 난이도** | 별도 기능적 목표 | 위계적으로 낮은 수준의 동일 교과 목표 |
| **적용 대상** | 중도·중복 장애학생 | 경도~중등도 장애학생 |

**예시**: 미술 시간에 일반학생은 '색 혼합 원리 이해'를 학습하는 동안, 장애학생은 '색깔 이름 말하기(의사소통 목표)' 또는 '가위 바르게 사용하기(소근육 운동 목표)'를 학습

## 5. 부분 참여(Partial Participation) 원칙

**정의**: 장애의 중증도와 관계없이 모든 학생이 연령에 적합한 활동에 어떤 형태로든 참여할 수 있다는 원칙

### 5-1. 부분 참여의 근거

- 완전한 독립 수행이 불가능해도 활동 참여 자체가 교육적 가치를 가짐
- 수동적 관찰도 사회적 참여의 한 형태
- 보조도구·지원을 활용하여 참여의 양과 질을 높임

### 5-2. 부분 참여와 삽입교수·중복 교육과정의 연계

| 개념 | 역할 |
|------|------|
| **부분 참여** | 활동 참여의 당위성·철학적 근거 제공 |
| **중복 교육과정** | 참여하면서 달성할 목표를 달리 설정하는 방법 |
| **삽입교수** | 참여 중 IEP 목표를 구체적으로 교수하는 절차 |

## 6. 기능적 일과 교수(Functional Routine-based Instruction)

**정의**: 일상적·반복적으로 발생하는 기능적 활동(일과)을 교수 맥락으로 활용하는 접근

### 6-1. 기능적 일과 교수의 적용

| 일과 유형 | 교수 목표 예시 |
|----------|--------------|
| **등교 일과** | 가방 정리, 출석 확인, 자리 찾기 |
| **식사 일과** | 식판 선택, 식사 도구 사용, 정리하기 |
| **화장실 일과** | 독립적 배변, 손 씻기, 복귀하기 |
| **귀가 일과** | 알림장 확인, 소지품 챙기기, 인사하기 |

### 6-2. 기능적 일과 교수의 장점

- 자연적 선행자극과 결과 활용 → 일반화 촉진
- 매일 반복되는 일과이므로 충분한 연습 기회 보장
- 삽입교수 기회가 자연스럽게 구조화됨
- 가정·지역사회로의 전이 가능성 높음

> **삽입교수-기능적 일과 연계**: 기능적 일과는 삽입교수의 '교수 기회 확인' 단계에서 핵심 자원이 됨. 일과 분석(task analysis)을 통해 IEP 목표를 체계적으로 삽입할 지점을 파악한다.
