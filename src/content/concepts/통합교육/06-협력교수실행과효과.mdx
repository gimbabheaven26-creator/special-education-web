## 1. 협력교수 6모델 선택 기준

**협력교수 모델 추가**: 기존 5모델(교수-지원·대안·평행·스테이션·상호)에 더해 실무에서는 **6모델** 체계로 분류하기도 함 (대안교수를 '보충·심화 교수'로 세분하거나, One Teach-One Observe를 별도 모델로 포함)

> ⚠️ **기출 포인트**: 어떤 상황에서 어떤 모델을 선택하는지 판단 기준을 묻는 문항 출제. 모델 자체 암기보다 선택 기준 이해가 핵심.

### 1-1. 모델 선택의 3가지 핵심 기준

| 기준 | 고려 사항 | 적합한 모델 |
|------|----------|-----------|
| **학생 수 및 구성** | 전체 수업 vs 소집단; 이질적 집단 구성 정도 | 전체: 교수-지원 / 소집단: 대안·스테이션 |
| **IEP 목표 특성** | 개별 집중 지원 필요 여부; 일반교육과정 접근 수준 | 개별 집중: 대안 / 통합: 평행·상호 |
| **교과 내용 특성** | 계열성(순차 학습)인가 병렬성(독립 영역)인가 | 계열: 교수-지원·상호 / 병렬: 스테이션 |

### 1-2. 상황별 모델 선택 가이드

| 상황 | 권장 모델 | 이유 |
|------|----------|------|
| 새 단원 도입; 핵심 개념 전체 전달 | 교수-지원 | 일관된 내용 전달; 지원 교사가 개별 확인 |
| 특정 학생 집중 지원; 결손 보충 | 대안교수 | 소집단 집중 교수 가능 |
| 교사 2인의 전문성이 대등한 경우 | 상호교수(팀티칭) | 동등한 책임·전문성 공유 |
| 독립적 연습·실습 구간 | 스테이션 교수 | 능동적 참여; 교사 각 스테이션 전담 |
| 같은 내용을 작은 집단으로 반복 | 평행교수 | 소집단 비율 증가; 상호작용 강화 |

## 2. 공동 계획의 중요성과 장벽

**공동 계획**: 협력교수 성패를 결정하는 가장 중요한 구조적 요소

### 2-1. 공동 계획이 필요한 이유

- 두 교사가 수업에서 각자의 역할과 담당 내용을 미리 합의
- 장애학생의 IEP 목표를 일반교육 수업 안에 반영하는 창구
- 교수 전달 방식의 차이로 인한 학생 혼란 예방
- 상호교수(팀티칭) 실행을 위한 필수 전제 조건

### 2-2. 공동 계획의 주요 장벽

| 장벽 유형 | 내용 | 영향 |
|----------|------|------|
| **시간 부족** | 교사 시간표에 공동 계획 시간이 확보되지 않음 | 비공식·즉흥 계획 → 역할 혼선 |
| **역할 갈등** | 일반교사-특수교사 간 교수 리더십 경쟁 | 한 교사가 보조자로 전락 |
| **전문성 차이** | 교과 내용 전문성(일반교사) vs 특수교육 전문성(특수교사) 불균형 | 의사소통 단절 |
| **철학적 불일치** | 통합교육 가치에 대한 관점 차이 | 협력 의지 저하 |
| **행정적 지원 부재** | 협력 시간 미확보; 학교 차원 지원 없음 | 협력 지속 불가 |

> ⚠️ **기출 포인트**: 공동 계획 시간 미확보는 협력교수 실패의 가장 빈번한 구조적 원인으로 제시된다.

## 3. 파라교사(교육보조원) 역할: 독립성 지원 vs 의존성 유발

**파라교사(Paraeducator)**: 특수교사·일반교사를 보조하여 장애학생 지원을 담당하는 교육보조 인력

### 3-1. 파라교사의 적절한 역할

| 적절한 역할 | 내용 |
|-----------|------|
| **근접 지원** | 필요한 순간에 적시 지원; 전반적 밀착 동행 최소화 |
| **독립성 증진** | 학생이 스스로 시도한 후 지원; 즉각 개입 자제 |
| **간헐적 지원** | 점진적으로 지원 수준 감소; 자립도 증가 목표 |
| **교사 지시 이행** | 특수교사·일반교사의 교수 계획 내에서 역할 수행 |

### 3-2. 과도한 파라교사 지원의 부작용 (의존성 유발)

> ⚠️ **기출 포인트**: 파라교사의 밀착 지원이 오히려 장애학생의 독립성과 또래 관계를 저해하는 역설적 결과를 낳는다.

| 부작용 | 내용 |
|-------|------|
| **분리 효과** | 파라교사가 장애학생을 또래로부터 물리적·사회적으로 격리 |
| **의존성 형성** | 모든 상황에서 파라교사 도움 기대 → 독립적 문제 해결 능력 저하 |
| **교사 책임 회피** | 일반교사가 장애학생 지도를 파라교사에게 전가 |
| **낙인** | 전용 보조원이 배정됨으로써 장애가 시각적으로 표시됨 |

### 3-3. 파라교사 역할의 올바른 구조화

- 파라교사는 학생이 아닌 **활동·환경**을 지원한다는 원칙 유지
- 지원 결정권은 특수교사에게; 파라교사는 수행자 역할
- 점진적 소거(fading) 계획 수립: 지원 수준을 단계적으로 감소

## 4. 협력교수 효과성 요인

**효과성 정의**: 협력교수가 장애학생과 일반학생 모두의 학업적·사회적 성과를 향상시키는 정도

### 4-1. 핵심 효과성 요인

| 요인 | 내용 |
|------|------|
| **상호 존중** | 두 교사가 서로의 전문성을 동등하게 인정 |
| **공동 책임** | 학급 내 모든 학생에 대한 책임을 양 교사가 공유 |
| **전문성 공유** | 교과 전문성 + 특수교육 전문성의 상호 학습 |
| **명확한 역할 합의** | 사전에 역할·책임 범위를 명시적으로 협의 |
| **지속적 의사소통** | 정기적 협의 시간; 교수 중 실시간 신호 체계 |
| **행정적 지원** | 관리자의 협력 시간 확보 및 자원 지원 |

### 4-2. 교사 태도 요인

| 태도 | 협력교수에 미치는 영향 |
|------|---------------------|
| 통합교육에 대한 긍정적 신념 | 협력 동기 유지; 어려움 극복 의지 |
| 특수교육 지식에 대한 개방성 | IEP 목표를 수업에 반영하는 적극성 |
| 교실을 '공동 소유'로 인식 | 역할 갈등 감소; 협력적 의사결정 |

## 5. 협력교수 성공 조건 체크리스트

> ⚠️ **기출 포인트**: 협력교수의 성공 조건을 구조적 조건(환경)과 관계적 조건(태도)으로 구분하여 제시하는 문항 출제.

### 구조적 조건 (학교·행정 차원)

- [ ] 정기적 공동 계획 시간이 시간표에 확보되어 있다
- [ ] 관리자가 협력교수의 가치를 인정하고 지원한다
- [ ] 학급 규모가 협력교수 실행에 적절하다
- [ ] 필요한 교수 자료·공간이 제공된다
- [ ] 전문성 개발(연수) 기회가 정기적으로 제공된다

### 관계적 조건 (교사 차원)

- [ ] 두 교사가 통합교육의 가치에 동의한다
- [ ] 서로의 전문성을 인정하고 존중한다
- [ ] 역할과 책임이 명확하게 합의되어 있다
- [ ] 의사소통이 개방적이고 솔직하다
- [ ] 장애학생에 대한 공동 책임 의식이 있다

### 교수 실행 조건 (수업 차원)

- [ ] 모델 선택이 수업 목표·학생 특성에 맞게 이루어진다
- [ ] 역할이 수업마다 적절히 교대된다
- [ ] IEP 목표가 수업 계획에 반영된다
- [ ] 두 교사 모두 학급 전체 학생과 상호작용한다

## 6. 협력교수 도전 과제와 해결 전략

| 도전 과제 | 해결 전략 |
|----------|----------|
| **공동 계획 시간 부족** | 짧은 주간 미팅 정례화; 이메일·메모 등 비동기 협의 활용; 관리자에게 시간 확보 요청 |
| **역할 불균등** (특수교사가 보조자로 인식) | 교대 역할 계획 명문화; 매 수업 역할 교대 원칙 수립 |
| **교수 방식 불일치** | 사전 시범·연습; 신호 체계 합의; 수업 후 피드백 공유 |
| **의사소통 단절** | 정기 협의 일정 확보; 불일치 발생 시 즉각 논의 |
| **파라교사 역할 혼선** | 파라교사 역할 범위 문서화; 특수교사가 파라교사 코칭 |
| **일반교사의 낮은 통합교육 신념** | 성공 사례 공유; 점진적 협력 확대; 연수 참여 |

> **종합**: 협력교수는 단순히 두 교사가 같은 공간에 있는 것이 아니다. 공동 계획 → 공동 실행 → 공동 평가의 순환 구조가 확립될 때 진정한 협력교수가 가능하다.
