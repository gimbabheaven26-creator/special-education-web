## 교육공학의 정의 및 5개 영역

교육공학은 **인간 학습과 관련된 모든 문제들을 분석하고 해결책을 고안·실행·평가·관리하기 위한 복합적이고 통합된 과정**이다.

> ⚠️ **기출 포인트**: 교육공학의 정의에서 "복합적이고 통합된 과정"이라는 표현이 핵심이다. 단순 도구나 매체가 아닌 과정(process) 개념으로 이해해야 한다.

### 5가지 영역

| 영역 | 내용 |
|------|------|
| **설계** | 학습에 대한 조건을 구체화하는 것 |
| **개발** | 설계에서 구체화된 내용을 물리적으로 완성하는 것 |
| **활용** | 학습을 위해 과정과 자원을 활용하는 것 |
| **관리** | 계획, 조직, 조정, 감독 등을 통해 교육공학을 통제하는 것 |
| **평가** | 교수와 학습의 적절성을 결정하는 것 |

> ⚠️ **기출 포인트**: 특수교육공학의 영역은 교육공학의 5가지 영역(설계·개발·활용·관리·평가)과 동일하다. 두 개념의 영역이 같다는 점이 자주 출제된다.

## 교수·학습 이론

### 행동주의 (객관주의)

- 학습 = 경험의 결과로 나타나는 **관찰 가능한 행동의 변화**
- 자극과 반응의 관계를 중심으로 학습을 설명
- 학습 활동의 **선행 조건이나 결과를 조작**함으로써 학습장애 학생의 학업 성취 향상 가능
- 학습 과제를 **세분화**하고, 학생 활동에 대한 **피드백 제공**
- 후속 자극의 변화가 학습 행동에 미치는 영향을 체계적으로 분석
- 매체 개발 방향: **훈련·연습형** (반복 연습 중심)

> ⚠️ **기출 포인트**: 행동주의 관점의 교육공학 적용 = "훈련·연습형 매체 개발"이 핵심 키워드다.

### 인지주의 (구성주의)

- 실제 문제 해결 상황을 비디오 등 매체를 활용하여 제공
- 교사와 학생 간 **활발한 상호작용**에 초점
- **근접 발달 영역(ZPD)** 내에서 필요한 도움 제공 → 과제 수행이 능숙해짐에 따라 도움 철회(비계 설정)
- 실생활 관련 과제와 경험을 활용하여 정보를 **능동적으로 구성**
- 학습 전략 개발·응용 방법 지도

#### 앵커드 교수 (상황 학습 이론)

구성주의 관점에서 매체를 이용하여 **실제 상황에서 일어날 수 있는 문제 해결 환경을 제공**하여 전문가들의 문제 해결 방법을 습득하도록 하는 교수 방법이다.

| 구분 | 내용 |
|------|------|
| **설계 원리** | 인지적 전략의 시연과 관찰 기회 제공, 협동·반성·명료화 기회 제공, 교사는 지식 전수자가 아닌 **학습 촉진자** 역할 |
| **장점** | 맥락적 학습 기회 제공, 학습 장려, 학습 활동에 활발한 참여, 지식의 사회적 구성, 문제 해결 능력 개발, 지식 전이 강화, 학생들의 지식 소유권 촉진 |

> ⚠️ **기출 포인트**: 앵커드 교수는 "구성주의 + 실제 상황 + 매체(비디오 등)"의 조합으로 출제된다. 교사의 역할이 "지식 전수자 → 학습 촉진자"로 전환된다는 점도 빈출 포인트다.

## 특수교육공학의 정의, 영역, 장점 및 특징

### 정의

특수교육공학은 **특수교육 대상자의 교육을 효율적으로 실시하기 위해 각 학교의 장에 의해 제공되는 장애인용 각종 교구, 학습 보조기, 보조공학기기, 정보 접근 기회**를 의미한다.

> ⚠️ **기출 포인트**: "각 학교의 장에 의해 제공된다"는 주체와, "교구·학습 보조기·보조공학기기·정보 접근 기회"라는 구성 요소가 함께 출제된다.

### 장점

| 장점 |
|------|
| 기본적인 학습 능력을 향상시키기 위한 훈련과 연습 가능 |
| 추상적인 수학·과학 개념에 대한 이해 향상 |
| 자료 처리 및 다루기에 용이 |
| 학습자들의 학습 활동 관리와 관련 기록 저장이 용이 |
| 장애 학생들을 위한 **접근성과 의사교환** 지원 |

### ABC 모델 이점

특수교육 대상자들에게 제공되는 기술 지원의 세 가지 이점이다.

| 구분 | 내용 |
|------|------|
| **A (Ability)** | 능력의 신장 |
| **B (Bypass)** | 매체의 대체 |
| **C (Compensation)** | 장애의 보상 |

> ⚠️ **기출 포인트**: ABC 모델은 영어 단어와 한국어 의미를 묶어서 암기해야 한다. 세 가지 이점을 순서·내용 모두 정확히 서술하는 문제가 출제된다.

### 특수교육공학의 특징

| 특징 | 내용 |
|------|------|
| **개별화 촉진** | 특수교육 대상자들에게 개별적인 교수를 가능하게 하여 학습 관련 수행 능력 향상 |
| **동기 유발** | 학습자들의 흥미를 유발하고 자신감과 친밀감을 가지고 수업에 임할 수 있도록 지원 |
| **실제 상황과 유사한 간접 경험 제공** | 지식의 전이와 일반화를 위해 실제 상황에서의 반복·연습이 필요한 정신지체·학습장애 학생에게 긍정적 효과 |
| **긍정적 태도와 자신감 함양** | 장애를 보완하거나 이전에 하지 못했던 것을 할 수 있도록 새로운 능력 부여 → 자아상과 가치 향상 |

> ⚠️ **기출 포인트**: 특수교육공학의 특징 중 주의할 점 — **과도한 화려함이나 너무 많은 다양성은 오히려 해로울 수 있다.** 적절한 수준의 기술 활용이 중요하다.
