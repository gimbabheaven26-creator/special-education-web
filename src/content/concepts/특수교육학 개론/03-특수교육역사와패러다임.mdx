## 패러다임 변화란?

흑백 TV가 컬러 TV로 바뀌면 채널이 같아도 보이는 것이 달라진다. 특수교육도 같은 "아이"를 보면서 시대마다 전혀 다른 방식으로 이해해왔다. 의료적 관점(고쳐야 할 결함), 재활 관점(기능 회복), 교육적 관점(성장 가능성), 사회적 관점(환경이 장벽)으로의 전환은 교육 방식 자체를 바꿔왔다.

---

## 1. 장애 모델 비교

| 모델 | 핵심 관점 | 교육적 함의 |
|------|-----------|-------------|
| **의료적 모델** | 장애 = 결함·질병 → 치료 대상 | 의사·전문가가 주도, 분리 환경 |
| **재활 모델** | 기능 손상 회복에 초점 | 치료 중심, 정상화 목표 |
| **교육적 모델** | 학습 가능성 강조 | 적절한 교육 환경 제공 |
| **사회적 모델** | 장애 = 사회적 장벽 문제 | 환경 수정, 접근성, 보편적 설계 |
| **문화적 모델** | 다양성의 한 형태 | 차이 존중, 당사자 주도 |

> ⚠️ **기출 포인트**: 사회적 모델과 의료적 모델의 차이는 단골 출제다. "사회적 모델"은 장애를 **개인의 문제**가 아닌 **사회·환경의 문제**로 본다.

## 2. 정상화 원리 (Normalization)

닐레 에릭 뱅크-미켈센(Niels E. Bank-Mikkelsen)이 제창하고, 울펜스버거(Wolfensberger)가 체계화.

| 원칙 | 내용 |
|------|------|
| **핵심 개념** | 장애인도 비장애인과 동일한 생활 리듬·방식을 누릴 권리 |
| **실천 방향** | 시설 수용 반대, 지역사회 생활 지원 |
| **교육적 적용** | 일반학교·학급에서 비장애 또래와 함께 교육 |
| **발전 개념** | 사회적 역할 가치화(SRV): 사회 내 긍정적 역할 수행 강조 |

## 3. 미국 특수교육법 역사

| 연도 | 법령 | 핵심 내용 |
|------|------|-----------|
| **1975** | EHA (전장애아교육법) | 모든 장애 아동 무상 공교육 보장, IEP 의무화 |
| **1990** | IDEA (장애인교육법) | EHA 개정·강화, 전환 서비스 추가, 자폐성장애·TBI 추가 |
| **1997** | IDEA 재승인 | 일반교육과정 접근 강조, 행동중재계획(BIP) 추가 |
| **2004** | IDEIA | 높은 자격 교사 기준, SLD 판별에 RTI 허용 |

> ⚠️ **기출 포인트**: IDEA는 6가지 핵심 원칙을 포함한다. FAPE(무상 적절 공교육), LRE(최소제한환경), IEP, 비차별 평가, 보호자 참여, 절차적 보호장치.

### IDEA 6대 원칙

| 원칙 | 영문 | 내용 |
|------|------|------|
| **무상 적절 공교육** | FAPE | 비용 없이 학생 요구에 맞는 교육 제공 |
| **최소제한환경** | LRE | 가능한 한 일반교육 환경에서 교육 |
| **비차별적 평가** | Nondiscriminatory Evaluation | 편향 없는 다면적 평가 |
| **개별화교육계획** | IEP | 개별 맞춤 교육 계획 문서 |
| **보호자 참여** | Parent Participation | 의사결정에 보호자 참여 보장 |
| **절차적 보호장치** | Procedural Safeguards | 법적 권리 보호, 이의제기 절차 |

## 4. 한국 특수교육법 역사

| 연도 | 사건 |
|------|------|
| **1977** | 「특수교육진흥법」 제정 — 최초 특수교육 법제화 |
| **1994** | 특수교육진흥법 전면 개정 — 통합교육 조항 신설 |
| **2007** | 「장애인 등에 대한 특수교육법」 제정 (특수교육진흥법 폐지) |
| **2008** | 특수교육법 시행 |
| **2013~** | 수차례 개정 — 발달지체 규정, 보조공학 강화 등 |

> ⚠️ **기출 포인트**: 1977년 특수교육진흥법이 최초 법제화, 2007년 현행 특수교육법 제정. 두 연도를 구별해야 한다.

## 5. 통합교육의 발전 단계

| 단계 | 특징 |
|------|------|
| **분리교육** | 특수학교·특수시설에서 완전 분리 |
| **주류화 (Mainstreaming)** | 일부 시간 일반학급 참여 (학생이 준비된 경우만) |
| **통합교육 (Integration)** | 일반학교에 물리적 통합, 특수학급 운영 |
| **완전통합 (Inclusion)** | 모든 학생이 일반학급에서 필요한 지원을 받으며 교육 |
| **완전한 통합 (Full Inclusion)** | 특수학급 없이 일반학급에서 모든 교육 제공 |

> ⚠️ **기출 포인트**: "주류화"는 학생이 일반교육에 **적응해야** 하는 반면, "완전통합"은 **학교와 환경이 학생에게 맞춰지는** 방향이다. 이 방향의 차이가 핵심이다.

## 6. 자기결정(Self-Determination)

> 자기결정: 외부의 영향이나 간섭 없이 자신의 삶에 관한 의사결정을 스스로 하는 것

| 구성 요소 | 내용 |
|-----------|------|
| **선택하기** | 선호·선택 표현 |
| **의사결정** | 정보 수집→분석→선택 |
| **문제 해결** | 장애물 파악·해결책 생성 |
| **목표 설정** | 단기·장기 목표 수립 및 조절 |
| **자기 옹호** | 자신의 권리와 필요 표현 |
| **자기효능감** | "나는 할 수 있다" 신념 |

특수교육에서 자기결정은 **전환교육**의 핵심 성과 목표이며, IEP 과정에서도 학생의 자기결정 역량 강화가 중요하다.
