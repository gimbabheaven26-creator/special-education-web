## 1. 학습장애의 정의

### 특수교육법상 정의
개인의 **내적 요인**으로 인해 듣기·말하기·주의집중·지각·기억·문제 해결 등의 학습 기능이나
읽기·쓰기·수학 등 학업 성취 영역에서 **현저한 어려움**이 있는 사람.

> ⚠️ **기출 포인트**: "내적 요인"이 핵심 — 외적(환경, 교육 기회 부족, 빈곤) 요인이 일차 원인이면 학습장애에서 제외.

### 정의의 5가지 주요 요소

| 요소 | 내용 |
|------|------|
| **내적 원인** | IDEIA: 심리처리 과정 장애 / NJCLD: 중추신경계 기능 장애 / 특수교육법: 개인 내적 요소(구체적 원인 미명시) |
| **이질성** | 어려움이 나타나는 학습 영역이 개인마다 다양 |
| **배제 요인** | 다른 장애·환경적 불리함이 일차 원인이면 제외 — 단, 중복장애 가능성은 인정 |
| **일생을 통해 지속** | 학령기에만 국한되지 않고 성인기까지 지속 |
| **진단 준거** | 불일치 준거 + 중재 반응 준거 포함 |

### 비언어성 학습장애(NLD)

언어 능력에는 강점, **비언어적 능력**(공간 지각, 운동, 사회성 기술)에서 현저한 어려움.

- **손상 부위**: 뇌 우반구 체계 결함
- **주요 특성**: 퍼즐·비언어 과제 수행 어려움, 추상 개념 이해 곤란, 사회적 맥락 파악 미흡, 신체 증상(두통·복통) + 우울·불안

## 2. 학습장애 특성

| 특성 | 내용 |
|------|------|
| **선택적 주의집중 부족** | 불필요한 자극 무시하고 중요 정보에 집중하는 데 어려움 |
| **초인지 능력 부족** | 과제에 필요한 전략 선택 및 자기 점검 곤란 |
| **평균 또는 그 이상의 지능** | 지능 자체 문제가 아님 — 배제 요인과 연결 |
| **지각·협응 기능 저하** | 시·공간 지각 문제, 글씨 엉망, 행 건너뜀 |
| **일반화 능력 부족** | 습득한 기술을 다른 상황에 적용하기 어려움 |

> ⚠️ **기출 포인트**: 장애 특징이 "몇 가지 나타났는가"보다 특정 현상이 **얼마나 오랫동안, 얼마나 심각하게** 나타나는지가 중요.

|------|-----------|-----------------|

| **접근** | 진단 중심 | 교육 예방·조기 발견 중심 |

| **교수 연계** | 미흡 | 진단-교수-평가 유기적 연결 |
| **조기 중재** | 어려움 | 강점 |

## 4. 불일치 모델(능력-성취 불일치)

**정의**: 기대되는 학업 성취 수준(지능 기반)과 실제 성취 수준 간에 **기준 이상의 차이**가 있을 때 학습장애로 봄.

> ⚠️ **'실패를 기다리는 모델'**: 학습 문제가 충분히 심화된 후에야 판별 가능 — RTI가 이 한계를 극복하기 위해 등장.

### 불일치 공식 4가지

| 공식 | 방법 | 장점 | 단점 |
|------|------|------|------|
| **학년 수준 편차** | EGL − AGL 차이(1~2학년 이상) | 계산 용이, 이해 쉬움 | 지능 미고려 → 지능↑ 과소판별, 지능↓ 과잉판별 / 평균 회귀 |
| **기대 학령 공식** | 생활연령 + 지능 + 재학연수 고려 | 지능 검사 오류 보완 | 기타 오류 잔존 |
| **표준 점수 비교** | IQ·성취를 표준점수(SD=15)로 변환 후 비교 | 측정적 단점 일부 해결 | 평균 회귀 잔존 → 지능↑ 과잉판별, 지능↓ 과소판별 |
| **회귀 공식** | 회귀 분석으로 기대 성취 산출 후 실제와 비교 | 정확성↑, 전체 IQ 범위 균등 판별 | 통계 복잡 → 현장 적용 어려움 |

> ⚠️ **기출 포인트**: 회귀 공식이 문제가 **가장 적은** 방법. 표준 점수 비교는 **평균 회귀** 문제가 잔존.

### 불일치 모델의 한계

- 지능이 학습 잠재력의 척도가 아님
- 학습장애/학습 부진 구분 어려움
- 조기 중재 불가 (충분한 불일치가 나타날 때까지 대기)
- 진단 결과가 교수 계획에 정보를 제공하지 못함

## 5. 중재 반응 모델(RTI)

**정의**: 학습 문제 학생을 조기 선별 → 중재 실시 → 중재 반응 정도에 따라 학습장애 적격성 결정.

**진단 기준**: 효과적인 중재에도 불구하고 또래 대비 반응이 현저히 낮을 때 학습장애 진단.

### RTI 장·단점

| 장점 | 단점 |
|------|------|
| 조기 선별·조기 중재 | 과학적으로 검증된 프로그램 부족 |
| 실패를 기다리는 일 감소 | 읽기 외 영역(쓰기·수학) 검증 연구 부족 |
| 학습장애/학습 부진 변별 가능 | 효과적 중재 기준과 반응 측정 합의 어려움 |
| 평가-교수-점검 유기적 연결 | 비언어적 학습장애 조기 판별 어려움 |
| 타당성·신뢰성 ↑(다단계 선별) | 심리 처리 과정 결함 정보 미제공 |

### RTI 유형 3가지

| 유형 | 특징 |
|------|------|
| **예언자-기준 모델** | 읽기 장애와 관련된 예언 요인에 중점 |
| **기능적 평가 모델** | 학업 수행을 환경 측면과 연결, 환경 요인 분석 후 중재 |
| **이중 불일치 모델** | 성취 수준 + 성장률 **모두** 또래보다 낮을 때 학습장애 진단 |

### 이중 불일치 모델

- **정의**: 학업 성취 수준 + 학업 성장률(발달 속도) **둘 다** 학급 동료보다 낮은 이중 불일치 → 학습장애
- **방법**: CBM(교육과정중심측정)으로 일반 교육과정에서 계속적 진전도 평가
- **장점**: 문제 확인 + 문제 증명 두 단계 적용 → 진단 타당성 ↑

### 3단계 중재 반응 모델

| 단계 | 대상 | 중재 | 평가 |
|------|------|------|------|
| **1단계** | 전체 학생 | 양질의 핵심 교수(보편적 지원) | CBM·표준화 학력 평가 |
| **2단계** | 1단계 미반응 학생(약 20~25%) | 소집단 보충 중재 | 진전도 모니터링 강화 |
| **3단계** | 2단계 미반응 학생(약 5%) | 개별화된 집중 중재 | 포괄적 평가 → 학습장애 진단 여부 결정 |

## 6. 인지처리과정 결함 접근

**정의**: 학습장애 진단 시 학업 성취도뿐 아니라 **인지 처리** 능력도 함께 평가해야 한다는 모델.

**진단 절차**:
1. 형식적·포괄적 평가 실시 (학업 성취 + 인지 처리)
2. 낮은 점수 확인 — **16백분위 이하**
3. 전체 IQ **70 이상** 확인
4. 다른 장애로 인한 결과가 아님을 확인

### 읽기 영역별 인지 처리 능력

| 읽기 영역 | 핵심 인지 처리 능력 |
|---------|---------------|
| **단어 인지** | 음운 인식, 빠른 자동 이름 대기, 음운 기억, 어휘 |
| **읽기 유창성** | 빠른 자동 이름 대기, 어휘, 표기 처리 |
| **읽기 이해** | 어휘, 문장 따라 말하기, 듣기 이해 |

### 장·단점

| 장점 | 단점 |
|------|------|
| 학습장애를 **적극적으로** 정의·측정하는 접근 | 이론적·실제적 근거 미충분 |
| 발달 연령·학년 수준 무관하게 판별 가능 | — |
| 특정 영역만 차이 나는 학생 가려낼 수 있음 | — |
| 중재 프로그램 기획에 유용 | — |
