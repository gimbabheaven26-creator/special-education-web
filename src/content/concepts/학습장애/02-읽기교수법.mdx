## 1. 읽기를 위한 선수 기술

### 프린트 인식
문자 언어가 어떻게 사용되는지 이해하는 능력.

| 하위 기술 | 내용 |
|---------|------|
| 프린트 기능 | 문어가 의미를 전달한다는 것을 이해 |
| 프린트 관례 | 글 읽는 방향(좌→우), 책 앞·뒤 식별, 단어 경계 인식 |
| 소리-단어 연결 | 교사가 읽는 단어를 손가락으로 가리키기 |

> ⚠️ **기출 포인트**: 프린트 인식은 향후 읽기 능력과 상관이 있으나 **예측력은 크지 않음** — 자모 지식과 대비 출제.

### 자모 지식
단어 인지·읽기 능력에 대한 **높은 예측력**을 지닌다.
- 자음자·모음자의 이름과 소리에 대한 지식
- 이름과 소리를 빠르고 정확하게 인출하는 능력

### 음운 인식
말소리를 변별하는 능력(말소리 민감도). **오직 청각에만 의존**하며 글자를 제공하지 않는다.

**발달 순서 (단위)**:
문장 내 단어 인식 → 끝소리 같은/다른 단어 인식 → 음절 인식 → 초성-각운 인식 → **음소 인식**

**발달 순서 (과제 유형)**:
변별 → 합성·분리·분절·수 세기 → **탈락과 대치**

| 음운 처리 개념 | 내용 |
|-------------|------|
| **음운 분절** | 단어를 구성하는 소리를 분석(단어 → 음절 → 음소) |
| **음성 혼성** | 분석된 소리를 하나로 결합(음소 → 음절 → 단어) |

> ⚠️ **기출 포인트**: 읽기 능력 **예측도가 높은** 과제: 음절 분리, 음절 탈락, 음절 대치, 음소 변별, 음소 대치. / 음소 인식 교수는 **1~2가지 유형**에 집중할 때 더 효과적.

## 2. 전통적 읽기 교수법 (다감각 중심)

시각·청각·촉각·운동 감각 모두 활용. **자극을 점차 줄여가는** 방식.

### Fernald 읽기 교수법 — 4단계

| 단계 | 특징 |
|------|------|
| **1단계** | 학생이 선택한 단어를 교사가 단어장에 크게 씀 → 학생은 **손가락으로 따라 쓰며 소리 내어 읽기** (4감각 관여) |
| **2단계** | 손가락 쓰기 활동 제외, 나머지 1단계와 동일 |
| **3단계** | 일상 읽기 자료에서 낯선 단어 보고 말하기 → 글쓰기 |
| **4단계** | 문맥·이전 학습 단어와의 유사성으로 낯선 단어 의미 **스스로 확인** |

- **특징**: 학생이 학습할 단어를 **직접 선택** (학습 동기 중시)

### Gillingham 읽기 교수법 — 3단계

| 단계 | 내용 |
|------|------|
| **1단계** | 문자와 음소의 대응 관계 습득 (다감각 방식) |
| **2단계** | 배운 문자로 단어 만들기 → 단어 카드 보관 |
| **3단계** | 단어 카드로 문장·이야기 작성 |

- **특징**: 음운 분석적 방법 / **단점**: 학습 동기 고려 없음

### Hegge-Kirk-Kirk 읽기 교수법 — 3단계
문자-음소 대응 관계를 **단순화**하여 **많은 연습 기회** 제공.

### 신경학적 각인 읽기 교수법
**함께 읽기 + 반복 읽기**의 결합. 교사 도움을 **점진적으로 소거**.

## 3. 단어 인지 능력 향상을 위한 교수법

### 단어 인지와 관련된 4가지 활동

| 활동 | 내용 |
|------|------|
| **형태 분석** | 문자의 시각적 특징·단서로 단어 인식 |
| **음소 분석** | 문자소-음소 대응 관계 분석 (초기 단계 주사용) |
| **음절 분석** | 음절에 해당하는 소리 분석 |
| **문맥 분석** | 앞뒤 문맥의 다른 단어·의미에 의존 → 학습장애 학생이 과다 의존하나 정확도 낮음 |

### 의미 중심 vs 해독 중심 프로그램

| 구분 | 의미 중심 | 해독 중심 |
|------|---------|---------|
| **접근** | 하향식 (whole→part) | 상향식 (part→whole) |
| **초점** | 의미 이해 | 낯선 단어 읽기 능력 |
| **대표** | 통언어적 접근, 일견 단어, LEA | 언어학적 접근, 음운 분석적 접근, 파닉스 |

**통언어적 접근(총체적 언어 교수)**:
- 읽기 능력이 '의미 있는' 읽기 활동 과정에서 **자연적으로 습득**된다고 봄
- 장점: 읽기·쓰기 통합, 흥미 유발 / 단점: 체계적 음운 분석 교수 미흡

**언어 경험적 접근(LEA)**:
- 학생의 **경험을 바탕**으로 읽기 지도
- 단계: 토의하기 → 받아쓰기 → 읽기 → 단어 학습 → 다른 자료 읽기

### 파닉스 교수 4가지 유형

| 유형 | 별칭 | 방법 |
|------|------|------|
| **합성 파닉스** | 명시적 파닉스 / 부분-전체 접근법 | 낱자별 소리를 따로 가르친 후 합쳐서 읽기 — **가장 효과적** |
| **분석 파닉스** | 암시적 파닉스 / 전체-부분 접근법 | 이미 아는 단어에서 낱자-소리 대응 관계를 귀납적으로 파악 |
| **유추 파닉스** | — | 알고 있는 단어·부분을 활용해 새 단어 읽기 |
| **임베디드 파닉스** | — | 글 읽는 맥락 안에 파닉스 교수를 삽입 (총체적 언어 프로그램의 일부) |

> ⚠️ **기출 포인트**: 합성 파닉스 = 명시적 파닉스 = 부분-전체. 분석 파닉스 = 암시적 = 전체-부분. 용어 혼동 주의.

## 4. 읽기 유창성 교수법

**읽기 유창성 = 정확도 + 속도 + 표현력**

- 단어 인지와 읽기 이해를 연결하는 **다리 역할**
- 유창성 부족 → 개별 단어 해독에 인지 자원 과다 사용 → 이해력 저하

### 효과적인 읽기 유창성 교수의 조건
- 단어의 **90% 이상** 정확하게 읽을 수 있는 글 선택
- 동일한 글을 **3번 이상** 소리 내어 반복 읽기
- **일주일에 3회 이상** 실시
- 오류 시 **즉시** 체계적 오류 교정

### 읽기 유창성 교수법 4가지

| 교수법 | 내용 | 특징 |
|------|------|------|
| **반복 읽기** | 90% 인식 가능한 50~200단어 자료를 유창성 기준 도달까지 반복 | 새 자료에도 유창성·이해 전이 효과 |
| **신경학적 각인 읽기** | 함께 읽기 + 반복 읽기 | 교사 도움 점진적 소거, 모델링 후 학생 주도 |
| **짝과 함께 반복 읽기** | 유창성 우수 또래와 짝지어 반복 읽기 | 각자 3분 읽기 → 오류 교정 → 각자 1분 '최대한 잘 읽기' → 점수 기록 |
| **끊어서 반복 읽기** | 의미 단위(구·절)로 끊어서 반복 읽기 | **표현력** 향상에 효과적 |

## 5. 어휘력 증진을 위한 교수법

### 어휘 지식의 분류 (질적 수준)

| 수준 | 내용 | 특징 |
|------|------|------|
| **결합 지식** (조금 이해) | 목표 어휘의 단순 정의를 아는 수준 | 단일 맥락에서만 의미 이해 |
| **이해 지식** (부분 이해) | 관련 어휘와 연결·범주화 가능 | 다양한 맥락에서 의미 이해 |
| **생성 지식** (충분 이해) | 여러 상황에 어휘를 적절히 적용 | 비슷한 어휘 간 구분, 다양한 범주 이해 |

### 어휘 교수법 개요

| 교수법 | 내용 |
|------|------|
| **직접 교수법** | 목표 어휘를 직접 가르침 — 효과적이나 모든 어휘 지도 불가 |
| **간접 교수법** | 다양한 맥락 노출로 자연 습득 — 다독, 우연 교수 |
| **키워드 기억 전략(핵심어법)** | 목표 어휘 + 알고 있는 키워드를 **시각적 이미지**로 연결 — 사회·과학 교과에 효과적 |
| **의미 지도** | 목표 어휘 중심으로 관련 어휘 열거 → 그래픽 조직자로 범주화 |
| **의미 특성 분석** | 개념(가로) × 특성(세로) 격자표로 +/- 분석 |

### 기억 전략

| 전략 | 내용 |
|------|------|
| **두문자법(약성어법)** | 기억할 정보의 첫 글자를 모아 새 단어 생성 |
| **어구 만들기(약문법)** | 첫 글자로 시작하는 단어를 조합해 어구 생성 |
| **핵심어법** | 친숙한 정보 + 새 단어를 시각 이미지로 연결 |
| **페그워드 전략** | 순서 정보를 소리 유사 단어와 연결 |

### 그래픽 조직자 유형

| 유형 | 용도 |
|------|------|
| **개념형** | 하나의 주요 개념 + 지원하는 사실·특성 |
| **계층형(위계형)** | 중요도에 따른 상·하위 배열 |
| **연속형(순서형)** | 시간 순서 또는 인과관계의 선형 제시 |
| **순환형** | 시작·끝 없는 지속적 연속 |
| **비교 대조형** | 벤 다이어그램 — 공통점·차이점 |
| **매트릭스형** | 주제 간 범주 분류 |

> ⚠️ **기출 포인트**: 그래픽 조직자는 **선행·수업·마무리 조직자** 3종류 출제. 선행 조직자는 수업 전 배경 지식 활성화.

## 6. 읽기 이해력 증진을 위한 교수법

**읽기 이해**: 자신의 선행 지식과 글 정보를 연결하여 의미를 형성하는 과정 → **초인지 기술** 요구.

### 읽기 이해 수준 3단계

| 수준 | 요구 능력 |
|------|---------|
| **단어 수준** | 음운 인식, 자모 지식, 낱자-소리 대응, 단어 인지 |
| **문장 수준** | 구·절 읽기 유창성, 어휘력, 문장 구조 이해 |
| **글 수준** | 글 구조 이해, 선행 지식, 읽기 이해 점검력 |

### 읽기 전·중·후 전략

| 시점 | 전략 | 내용 |
|------|------|------|
| **읽기 전** | 브레인스토밍 | 선행 지식 활성화 |
| **읽기 전** | 예측하기 | 제목·소제목·그림으로 내용 예측 → 능동적 독자 |
| **읽기 중** | 이야기 문법 | 인물·배경·사건·문제·결말 파악 |
| **읽기 중** | 중심 내용 파악 | 각 문단의 무엇/누구 → 중요 내용 → 10어절 이내 표현 |
| **읽기 후** | 요약하기 | 불필요 내용 제거 + 중요 내용 정리 |
| **읽기 후** | 질문 만들기·답하기 | 자기 이해 점검 |

### 다전략 교수 (전략 수 최대 4개로 제한)

| 교수법 | 전략 | 특징 |
|------|------|------|
| **상보적 교수(호혜적 교수)** | 예측하기 / 질문 만들기 / 요약하기 / 명료화하기 | 교사-학생 구조화된 대화로 이해력 향상 |
| **협력 전략적 읽기(CSR)** | 전(브레인스토밍·예측) / 중(점검·중심내용) / 후(질문·요약) | 상보적 교수 + 직접 교수 + 협력 활동 결합 |
| **K-W-L** | K(이미 아는 것) → W(배우고 싶은 것) → L(배운 것) | 선행 지식 활성화 + 읽기 내용 요약 |
| **SQ3R** | 훑기(Survey) → 질문(Question) → 읽기(Read) → 되새기기(Recite) → 검토(Review) | 설명문으로 된 내용 교과에 효과적 |

> ⚠️ **기출 포인트**: 상보적 교수 = 호혜적 교수. 4가지 전략(예측·질문·요약·명료화) 순서와 내용 정확히 암기.
