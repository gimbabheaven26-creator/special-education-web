## 쓰기 이해 및 지도 개요

쓰기 능력 향상을 위해서는 **글씨 쓰기**, **철자**, **작문** 등 각 영역별 특성을 이해하고 효과적인 교수법을 적용해야 한다. 특히 작문은 쓰기 교수의 궁극적인 목표로, 과정적 접근과 자기 조절 전략 교수, 글의 구조에 대한 명시적 교수를 통해 학생의 사고력 증진과 작문 능력 향상을 도모해야 한다.

| 영역 | 핵심 내용 |
|------|-----------|
| 글씨 쓰기 | 개관 / 평가(3P, 4SPC, 양적) / 교수법(시각 단서, 기억 인출, 베껴 쓰기) |
| 철자 | 영향 변인(음운·표기·형태 처리) / 오류 유형 / 교수법 |
| 작문 | 목표 및 특성 / 효과적 교수의 특징 / 교수법(과정적 접근, SRSD, 글의 구조) |

## 글씨 쓰기

### 글씨 쓰기 개관

- **정의**: 손으로 글자를 쓰는 능력
- **영향 요인**: 소근육 운동 기술보다 **표기 처리**(낱자 및 글자의 형태에 대한 인식)에 더 많은 영향을 받음
- **학습장애 학생의 특성**:
  - 글씨를 지나치게 천천히 씀 (속도)
  - 글자 크기가 크거나 일정하지 않음 (크기)
  - 글자 형태가 이상함 (형태)
- **유창한 자동적 쓰기의 중요성**: 글씨를 단정하게 쓰는 데 과도한 시간·노력을 들이면, 쓰기의 본래 목적인 의사소통 기능을 발휘할 수 없게 됨

### 글씨 쓰기 평가

| 평가 유형 | 약칭 | 평가 항목 |
|-----------|------|-----------|
| 일반적(형식적) 글씨 쓰기 평가 | **3P** | 바른 자세(Posture), 올바른 연필 쥐기(Pencil hold), 올바른 종이 위치(Paper position) |
| 질적인 측면의 글씨 쓰기 평가 | **4SPC** | 글자 모양(Shape), 크기(Size), 기울기(Slant), 띄어쓰기(Spacing), 글씨 위치(Placement), 연결선(Connections) |
| 양적인 측면의 글씨 쓰기 평가 | — | 제한된 시간 안에 쓰는 양 |

> ⚠️ **기출 포인트**: 3P는 형식적 평가(자세·연필·종이), 4SPC는 질적 평가(모양·크기·기울기·띄어쓰기·위치·연결선)로 구분하여 기억할 것.

### 효과적인 글씨 쓰기 교수법

| 교수법 | 설명 |
|--------|------|
| **시각 단서 교수법** | 필순과 진행 방향을 화살표·번호로 표시한 학습지를 사용하여 시각적으로 보여주며 가르침 |
| **기억 인출 교수법** | 글자를 주의 깊게 살펴본 후 가림판으로 가린 상태에서 기억하여 쓰도록 함 |
| **결합 교수법** | 시각 단서 + 기억 인출을 결합한 방법이 **가장 효과적** |
| **베껴 쓰기** | 교사가 먼저 시범을 보인 후 학생이 같은 글자를 베껴 쓰도록 함 |

**쓰기 유창성 향상을 위한 베껴 쓰기 전략**:
- 제한된 시간 동안 베껴 쓰기를 한 후 학생이 베껴 쓴 글자의 수를 기록
- 연필 대신 손가락으로 글자를 따라 쓰는 방법(tracing) 활용

## 철자

### 철자 개관

- **정의**: 단어를 맞춤법에 맞게 쓰는 것 (소리대로 적되, 어법에 맞도록 쓰는 것이 원칙)
- **중요성**: 향후 작문 능력을 예측하는 중요한 변인
- **쓰기 유창성**: 글자나 단어를 알아볼 수 있도록 정확하고 속도감 있게 쓰는 능력으로, 정확성과 속도 모두 작문과 직간접적으로 연결됨

### 철자 능력에 영향을 주는 변인

| 처리 유형 | 설명 |
|-----------|------|
| **음운 처리** | 낱자 및 글자의 소리에 대한 인식, 낱자-소리 대응 관계에 대한 인식 |
| **표기 처리** | 낱자 및 글자의 형태에 대한 인식 (글자의 시각적 형태를 인식하는 능력) |
| **형태 처리** | 형태소에 대한 인식 |

### 철자 오류 유형

| 오류 유형 | 원인 | 구체적 유형 |
|-----------|------|-------------|
| **음운 처리 능력 오류** | 낱자-소리 대응 관계를 제대로 적용하지 않음 | 소리 나는 대로 쓰이는 단어에서의 오류 |
| **표기 처리 능력 오류** | 음운 변동이 적용되는 단어를 정확하게 쓰지 못함 | 같은 소리가 나는 다른 낱자로 대치, 전체·일부 단어를 소리 나는 대로 표기, 발음상 구분 안 되는 글자에서의 오류 |
| **형태 처리 능력 오류** | 단어를 구성하는 형태소에 대한 인식 부족 | 어간과 어미의 경계 혼동, 시제 선어말 어미 오류, 어미 변환 오류, 동음이의어 혼동 |

> ⚠️ **기출 포인트**: 한글 철자 오류에서 **가장 빈번하게** 나타나는 것은 **표기 처리 능력 오류**이다. 한글의 음운 변동 현상 때문에 소리만으로 올바른 표기를 할 수 없고, 표기 처리 능력이 요구된다.

### 철자 교수법

#### 음운 처리 중심 교수법

- 음운 처리 오류를 보이는 경우, 학생이 어떤 자음·모음에서 어떠한 오류(생략, 첨가, 대치)를 보이는지 평가
- **교수 순서**: 자주 사용하는 낱자-소리 대응 관계(자음, 모음) → 이중 모음과 겹자음 순
- **유의점**: 시각적 형태나 발음이 비슷한 낱자를 동시에 가르치지 않도록 함
- **파닉스 교수**: 음운 처리 문제가 있을 경우 낱자-소리 대응 관계를 활용한 파닉스 교수법이 효과적

#### 표기 처리 중심 교수법

- 한글에서 음운 변동 현상 때문에 표기 처리 오류가 가장 빈번하게 발생
- **Word sorting**: 음운 변동 규칙별로 단어를 묶어 소개하고, 같은 음운 변동 규칙이 적용되는 단어끼리 분류하는 활동
  - 처음에는 두 가지 음운 변동 규칙을 비교하고, 학생의 반응에 따라 점차 규칙의 수를 늘림
- 문장 안에서 단어의 쓰임을 인식할 수 있도록 하여 단어의 의미 파악 및 기본형과 연결

#### 형태 처리 중심 교수법

- 형태 처리 오류는 단어를 구성하는 형태소에 대한 인식 부족으로 발생
- **용언의 기본형과 변형을 연결하여 교수하는 절차**:
  1. 모든 단어를 읽기
  2. 어간과 어미를 명확히 알려주기
  3. 교사가 단어 분류 활동 시범 보이기
  4. 학생이 단어를 분류하도록 하기
  5. 학생이 분류한 단어를 점검하도록 하기
  6. 확인하기
  7. 가리고 베껴 쓰고 비교하기
- 문장과 함께 제시하여 어간과 어미를 인식하도록 도움

#### 기타 철자 교수법

| 교수법 | 설명 |
|--------|------|
| **자기 교정법** | 학생 자신이 쓴 단어와 정답을 비교 → 잘못 철자한 단어 확인·수정 → 바르게 베껴 쓰기 |
| **지속적인 시간 지연법** | 처음 1초 후 → 점차 3초, 6초, 9초 후에 단어를 기억하여 쓰도록 함 |
| **목표 단어 반복 쓰기** | 전통적인 철자 학습 방법으로, 목표 단어를 반복하여 베껴 쓰기 |
| **우연 학습 접근법** | 철자는 자연적으로 습득되므로 체계적·의도적 지도가 필요 없다는 관점 |
| **단어 목록에 의한 학습** | 단어 목록을 통해 철자를 익히게 함 |
| **일반화에 의한 철자 학습** | 이전에 배운 적 없는 단어도 일반화를 통해 학습 가능하다고 가정 |

## 작문

### 목표 및 특성

- **쓰기 교수의 궁극적 목표**: 작문 능력 향상 및 사고력 증진·활성화
- **잘 쓴 글**: 주제와 관련된 중심 아이디어와 뒷받침하는 정보·예들이 응집력 있게 조직된 글

**학습장애 학생의 일반적인 작문 특성**:
- 주제와 관련된 생각들을 단순히 나열하는 형태의 글 작성
- 글의 내용을 산출하는 초안 작성만을 글쓰기의 과정으로 인식
- 글쓰기 전 계획 단계를 거의 거치지 않음
- 초안 검토 및 수정에 어려움
- 수정 시 내용 수정보다 기계적 측면(구두점, 맞춤법 등) 교정에 집중하는 경향

### 효과적인 작문 교수의 특징

| 특징 | 설명 |
|------|------|
| **명시적·직접적·체계적 교수** | 작문 교수 유형과 무관하게 모든 작문 교수는 명시적·직접적·체계적으로 제공되어야 함 |
| **쓰기 과정에 대한 명시적 전략 교수** | 계획하기, 초안 작성하기, 수정하기를 명시적·체계적으로 가르침 |
| **자기 조절 전략 교수** | 쓰기 과정 명시적 전략 교수 + 자기 조절 요소(목표 설정, 자기 점검, 자기 강화) 포함 |
| **글의 구조에 대한 명시적 교수** | 글의 장르별 구조를 명시적·체계적으로 가르침 |
| **계획하기 활동** | 생각 꺼내기 활동을 통해 아이디어 생성 및 조직 |
| **요약하기에 대한 명시적 교수** | 읽기와 결합된 작문 교수 (읽은 내용을 요약하여 쓰기) |
| **목표 설정 및 자기 점검** | 글쓰기 목표(작문의 양, 전략 사용 등)를 설정하고 과정에서 달성 여부를 점검 |
| **안내된 피드백** | 교사나 또래 친구들이 학생 작문에 피드백 제공 |
| **또래 교수 활용** | 쓰기 과정(계획, 초안 작성, 수정 등)에서 또래와 함께 작업 |
| **문장 작성에 대한 명시적 교수** | 문장 구조, 특히 복문 산출에 대한 명시적 교수 제공 |

**작문 지도를 위한 지침**:
- 다양한 쓰기 과제 제시
- 다른 과목과 쓰기를 통합
- 효과적 쓰기를 위한 쓰기 단계를 따르도록 함 (쓰기 과정적 접근)
- 어문 규정에 익숙하게 하고 쓰기를 자동화
- 쓰기 향상을 위한 목표를 세우고 달성하도록 함

### 효과적인 작문 교수법

#### 쓰기 과정적 접근

**단계**:

| 단계 | 내용 |
|------|------|
| 1. 계획하기 | 생각 꺼내기, 아이디어 생성 및 조직 |
| 2. 초안 작성하기 | 글의 내용 산출 |
| 3. 내용 수정하기 | 내용 검토 및 수정 |
| 4. 편집하기 | 기계적 측면(구두점, 맞춤법 등) 교정 |
| 5. 게시하기 | 완성된 글 발표·공유 |

**교사가 해야 할 일**:
- 쓰기 과정의 각 단계마다 모델링 제공
- 교사 및 다른 학생과의 협력적 작업을 통해 쓰기 과정 진행
- 각 단계를 촉진할 수 있도록 단서를 지속적으로 제공
- 학생이 자기 주도적으로 점검·수정할 수 있도록 훈련

**장점**:
- 글쓰기 활동이 지속적으로 일어나고 반복됨
- 자기 주도적 학습을 강조하는 교육 환경 조성
- 읽고 쓰기를 통합하는 학습 강조

**효과**: 문제 해결 능력, 비판적 사고 기술, 긍정적인 자아 개념 발달

> ⚠️ **기출 포인트**: 쓰기 과정적 접근의 유의점 — 글씨 쓰기나 철자와 같은 **기능적 훈련을 제공하지 않거나**, 매 단계마다 **구체적이고 적극적인 도움이 제공되지 않으면** 이 접근은 적절하지 않다.

#### 자기 조절 전략 교수 (SRSD: Self-Regulated Strategy Development)

자기 조정 전략 발달(SRSD)은 작문 과정에서 **자기 조절의 역할**을 강조하는 학습 전략으로, 쓰기 과정적 접근을 명시적이고 체계적으로 교수하는 것이다.

**목표**:
- 학생들의 쓰기 과정과 관련된 지식과 기술 전략 발달
- 학생이 스스로 점검하고 지속적으로 글을 쓰는 데 필요한 능력을 계속적으로 발달시키도록 지원
- 학생들이 글쓰기에 대한 긍정적인 태도를 발달시키도록 촉진

**SRSD 6단계**:

| 단계 | 내용 |
|------|------|
| 1. 배경 지식 형성하기 | 전략 사용에 필요한 선행 지식·기술 형성 |
| 2. 논의하라 | 전략의 목적·이점에 대해 교사와 학생이 함께 토의 |
| 3. 시범을 보여라 | 교사가 전략 사용을 직접 시범 |
| 4. 외우도록 하라 | 학생이 전략의 단계와 절차를 암기 |
| 5. 지원하라 | 교사의 도움을 받으며 연습 |
| 6. 독립적으로 사용하게 하라 | 학생이 스스로 전략을 사용 |

> ⚠️ **기출 포인트**: SRSD는 "자기 조정 전략 발달"이라고도 하며, 쓰기 과정적 접근을 기반으로 자기 조절 요소(목표 설정, 자기 점검, 자기 강화)를 추가한 교수법이다. 6단계 순서를 정확히 암기할 것.

#### 글의 구조에 대한 교수

**교사의 역할**:
- 각각의 장르별로 다른 글의 구조를 명확하게 소개
- 다양한 예를 제시
- 학생이 초안을 작성하는 과정 중에서 단서를 충분히 제공

**글의 구조 유형**:

| 유형 | 특징 |
|------|------|
| 이야기 글 | 사건·인물·배경의 서사적 구조 |
| 설명글 | 정보를 논리적으로 전달하는 구조 |
| 논설문 | 주장과 근거로 독자를 설득하는 구조 |
