## 8. 내용 교과 지원 전략

내용 교과 지원 전략은 교과서의 중심 내용과 주요 어휘 학습을 돕기 위해 학습 안내지, 워크시트, 안내된 노트와 같은 학습 자료를 활용하여 학생들의 학습 동기를 높이고 이해를 증진시키는 데 중점을 둔다.

| 전략 유형 | 주요 도구 |
|---|---|
| 내용 교과 지원 전략 | 학습 안내지, 워크시트, 안내된 노트 |
| 기억 전략 | 핵심어 중심 기억 보조 |
| 그래픽 조직자 | 개념 간 관계 시각화 |

### 8.1. 학습 안내지

| 도구 | 의미 | 주요 특징 |
|---|---|---|
| **학습 안내지 (study guide)** | 교과서의 중심 내용 및 주요 어휘에 관한 질문으로 구성 | 질문·핵심어로 학생을 학업 정보로 이끌고, 조직화된 구조로 정보 학습·유지를 지원. 시험 전 독자적 복습 및 수업 중 활용 모두 효과 입증 |
| **워크시트 (framed outline)** | 중심 내용 및 주요 어휘에 관한 개요 제시 | 아웃라인에 질문 형태로 구성 가능. 핵심 단어를 빈칸으로 제시하여 학생이 수업을 들으며 채우도록 함 |
| **안내된 노트 (guided note)** | 수업 내용의 요약문에 빈칸을 두어 학생이 수업 중 채우도록 제공하는 유인물 | 보편적 학습 설계 원리 반영. 학생이 수업 내용을 보고·듣고·쓰며 적극 참여 유도. 강의 마지막 약 10분 복습 시 효과 증진 |

> ⚠️ **기출 포인트**: 안내된 노트(guided note)는 보편적 학습 설계(UDL) 원리를 따르며, 학생의 반응 증가·노트 필기 정확성 향상·수업 내용 이해 증진의 세 가지 장점이 자주 출제된다.

**안내된 노트의 장점 (3가지)**

1. 학생들의 반응을 증가시키고, 노트 필기의 정확성과 효율성을 향상
2. 수업 내용을 효과적으로 전달하며 학문·과목에 상관없이 내용 조직 및 이해 증진
3. 학생이 교사에게 더 분명하게 질문할 수 있도록 하고, 교사가 수업을 더 주의 깊게 준비하며 특정 과제에 집중하도록 함

## 9. 학습장애 학생들을 위한 교수 방법 및 학습 전략

학습장애 학생들을 위한 효과적인 교수법은 직접 교수와 정밀 교수를 통해 학습 목표를 명확히 하고 개별화된 학습 진전을 모니터링하며, 전략 중재 모형·과정 중심 교수법·통합 전략 교수법과 같은 학습 전략을 활용하여 학생들이 전략을 습득하고 일반화할 수 있도록 지원해야 한다.

| 분류 | 방법 |
|---|---|
| **학습 방법** | 직접 교수, 정밀 교수 |
| **학습 전략** | 호혜적 교수법(상보적 교수법), 전략 중재 모형, 과정 중심 교수법, 통합 전략 교수법 |

### 9.1. 학습 방법

#### 직접 교수 (Direct Instruction)

학업에 초점을 맞추어 학생들이 의도적으로 참여하며, 교사가 구조적으로 위계 제시된 교재를 활용하여 주도적으로 진행하는 수업이다.

**개념**

- 행동주의 이론에 근거한 교수 전략으로 교수 자료와 교수 절차를 순서화
- 과잉 학습을 통해 학습이 이루어질 수 있도록 빠른 속도로 수업 진행
- 과제 분석을 통해 교수 내용을 기능적으로 분석하고 즉각적인 교정적 피드백 제공
- 지도할 내용을 논리적으로 계열화
- 교사의 구체적인 시범을 학생이 관찰하여 사고나 기능을 습득
- 언어적 상호작용을 통해 학습 내용 지도
- 학습의 통제가 교사에서 학생으로 전이

**특징**

- 철저하게 학습 향상을 위한 피드백 제공
- 잘못된 반응에 대한 정확하고 신속한 교정
- 학습 진도를 빠르게 이끌어 숙달 정도를 높임
- 교수 활동 종료 후 지속적으로 학업 성취 평가(평가 내용은 교수 활동과 밀접 관련)

> ⚠️ **기출 포인트**: 직접 교수의 8단계 순서가 자주 출제된다.

**직접 교수의 8단계**

| 순서 | 단계 |
|---|---|
| 1 | 학습 목표 |
| 2 | 주의 집중 단서 |
| 3 | 예상 |
| 4 | 검토, 선행 학습 확인 및 목표 진술 |
| 5 | 교사 시범 |
| 6 | 안내된 연습 |
| 7 | 독립적 연습 |
| 8 | 마무리 |

---

#### 정밀 교수 (Precision Teaching) = 정확 교수 / 직접 매일 측정 방법

특정한 교수 방법이 아닌, 학생의 학업 수행을 면밀히 모니터링하기 위한 방법이다.

**특징 (4가지)**

1. 도표화(차트화)
2. 매일 측정
3. 정확 반응과 부정확 반응의 수에 의해 검증
4. 교수적 모니터링

**개념**

- 학생이 학습해야 할 목표 행동을 정확히 지적하고 기능 습득 시 진전을 도표화
- 과제의 성공을 정확도뿐 아니라 **과제 수행률(유창성)** 측면에서도 측정 → 학습장애 학생에게 매우 효과적
- 교육과정 중심 사정(CBA)의 한 방법
- 시간 제한 검사를 통한 유창성·자료 비율, 수행 결과 도표화와 평가, 필요 시 교육과정·교수 변화 → CBM과 유사

> ⚠️ **기출 포인트**: 정밀 교수는 교육과정 중심 사정(CBA)의 한 방법이며 CBM과 유사점이 자주 비교 출제된다. '유창성'을 행동 척도로 삼는다는 점에 주목할 것.

**5가지 원리**

| 원리 | 내용 |
|---|---|
| 1 | 학습자가 가장 잘 안다 |
| 2 | 직접적으로 관찰 가능한 행동에 초점 |
| 3 | 빈도수는 행동의 척도이다 (유창성) |
| 4 | 성공 여부를 위한 표준 차트를 사용한다 |
| 5 | 환경 조건에 대한 기술이다 |

**장점**

- 학습장애 학생의 필요에 따라 교수적 결정을 내릴 수 있음
- 개별적 교수 가능
- 학생이 차트로 진전을 확인하면서 목표 달성에 더 많은 책임감을 가지는 경향
- 일일 단위 교육적 결정이 가능 → 2~3주 단위 시험에서 나타나는 시간 낭비 최소화
- 자료 차트를 통해 다른 교사들과의 의견 교환이 용이

#### 2. 전략 중재 모형 (Strategy Intervention Model)

중등학교에 재학 중인 학습장애 학생을 위해 개발된 모형으로, "여러 가지 전략을 이 모형을 통하여 교수하자"는 목표 아래 전반적인 학습 활동의 성공적 수행에 요구되는 구체적 학습 전략을 포함한다.

**3가지 요소**

| 요소 | 내용 |
|---|---|
| 획득 요소 | 전략 습득 |
| 저장 요소 | 전략 저장 |
| 표현 요소 | 능력에 대한 증명과 표현 |

> ⚠️ **기출 포인트**: 전략 중재 모형의 8단계 순서가 자주 출제된다. 특히 '소리 내어 연습하기(구두 연습)'와 '학년 수준 연습'의 위치에 주의.

**8단계**

| 순서 | 단계 |
|---|---|
| 1 | 사전 검사 |
| 2 | 전략 설명 |
| 3 | 전략 시범 |
| 4 | 소리 내어 연습하기 (= 구두 연습) |
| 5 | 통제된 (안내된) 연습 |
| 6 | 학년 수준 연습 (= 진보된 연습과 피드백) |
| 7 | 사후 검사 |
| 8 | 일반화 |

---

#### 3. 과정 중심 교수법 (Process-Based Instruction)

임상적 상황이나 특수 학급에서 사용되는 학습 전략 프로그램을 **일반 학급 상황에서 사용할 수 있도록** 한 학급 통합 모형이다.

**4가지 구성 요소**

| 구성 요소 |
|---|
| 전략 계획 |
| 부호화 전략 |
| 학습 활동 |
| 교과 내용 |

**5단계**

| 순서 | 단계 |
|---|---|
| 1 | 평가 활동 |
| 2 | 소개 활동 |
| 3 | 전략 개발 활동 |
| 4 | 연습 활동 |
| 5 | 통합/일반화 활동 |

---

#### 4. 통합 전략 교수법 (Integrated Strategy Instruction)

임상 상황이나 특수 학급 상황에서 활용하기 위해 개발되었으며, 학습장애 학생들이 일반 학급에서 학습 전략을 일반화하여 적용하는 데 제한점이 있으므로 **일반 학급 상황에서 학습 내용과 함께 학습 전략 학습이 일어나도록** 하는 전략 활동이다.

**4단계**

| 순서 | 단계 |
|---|---|
| 1 | 소개 |
| 2 | 구조화 |
| 3 | 적용 |
| 4 | 확장 |

> ⚠️ **기출 포인트**: 과정 중심 교수법(5단계)과 통합 전략 교수법(4단계)은 모두 일반 학급에서의 전략 일반화를 목표로 하나, 개발 배경과 단계 구성에서 차이가 있다. 두 모형의 단계 수와 각 단계 명칭을 구분해서 암기할 것.

| 구분 | 과정 중심 교수법 | 통합 전략 교수법 |
|---|---|---|
| 개발 배경 | 임상·특수학급 → 일반학급 통합 모형 | 임상·특수학급 개발 → 일반학급 일반화 목표 |
| 단계 수 | 5단계 | 4단계 |
| 단계 구성 | 평가→소개→전략개발→연습→통합/일반화 | 소개→구조화→적용→확장 |
