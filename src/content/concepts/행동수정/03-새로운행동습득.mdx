## 개요

새로운 행동을 가르치는 것은 악보 없이 피아노를 배우는 것과 같다. **변별 훈련**으로 어떤 상황에서 연주할지 알고, **촉구**로 손가락 위치를 잡아주고, **행동 형성**으로 단음부터 화음까지 단계를 높이고, **행동 연쇄**로 곡 전체를 이어 붙인다.

| 전략 | 핵심 원리 | 주요 적용 |
|------|----------|---------|
| 변별 훈련 | SD vs S△ 차별 강화 | 어떤 자극에 반응할지 구분 |
| 촉구와 용암 | 부가 자극 → 점진 제거 | 오류 없는 학습 지원 |
| 행동 형성 | 점진적 접근 + 차별 강화 | 전혀 없는 행동 새로 만들기 |
| 행동 연쇄 | 과제 분석 + 순차 강화 | 복잡한 다단계 행동 |
| 모방(모델링) | 관찰 학습 | 시범-모방으로 빠른 습득 |

## 1. 변별 훈련과 자극 통제

### 핵심 개념

| 용어 | 정의 |
|------|------|
| **변별(discrimination)** | 특정 자극과 다른 자극의 차이를 구분하는 능력. 차별 강화의 결과로 학습됨 |
| **변별 자극(SD)** | 특정 반응에 강화가 주어질 것을 알려주는 자극. 행동의 *원인*이 아니라 발생 *가능성을 높이는* 자극 |
| **S델타(S△)** | 해당 반응에 강화가 없을 것을 알려주는 자극 |
| **자극 통제** | SD가 확립되어 특정 반응이 특정 자극에만 나타나는 상태 |
| **자극 과잉 선택** | 변별 자극과 무관한 자극이나 SD의 일부분에만 반응하는 경향 |

> ⚠️ **기출 포인트**: SD와 S△의 구분, 그리고 "변별 자극은 행동의 원인이 아니다"는 진술이 자주 출제됩니다. 자극 과잉 선택 시 **델타 자극의 체계적 선별 계획**이 필요합니다.

### 자극 통제 확립 과정

```
SD에 대한 반응 → 차별적 강화 ✓
S△에 대한 동일 반응 → 강화 없음 ✗
            ↓ 반복
     자극 통제 확립
```

## 2. 촉구와 용암

### 2.1 촉구(Prompt) 개념

**촉구**: 변별 자극이 바람직한 반응을 야기할 가능성을 높이는 **추가적인 자극**

- 목적: 아동이 **교사 촉구 없이 독립적으로** 반응하도록 만드는 것 → 결국 촉구는 제거되어야 함
- 원칙: **최소 개입 촉구** — 가장 적게 개입하고 독립성을 높임

### 2.2 촉구의 유형

| 대분류 | 유형 | 정의 |
|--------|------|------|
| **자극 촉구** | 자극 내 촉구 | SD의 위치·크기·모양·색깔·강도를 변화시킴 |
| **자극 촉구** | 자극 외 촉구 | 다른 자극 추가 또는 SD에 단서(힌트) 제공 |
| **반응 촉구** | 시각적 촉구 | 그림·사진·색깔·그래픽 등 시각적 단서 |
| **반응 촉구** | 언어적 촉구 | 지시·힌트·질문·정의·규칙 언어 제시 (수화 포함) |
| **반응 촉구** | 몸짓 촉구 | 신체 접촉 없이 동작·자세로 정반응 유도 |
| **반응 촉구** | 모델링 촉구 | 정반응 행동을 시범으로 보임 |
| **반응 촉구** | 신체적 촉구 | 신체 접촉으로 행동 유발 (부분/전반적) |

> ⚠️ **기출 포인트**: 자극 촉구 vs 반응 촉구 분류, 자극 내 촉구 vs 자극 외 촉구 구분이 자주 출제됩니다.
> 자극 촉구는 준비 시간이 더 걸리지만 **교사 부재 상황에서도 독립성을 향상**시킵니다.

### 2.3 촉구의 용암(Fading) — 4가지 방법

| 방법 | 핵심 원리 | 장점 | 단점 |
|------|----------|------|------|
| **최대-최소 촉구법** (도움 감소법) | 처음 최대 촉구 → 정반응 시 점차 감소 | 초기 오류·좌절 방지 | 필요 이상 많은 촉구 제공 가능 |
| **최소-최대 촉구법** (도움 증가법) | SD만 제시 → 무반응 시 점차 증가 | 독립 반응 기회 극대화 | 초기 오류 발생 가능 |
| **시간 지연법** | 촉구 형태 유지, **제시 시간만** 지연 | 독립성 촉진·낙인 최소화, 사회적 수용도 높음 | — |
| **점진적 안내 감소** | 신체 촉구 수준 점진 감소 → 그림자 방법 | 심한 장애 학생에 적합 | — |

#### 시간 지연법 세부

| 유형 | 특징 |
|------|------|
| **0초 지연 (동시 촉구)** | SD와 촉구를 동시 제시. 유지·일반화 효과 우수, 또래 교수·준전문가 활용 용이 |
| **고정 시간 지연** | 모든 시도에서 지연 시간 동일 |
| **점진적 시간 지연** | 회기마다 지연 시간을 조금씩 늘려감 |

**그림자 방법**: 교사의 손을 학생 움직이는 부위에 가까이 두되 접촉하지 않고 필요 시 도울 준비 상태를 유지하는 것

> ⚠️ **기출 포인트**: 시간 지연법은 "촉구의 **형태**가 변하지 않고 **시간**이 변한다"는 점이 다른 용암 방법과의 핵심 차이입니다.

## 3. 행동 형성법(Shaping)

### 3.1 정의

현재 **나타나지 않는 표적 행동**을 발생시키기 위해 표적 행동에 점진적으로 가까운 행동을 **체계적으로 차별 강화**하여 새로운 행동을 형성시키는 것

- **도달점 행동**: 최종 목표 행동
- 점점 강화 기준을 높여(까다롭게) 보다 복잡하고 정교한 행동을 습득시킴

### 3.2 두 가지 핵심 원리

| 원리 | 내용 |
|------|------|
| **차별 강화** | 물리적으로 다른 행동 중 **하나는 강화**, **나머지는 소거** (3요소: 다른 행동 개입 + 한 행동 강화 + 나머지 소거) |
| **점진적 접근** | 도달점 행동에 가까운 반응을 선택해 강화. 강화 기준을 단계적으로 상향 조정 |

### 3.3 장단점

| 구분 | 내용 |
|------|------|
| **장점** | 벌·혐오 절차 없음, 새 기술 습득에 효과적, 긍정적 절차 |
| **단점** | 시간 소요 많음, 교사 지속 점검 필요, 진전 속도 불균등 |
| **주의점** | 바람직하지 않은 행동도 우연히 형성될 수 있으므로 세심한 관찰 필요 |

### 3.5 행동 형성 vs 유사 개념 비교

| 비교 항목 | 행동 형성 | 촉구 용암 | 행동 연쇄 |
|----------|----------|----------|----------|
| **목적** | 새 행동 창조 | 이미 학습된 행동을 다른 자극 통제 하에 | 다단계 행동 연결 |
| **조작 대상** | **후속 결과** 조작 | **선제 자극** 조작 | 각 단계 자극-반응 연결 |
| **행동 변화** | 행동 자체가 변함 | 행동은 불변, 선제 자극만 변함 | 각 단계 행동이 목표의 일부 |
| **접근 과정** | 점진적 접근이 목표 행동이 되지 않음 | — | 각 단계가 목표 행동의 구성요소 |

## 4. 행동 연쇄법(Chaining)

### 4.1 정의

과제 분석을 통해 표적 행동을 하위 과제로 세분 → 순서에 따라 **앞에서 또는 뒤에서부터 누가적으로 연결·강화**하는 방법

**행동 연쇄의 원리**: 하나의 단위 행동이 다음 단계의 **변별 자극** 역할을 하면서 동시에 직전 행동을 증가시키는 **강화 역할**도 함

### 4.2 과제 분석

| 개념 | 내용 |
|------|------|
| **과제 분석** | 복잡한 목표 행동을 가르칠 수 있는 하위 과제로 나누는 것 |
| **타당성 검증 방법** | ① 숙련자 수행 관찰, ② 전문가 평가·조언, ③ 연구자가 직접 수행 |

### 4.3 성취 수준 평가 — 단일 기회법 vs 다수 기회법

| 방법 | 특징 |
|------|------|
| **단일 기회법** | 오류 발생 즉시 평가 중단. 순서대로 수행 능력만 평가 |
| **다수 기회법** | 오류 발생 시 교사가 올바른 상태로 교정 → 다음 단계 계속 수행. 전체 단계 평가 가능 |

> 평가 과정에서는 **훈련 효과가 개입되지 않도록** 주의

### 4.4 세 가지 훈련 방법

| 방법 | 시작점 | 특징 | 장점 | 단점 |
|------|--------|------|------|------|
| **전진형** | 첫 번째 단계 | 앞에서부터 누가적으로 연결 | 초기 단계 짧아 한 회기에 다수 시행 가능 | 훈련 진행 시 과제 양 증가 → 저항감 유발 가능 |
| **후진형** | 마지막 단계 | 뒤에서부터 역순으로 연결 | **모든 훈련에서 자연 강화 제공**, 완성 만족감, 독립성·자신감 향상 | 시간 소요 많음, 회기당 시행 수 적음 |
| **전체 과제** | 전체 동시 | 매 시행에서 전체 단계 수행 | 더 자연스럽고 빠른 학습 | 하위 과제 수가 많으면 어려움 |

#### 전진형 vs 후진형 비교 심화

| 비교 항목 | 전진형 | 후진형 |
|----------|--------|--------|
| 첫 교수 단계 | 첫 번째 구성 요소 | 마지막 구성 요소 |
| 강화인 종류 | 마지막 단계 제외 → 인위적 강화인 | **모든 단계**에서 자연적 강화인 |
| 적합 대상 | 과제 대부분 습득한 학습자 | **장애 정도가 심한** 개인 |

**전체 과제 연쇄 적용 조건**: ① 하위 과제 대부분 이미 습득, ② 하위 과제 수가 적고 단순, ③ 모방 능력 있음, ④ 장애 정도가 심하지 않음

> ⚠️ **기출 포인트**: 후진형은 "모든 훈련에서 자연적 강화인 제공"이 핵심 장점입니다. 전진형·후진형·전체 과제의 3가지 방법 구분과 적용 조건이 자주 출제됩니다.

## 5. 모방하기(Modeling)

### 5.1 이론적 기반

**반두라(Bandura)의 관찰 학습 이론** — 다른 사람의 행동을 관찰·모방하여 새로운 행동을 학습

- 촉구의 한 유형이기도 함
- 빠른 시간 내 새로운 행동을 가르치는 데 효과적

### 5.2 효과적인 모방을 위한 조건

| 요인 | 내용 |
|------|------|
| **관찰자(아동) 특성** | 주의 집중 능력, 인지 능력 |
| **모델 특성** | 연령·특성 유사성, 문제 공유성, 능력 우월성 |
| **제시 방법** | 직접 제시(실제 모델 관찰) vs 간접 제시(비디오 등) |

**지침**:
- 쉬운 행동 → 어려운 행동 순서로 제시
- 여러 모델을 사용하는 것이 한 명보다 효과적
- 모방 후 **가능한 한 빨리 모방 기회** 제공
- 모델이 강화받는 장면을 아동이 직접 볼 때 효과 증가

### 5.3 모델링의 3가지 기능

| 기능 | 내용 |
|------|------|
| **반응 촉진** | 모델 행위가 아동에게 유사 행동을 하도록 동기화 (사회적 촉진) |
| **금지/탈금지** | 금지: 모델이 벌받는 것 관찰 → 유사 행동 억제 / 탈금지: 모델이 부정적 결과 없이 위협 활동 수행 → 행동 억제 해제 |
| **관찰 학습** | 타인 행동 관찰로 간접적 학습 |

### 5.4 모델링의 종류

| 종류 | 정의 |
|------|------|
| **선행 모델링** | 반응 요구 전 각 단계에서 올바른 반응을 먼저 시범 후 모방 요구 |
| **실수-의존 모델링** | 아동 오류 시에만 모델링 |
| **부분 모델링** | 정반응의 일부만 시범, 나머지는 아동이 완성 |

### 5.5 비디오 모델링 4가지 유형

| 유형 | 내용 | 장점 | 단점 |
|------|------|------|------|
| **기본 비디오 모델링** (또래/성인) | 또래·성인의 목표 기술 수행 비디오 | 연출 용이, 유지 우수 | 모델 관심도 낮을 수 있음, 일반화 불일관 |
| **비디오 자기 모델링** | 아동 **자신**이 목표 기술 수행하는 모습 (촉구 편집 제거) | 자신감 증진, 높은 관심·일반화 | 연출·편집 복잡 |
| **1인칭 시점 비디오** | 학습자 관점에서 과제 수행 촬영 | 연출 용이, 실제 생활 묘사, 편집 적음 | 행동 범위 제한적, 연구 부족 |
| **비디오 촉진** | 단계별로 나눈 짧은 클립을 순서대로 제시 → 각 단계 수행 후 다음 클립 | 단계별 독립 수행 촉진 | — |

> ⚠️ **기출 포인트**: **비디오 자기 모델링**은 자신이 성공적으로 수행하는 모습만 담고 촉구는 편집으로 제거한다는 점이 핵심입니다. 일반 비디오 모델링과의 차이를 명확히 구분하세요.

비디오 모델링의 공통 장점:
- 일반화 촉진
- 실제 모델링보다 빠른 기술 습득
- 아동의 독립성 증진 및 촉구 의존성 감소

## 6. 핵심 비교 총정리

### 촉구 유형 위계 (약함 → 강함)

```
시각적 촉구 → 언어적 촉구 → 몸짓 촉구 → 모델링 촉구 → 신체적 촉구
```

### 용암 방법 비교

| 방법 | 무엇이 변하는가 | 시작 강도 |
|------|--------------|----------|
| 최대-최소 | 촉구의 **양/강도** | 최대 → 감소 |
| 최소-최대 | 촉구의 **양/강도** | 최소 → 증가 |
| 시간 지연 | 촉구의 **제시 시간** | 0초 → 점진 증가 |
| 점진적 안내 감소 | 신체 촉구의 **수준** | 전반적 → 그림자 |
