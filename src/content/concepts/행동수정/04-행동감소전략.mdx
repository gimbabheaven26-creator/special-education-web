## 개요: 4수준 행동 감소 전략

문제 행동을 감소시킬 때는 **최소 강제 대안의 원칙**에 따라, 강제성이 낮은 수준부터 단계적으로 적용한다. 혐오 자극은 최후 수단이다.

| 수준 | 전략 범주 | 대표 기법 |
|------|-----------|-----------|
| **수준 I** | 강화 중심 전략 (차별 강화) | DRL, DRO, DRI, DRA, NCR |
| **수준 II** | 소거 | 강화 종료, 감각 소거 |
| **수준 III** | 바람직한 자극 제거 | 반응 대가, 타임아웃 |
| **수준 IV** | 혐오 자극 제시 | 무조건/조건 혐오 자극, 과잉 교정 |

> ⚠️ **기출 포인트**: 수준이 높아질수록 강제성·혐오성이 강해진다. 시험에서는 "최소 강제 대안의 원칙"을 언급하며 특정 기법을 선택한 이유를 묻는 문제가 출제된다.

---

## 1. 전제 원칙

### 1-1. 최소 강제 대안의 원칙

- 동일하게 효과적인 절차 중 **가장 덜 혐오적인 절차**를 우선 선택한다.
- 덜 강제적이지만 비효과적인 절차 vs 혐오적이지만 효과적인 절차 → **효과적인 절차** 선택.

### 1-2. 도전 행동의 기능에 근거한 중재

- 기능 확인이 가능하면 **기능에 기반한 중재**를 선정한다.
- 행동 형태만 보고 중재를 선택하면 행동이 원상태로 돌아가거나 다른 문제 행동으로 대체될 수 있다.

---

## 2. 수준 I: 강화 중심 전략 (차별 강화)

**정의**: 바람직한 행동에는 강화, 바람직하지 못한 행동에는 강화를 제공하지 않아 문제 행동을 감소시키는 방법.

> ⚠️ **기출 포인트**: 5가지 차별 강화(DRL, DRO, DRI, DRA, NCR)의 **정의와 차이점** 구별이 핵심이다. 특히 DRI vs DRA 비교표는 단골 출제 항목이다.

### 2-1. DRL — 저빈도 차별 강화 (Differential Reinforcement of Low rates)

**정의**: 정해진 시간 안에 표적 행동 발생 수가 기준치 **이하**일 때 강화를 제공해 반응 비율을 점차 감소시키는 방법. 표적 행동을 수용 가능한 수준으로 **줄이는** 것이 목적(제거 아님).

| 구분 | 내용 |
|------|------|
| **장점** | 혐오 자극 없이 정적 강화만으로 빈도 감소; 점진적 행동 감소; 습관화된 행동에 적합 |
| **단점** | 신속한 변화 불가; 자신·타인에게 위험한 행동에는 부적합 |

**사용 지침**:
1. 기초선 기록 → 평균 발생 수를 초기 DRL 기준치로 설정
2. 처음 강화 기준은 평균 또는 평균보다 약간 적은 양
3. 성공 시 기준을 점차 낮춰 원하는 수준까지 단계적 감소

**종류**: 전체 회기 DRL / 간격 DRL / 반응 간격 DRL / 기준 변동 DRL

---

### 2-2. DRO — 다른 행동 차별 강화 (Differential Reinforcement of Other behaviors)

**정의**: 정해진 시간 간격 동안 표적 행동이 **전혀 발생하지 않을 때(빈도 = 0)** 강화를 제공하는 방법. 어떤 다른 행동이 발생하든 상관없이 시간 간격 종료 즉시 강화.

| 구분 | 내용 |
|------|------|
| **단점 ①** | 의도치 않게 다른 문제 행동을 강화할 수 있음 |
| **단점 ②** | "행동 진공 상태" 발생 → 대체 행동을 별도로 지도해야 함 |
| **단점 ③** | 교사 강화 < 문제 행동 강화 이면 효과 없음 |

**사용 절차**:
1. 기초선 기반으로 시간 간격 설정 (평균 간격보다 **약간 짧게**)
2. 성공 시 간격을 점차 늘림
3. 간격 내 문제 행동 발생 → 그 시점부터 간격 재시작

**종류**: 전체 회기 DRO / 간격 DRO / 영속적 DRO

---

### 2-3. DRI — 상반 행동 차별 강화 (Differential Reinforcement of Incompatible behaviors)

**정의**: 문제 행동과 **동시에 발생할 수 없는** 바람직한 행동(상반 행동)을 강화하는 방법.

- 상반 행동은 기능적으로 동일할 필요 없고, 개발 범위가 좁다.
- 문제점: 상반되는 바람직한 행동을 찾기 어려움.

**사용 절차**:
1. 문제 행동 정의
2. 적절한 상반 행동 선정 및 조작적 정의
3. 상반 행동이 행동 목록에 없으면 먼저 지도
4. 연속 강화 계획으로 상반 행동 확립까지 즉각·지속 강화
5. 부적절 행동 발생 시 소거 적용; 지속 시 최소한의 처벌 사용

---

### 2-4. DRA — 대체 행동 차별 강화 (Differential Reinforcement of Alternative behaviors)

**정의**: 문제 행동에는 강화하지 않고, 문제 행동을 **대신할 수 있는 바람직한 행동**(대체 행동)을 강화하는 방법.

- 대체 행동은 문제 행동과 **기능적으로 동일**하고 사회적으로 용인됨.
- 대체 행동은 문제 행동보다 **힘을 덜 들이고 즉각적인 결과**를 얻을 수 있어야 함.
- "바람직한 행동 강화 + 바람직하지 않은 행동 소거"의 결합 전략.

### DRI vs DRA 비교

| 구분 | DRI (상반 행동 차별 강화) | DRA (대체 행동 차별 강화) |
|------|---------------------------|---------------------------|
| **공통점** | 문제 행동은 강화 안 함; 문제 행동을 대신할 사회적으로 용인된 행동을 강화 | 동일 |
| **동시 발생 가능성** | 문제 행동과 상반 행동이 **동시에 발생 불가** | 문제 행동과 대체 행동이 **동시에 나타날 수 있음** |
| **기능 동일성** | 기능이 **동일할 필요 없음** | 기능이 **동일해야 함** |
| **개발 범위** | 좁음 (찾기 어려움) | **넓음** |

> ⚠️ **기출 포인트**: DRI와 DRA의 차이는 "기능 동일성"과 "동시 발생 가능성"이 핵심 변별 기준이다.

---

### 2-5. NCR — 비유관 강화 (Non-Contingent Reinforcement)

**정의**: 학생의 행동 수행과 **상관없이**, 문제 행동을 유지시켜 온 강화 인자를 미리 설정된 시간 간격에 따라 제공하는 방법.

- 문제 행동의 **동기·요구 자체를 제거**하는 전략.
- 본질적으로 문제 행동을 '소거'하는 절차.

| 구분 | 내용 |
|------|------|
| **장점** | 활용 쉬움; 긍정적 학습 환경 조성; 바람직한 행동과 우연 일치 기회 많음 |
| **단점** | 바람직한 행동 동기까지 감소 가능; 문제 행동이 우연히 강화될 우려; 부적 강화 시 수업 지장 |

**효과 향상 방법**:
- 기능성 평가로 문제 행동 강화 후속 자극 명확히 파악
- 후속 자극을 문제 행동과 **무관하게 풍족히** 제공
- 초기에 밀집된 강화 계획 사용
- **소거 전략과 함께** 사용할 때 효과 극대화

---

## 3. 수준 II: 소거

**정의**: 문제 행동을 유지시켜 온 강화 인자를 **더 이상 제공하지 않거나 보류**해 행동을 감소시키는 방법.

> ⚠️ **기출 포인트**: 소거 폭발, 소거 저항, 자발적 회복은 단골 출제 개념이다. 각각의 정의와 교사 대응 방법을 정확히 숙지할 것.

### 3-1. 소거의 3가지 주요 현상

| 현상 | 정의 | 교사 대응 |
|------|------|-----------|
| **소거 저항** (유지/지연 반응) | 강화 보류 후에도 행동이 얼마간 **지속**되는 것. 간헐적 강화 스케줄로 유지된 행동에서 특히 두드러짐. | 일관성 있게 무시를 유지 |
| **소거 폭발** (행동의 발작) | 소거 초기에 표적 행동의 **빈도·강도가 크게 증가**하는 현상. "나아지기 전에 더 심해지는 과정". | 행동 증가에도 강화를 절대 제공하지 않음 |
| **자발적 회복** | 완전히 제거된 줄 알았던 행동이 **일정 시간 후 다시 출현**하는 현상. | 반드시 무시; 강화 제공 시 간헐 강화로 재학습되어 제거가 더 어려워짐 |

### 3-2. 소거의 장단점

| 장점 | 단점 |
|------|------|
| 바람직하지 않은 행동 감소에 매우 효과적 | 문제 행동 유지 강화 요인 찾기·통제 어려움 |
| 결과가 오래 지속됨 | 환경 통제가 완벽해야 함 |
| 혐오 자극 직접 제시 없음 | 소거 폭발 시 포기 가능성 |
| 학생 자존감 보호 | 좌절 유발 가능성 |
| 차별 강화와 병행 시 효과 극대화 | 소거 후 어떤 행동이 학습될지 알 수 없음 |

### 3-3. 감각 소거

어떤 행동이 **자동적 강화**에 의해 유지될 때, 그 행동이 가져오는 **자연적 감각적 결과를 제거**해 행동을 감소시키는 기법.

### 3-4. 효과적 사용 지침

- 모든 강화인을 찾아 **일관성 있게** 제거
- 차별 강화(특히 DRA·DRI)와 **병행** 사용
- 학생에게 소거 절차 **사전 고지**
- 소거 폭발 시 공격적 행동에 대비
- 자신·타인에게 위험한 행동에는 단독 적용 지양
- 바람직한 행동에 의도치 않은 소거 발생 주의

## 4. 수준 III: 바람직한 자극의 제거

### 4-1. 반응 대가

**정의**: 표적 행동 발생 시 아동이 가진 강화 인자를 **일정량 회수**하는 부적 벌의 일종.

- **소거와의 차이**: 소거는 문제 행동으로 얻던 강화 자체를 차단하는 것. 반응 대가는 회수하는 강화제가 **문제 행동과 관련이 없다**.

| 장점 | 단점 |
|------|------|
| 사용 쉬움; 다양한 활용 가능 | 부적절한 행동에 더 많은 주의를 줄 수 있음 |
| 행동 감소 비교적 빠름 | 강화물 모두 잃고 포기 가능 → 적절한 행동 강화 전략 병행 필요 |
| 행동 변화 오래 유지 | |

**절차**:
1. 반응 대가를 적용할 문제 행동 정의
2. 학생에게 강화제 지속 획득 가능성 확인
3. 문제 행동별 잃게 될 강화제 양 결정 (세심한 주의 필요)
4. 제거할 강화제는 제거하기 쉬워야 함
5. 구체적 행동에만 적용; 학생에게 절차 명확히 설명

**토큰 경제와의 병행**:
- 토큰 경제 목적이 바람직한 행동 강화이고 **경쟁 문제 행동이 없을 때** → 반응 대가 미포함
- 경쟁 문제 행동이 있다면 반응 대가를 **토큰 경제 일정 기간 실행 후** 도입
- 토큰을 **모두 잃게 만들어서는 안 됨**

> ⚠️ **기출 포인트**: 반응 대가와 소거의 차이(강화제와 문제 행동의 관련성 유무)는 출제 빈도가 높다.

---

### 4-2. 타임아웃 (정적 강화로부터 타임아웃)

**정의**: 문제 행동 발생 시 학생이 정적 강화를 받지 못하도록 일정 시간 동안 **강화제 접근을 차단**하는 것.

**효과적인 적용 조건**:
- 타임아웃 장소 → **정적 강화 가능성 전혀 없는 곳**
- 학생이 떠나는 장소 → **정적 강화 가능성 높은 곳**
- (두 조건 모두 충족되어야 효과적)

| 장점 | 단점 |
|------|------|
| 긍정적 강화 프로그램과 병행 용이 | 어떤 학생에게는 타임아웃 자체가 강화로 작용 가능 |
| 효과 즉각적이고 지속 시간 김 | 빈번한 사용 시 학업 성취에 부정적 영향 |
| 비배제 타임아웃: 교육 환경 배제 없이 사용 가능 | |

### 타임아웃 유형 (강제성 순서: 낮음 → 높음)

| 유형 | 정의 | 예시 |
|------|------|------|
| **비격리-비배제 타임아웃** | 격리·배제 없이 강화제 접근 차단 | 불 끄기, 타임아웃 리본, 유관 관찰, 시각적 선별 |
| **비격리-배제 타임아웃** | 격리 없이 배제하여 강화제 접근 차단 | 벽 보고 있기, 교실 뒤 "생각하는 의자" |
| **격리 타임아웃** | 별도 장소로 격리 | 타임아웃 전용 방 |

> ⚠️ **기출 포인트**: 격리 타임아웃은 가장 강제적 방법으로, 가장 심각한 문제 행동에만 적용해야 한다. 타임아웃이 강화로 작용할 수 있는 상황(수업 회피 기능)도 자주 출제된다.

---

## 5. 수준 IV: 혐오 자극 제시

> ⚠️ **기출 포인트**: 수준 IV는 최후 수단이며, 항상 다른 수준 전략을 먼저 시도한 후 적용해야 한다. 과잉 교정의 두 가지 유형(복원 vs 긍정적 연습)의 차이를 정확히 구별할 것.

### 5-1. 벌의 이점과 단점

| 이점 | 단점 |
|------|------|
| 행동 발생 신속 정지; 효과 오래 지속 | 교사-학생 관계 약화 |
| 수용/불가 행동 간 명확한 변별 제공 | 학생의 공격적 행동 유발 |
| 다른 학생에게 행동 결과 시범 | 부적절한 행동의 모델링 효과 |
| | 효과 유지·일반화 어려움 |
| | 바람직한 행동 학습 없음 |
| | 교사가 부적 강화될 수 있음 |

### 5-2. 행동의 대비 현상

한 자극 상황에서 특정 행동이 벌을 받으면, 그 상황에서는 감소하지만 **벌 없는 다른 상황**에서는 오히려 이전보다 **빈도가 증가**하는 현상.

**감소·예방 방법**:
- 관련된 모든 상황·자극 조건에서 표적 행동 벌함
- 강화 차단 또는 최소화
- 바람직한 대체 행동 마련 및 강화
- 문제 행동이 일어날 수 없는 환경 조성

### 5-3. 혐오 자극 유형

| 유형 | 정의 |
|------|------|
| **무조건 혐오 자극** | 신체적 고통·불편 유발 자극. 이전 경험 없이 즉각 행동 변화 유발. "보편적/자연적/학습되지 않은 벌인자" |
| **조건 혐오 자극** | 무조건 혐오 자극과 짝짓기되어 **학습된** 혐오 자극 |
| **부적 연습** | 부적절한 행동을 반복하게 하여 **지치게 하거나 포화** 상태로 만들어 감소 |
| **유관 훈련** | 부적절한 행동과 **아무 관련 없는** 신체적 운동 동작을 반복 |
| **신체적 구속** | 문제 행동 관련 신체 부위를 **움직이지 못하도록** 구속 |

### 5-4. 과잉 교정 절차

**정의**: 부적절한 행동에 대한 후속 결과로 적절한 행동을 **반복적으로 실시**하는 절차.

- 벌의 **억제 효과** + 긍정적 연습의 **교육적 효과**를 함께 추구하는 전략.

| 유형 | 정의 |
|------|------|
| **복원 과잉 교정** (원상 회복 과잉 교정) | 문제 행동으로 발생한 손상을 원상태로 복원하고, **문제 행동 발생 이전보다 더 좋은 상태**로 만들어 놓는 것 |
| **긍정적 연습 과잉 교정** (정적 연습 과잉 교정) | 문제 행동 후 올바른 형식으로 **반복 수행**하게 하거나, 문제 행동과 상반되는 바람직한 행동을 일정 시간·횟수 **반복** 요구 |

**사용 지침**:
1. 문제 행동 발생 즉시 조용하고 차분한 목소리로 지적 및 이유 설명
2. 학습자가 수행해야 할 과잉 교정 과정 설명
3. 가능한 한 신속히 실행
4. 일상 활동 중 적절한 행동 수행 시 칭찬·강화 제공
5. 전 과정에서 학습자 지시·감독

**단점·역효과**:
- 절차 시행에 오래 걸림
- 학생이 순응하지 않거나 공격적이 될 수 있음
- 교사의 온전한 관심이 대상 학생에게 강화로 작용 가능

> ⚠️ **기출 포인트**: 복원 과잉 교정은 "발생한 손상 복원 + 더 좋은 상태"가 핵심. 긍정적 연습 과잉 교정은 "올바른 형식의 반복 연습"이 핵심. 두 유형의 예시를 들어 구별하는 문제가 자주 출제된다.

## 6. 핵심 개념 총정리

| 개념 | 한 줄 정의 |

| DRA | 문제 행동과 기능 동일한 대체 행동 강화 |
| NCR | 행동 무관하게 시간 간격으로 강화 제공 |
| 소거 폭발 | 소거 초기 빈도·강도 급증 현상 |
| 자발적 회복 | 소거 후 일정 시간 뒤 행동 재출현 |
| 반응 대가 | 문제 행동 시 보유 강화제 일정량 회수 |
| 타임아웃 | 일정 시간 강화제 접근 차단 |
| 복원 과잉 교정 | 손상 복원 + 이전보다 더 좋은 상태 만들기 |
| 긍정적 연습 과잉 교정 | 올바른 행동 또는 상반 행동 반복 연습 |
| 행동 대비 현상 | 벌 있는 상황엔 감소, 벌 없는 상황엔 증가 |
