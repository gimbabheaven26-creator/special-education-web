## 5. 자료 수집 — 행동의 직접 관찰과 측정

행동을 객관적으로 이해하고 효과적인 중재를 적용하려면, 행동의 특성에 맞는 관찰 기록 방법을 선택하고 신뢰도를 확보해야 한다.

> ⚠️ **기출 포인트**: 각 기록법의 **정의·특징·장단점·적합한 행동 유형**이 단골 출제 소재다. 특히 전체 간격 vs 부분 간격, 관찰자 간 신뢰도 산출 공식이 자주 출제된다.

---

### 5.1 체계 선택 기준

| 유형 | 설명 | 수량화 | 예시 |
|------|------|--------|------|
| **행동 묘사 관찰 기록** | 행동을 이야기 형식으로 서술 | 불가 | 일화 기록, A-B-C 기록 |
| **행동 결과물 중심 기록** | 반영구적으로 남는 산물 관찰 | 가능 | 영속적 산물 기록 |
| **행동 특성 중심 기록** | 빈도·강도·지속시간 등 특성 측정 | 가능 | 사건 기록법, 지속 시간 기록법 |
| **시간 중심 관찰 기록** | 시간 구간 내 행동 발생 여부 기록 | 가능 | 간격 기록법, 시간표집법 |

**행동 특성 용어 정리**

| 용어 | 정의 |
|------|------|
| 빈도 | 행동 발생 수 |
| 비율 | 빈도를 시간으로 나눈 값 |
| 지속 시간 | 행동이 지속된 시간 길이 |
| 지연 시간 (잠복기) | 선행 자극 → 반응 시작까지 걸린 시간 |
| 형태 | 행동의 외형 |
| 힘(강도) | 행동의 세기 |
| 위치 | 행동이 일어난 장소 |

---

### 5.2 일화 기록 (Anecdotal Recording)

이야기 형식으로 자연스러운 장면의 행동을 기술하는 비공식적 방법이다.

**기록 포함 요소**: 환경 정보, 아동의 행동·말, 타인의 행동·말, 시간대, 관찰자 의견(별도 구분)

| 구분 | 내용 |
|------|------|
| **장점** | 전후 맥락 정보 획득, 기능 평가 수단, 사전 준비 간편 |
| **단점** | 양적 자료 제공 어려움, 시간 소요, 결과 일반화 어려움 (주관적) |
| **활용** | 기능 평가(FBA)의 첫 단계, 심각한 행동의 장기 절차 시작점 |

> ⚠️ **기출 포인트**: 일화 기록은 **평가보다 분석**에 유용하며, 과거형으로 서술, 관찰자의 주관적 해석 개입 금지가 핵심 특징이다.

---

### 5.3 영속적 산물 기록 (Permanent Product Recording)

행동의 결과가 반영구적으로 남는 경우에 사용한다. 교사가 행동 발생 순간에 직접 관찰하지 않아도 된다.

| 구분 | 내용 |
|------|------|
| **장점** | 실시간 관찰 불필요, 접근 어려운 시간·장소 행동 측정 가능, 관찰 일치도 높음 |
| **단점** | 즉시 미기록 시 결과 소실 위험, 행동의 강도·형태·시간 양상 설명 불가 |

---

### 5.4 관찰 기록 체계

#### 사건 기록법 (Event Recording) = 빈도 기록법

관찰 기간 동안 **행동 발생 횟수**를 직접 기록한다.

- 시작과 끝이 명백한 **불연속적 행동**에 적합
- 자료는 빈도 수 또는 비율로 요약

**활용 불가 경우**
- 매우 높은 빈도로 발생하여 정확한 수 기록이 어려운 행동
- 지속 시간이 긴 연장 행동

| 구분 | 내용 |
|------|------|
| **장점** | 사용 간편, 다양한 학업 행동에 적용 용이, 시간 흐름에 따른 행동 분포 파악 가능 |
| **단점** | 행동 형태 설명 불가, 극히 짧은 간격 또는 장시간 행동에 부적절 |
| **신뢰도 산출** | (작은 수 ÷ 큰 수) × 100 |

**변형: 통제 제시 기록법 (반응 기회 기록법)**
- 교사/치료자가 반응 기회를 통제하며, 시도(선행 자극–반응–후속 결과)마다 발생 여부 기록

---

#### 간격 기록법 (Interval Recording) = 등간/동간 기록

전체 관찰 시간을 짧은 간격('초' 단위)으로 나눠 각 간격에서 행동 발생 유무(+/–)를 기록한다.

| 유형 | 기록 조건 | 적합한 행동 | 경향성 |
|------|----------|------------|--------|
| **전체 간격 기록법** | 간격 **처음부터 끝까지** 행동이 지속될 때 발생으로 기록 | 어느 정도 지속되는 안정된 행동 | 실제보다 **과소** 추정 |
| **부분 간격 기록법** | 간격 내 **잠깐이라도** 나타나면 발생으로 기록 | 짧고 빈도 높은 과잉 행동 | 실제보다 **과대** 추정 |

> ⚠️ **기출 포인트**: 전체 간격은 과소 추정, 부분 간격은 과대 추정이라는 방향성 차이가 핵심이다.

---

#### 시간표집법 (Time Sampling) = 순간표집법

전체 관찰 간격을 '분' 단위로 나누고, **간격의 마지막 순간**에 행동이 있으면 발생으로 기록한다.

- 관찰자가 간격 전체를 지속 관찰하지 않아도 됨
- **지속 시간이 길거나 저빈도 행동** 기록에 적합
- 여러 아동 동시 관찰에 유용

| 구분 | 내용 |
|------|------|
| **장점** | 교수 중 병행 관찰 가능, 교사 시간 절약, 여러 아동 동시 관찰 가능 |
| **단점** | 관찰 간격 내 행동 빈도 정보 없음, 간격이 길수록 실제와 괴리 증가 |

**변형 유형**

| 유형 | 설명 |
|------|------|
| **플라체크(Placheck)** | 집단 전체 활동 측정·평가에 활용 |
| **원탁형** | 집단 구성원 각각을 순환 관찰하여 집단 행동 산정 |
| **관찰 간격 변형** | 관찰 중 놓치는 단점을 보완하기 위해 간격 구조 변형 |

---

#### 지속 시간 기록법 (Duration Recording)

행동의 **시작 시각과 종료 시각**을 기록하여 지속 시간을 산출한다.

- 시작과 끝이 명확하고 **비교적 오랫동안 지속되는 행동**에 적합

| 접근법 | 산출 방법 |
|--------|----------|
| **전체 지속 시간** | 관찰 시간 내 총 지속 시간 합산 |
| **평균 지속 시간** | 각 발생의 지속 시간 평균 |
| **지속 시간 백분율** | (총 지속 시간 ÷ 총 관찰 시간) × 100 |

**관찰자 간 신뢰도**

| 방법 | 설명 |
|------|------|
| 총 지속 시간 일치도 | 두 관찰자의 전체 지속 시간을 비교하여 백분율 환산 |
| 평균 지속 시간 일치도 | 각 발생별로 비교 후 평균 → 더 보수적이고 신뢰성 높음 |

---

#### 반응 시간 기록법 (Latency Recording)

**선행 사건 → 표적 행동 시작까지 지연된 시간**을 기록한다.

- 지시에 따르는 능력(순응도) 측정에 특히 유용
- 디지털 스톱워치 사용 권장
- 결과: 전체 지연 시간 ÷ 횟수 = **평균 지연 시간**

---

### 5.5 기록법 비교 요약

| 기록법 | 단위 | 적합한 행동 | 핵심 특징 |
|--------|------|------------|----------|
| 사건 기록법 | 횟수 | 불연속, 명확한 시작·끝 | 가장 직접적 빈도 측정 |
| 전체 간격 기록법 | 초 단위 간격 | 지속적·안정된 행동 | 과소 추정 경향 |
| 부분 간격 기록법 | 초 단위 간격 | 짧고 고빈도 행동 | 과대 추정 경향 |
| 시간표집법 | 분 단위 간격 | 저빈도·장시간 지속 행동 | 마지막 순간 관찰 |
| 지속 시간 기록법 | 시간 | 오래 지속되는 행동 | 지속 시간 길이 초점 |
| 반응 시간 기록법 | 시간 | 지시 순응 행동 | 자극 → 반응 간 지연 시간 |

---

### 5.6 행동 관찰과 측정의 신뢰도

#### 신뢰도에 영향을 미치는 변수

| 변수 | 설명 | 대처 방법 |
|------|------|----------|
| **반응성** | 관찰 대상이 관찰 사실을 의식하여 행동이 달라짐 | 비디오 녹화, 적응 기간 설정 |
| **관찰의 복잡성** | 여러 행동을 동시 관찰할수록 일관성 저하 | 관찰자 훈련 |
| **관찰자 표류** | 시간 경과에 따라 관찰 기준이 점진적으로 변화 | 관찰자 간 접촉 최소화, 정기적 검토, 새 관찰자 투입 |
| **관찰자의 기대** | 중재 목적 인지 시 기준이 느슨해짐 | 관찰자는 중재 목적을 모르게 함 |

> ⚠️ **기출 포인트**: **관찰자 표류**는 정의와 방지 방법이 세트로 출제된다. "시간이 흐르면서 관찰 기준이 점진적으로 변화"라는 정의를 그대로 암기할 것.

#### 신뢰도 측정 방법

**빈도 기록법**

| 방법 | 산출 방식 |
|------|----------|
| 총 발생 횟수 일치도 | (작은 수 ÷ 큰 수) × 100 |
| 평균 발생 횟수 일치도 (간격 내 빈도 신뢰도) | 각 간격별 (적은 빈도 ÷ 많은 빈도) 평균 |
| 정확 발생 횟수 일치도 | 일치 간격 수 ÷ 전체 간격 수 × 100 |
| 시행별 일치도 | 매 시행에서 표적 행동 수행 여부를 두 관찰자가 각각 평가하여 IOA 산출 |

**시간 간격별 기록법**

| 방법 | 특징 |
|------|------|
| 전체 일치도 | 적은 수 ÷ 많은 수 × 100 (덜 엄격) |
| 시간 간격 일치도 (총 간격 일치도) | 일치 간격 ÷ (일치 + 불일치) × 100 |
| 발생 일치도 (채점 간격 일치도) | 발생 일치 간격 ÷ (발생 일치 + 불일치) × 100 (발생에 집중) |
| 비발생 일치도 (비채점 간격 일치도) | 비발생 일치 간격 ÷ (비발생 일치 + 불일치) × 100 (비발생에 집중) |

## 6. 단일대상 연구 설계

독립 변인과 종속 변인의 **기능적 관계**를 체계적으로 입증하기 위한 연구 방법이다. 집단 비교 대신 개인 내 반복 측정으로 중재 효과를 검증한다.

> ⚠️ **기출 포인트**: 각 설계의 **기능적 관계 입증 방식**, **윤리적 문제**, **사용 불가 조건**이 반복 출제된다. 설계 이름과 특징의 1:1 매핑을 완성해 두어야 한다.

---

### 6.1 핵심 용어

| 용어 | 정의 |
|------|------|
| **독립 변인** | 행동 변화를 위해 적용하는 중재·교수 방법 (교사가 조정) |
| **종속 변인** | 변화되어야 할 표적 행동 |
| **기능적 관계** | 독립 변인의 변화로 종속 변인의 변화를 예측 가능한 관계 |
| **외생 변인** | 통제되지 않은 변인 |
| **내적 타당도** | 종속 변인 변화가 독립 변인에 의한 것임을 확신하는 정도 |
| **외적 타당도** | 연구 결과를 일반화할 수 있는 정도 |

---

### 6.2 기초선 (Baseline)

어떠한 중재도 적용하지 않은 자연 상태에서의 수행 수준이다.

**기초선의 3가지 기능**

| 기능 | 설명 |
|------|------|
| **기술적 기능** | 학생의 현재 수행 수준 설명 |
| **예언적 기능** | 중재 없을 때 미래 수행 수준 예측 |
| **시험적 기능** | 예측과 실제 수행 간 차이를 확인하여 중재 효과 검증 |

**기초선 안정성 평가 — 중재 투입 시기 결정**

| 기준 | 설명 |
|------|------|
| **자료의 다양성** | 파동이 클수록 결론 내리기 어려움 → 안정될 때까지 기다림 |
| **자료의 경향성** | 동일 방향으로 3개 이상 연속 자료점 = 경향성 있음 |

- 상향 기초선(증가 경향) → 중재 목표는 **감소**
- 하향 기초선(감소 경향) → 중재 목표는 **증가**
- 바람직한 방향으로 이미 변화 중이면 → 중재 도입 **잠시 보류**

---

### 6.3 자료의 시각적 분석

| 분석 요소 | 설명 |
|----------|------|
| **수준 (Level)** | 자료의 평균값. 상황 간 평균선 비교로 변화 확인 |
| **경향 (Trend)** | 자료의 방향과 변화 정도. 경향선 기울기로 판단 |
| **변화율 (Variability)** | 경향선 중심으로 자료가 퍼진 범위. 클수록 안정성 낮음 |
| **중첩 정도 (Overlap)** | 두 상황 간 자료값이 겹치는 정도. 중첩 적을수록 효과 큼 |
| **효과의 즉각성** | 한 상황 마지막 자료 ↔ 다음 상황 첫 자료 간 차이. 클수록 기능적 관계 강함 |

**시각적 분석의 장단점**

| 구분 | 내용 |
|------|------|
| **장점** | 통계 전문 기술 불필요, 개별 학생 데이터 기반 결정 가능, 중재 중 형성 평가 가능 |
| **단점** | 명확한 원칙 없음, 주관 의존, 자료의 연속적 의존성 높음 |

---

### 6.4 단일대상 연구 설계의 실제

#### AB 설계

가장 기본 형태. A = 기초선, B = 중재.

| 구분 | 내용 |
|------|------|
| **장점** | 중재 전후 비교 용이, 간편, 반전 어려운 경우 사용 가능 |
| **단점** | **기능적 관계 입증 불가** (외생 변인 배제 못함) |

---

#### ABA 설계

기초선(A1) → 중재(B) → 기초선 복귀(A2). 기능적 관계를 입증할 수 있는 가장 단순한 설계.

| 구분 | 내용 |
|------|------|
| **장점** | 기능적 관계 입증 가능 |
| **단점** | 기초선 상태에서 실험 종료 → **윤리적 문제** |

---

#### ABAB 설계 (= 반전 설계 / 중재 제거 설계)

중재의 반복 적용(B1)과 철회(A2), 재적용(B2)으로 기능적 관계를 **강력하게** 입증한다.

**기능적 관계 입증 3단계**
1. **예언** — 기초선(A1)으로 중재 없을 때 미래 행동 예측
2. **예언의 증명** — 중재(B1) 도입 후 행동 변화 확인
3. **효과의 복제** — 중재 철회(A2) 후 행동 복귀, 재도입(B2) 후 재변화 확인

| 구분 | 내용 |
|------|------|
| **장점** | ABA보다 강력한 기능적 관계 입증, 중재 단계에서 종료 가능 (윤리적), 내적 타당도 강화 |
| **단점** | A2 구간을 위한 중재 철회 시 윤리적 문제, 반전 불가 행동에 부적절 |

**사용 부적절 및 변형**

| 상황 | 변형 |
|------|------|
| 기초선이 비윤리적으로 긴 경우 | A를 짧게 → **A(짧게)BAB** |
| 표적 행동이 폭력적이거나 전혀 나타나지 않는 경우 | A1 완전 생략 → **BAB** |
| 더 확실한 입증 필요 | **ABABAB** (윤리적·현실적 문제 여전히 존재) |

> ⚠️ **기출 포인트**: A2 구간 자료가 A1 평균에 근접하거나, A2 방향이 B1 방향과 **뚜렷한 반대 경향**이면 기능적 관계 입증으로 본다.

---

#### 중다 기초선 설계 (= 복식/복합 기초선 설계)

ABAB의 단점(반전 필요)을 극복. 하나의 독립 변인을 **시간차를 두고 3개 이상**의 행동/상황/대상에 순차 적용한다.

**기능적 관계 입증을 위한 2가지 가정**

| 가정 | 내용 |
|------|------|
| **기능적 독립성** | 중재 미적용 기초선은 다른 기초선의 중재 영향을 받지 않아야 함 |
| **기능적 유사성** | 각 기초선은 동일 중재에 유사하게 반응해야 함 |

**3가지 유형 비교**

| 유형 | 적용 대상 | 특징 |
|------|----------|------|
| **행동 간** | 동일 인물의 여러 행동 | 행동 간 동시 변화(공변) 주의 |
| **상황 간** | 동일 행동, 여러 상황 | 상황별 외생 변인 통제 어려움 |
| **대상 간** | 여러 피험자의 동일 행동 | 대상 간 공변 주의, 다수 관리 어려움 |

| 구분 | 내용 |
|------|------|
| **장점** | 반전 불필요, 일반 교육 환경과 유사, 현장 적용 용이 |
| **단점** | 다수 기초선 동시 측정 어려움, 기초선 기간 길어짐, 즉각 중재 필요 행동에 부적절 |

**중다 간헐 기초선 설계**: 아직 중재 미적용 기초선에서 매 회기가 아닌 **간헐적으로** 자료 수집.
- 사용 조건: 인력·시간 부족, 대상자의 심한 반응성, 안정성 전제 가능할 때

---

#### 준거 변경 설계 (= 기준 변동 설계)

종속 변인의 **점진적·단계적** 변화를 이루고자 할 때 사용한다. 중재 구간은 수준을 단계별로 높이는 하위 구간으로 구성된다.

**적합한 경우**: 최종 목표가 기초선에서 상당히 멀 때, 형성 절차의 효과 측정

**실행 절차**
1. 목표 행동 정의 및 기초선 자료 수집
2. 첫 중간 준거 결정 (기초선 평균, 기초선 최고 수준, 전문가 판단 등)
3. 각 하위 구간에서 준거 달성 시 강화 → 준거 상향 반복
4. 그래프 작성

**신뢰성 증가 방법**
- 안정된 비율 확립까지 하위 구간 유지
- 하위 구간 회기 수 변경
- 중간에 이전 단계 낮은 기준 재제시
- 구간별 증감량 다양화

| 구분 | 내용 |
|------|------|
| **장점** | 반전 불필요, 행동을 긍정적 방향으로 지속 변화시키며 기능적 관계 확립 가능 |
| **단점** | 빠른 수정 필요 행동에 부적절, 새로운 행동 발달에 부적절, 기준 설정이 주관적 |

---

#### 교대 중재 설계 (= 중재 교대 / 교대 처치)

하나의 종속 변인에 둘 이상의 독립 변인을 **빠르게 교대**로 투입하여 효과를 비교한다.

- 각 처치 전 변별 자극(신호·단서) 제시로 현재 조건을 학생에게 명확히 알림
- 마지막 단계에서는 가장 효과적인 중재를 단독 실시 가능

| 구분 | 내용 |
|------|------|
| **장점** | 기초선 측정 최소화 가능, 중재 효과 빠른 비교, 내적 타당도 위협 감소 |
| **단점** | 중재 간 간섭(공변) 효과 완전 배제 불가, 변인 통제·균형 복잡 |

**핵심 판독 방법**: 그래프에서 두 중재 자료선 간의 **수직적 거리** → 크면 효과 차이 큼, **분할(separation)** 발생 시 차별적 효과 있음

**중다 기초선 설계와의 차이**

| 구분 | 중다 기초선 설계 | 교대 중재 설계 |
|------|----------------|--------------|
| 독립 변인 | 1개 | 2개 이상 |
| 종속 변인 | 여러 개 | 1개 |
| 목적 | 중재 효과 입증 | 중재 간 효과 비교 |

---

#### 조건 변경 설계 (= ABC 설계 / 중다 중재 설계)

하나의 종속 변인에 여러 처치를 **순서적으로** 도입하여 가장 효과적인 중재를 탐색한다. 교대 중재(번갈아 적용)와 달리 처치가 순차적으로 도입된다.

**3가지 기본형**

| 유형 | 특징 |
|------|------|
| **ABC 설계** | 처치 효과 순차 탐색, 도움 체계적 제거에 활용 |
| **ABAC 설계** | 기초선으로 처치들을 분리 → 처치 간 간섭 효과 방지 |
| **ABACAB 설계** | 가장 성공적인 처치를 기초선 후 재실행 → 기능적 관계 입증 가능 |

| 구분 | 내용 |
|------|------|
| **장점** | 여러 독립 변인 효과 비교 가능, 시간 절약, 근접 중재 간 비교 가능 |
| **단점** | 기본적으로 **기능적 관계 입증 불가**, 중재 효과의 누적 가능성, 전이 효과 가능성 |

> ⚠️ **기출 포인트**: ABC 설계는 기능적 관계를 입증하지 못하며, ABACAB 설계만 복제 구간이 있어 입증 가능하다는 점이 자주 출제된다.

### 6.5 단일대상 설계 비교 총정리

| 설계 | 기능적 관계 입증 | 반전 필요 | 윤리적 문제 | 핵심 특징 |
|------|----------------|----------|------------|----------|
| AB | 불가 | 없음 | 없음 | 가장 기본형 |
| ABA | 가능 | 있음 | 기초선 종료 문제 | 단순하지만 윤리 문제 |
| **ABAB** | 강력 | 있음 | 철회 시 문제 | 반복으로 내적 타당도 강화 |
| **중다 기초선** | 가능 | 없음 | 기초선 연장 문제 | 반전 불가 행동에 최적 |
| 준거 변경 | 가능 (단계별) | 선택적 | 적음 | 점진적 목표 행동에 적합 |
| **교대 중재** | 비교 가능 | 없음 | 적음 | 중재 간 효과 비교 |
| 조건 변경(ABC) | 기본 불가 | 없음 | 적음 | 탐색적 사용 |
