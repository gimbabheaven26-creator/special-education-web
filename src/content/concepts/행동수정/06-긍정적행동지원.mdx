## 1. 긍정적 행동지원(PBS) 기본 개념

### 1.1 정의와 목적

> 병원에서 증상만 억누르는 약을 주는 대신, 생활 습관 전체를 바꾸도록 코칭하는 것이 PBS이다. 문제 행동 자체를 제거하는 것이 아니라 그 원인 환경을 바꾸고 더 나은 삶의 방식을 만드는 것이 목표이다.

**정의**: 문제 행동의 이유를 이해하고, 문제 행동이 왜 발생했는지에 대한 가설에 따라 개인의 독특한 사회적·환경적·문화적 배경에 적합한 종합적인 해결책을 제공하는 문제 해결 접근 방법이다.

**목적**:
- 단기 문제 행동 감소가 아닌 **장기간 지속되는 삶의 질 향상**
- 문제 행동 감소 및 건강·안전 증진
- 기능적 기술 향상, 긍정적 학습 환경 조성

**PBS에 영향을 미친 주요 요소**: 자기 결정, 개인 중심 계획, 통합, 응용 행동 분석

> ⚠️ **기출 포인트**: PBS의 목표는 **단기 문제 행동 감소가 아니라 장기적 삶의 질 향상**임을 강조. 전통적 행동 수정과의 차이가 단골 출제 비교 문항.

---

### 1.2 전통적 행동 수정 vs. 긍정적 행동지원 비교

| 구분 | 전통적 행동 수정 | 긍정적 행동지원 |
|------|----------------|----------------|
| 관점 | 문제 행동은 타인에게 해를 주므로 제거되어야 함 | 문제 행동은 학습된 것이며 의사소통의 목적을 가짐 |
| 평가 | 빈도·강도·지속 기간 등 행동 양상 파악 | 문제 행동의 원인 파악을 위한 **기능 평가** 실시 |
| 목표 | 단기 문제 행동 감소·제거 | 사회적으로 용인되는 방법으로 욕구를 표현하도록 지원 |
| 중재 | 강화 축소 또는 벌 제공 | 안전감·신뢰감 발달, 대안적 의사소통 교수, 환경 재구성 |

**행동 수정 → 응용 행동 분석 → 긍정적 행동지원** 발전 계보:
- 행동 수정: 바람직한 행동 증가 + 문제 행동 감소에 초점
- 응용 행동 분석(ABA): 행동 수정 + **개별화** 원리 + 행동의 기능에 대한 관심
- 긍정적 행동지원(PBS): ABA + **기능 분석** + **문제 행동 예방** 원리

---

### 1.3 PBS의 주요 요소

| 주요 요소 | 핵심 내용 |
|----------|---------|
| 생태학적 접근 | 문제 행동은 장애가 아닌 **환경적 사건·조건** 때문에 발생할 수 있음 |
| 진단 기반 접근 | 기능 진단에 의해 이루어지며, 개인 중심 접근 적용 |
| 맞춤형 접근 | 동일 중재를 여러 학생에게 적용하지 않음. 개인의 필요와 환경에 맞추어 구성 |
| 예방·교육 중심 접근 | 처벌이 아닌 **예방과 교수**가 목표. 환경 변화 + 대처 기술 교수 |
| 삶의 방식·통합 중심 접근 | 문제 행동 감소만이 아닌 **폭넓은 삶의 질 향상**과 통합 촉진 |
| 종합적 접근 | 예방·대체 기술 교수·반응·삶의 방식 개선을 위한 다양한 중재 적용 |
| 팀 접근 | 부모·교사·행정가·전문가·학생·가족 등 팀 노력 필요 |
| 대상 존중 접근 | 모든 참여자, 특히 가족과 장애 학생에게 의미 있는 중재 목표 설정 |

## 2. 긍정적 행동지원의 실제

### 2.1 표적 행동 식별과 우선순위

**표적 행동 선정**: 관찰·측정 가능한 용어로 정의할 행동 (바람직한 행동 또는 바람직하지 않은 행동 모두 가능)

**우선순위 결정**:

| 순위 | 유형 | 예시 |
|------|------|------|
| 1순위 | **파괴 행동** — 자신·타인에게 직접 해가 되거나 위협적인 행동 | 때리기, 물기, 자해 |
| 2순위 | **방해 행동** — 즉각적 해는 없으나 지속 시 학습·상호작용에 부정적 영향 | 학습 방해, 상동 행동 |
| 3순위 | **경미한 방해 행동** — 사회적 수용을 어렵게 하며 방해 행동으로 발전 가능 | 반향어, 틱 |

**조작적 정의 필요성**: 서로 다른 관찰자들이 행동 발생 여부에 동의할 수 있도록, 행동을 다른 행동들과 변별할 수 있도록 하기 위함

**행동 목표 양식**: `[아동/학습자]가 [조건]에서/할 때, [기준] 수준으로 [행동]할 것이다.`

행동 목표의 네 가지 구성 요소: **학습자 / 행동 / 조건 / 기준**

행동의 여섯 가지 차원: 빈도, 지속 시간, 지연 시간, 위치, 형태, 강도

### 2.2 기능 평가(FBA)

> 마치 의사가 증상만 보고 약을 처방하지 않고 혈액 검사, 문진, 영상 검사를 순서대로 시행하듯, FBA도 간접 → 직접 → 실험의 순서로 이루어진다.

**정의**: 특정 행동을 신뢰할 수 있게 예언하고, 그 행동을 지속시키는 환경 내 사건을 정의하기 위해 이루어지는 일련의 활동 과정

**목적**: 문제 행동을 유발·유지하게 하는 환경적 원인을 찾아 가장 효과적인 중재를 적용

#### 기능 평가의 세 가지 방법

| 방법 | 정의 | 장점 | 단점 |
|------|------|------|------|
| **간접적 방법** | 기존 기록·정보 제공자 진술에 의존 (면담, 척도, 체크리스트, 기록물 분석) | 전체적 정보 신속 수집, 실시 용이 | 객관성 낮음, 정보 구체성 부족 |
| **직접적 방법** | 자연스러운 상황에서 직접 관찰 (일화 기록, 산점도, ABC 관찰) | 자연 환경에서 객관적 자료 수집 | 시간 많이 소요, 빈도 낮은 행동에 한계 |
| **기능 분석** (실험적 평가) | 선행 사건·후속 결과를 실험적으로 조작하여 기능 검증 | 행동-환경 기능적 관계를 가장 명확히 밝힘 | 위험한 행동 적용 불가, 시간·비용·인력 대량 소요 |

> ⚠️ **기출 포인트**: **기능 분석(실험적 평가)**의 제한점과 주의점이 자주 출제됨. "부모의 사전 동의 필요", "심한 자해 행동에 적용 불가"는 반드시 기억.

#### 직접 관찰 방법 상세 비교

| 방법 | 특징 | 장점 | 단점 |
|------|------|------|------|
| **일화 기록** | 이야기식 비공식 기록, 일정한 형식 없음 | 자유로운 관찰 | 주관적, 체계성 부족 |
| **산점도 (행동 분포 관찰)** | 시간·날짜별 행동 발생 여부 표시 | 작성 쉬움, 문제 시간대 파악 | 선행·후속 정보 없음, 빈도 파악 불가 |
| **ABC 관찰** | 선행 사건–행동–후속 결과 시간 흐름대로 기록 | 기능 파악에 직접적으로 유용 | 시간 소요, 빈번한 행동 기록 어려움 |
| **ABC 관찰 검목표** | ABC 기록지를 발전시킨 체크리스트 형태 | 사전에 확인된 내용을 체계적으로 기록 | 예상치 못한 사건 누락 가능 |
| **행동의 기능 평가 관찰지** | ABC 검목표를 더 발전시킨 형태 | 선행·후속 사건 찾기 용이 | — |

#### 행동의 네 가지 기능

| 기능 | 설명 |
|------|------|
| **사회적 정적 강화** | 대인 관계 상황에서 다른 사람들에 의해 정적 강화가 제공됨 (관심 얻기, 물건 얻기) |
| **사회적 부적 강화** | 대인 관계 상황에서 혐오 자극이 제거됨 (과제 회피, 요구 회피) |
| **자동적 정적 강화** | 행동의 결과로 내적 감각 자극이 자동으로 뒤따름 (자기 자극) |
| **자동적 부적 강화** | 행동의 결과로 불쾌한 상황이나 신체적 불편이 자동적으로 제거됨 |

> ⚠️ **기출 포인트**: 행동 기능 4가지를 예시와 함께 연결하는 서술형 출제 빈도 높음. "주의 끌기 = 사회적 정적 강화", "과제 회피 = 사회적 부적 강화"가 대표 예시.

### 2.3 가설 설정

**가설 문장의 4요소**: 아동 이름 + 선행 사건 + 문제 행동 + 추정되는 기능

예시: `"민수는 어려운 수학 과제가 제시될 때(선행 사건), 소리를 지르는 행동(문제 행동)을 하며, 이는 과제를 회피하기 위한 것으로 추정된다(기능)."`

### 2.4 행동지원계획(BSP) 수립

행동지원계획의 네 가지 중재 요소 (동시 수행 가능):

| 중재 요소 | 내용 |
|----------|------|
| **선행 사건/배경 사건 중재** | 문제 행동을 일으키는 사건을 수정·제거. **동기화 조작**(EO: 동기 설정 조작, AO: 동기 해지 조작) 포함 |
| **대체 행동 교수** | 문제 행동의 기능을 대체하는 바람직한 행동 교수 |
| **문제 행동에 대한 반응** | 다른 사람들의 반응을 효과적·교육적 방향으로 전환하는 전략 |
| **장기 지원** | 삶의 방식 변화, 지속적 지원 전략 실행 |

#### 대체 행동 교수의 세 가지 기술 유형

| 기술 유형 | 정의 |
|----------|------|
| **교체 기술** | 문제 행동과 동일한 기능을 가지면서 사회적으로 바람직한 대안 행동 |
| **대처 및 안전 기술** | 힘들고 재미없는 상황에서 문제 행동 없이 인내하며 대처하는 기술 |
| **일반적 적응 기술** | 문제 행동 예방 + 의미 있는 생활 향상에 기여하는 기술 |

**교체 기술 선정 기준 (반응 효율성)**: 노력, 질/강도, 결과의 즉각성, 일관성, 처벌 개연성

#### 장기 지원 — 삶의 방식 변화 5요소

1. 선택의 기회
2. 학교 및 지역사회 통합
3. 다른 사람과의 관계
4. 가치 있는 역할 수행
5. 전반적인 건강과 안녕

#### 위기 관리 계획

**목적**: 심각한 문제 행동으로 인한 위험 상황에서 학생과 타인을 보호

**포함 요소**: 위기 상황 정의 / 대응 담당자 / 팀 호출 방법 / 역할 / 위기 중재 장소 / 발생 원인 기록

**주의점**: 위기 관리 절차는 가끔만 사용되어야 하며, 중재 중에도 계속 필요하다면 **지원 계획이 부적절하다는 신호**이다.

---

### 2.5 계획의 실행–평가–수정

- 중재 전략 성과 점검 후 수정 필요 여부 평가
- 평가는 **자주 정기적으로 직접 평가하는 형성 평가** 방식으로 진행

## 3. 학교차원 긍정행동지원(SWPBS)

### 3.1 개념 및 목적

**정의**: 학교 안의 모든 학생이 사회적·학업적 성취를 달성하는 데 필요한 행동을 지원하고 긍정적 사회 문화를 정착시키기 위한 체계적 접근 방식

**목적**:
- 시스템·절차 개선을 통해 교사의 긍정적 행동 변화 촉진
- 학생의 행동 변화를 통한 학교 환경 변화

#### 개인 PBS vs. 학교차원 PBS

| 구분 | 개인 PBS | 학교차원 PBS(SWPBS) |
|------|---------|-------------------|
| 대상 | 심각한 문제 행동을 보이는 소수 아동 | **학교 소속 모든 아동** |
| 초점 | 한 개인의 행동 지원, 모든 환경 고려 | PBS 이념을 학교 전체에 실천 |
| 지원 주체 | 개별 지원 팀 | 학교 차원의 행동 지원 팀 |

**조직의 네 가지 특징**: 공동 비전/가치 + 공동 언어 + 공동 경험 + 질 높은 리더십

> ⚠️ **기출 포인트**: "개별 PBS와 SWPBS의 차이"는 단골 비교 문항. SWPBS는 대상이 **학교 전체** 학생이며, 행동 지원 단위가 개인이 아닌 **학교**임을 명확히.

---

### 3.2 연속적 행동지원 체계 (3단계 예방)

> 의료 예방 개념과 같다. 전체 인구 대상 건강 교육(1차), 고위험군 집중 관리(2차), 만성 질환자 개별 치료(3차) — SWPBS의 3단계도 동일한 논리이다.

| 단계 | 명칭 | 대상 | 목표 | 중재 |
|------|------|------|------|------|
| **1차 예방** | 보편적/범단체적 | 모든 학생·교직원 | 문제 행동의 **새로운 발생 예방** | 공동 가치·기대 행동 교수 + 강화 (보편적 중재) |
| **2차 예방** | 소집단 중재 | 1차에 미반응 학생, 고위험 가능 학생 | 기존 문제 행동의 **출현율 감소** | 교사 관심·점검 증가, 사회적/학업적 기술 향상 소집단 지도 |
| **3차 예방** | 개별화 중재 | 지속적 문제 행동 학생 | 문제 행동의 **강도·복잡성 감소** | 개별화 중재 + 보편적 중재 내용과 연계 |

**중요 원칙**: 삼각형의 각 층에 해당하는 인원이 고정되어 있지 않으며, 세 체계는 유기적으로 연결된다.

#### RTI와의 비교

| 구분 | 공통점 | SWPBS 차별점 |
|------|--------|------------|
| — | 다층 체계 모델, 증거 기반 중재, 자료 수집, 선별·진단·진보 점검 | 모든 수준의 지원이 **보편적 시스템의 틀 안에** 서로 연결되어 있음 |

**연속적 행동지원 체계의 효과**: 바람직한 기대 행동 반복 연습·보상 기회 지속, 소집단·개별 중재 효과의 유지·일반화 가능성 향상

---

### 3.3 SWPBS의 핵심 4요소

| 요소 | 정의 |
|------|------|
| **시스템** | 정확하고 지속 가능한 실제의 실행과 자료의 효율적 사용을 위한 **지원 구조** |
| **자료** | 성과·실제·시스템을 선택·점검·평가하기 위해 수집·사용하는 **정보** |
| **실제** | 성과를 성취하기 위한 **증거 기반의 중재와 전략** (연구로 효과 입증된 것) |
| **성과** | 학업과 사회성에서 지향하는 **목표·지표** (SWPBS가 궁극적으로 달성하려는 목표) |

> ⚠️ **기출 포인트**: SWPBS 핵심 4요소(시스템-자료-실제-성과)는 단답형·서술형 모두에서 출제됨. 각 요소의 정의와 예시를 연결하여 암기.

---

### 3.4 의사 결정 과정

- **원칙**: 주관적 판단이 아닌 **객관적 자료에 근거한 의사 결정**
- **활용 자료**: 보관된 기록, 평가서/설문지, 면담, 직접 관찰 자료

## 4. PBS 전체 흐름 요약

| 단계 | 내용 |
|------|------|
| 1. 문제 행동 식별 | 표적 행동 선정 → 우선순위 결정 → 조작적 정의 → 행동 목표 설정 |
| 2. 기능 평가(FBA) | 간접적 방법 → 직접적 방법 → 기능 분석 (필요 시) |
| 3. 가설 설정 | 선행 사건 + 문제 행동 + 추정 기능 포함한 검증 가능한 가설 문장 |
| 4. BSP 수립 | 선행 사건 중재 + 대체 행동 교수 + 문제 행동 반응 + 장기 지원 |
| 5. 실행–평가–수정 | 형성 평가를 통한 주기적 점검 및 수정 |
